# <a name="microsoft-azure-documentation"></a>Dokumentace k Microsoft Azure

Vítá vás open source [dokumentaci](https://docs.microsoft.com/azure) z [Microsoft Azure](https://azure.microsoft.com). Přečtěte si prosím tento soubor README k pochopení, jak vám může být užitečné při přispívání do dokumentace k Microsoft Azure.

## <a name="getting-started"></a>Začínáme

Přispívání do open source je víc než jenom WSUS budou poskytovat aktualizace, to je také nám dáte vědět, když se vyskytl problém. Přečtěte si naše [přispívání pokyny](.github/CONTRIBUTING.md) Další.

### <a name="prerequisites"></a>Požadavky

Jste se rozhodli poskytují příspěvku, který je skvělé! Existuje několik nástrojů, které jsou potřeba k přispívání do dokumentace.

Přispívání do dokumentace vyžaduje účet GitHub, pokud nemáte účet, postupujte podle pokynů pro [nastavení účtu GitHub](https://docs.microsoft.com/contribute/get-started-setup-github) z naší příručky pro přispěvatele.

#### <a name="download"></a>Ke stažení

Nainstalujte nástroje pro následující:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* [Docs Authoring Pack](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) rozšíření pro Visual Studio Code

#### <a name="install"></a>Instalace

Postupujte podle pokynů uvedených v [instalace nástrojů pro tvorbu obsahu](https://docs.microsoft.com/contribute/get-started-setup-tools) z naší příručky pro přispěvatele.
