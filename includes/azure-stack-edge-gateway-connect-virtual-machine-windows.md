Připojte se k VIRTUÁLNÍmu počítači s Windows pomocí protokol RDP (Remote Desktop Protocol) (RDP) přes IP adresu, kterou jste předali během vytváření virtuálního počítače.

1. V klientovi spusťte protokol RDP. Pokračujte na **začátek** a zadejte **mstsc**.
1. Zadejte IP adresu virtuálního počítače a přihlašovací údaje pro přístup, které jste použili v souboru parametrů šablony virtuálního počítače.

    ![Připojení přes protokol RDP k virtuálnímu počítači s Windows](media/azure-stack-edge-gateway-connect-vm-windows/connect-vm-rdp-1.png)
1. Možná budete muset schválit připojení k nedůvěryhodnému počítači. 
1. Nyní jste přihlášeni k VIRTUÁLNÍmu počítači spuštěnému na zařízení.
