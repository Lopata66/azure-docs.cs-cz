### <a name="open-the-mfa-page"></a><a name="mfa"></a>Otevření stránky Vícefaktorové finanční hod.

1. Přihlaste se k portálu Azure.
2. Přejděte do **služby Azure Active Directory -> Všichni uživatelé**.
3. Vyberte **vícefaktorové ověřování,** chcete-li otevřít stránku vícefaktorového ověřování.

   ![Přihlášení](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Vybrat uživatele

1. Na stránce **vícefaktorového ověřování** vyberte uživatele, pro kterého chcete povolit vícefaktorové ověřování.
2. Vyberte **Povolit**.

   ![Vyberte](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
