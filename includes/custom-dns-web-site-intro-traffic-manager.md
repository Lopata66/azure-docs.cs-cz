Pokud používáte Microsoft Azure Traffic Manager vyrovnávat zatížení provozu na svůj web Azure, webu lze poté přistupovat pomocí  **\*. trafficmanager.net** domény název přiřazený nástrojem Azure. Můžete také přiřadit vlastní název domény, jako je například www\.contoso.com s vaším webem, aby bylo možné zadat více rozpoznatelných název domény pro vaše uživatele.
