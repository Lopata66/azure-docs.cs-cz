Pokud používáte Microsoft Azure Traffic Manager vyrovnávat zatížení provozu na svůj web Azure, webu lze poté přistupovat pomocí  **\*. trafficmanager.net** domény název přiřazený nástrojem Azure. Název vlastní domény, třeba www.contoso.com, můžete také přiřadit svůj web tak, aby bylo možné zadat více rozpoznatelných název domény pro vaše uživatele.
