Ve výstupu `identity` příkazu ukazuje oddíl identitu typu `SystemAssigned` nastavenou v úloze. `principalId` Je ID objektu služby identity:

```console
[...]
  "identity": {
    "principalId": "xxxxxxxx-2703-42f9-97d0-xxxxxxxxxxxx",
    "tenantId": "xxxxxxxx-86f1-41af-91ab-xxxxxxxxxxxx",
    "type": "SystemAssigned",
    "userAssignedIdentities": null
  },
  "location": "eastus",
[...]
``` 
Pomocí příkazu [AZ ACR Task show][az-acr-task-show] uložte principalId do proměnné pro použití v pozdějších příkazech:

```azurecli
principalID=$(az acr task show --name dockerhubtask --registry myregistry --query identity.principalId --output tsv)
```
