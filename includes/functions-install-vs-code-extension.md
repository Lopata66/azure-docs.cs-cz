## <a name="install-the-azure-functions-extension"></a>Instalace rozšíření Azure Functions

Rozšíření Azure Functions můžete použít k vytvoření a testování funkcí a jejich nasazení do Azure.

1. V kódu Visual Studia otevřete **rozšíření** a vyhledejte **funkce Azure**nebo [vyberte tento odkaz v kódu Visual Studia](vscode:extension/ms-azuretools.vscode-azurefunctions).

1. Vyberte **Instalovat,** chcete-li nainstalovat rozšíření pro kód sady Visual Studio:

    ![Instalace rozšíření pro Azure Functions](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. Po instalaci vyberte ikonu Azure na panelu aktivit. Měla by se na postranním panelu zobrazit oblast Azure Functions.

    ![Oblast Azure Functions na postranním panelu](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)
