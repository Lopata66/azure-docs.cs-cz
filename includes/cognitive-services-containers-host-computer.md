**Hostitele** je x64 počítač, který spustí kontejner dockeru. Může být počítač připojený k vlastních prostorách nebo v dockeru, který hostuje službu v Azure včetně:

* [Azure Kubernetes Service](../articles/aks/index.yml)
* [Azure Container Instances](../articles/container-instances/index.yml)
* [Kubernetes](https://kubernetes.io/) clusteru nasadí do [Azure Stack](/azure-stack/operator). Další informace najdete v tématu [nasazení Kubernetes pro Azure Stack](/azure-stack/user/azure-stack-solution-template-kubernetes-deploy).
