### <a name="prerequisites"></a>Požadavky
* A [ProjectOnline](https://products.office.com/Project/project-online-with-project-for-office-365) účtu 

Před použitím svého účtu pro Project online. v aplikaci logiky, musí autorizovat aplikaci logiky se připojit ke svému účtu pro Project online. Naštěstí vám pomůžou snadno z v rámci aplikace logiky na webu Azure Portal. 

Tady jsou kroky k autorizaci aplikace logiky se připojit ke svému účtu pro Project online:

1. Chcete-li vytvořit připojení k ProjectOnline, v návrháři aplikace logiky, vyberte **rozhraní API spravovaná Microsoftem zobrazit** v rozevíracím seznamu zadejte *ProjectOnline* do vyhledávacího pole. Výběr triggeru nebo akce, které budete chtít použít:  
   ![Krok 1 pro Project online.](./media/connectors-create-api-projectonline/projectonline-1.png)
2. Pokud jste ještě nevytvořili žádná připojení pro ProjectOnline před, zobrazí výzva zadat přihlašovací údaje pro Project online. Tyto přihlašovací údaje se použije k autorizaci aplikace logiky se připojit k a přístup k datům v účtu pro Project online:  
   ![Krok 2 pro Project online.](./media/connectors-create-api-projectonline/projectonline-2.png)
3. Zadejte své ProjectOnline uživatelské jméno a heslo k autorizaci aplikace logiky:  
   ![Krok 3 pro Project online.](./media/connectors-create-api-projectonline/projectonline-3.png)   
4. Všimněte si, připojení bylo vytvořeno a teď jste zdarma postupujte podle dalších kroků ve vaší aplikaci logiky:  
   ![Krok 4 pro Project online.](./media/connectors-create-api-projectonline/projectonline-4.png)
