1. Klikněte pravým tlačítkem na projekt pro Windows Store, klikněte na tlačítko **nastavit jako spouštěný projekt**, stiskněte klávesu F5 ke spuštění aplikace pro Windows Store.

    Po spuštění aplikace, zařízení je registrována pro nabízená oznámení.
2. Zastavit aplikaci Windows Store a opakujte předchozí krok pro aplikace Windows Phone Store.

    V tomto okamžiku jsou obou zařízení registrovaná pro příjem nabízených oznámení.

3. Znovu spusťte aplikaci Windows Store a zadejte text v **vložení úkolu**a potom klikněte na tlačítko **Uložit**.

    Všimněte si, že po dokončení insert Windows Store a aplikací Windows Phone, dostanete nabízené oznámení ze služby nabízených oznámení Windows. I když není aplikace spuštěna, zobrazí se oznámení na Windows Phone.
