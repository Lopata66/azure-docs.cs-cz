**Jak se mi pošle oznámení o vyřazení sady SDK?**

Microsoft bude před koncem podpory pro vyřazení sady SDK poskytovat předběžné oznámení o 12 měsících, aby bylo možné plynulé přechody na podporovanou sadu SDK. Budeme vás informovat pomocí různých komunikačních kanálů: Azure Portal, aktualizace Azure a přímá komunikace pro přiřazené správce služeb.

**Můžu během 12 měsíců vytvářet aplikace pomocí Azure Cosmos DB SDK pro vyřazení?** 

Ano, v období 12 měsíců budete moct vytvářet, nasazovat a upravovat aplikace pomocí Azure Cosmos DB SDK pro vyřazení. Doporučujeme migrovat na novější podporovanou verzi sady Azure Cosmos DB SDK v období 12 měsíců, podle potřeby. 

**Co se stane po datu vyřazení, co se stane s aplikacemi, které používají nepodporovanou sadu Azure Cosmos DB SDK?** 

Po datu vyřazení již Azure Cosmos DB nebude provádět opravy chyb, přidávat nové funkce nebo poskytovat podporu vyřazení verzí sady SDK. Pokud nechcete upgradovat, požadavky odeslané z vyřazených verzí sady SDK budou nadále obsluhovány službou Azure Cosmos DB. 

**Které verze SDK budou mít nejnovější funkce a aktualizace?**

Nové funkce a aktualizace budou přidány pouze do nejnovější podverze nejnovější podporované hlavní verze sady SDK. Doporučujeme vždy používat nejnovější verzi, abyste mohli využívat nové funkce, vylepšení výkonu a opravy chyb. Pokud používáte starou nevyřazenou verzi sady SDK, požadavky na Azure Cosmos DB budou pořád fungovat, ale nebudete mít přístup k žádným novým funkcím.  

**Co mám dělat, když nemůžu aktualizovat aplikaci před datem přerušení?**

Doporučujeme, abyste provedli upgrade na nejnovější sadu SDK co nejdříve. Po označení sady SDK k vyřazení budete mít 12 měsíců, abyste aplikaci aktualizovali. Pokud nemůžete aktualizovat podle data vyřazení, požadavky odeslané z vyřazených verzí sady SDK budou nadále obsluhovány Azure Cosmos DB, takže vaše běžící aplikace budou nadále fungovat. Azure Cosmos DB však již nebude provádět opravy chyb, přidávat nové funkce nebo poskytovat podporu vyřazení verzí sady SDK. 

Pokud máte plán podpory a potřebujete technickou podporu, [kontaktujte nás](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview) podáním lístku podpory.
