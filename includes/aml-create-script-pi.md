Ve stejném adresáři jako **aml_config/config.json** vytvořte skript a dejte mu název **pi.py**.

Zkopírujte do tohoto skriptu následující kód:
    
   ```python
   import random, math
   from azureml.core import Run
    
   run = Run.get_submitted_run()
    
   pi_counter = 0
   n_iter = 100000
   run.log("Number of iterations",n_iter)
    
   for i in range(1,n_iter):
       x = random.random()
       y = random.random()
       if x*x + y*y < 1.0:
           pi_counter += 1
       pi_value = 4.0*pi_counter / i
       if i%10000==0:
           error = math.pi-pi_value
           print(i, pi_value, error)
           run.log_row("Pi estimate",iteration=i,pi_value=pi_value)
           run.log_row("Error",iteration=i,error=error)
   ```

Všimněte si volání `log_row` na konci.  Po spuštění tohoto skriptu se tyto hodnoty zobrazí ve vašem pracovním prostoru.
