## <a name="versions-of-the-azure-cli"></a>Verze Azure CLI

Aktuálně jsou dostupné dvě verze rozhraní příkazového řádku Azure (CLI):

* [Azure CLI](../articles/storage/common/storage-azure-cli.md): Aktuální verze rozhraní příkazového řádku Azure pro použití s modelem nasazení Resource Manager.
* [Azure classic CLI](../articles/storage/common/storage-azure-cli-nodejs.md): Starší verze rozhraní příkazového řádku Azure, určené pouze pro použití s modelem nasazení classic.
