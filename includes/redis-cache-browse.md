Pokud jste mezipaměť na řídicí panel nepřipnuli, najdete ji na webu [Azure Portal](https://portal.azure.com) po kliknutí na **Všechny služby**.

![Azure Cache pro procházení redis](media/redis-cache-browse/redis-cache-browse.png)

Pokud chcete zobrazit mezipaměti, klikněte na **Všechny služby** a vyhledejte **Azure Cache for Redis**. 

Pokud chcete zobrazit a nakonfigurovat nastavení určité mezipaměti, vyberte ji.

![Seznam mezipaměti Azure pro redis procházení mezipaměti](media/redis-cache-browse/redis-caches.png)

Mezipaměť můžete zobrazit a nakonfigurovat z okna **Azure Cache for Redis.**

![Azure Cache pro redis všechna nastavení](media/redis-cache-browse/redis-cache-blade.png)
