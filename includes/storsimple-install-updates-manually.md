#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Instalace pravidelných aktualizací prostřednictvím klasického portálu Azure
1. Na stránce **Zařízení** vyberte zařízení, do kterého chcete instalovat aktualizace.
2. Přejděte na**Údržbu** **zařízení** > a přejděte dolů na **Aktualizace softwaru**.
3. Chcete-li vyhledat aktualizace, klikněte v dolní části stránky na **Zkontrolovat aktualizace.**
4. Pokud jsou k dispozici aktualizace softwaru, zobrazí se zpráva. Chcete-li začít aktualizovat zařízení, klepněte na **tlačítko Instalovat aktualizace.**
   
    Budete upozorněni, když je aktualizace úspěšně nainstalována.
