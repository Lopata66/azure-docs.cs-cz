> [!NOTE]
>
> V tomto článku se nedávno aktualizoval použití příkazů z modulu Azure PowerShell Az nové. Tento modul je nový modul prostředí PowerShell pro Azure, které jsou kompatibilní s verzí prostředí PowerShell 5.x a 6. AzureRM budou i nadále získávat aktualizace kritické opravu, ale nové funkce budou v modulu Az.
>
> * Další informace o modulu Az a nepřetržitou podporu pro modul AzureRM, naleznete v tématu [představení nového modulu Azure PowerShell Az](/powershell/azure/new-azureps-module-az).
> * Modul Az je již k dispozici ve službě Cloud Shell s kompatibilitou AzureRM automaticky povolené.
>   Pokyny k instalaci modulu Az najdete v tématu [instalace Azure Powershellu](/powershell/azure/install-az-ps).
> * Chcete-li povolit příkazy kompatibility AzureRM, spusťte `Enable-AzureRmAlias` po instalaci.
>   Další informace o kompatibilitě naleznete v tématu [migrace ze služby Azure AzureRM Powershellu na Az](/powershell/azure/migrate-from-azurerm-to-az).
