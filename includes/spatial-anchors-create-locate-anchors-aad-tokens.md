### <a name="azure-active-directory-authentication"></a>Ověřování Azure Active Directory

Prostorové kotvy Azure také umožňují aplikacím ověřování pomocí tokenů služby Azure AD (Active Directory) pro uživatele. Můžete například použít tokeny Azure AD pro integraci s prostorovými kotvami Azure. Pokud podnik zachovává uživatele ve službě Azure AD, můžete v sadě Azure prostorové kotvy sady Azure poskytovat token Azure AD. Díky tomu můžete přímo ověřit službu Azure prostorových kotev pro účet, který je součástí stejného tenanta Azure AD.
