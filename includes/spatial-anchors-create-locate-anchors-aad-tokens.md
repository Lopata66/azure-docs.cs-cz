### <a name="azure-active-directory-authentication"></a>Ověřování Azure Active Directory

Azure Spatial Anchors také umožňuje aplikacím ověřování pomocí tokenů Azure AD (Active Directory). Tokeny Azure AD můžete například použít k integraci s azure spatial anchors. Pokud enterprise udržuje uživatele ve službě Azure AD, můžete zadat token Azure AD v Azure Spatial Anchors SDK. To vám umožní ověřit přímo službu Azure Spatial Anchors pro účet, který je součástí stejného klienta Azure AD.
