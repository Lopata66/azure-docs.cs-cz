### <a name="azure-active-directory-authentication"></a>Ověřování Azure Active Directory

Azure prostorových kotvy také umožňuje aplikacím, aby ověřoval tokeny uživatele Azure AD (Active Directory). Například můžete použít tokeny služby Azure AD pro integraci s Azure prostorových ukotvení. Pokud organizace udržuje uživatele ve službě Azure AD, můžete zadat token uživatele Azure AD v sadě SDK Azure prostorových ukotvení. To umožňuje ověřovat přímo ke službě Azure prostorových ukotvení pro účet, který je součástí stejného tenanta služby Azure AD.
