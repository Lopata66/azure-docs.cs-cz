## <a name="more-information-about-bots"></a>Další informace o roboty

Další informace o používání této služby s roboty najdete v následujících zdrojích:

|Prostředek|Účel|
|--|--|
|[Azure bot Service](https://dev.botframework.com/)|Služba Azure bot Service poskytuje úplnou cloudovou webovou službu s koncovým bodem bot. Služby využívají [robot Framework](https://github.com/Microsoft/botframework), který je k dispozici v několika jazycích.|
|[Architektura robotů](https://github.com/Microsoft/botframework)|Microsoft bot Framework je komplexní platforma pro vytváření prostředí AI v konverzaci na podnikové úrovni.|
|[Emulátor pro robot Framework](https://github.com/Microsoft/botframework#Bot-Framework-Emulator)|Emulátor rozhraní robot Framework je desktopová aplikace pro různé platformy, která umožňuje vývojářům robotům testovat a ladit roboty sestavené pomocí sady robot Framework SDK. Emulátor rozhraní robot Framework můžete použít k otestování roboty spuštěného místně na vašem počítači nebo pro připojení k roboty spuštění vzdáleně.|
|[Nástroje robota](https://github.com/Microsoft/botbuilder-tools)|Nástroje pro robot Framework jsou kolekce nástrojů příkazového řádku pro různé platformy, které jsou navržené tak, aby pokryly ucelený pracovní postup pro vývoj robotů. |
|[Ukázky pro bot Builder](https://github.com/Microsoft/BotBuilder-Samples)|Plně vyvinuté ukázky robota jsou navržené tak, aby se předvedly scénáře, které budete muset implementovat, abyste mohli vytvářet skvělé roboty.|
