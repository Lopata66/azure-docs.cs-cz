[Šablona Resource Manageru](/azure/azure-resource-manager/templates/overview) je soubor JSON (JavaScript Object Notation), který definuje infrastrukturu a konfiguraci projektu. Tato šablona využívá deklarativní syntaxi, která umožňuje prohlásit, co máte v úmyslu vytvořit, aniž by k tomu bylo nutné psát sekvence programových příkazů. Pokud chcete získat další informace o vývoji šablon Resource Manageru, přečtěte si [dokumentaci k Resource Manageru](/azure/azure-resource-manager/) a [referenční informace k šablonám](/azure/templates).
