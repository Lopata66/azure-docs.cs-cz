Ke službám Azure Cognitive Services můžete přistupovat prostřednictvím dvou různých prostředků: prostředek s více službami nebo prostředek s jednou službou.

* Prostředek pro více služeb:
    * Získejte přístup k více službám Azure Cognitive Services s jedním klíčem a koncovým bodem.
    * Konsoliduje fakturaci ze služeb, které používáte.
* Prostředek jedné služby:
    * Získejte přístup k jedné službě Azure Cognitive Service s jedinečným klíčem a koncovým bodem pro každou vytvořenou službu. 
    * Pomocí bezplatné vrstvy vyzkoušejte službu.
