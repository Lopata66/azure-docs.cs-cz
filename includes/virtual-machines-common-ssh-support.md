## <a name="supported-ssh-key-formats"></a>Podporované formáty klíčů SSH

Azure aktuálně podporuje SSH protokol 2 (SSH-2) RSA dvojice veřejných soukromých klíčů s minimální délkou 2048 bitů. Ostatní formáty klíčů, jako je ED25519 a ECDSA nejsou podporovány.
