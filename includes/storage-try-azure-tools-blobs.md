> [!TIP]
> 
> Správa prostředků úložiště objektů Blob v Azure pomocí Průzkumníka služby Azure Storage. 
> [Průzkumník služby Azure Storage](https://azure.microsoft.com/features/storage-explorer/) je bezplatná samostatná aplikace od Microsoftu, která vám umožní [spravovat prostředky úložiště objektů Blob v Azure](../articles/vs-azure-tools-storage-explorer-blobs.md). Pomocí Průzkumníka služby Azure Storage, můžete můžete vizuálně vytvářet, číst, aktualizovat, odstranit kontejnery objektů blob a objekty BLOB, i správa přístupu k objektům BLOB kontejnerům a objektům BLOB.
