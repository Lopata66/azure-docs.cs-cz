Musíte mít [rozhraní API služeb Cognitive Services předplatné](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account) s přístupem k rozhraní API pro analýzu textu. Pokud předplatné nemáte, můžete si [vytvořit účet](https://azure.microsoft.com/try/cognitive-services/?api=bing-web-search-api) zdarma. Než budete pokračovat, budete potřebovat klíč předplatného pro analýzu textu po aktivaci účtu k dispozici.
