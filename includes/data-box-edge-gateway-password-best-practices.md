Mějte na paměti tyto osvědčené postupy:

- Doporučujeme uložit všechna hesla na bezpečném místě, abyste nemuseli resetovat heslo, pokud je zapomenuto. Služba správy nemůže načíst existující hesla. Může je resetovat jenom přes portál Azure. Pokud resetujete heslo, nezapomeňte před resetováním upozornit všechny uživatele.
- K rozhraní prostředí Windows PowerShell zařízení můžete přistupovat vzdáleně přes protokol HTTP. Z bezpečnostních důvodů byste měli používat protokol HTTP pouze v důvěryhodných sítích.
- Ujistěte se, že hesla zařízení jsou silná a dobře chráněná. Postupujte podle [doporučených postupů pro heslo](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management).
