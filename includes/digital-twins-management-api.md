V příkladech níže `YOUR_MANAGEMENT_API_URL` odkazuje na identifikátor URI digitální dvojče rozhraní API:

```plaintext
https://YOUR_INSTANCE_NAME.YOUR_LOCATION.azuresmartspaces.net/management/api/v1.0
```

| Name | Nahradit hodnotou |
| --- | --- |
| YOUR_INSTANCE_NAME | Název instance digitální dvojče Azure |
| YOUR_LOCATION | Oblast, který je hostitelem vaší instance |
