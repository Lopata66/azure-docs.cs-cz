V níže uvedených `YOUR_MANAGEMENT_API_URL` příkladech odkazuje na identifikátor URI digitálních vláken rozhraní API:

```plaintext
https://YOUR_INSTANCE_NAME.YOUR_LOCATION.azuresmartspaces.net/management/api/v1.0
```

| Name | Nahradit hodnotou |
| --- | --- |
| YOUR_INSTANCE_NAME | Název instance digitálního vlákna Azure |
| YOUR_LOCATION | Oblast, ve které je vaše instance hostována |
