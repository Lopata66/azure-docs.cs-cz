V příkladech níže `yourManagementApiUrl` odkazuje na identifikátor URI digitální dvojče rozhraní API:

```plaintext
https://yourInstanceName.yourLocation.azuresmartspaces.net/management/api/v1.0/
```

| Vlastní název | Nahradit hodnotou |
| --- | --- |
| Název_vaší_instance | Název instance digitální dvojče Azure |
| yourLocation | Jaké oblasti serveru vaší instance je hostován aplikací |
