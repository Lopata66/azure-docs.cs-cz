| **Zdrojů** | **Limit** |
| --- | --- |
| Maximální počet prostředků sdílení dat na jedno předplatné Azure |50 |
| Maximální počet odeslaných sdílených složek na prostředek sdílení dat |100 |
| Maximální počet přijatých sdílených složek na prostředek sdílení dat |100 |
| Maximální počet pozvánek na odeslanou akcii |100 |
| Maximální počet předplatných akcií na odeslanou akcii |100 |
| Maximální počet datových sad na sdílenou složku |100 |
| Maximální počet plánů snímků na sdílenou složku |1 |
