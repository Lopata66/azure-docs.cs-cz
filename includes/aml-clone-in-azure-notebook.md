### <a name="azure-notebooks---free-jupyter-based-notebooks-in-the-azure-cloud"></a>**Azure Notebooks** – bezplatné poznámkové bloky v cloudu Azure založené na Jupyteru
Ve službě Azure Notebooks je už nainstalovaná a nakonfigurovaná sada SDK.
  
1. Proveďte [rychlý start Začínáme](../articles/machine-learning/service/quickstart-get-started.md), při kterém vytvoříte pracovní prostor a spustíte službu Azure Notebooks.
1. Přejděte do služby [Azure Notebooks](https://notebooks.azure.com/).
1. V knihovně `Getting Started`, kterou jste vytvořili v rámci rychlého startu, přejděte do složky `tutorials`.
1. Otevřete poznámkový blok.

### <a name="your-own-jupyter-notebook-server"></a>**Váš vlastní server poznámkového bloku Jupyter**
1. Proveďte [rychlý start Začínáme se sadou Python SDK](../articles/machine-learning/service/quickstart-create-workspace-with-python.md), ve kterém nainstalujete sadu SDK a vytvoříte pracovní prostor.
1. Naklonujte [úložiště GitHub](https://aka.ms/aml-notebooks).
1. Zkopírujte adresář `aml_config`, který jste vytvořili v rámci rychlého startu, do naklonovaného adresáře.
1. Spusťte server poznámkového bloku v naklonovaném adresáři.
1. Přejděte do složky `tutorials`.
1. Otevřete poznámkový blok.
