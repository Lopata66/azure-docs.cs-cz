* [Python 2.x nebo 3.x](https://www.python.org/downloads/). Ujistěte se, že používáte 32bitovou, nebo 64bitovou instalaci podle požadavků vašeho nastavení. Po zobrazení výzvy v průběhu instalace nezapomeňte přidat Python do proměnné prostředí pro konkrétní platformu. Pokud používáte Python 2.x, možná bude nutné [nainstalovat nebo upgradovat *pip*, systém pro správu balíčků Pythonu](https://pip.pypa.io/en/stable/installing/).

* V případě potřeby nainstalujte balíček [azure-iot-device](https://pypi.org/project/azure-iot-device/) pomocí příkazu`pip install azure-iot-device`

* V případě potřeby nainstalujte balíček [azure-iothub-service-client](https://pypi.org/project/azure-iothub-service-client/) pomocí příkazu`pip install azure-iothub-service-client`
