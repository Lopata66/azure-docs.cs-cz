## <a name="create-a-repository"></a>Vytvořte úložiště

V tomto článku se používá úložiště šablon GitHubu, které vám usnadní začít. Šablona obsahuje úvodní aplikaci, která se používá k nasazení pomocí statického Web Apps Azure.

# <a name="no-framework"></a>[Žádná architektura](#tab/vanilla-javascript)

1. Ujistěte se, že jste přihlášeni k GitHubu, a přejděte do následujícího umístění a vytvořte nové úložiště:
    1. https://github.com/staticwebdev/vanilla-basic/generate
1. Pojmenování úložiště **My-First-static-Web-App**

> [!NOTE]
> Pro vytvoření webové aplikace se v Azure static Web Apps vyžaduje aspoň jeden soubor HTML. Úložiště, které vytvoříte v tomto kroku, zahrnuje jeden soubor _index.html_ .
