> [!IMPORTANT]
> Tato funkce je ve verzi Preview. Pokud ji chcete používat, je nutné nainstalovat [rozšíření Event Grid ](/cli/azure/azure-cli-extensions-list) pro [Azure CLI](/cli/azure/install-azure-cli). Můžete si je nainstalovat pomocí příkazu `az extension add --name eventgrid`. Pokud jste dříve nainstalovali rozšíření, aktualizujte jej s použitím `az extension update -n eventgrid`. Pokud používáte rozhraní REST API, zkontrolujte, že používáte `api-version=2018-09-15-preview`.
