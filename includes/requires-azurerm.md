> Pomocí této funkce Azure z prostředí PowerShell vyžaduje `AzureRM` nainstalovaným modulem. Jde o starší modul, který je k dispozici jenom pro Windows PowerShell 5.1, který už nebude dostávat nové funkce.

> 2. [Instalaci modulu AzureRM](/powershell/azure/azurerm/install-azurerm-ps) z relace prostředí PowerShell 5.1.
> 3. [Stáhnout a nainstalovat PowerShell Core 6.x nebo novější](/powershell/scripting/install/installing-powershell-core-on-windows).
> 4. [Instalace modulu Az](/powershell/azure/install-az-ps) v relaci prostředí PowerShell Core.
