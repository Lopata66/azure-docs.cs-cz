## <a name="prerequisites"></a>Požadavky

* Účet [MailChimp](https://www.MailChimp.com/) 

Než budete moct používat svůj účet MailChimp v aplikaci Logika, musíte autorizovat aplikaci Logika pro připojení k účtu MailChimp. Naštěstí to můžete udělat snadno z aplikace Logika na webu Azure Portal. 

Tady jsou kroky k autorizaci aplikace Logika pro připojení k účtu MailChimp:

1. Chcete-li vytvořit připojení k MailChimp, v návrháři aplikace Logika vyberte zobrazit **rozhraní API spravovaná společností Microsoft** v rozevíracím seznamu a pak do vyhledávacího pole zadejte *MailChimp.* Vyberte aktivační událost nebo akci, kterou chcete použít:  
   ![MailChimp krok 1](./media/connectors-create-api-mailchimp/mailchimp-1.png)
2. Pokud jste nevytvořili žádné připojení k MailChimp dříve, budete vyzváni k zadání mailchimp pověření. Tato pověření se použijí k autorizaci aplikace Logiky pro připojení k datům vašeho účtu MailChimp a k přístupu k nim:  
   ![MailChimp krok 2](./media/connectors-create-api-mailchimp/mailchimp-2.png)
3. Zadejte své uživatelské jméno a heslo MailChimp pro autorizaci aplikace Logika:  
   ![MailChimp krok 3](./media/connectors-create-api-mailchimp/mailchimp-3.png)   
4. Všimněte si, že připojení bylo vytvořeno a nyní můžete pokračovat v dalších krocích v aplikaci Logika:  
   ![MailChimp krok 4](./media/connectors-create-api-mailchimp/mailchimp-4.png)
