Chcete-li zobrazit událost zařízení, proveďte následující kroky v Azure Portal.

1. V Azure Portal přejdete na prostředek Data Box Edge/Data Box Gateway a pak přejdete na **sledování událostí zařízení >**.
2. Vyberte událost a zobrazte podrobnosti výstrahy. Pro vyřešení podmínky výstrahy proveďte příslušnou akci.

    ![Výběr události a zobrazení podrobností](media/data-box-edge-gateway-view-device-events/view-device-events.png)
