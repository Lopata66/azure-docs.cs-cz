> [!TIP]
> Pomocí příkazu [imblin dockeru](https://docs.docker.com/engine/reference/commandline/images/) můžete zobrazit seznam stažených iobrazů kontejnerů. Následující příkaz například uvádí ID, úložiště a značku každé stažené bitové kopie kontejneru formátované jako tabulka:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
