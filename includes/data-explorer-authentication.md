## <a name="authentication"></a>Ověřování

Chcete-li spustit následující příklad, potřebujete aplikaci Azure Active Directory (Azure AD) a instancí služby, které mají přístup k prostředkům. Pokud chcete vytvořit bezplatnou aplikaci Azure AD a přidat přiřazení rolí na úrovni předplatného, přečtěte si informace [o vytvoření aplikace Azure AD](/azure/active-directory/develop/howto-create-service-principal-portal). Potřebujete také ID adresáře (klienta), ID aplikace a tajný klíč klienta.
