Můžete také použít [emulátor Azure Cosmos DB](https://docs.microsoft.com/azure/cosmos-db/local-emulator) `https://localhost:8081`s identifikátorem URI aplikace . Klíč, který chcete použít s emulátorem, naleznete [v tématu Ověřování požadavků](../articles/cosmos-db/local-emulator.md#authenticating-requests).
