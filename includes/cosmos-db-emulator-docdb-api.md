Můžete [vyzkoušet Azure Cosmos DB zdarma](https://azure.microsoft.com/try/cosmosdb/) bez předplatného Azure, bezplatného zpoplatnění a závazků nebo vytvořit [Azure Cosmos DB účet úrovně free](../articles/cosmos-db/optimize-dev-test.md#azure-cosmos-db-free-tier)s prvním 400 ru/s a 5 GB úložištěm zdarma. [Emulátor Azure Cosmos DB](https://docs.microsoft.com/azure/cosmos-db/local-emulator) můžete použít také s identifikátorem URI `https://localhost:8081`. Klíč, který se má použít s emulátorem, najdete v tématu [ověřování požadavků](../articles/cosmos-db/local-emulator.md#authenticating-requests).
