> Tento článek obsahuje obsah, který vyžaduje Azure classic CLI. Aktuální verze rozhraní příkazového řádku Azure nemá podporu pro funkce popsané v tomto článku, takže rozhraní příkazového řádku classic je povinný. 
> 
> Rozhraní příkazového řádku classic může být nainstalovaná na straně po boku s moderní rozhraní příkazového řádku Azure, ale doporučujeme používat rozhraní příkazového řádku Azure pro všechny nové skripty a nasazení, kde je k dispozici podpora funkce. Pokud chcete nainstalovat rozhraní příkazového řádku classic, najdete v článku [instalace Azure classic CLI](/cli/azure/install-classic-cli) a instalaci aktuální rozhraní příkazového řádku najdete v tématu [instalace Azure CLI](/cli/azure/install-azure-cli).
