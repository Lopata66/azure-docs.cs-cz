> [!IMPORTANT]
> Tento článek obsahuje obsah, který vyžaduje Azure classic CLI. Aktuální verze rozhraní příkazového řádku Azure nemá podporu pro funkce popsané v tomto článku, takže rozhraní příkazového řádku classic je povinný. 
> 
> Rozhraní příkazového řádku classic je možné nainstalovat souběžně s moderní rozhraní příkazového řádku Azure, ale doporučujeme rozhraní příkazového řádku Azure pro všechny nové skripty a nasazení, kde je k dispozici podpora funkce. Pokud chcete nainstalovat rozhraní příkazového řádku classic, najdete v článku [instalace Azure classic CLI](https://docs.microsoft.com/cli/azure/install-classic-cli?view=azure-cli-latest). 
> Instalaci aktuální rozhraní příkazového řádku najdete v tématu [instalace rozhraní příkazového řádku Azure](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest).
