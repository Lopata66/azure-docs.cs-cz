## <a name="set-up-authentication"></a>Nastavení ověřování

Přístup k této službě, budete muset zadat klíč účtu, přístupový token nebo ověřovací token AAD.

### <a name="account-keys"></a>Klíče účtu

Klíče účtu jsou přihlašovací údaje, které umožňuje vaší aplikaci k ověřování ve službě Azure prostorových kotvy. Zamýšlený účel klíče účtu se můžete rychle začít. Zejména ve fázi vývoje integraci vaší aplikace s Azure prostorových kotvy vztahů. V důsledku toho můžete použít klíče účtu jejich vložením do klientské aplikace během vývoje. Jak budete nad rámec vývoje, má důrazně doporučujeme pro přesun do mechanismus ověřování, který je provozní úrovni, podporuje přístupové tokeny nebo ověřování uživatelů na základě AAD. K získání klíče účtu služby pro vývoj, navštivte svůj účet prostorových kotev vztahů Azure a přejděte na kartu "Klíče".
