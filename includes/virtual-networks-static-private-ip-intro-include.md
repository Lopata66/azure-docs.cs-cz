Virtuální počítač (VM) je automaticky přiřadí privátní IP adresu z rozsahu, který určíte, založené na podsíť, ve které je nasazena v. Adresa je uchován virtuálního počítače, až do odstranění virtuálního počítače. Azure dynamicky přiřadí další dostupnou privátní IP adresu z podsítě, které můžete vytvářet virtuální počítače. Přiřadíte statickou IP adresu, pokud chcete konkrétní IP adresu z podsítě přiřadit k virtuálnímu počítači.
