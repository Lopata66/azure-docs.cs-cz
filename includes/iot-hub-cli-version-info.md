> [!NOTE]
> Tento článek používá nejnovější verzi rozšíření Azure IoT s názvem `azure-iot`. Starší verze se nazývá `azure-iot-cli-ext`. V jednom okamžiku byste měli mít nainstalovanou jenom jednu verzi. K ověření aktuálně nainstalovaných rozšíření můžete použít `az extension list` příkazu.
>
> Pro odebrání starší verze rozšíření použijte `az extension remove azure-iot-cli-ext`.
> Pomocí `az extension add -name azure-iot` přidejte novou verzi rozšíření. 
>
> Pokud chcete zjistit, jaká rozšíření jste nainstalovali, použijte `az extension show `.
>
