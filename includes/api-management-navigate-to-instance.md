## <a name="go-to-your-api-management-instance"></a>Přejít k instanci služby API Management

1. Na webu Azure Portal vyhledejte a vyberte **služby správy rozhraní API**.

   ![Vybrat služby správy rozhraní API](media/api-management-navigate-to-instance/view-apim1.png)

1. Na obrazovce **Správa rozhraní API** vyberte instanci správy rozhraní API.

   ![Výběr instance správy rozhraní API](media/api-management-navigate-to-instance/view-apim2.png)
