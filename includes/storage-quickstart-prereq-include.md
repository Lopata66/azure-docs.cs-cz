Pro přístup k Azure Storage budete potřebovat předplatné Azure. Pokud ještě nemáte předplatné, vytvořte si před zahájením [bezplatný účet.](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)

Veškerý přístup k Azure Storage probíhá prostřednictvím účtu úložiště. Pro tento rychlý start vytvořte účet úložiště pomocí [portálu Azure](https://portal.azure.com/), Azure PowerShellu nebo Azure CLI. Nápovědu k vytvoření účtu úložiště najdete [v tématu Vytvoření účtu úložiště](../articles/storage/common/storage-account-create.md).
