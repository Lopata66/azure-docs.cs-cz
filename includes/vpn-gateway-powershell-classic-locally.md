Při práci s klasickým modelem nasazení nemůžete používat Azure Cloud Shell. Místo toho je nutné nainstalovat nejnovější verzi rutin prostředí PowerShell azure service management (SM) místně v počítači. Tyto rutiny se liší od rutin AzureRM nebo Az. Informace o instalaci rutin SM naleznete v [tématu Instalace rutin správy služeb](/powershell/azure/servicemanagement/install-azure-ps). Další informace o Azure PowerShellu obecně najdete v [dokumentaci k Azure PowerShellu](/powershell/azure/).
