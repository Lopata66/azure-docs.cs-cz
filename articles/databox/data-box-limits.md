## <a name="data-box-limits"></a>Limity datových schrápte

- Data Box může uložit maximálně 500 milionů souborů.
- Data Box podporuje maximálně 512 kontejnerů nebo sdílených položek v cloudu. Z adresářů nejvyšší úrovně v rámci sdílené složky uživatele se stanou kontejnery nebo sdílené složky Azure v cloudu.
