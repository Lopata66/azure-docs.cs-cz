# <a name="use-cli-to-configure-active-geo-replication-for-a-pooled-database-in-azure-sql-database"></a>Použití rozhraní příkazového příkazu ke konfiguraci aktivní geografické replikace pro sdruženou databázi v Azure SQL Database

Tento příklad skriptu Azure CLI konfiguruje aktivní geografickou replikaci pro sdruženou databázi v Azure SQL Database a převezme ji do sekundární repliky databáze.

Pokud se rozhodnete nainstalovat a používat rozhraní příkazového řádku (CLI) místně, musíte mít spuštěnou verzi Azure CLI 2.0 nebo novější. Verzi zjistíte spuštěním příkazu `az --version`. Pokud potřebujete instalaci nebo upgrade, přečtěte si téma [Instalace rozhraní příkazového řádku Azure CLI](/cli/azure/install-azure-cli).
