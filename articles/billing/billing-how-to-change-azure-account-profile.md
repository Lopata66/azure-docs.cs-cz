# <a name="change-your-azure-account-profile-information"></a>Změnit informace z vašeho profilu účtu Azure

Informace z vašeho profilu slouží vás kontaktovat ohledně vašeho účtu Azure. Je nutné jej uschovat ho aktualizovat tak, aby Microsoft vás může upozornit, o důležitých oznámeních.

E-mailovou adresu v profilu správce účtu Microsoft používá k odesílání důležité fakturace, služby a doporučení související oznámení o účtu. Doporučujeme vám, že zadáte e-mailovou adresu, která pravidelně kontrolovat. Jako správce účtu můžete také aktualizovat svůj profil, chcete-li změnit poštovní adresu a telefonní číslo pro účet.

E-mailovou adresu v profilu uživatele uživatelský účet, který není správcem účtu Microsoft používá k odesílání oznámení týkající se služby a doporučení pro účet. Všechny fakturační oznámení se odesílají na účet správce. Můžete také aktualizovat svůj profil, chcete-li změnit svou poštovní adresu a telefonní číslo, ale nezmění účet poštovní adresu nebo telefonní číslo.

## <a name="change-your-profile-information"></a>Změnit informace z vašeho profilu

1. Přihlaste se do [Centra účtů Azure](https://account.azure.com/Profile).
1. Vyberte tlačítko **Upravit podrobnosti** a pak aktualizujte informace o **profilu**.

   ![Příklad zobrazující informace z vašeho profilu Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Odesílání oznámení do další e-mailovou adresu

Můžete mít oznámení o službách a doporučení pro účet odesílat e-mailovou adresu kromě e-mailovou adresu účtu správce spojené s vaším účtem. E-mailovou adresu účtu správce získá důležitá oznámení o všech předplatných v rámci účtu, včetně aktualizace týkající se fakturace.

### <a name="change-the-account-administrator-email-address-in-your-profile"></a>Změna e-mailovou adresu účtu správce ve vašem profilu

Správce účtu je osoba, která nastavila účet Azure a která dostává důležitá e-mailová oznámení týkající se všech předplatných v rámci účtu. Doporučujeme zadat kontaktní e-mailovou adresu, kterou pravidelně kontroluje správce účtu.

### <a name="updating-your-profile-email-doesnt-update-your-sign-in-email-address"></a>Aktualizuje se profil e-mailu neaktualizuje přihlašovací e-mailovou adresu

Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu. Chcete-li změnit přihlašovací e-mailovou adresu, budete muset převést vlastnictví účtu.

### <a name="updating-your-profile-address-doesnt-update-your-credit-card-billing-address"></a>Aktualizace vašeho profilu adresy neaktualizuje fakturační adresa vaší platební karty

Zjistěte, jak aktualizovat vaše fakturační údaje, najdete v článku [změně platební karty použité k úhradě předplatného Azure](billing-how-to-change-credit-card.md).

### <a name="you-cant-update-your-country"></a>Nelze aktualizovat svou zemi

Z důvodu technická omezení nemůžeme změnit zemi pro existující účet. Můžete však vytvořit účet v požadovanou zemi a obraťte se na podporu Azure o migraci vašich služeb k tomuto účtu.

## <a name="service-and-marketing-emails"></a>Služby a uvádění na trh e-mailů

Tým Microsoft Azure odesílá e-maily týkající se služeb vás informuje o kritické výstrahy. Tým také odesílá doporučení účtu e-mailem, označované také jako marketingových e-mailů, můžete optimalizovat a škálujte své projekty.

Zobrazí se výzva v [webu Azure portal](https://portal.azure.com) ověření nebo aktualizovat svou e-mailovou adresu každých 90 dní. Účelem oznámení je Ujistěte se, že společnost Microsoft aktualizovala kontaktní údaje v případě jakékoli urgentní oznámení týkající se služeb. Microsoft pošle e-mailů vaší e-mailovou adresu profil s Azure informace o účtu pro:

- Oznámení služby
- Výstrahy zabezpečení
- Fakturační účely
- Podpora
- Marketing komunikace
- Doporučení osvědčených postupů, na základě vašeho využití Azure

Zadejte e-mailovou adresu, ve které chcete dostávat zprávy o vašem účtu. Tak, že zadáte e-mailovou adresu, jste vyjádří výslovný souhlas a přijímat komunikační sdělení od společnosti Microsoft.

![Příklad zobrazující aktualizace pole vaše kontaktní informace](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Změňte kontaktní e-mailovou adresu

Můžete změnit kontaktní e-mailovou adresu, pomocí následujících metod:

**Možnost 1**: Přihlaste se do [centra účtů Azure](https://account.azure.com/Profile). Vyberte **upravit podrobnosti** a pak aktualizujte informace o profilu e-mailovou adresu. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

**Možnost 2**: Přejděte [kontaktní informace](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) oblast na webu Azure Portal a zadejte e-mailovou adresu. Adresa, které poskytnete obdrží služby a marketingových materiálů. E-mailovou adresu v System Center profilu účtu Azure se aktualizuje tak, aby odpovídaly automaticky. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

**Možnost 3**: V [webu Azure portal](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade), vyberte ikonu s vašimi iniciálami nebo obrázek. Potom vyberte příslušnou kontextovou nabídku (**...** ). V dalším kroku vyberte **Moje kontaktní informace** v nabídce a zadejte e-mailovou adresu. Adresa, které poskytnete obdrží služby a marketingových materiálů. E-mailovou adresu ve vašem profilu účtu centra Azure se aktualizuje tak, aby odpovídaly automaticky. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

![Příklad zobrazující aktualizace kontaktní údaje v Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Odhlásit marketingových e-mailů

Odhlásit z příjmu marketingových e-mailů:

1. Přejděte [formuláři žádosti o](https://account.microsoft.com/profile/permissions-link-request) odeslat žádost pomocí vašeho profilu e-mailovou adresu. Zobrazí se odkaz e-mailem na aktualizujte předvolby.
2. Tento odkaz zobrazí požadované typy marketingovou komunikaci, která e-mailová adresa je povoleno v. Vymazat všechna témata, které chcete odhlásit a pak klikněte na tlačítko **Uložit**.


![Příklad zobrazující možnosti oprávnění spravovat komunikace](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

### <a name="you-still-receive-service-notifications-if-you-opt-out-of-marketing-communications"></a>Pokud jste výslovný nesouhlas s marketingovou komunikaci stále přijímat oznámení o službách

Pokud jste zadali e-mailovou adresu pro komunikaci, zobrazí se oznámení služeb na základě vašeho účtu. Když je odhlásit, je jediným odhlásit marketingových komunikace – budete stále dostávat oznámení služby.
