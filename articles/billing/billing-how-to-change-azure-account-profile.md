# <a name="change-the-profile-information-for-your-azure-account"></a>Změnit informace profilu pro váš účet Azure

Společnost Microsoft používá e-mailovou adresu v profilu účtu správce k odesílání důležité fakturace, služby a doporučení související oznámení o účet Azure. Doporučujeme vám, že zadáte e-mailovou adresu, která pravidelně kontrolovat. Jako správce účtu může také aktualizovat svůj profil, chcete-li změnit poštovní adresu a telefonní číslo pro účet.

Společnost Microsoft používá e-mailovou adresu v profilu uživatele uživatelský účet, který není správcem účtu k odesílání oznámení týkající se služby a doporučení pro účet. Všechny fakturační oznámení se odesílají na účet správce. Můžete také aktualizovat svůj profil, chcete-li změnit svou poštovní adresu a telefonní číslo, ale nezmění účet poštovní adresu nebo telefonní číslo.

## <a name="change-your-profile-information"></a>Změnit informace z vašeho profilu

1. Přihlaste se do [Centra účtů Azure](https://account.azure.com/Profile).
1. Vyberte tlačítko **Upravit podrobnosti** a pak aktualizujte informace o **profilu**.

   ![Příkladem informací o profilu Azure](./media/billing-how-to-change-azure-account-profile/profile.png)

### <a name="send-notifications-to-an-additional-email-address"></a>Odesílání oznámení do další e-mailovou adresu

Můžete mít oznámení o službách a doporučení pro účet odesílat e-mailovou adresu kromě e-mailovou adresu správce účtu, který je spojen s vaším účtem. E-mailovou adresu účtu správce získá důležitá oznámení o všech předplatných v rámci účtu, včetně aktualizace týkající se fakturace.

### <a name="change-the-account-administrators-email-address-in-your-profile"></a>Změnit správce účtu e-mailovou adresu ve vašem profilu

Správce účtu je osoba, která nastavila účet Azure a která dostává důležitá e-mailová oznámení týkající se všech předplatných v rámci účtu. Doporučujeme zadat kontaktní e-mailovou adresu, kterou pravidelně kontroluje správce účtu.

### <a name="update-your-sign-in-email-address"></a>Aktualizovat přihlašovací e-mailovou adresu

Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu. Chcete-li změnit přihlašovací e-mailovou adresu, budete muset převést vlastnictví účtu.

### <a name="update-the-billing-address-for-your-credit-card"></a>Aktualizace fakturační adresa vaší platební karty

Zjistěte, jak aktualizovat vaše fakturační údaje, najdete v článku [změně platební karty použité k úhradě předplatného Azure](billing-how-to-change-credit-card.md).

### <a name="update-your-countryregion"></a>Aktualizujte svou zemi/oblast

Z důvodu technická omezení nemůžeme změnit zemi/oblast pro existující účet. Ale můžete vytvořit účet v požadovanou zemi/oblast a pak požádejte podporu Azure o migraci vašich služeb k tomuto účtu.

## <a name="service-and-marketing-emails"></a>Služby a uvádění na trh e-mailů

Se zobrazí výzva k v [webu Azure portal](https://portal.azure.com) ověření nebo aktualizovat svou e-mailovou adresu každých 90 dní. Microsoft pošle e-mailů vaší e-mailovou adresu profil s Azure informace o účtu pro:

- Oznámení služby
- Výstrahy zabezpečení
- Fakturační účely
- Podpora
- Marketing komunikace
- Doporučení osvědčených postupů, na základě vašeho využití Azure

Zadejte e-mailovou adresu, ve které chcete dostávat zprávy o vašem účtu. Tak, že zadáte e-mailovou adresu, které jste vyjádření výslovného souhlasu s přijímat komunikační sdělení od společnosti Microsoft.

![Příklad výzvy k aktualizaci vaše kontaktní informace](./media/billing-how-to-change-azure-account-profile/update-contact-information.png)

### <a name="change-your-contact-email-address"></a>Změňte kontaktní e-mailovou adresu

Kontaktní e-mailovou adresu můžete změnit pomocí jedné z následujících metod:

* Přihlaste se do [Centra účtů Azure](https://account.azure.com/Profile). Vyberte **upravit podrobnosti** a pak aktualizujte profil e-mailovou adresu. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

* Přejděte [kontaktní informace](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade) oblast na webu Azure Portal a zadejte e-mailovou adresu. Adresa, které poskytujete obdrží služby a marketingových materiálů. E-mailovou adresu v System Center profilu účtu Azure se aktualizuje tak, aby odpovídaly automaticky. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

* V [webu Azure portal](https://portal.azure.com/#blade/HubsExtension/ContactInfoBlade), vyberte ikonu s vašimi iniciálami nebo obrázek. Potom vyberte příslušnou kontextovou nabídku ( **...** ). V dalším kroku vyberte **Moje kontaktní informace** v nabídce a zadejte e-mailovou adresu. Adresa, které poskytujete obdrží služby a marketingových materiálů. E-mailovou adresu ve vašem profilu účtu centra Azure se aktualizuje tak, aby odpovídaly automaticky. Aktualizuje se profil e-mailovou adresu neaktualizuje přihlašovací e-mailovou adresu.

![Příklad aktualizuje e-mailovou adresu v Azure](./media/billing-how-to-change-azure-account-profile/azure-contact-information.png)

### <a name="opt-out-of-marketing-emails"></a>Vyjádřit výslovný nesouhlas marketingových e-mailů

Chcete-li vyjádřit výslovný nesouhlas přijímají marketingových e-mailů:

1. Přejděte [formuláři žádosti o](https://account.microsoft.com/profile/permissions-link-request) odeslat žádost s použitím vašeho profilu e-mailovou adresu. Obdržíte odkaz e-mailem na aktualizujte předvolby.
2. Vyberte odkaz k otevření **spravovat oprávnění komunikace** stránky. Tato stránka zobrazuje typy marketingovou komunikaci, která e-mailová adresa bude poskytnut souhlas k. Vymazat všechna témata, které chcete optimalizované z a pak vyberte **Uložit**.


![Příklad stránky pro správu oprávnění ve komunikace](./media/billing-how-to-change-azure-account-profile/manage-communication-permissions.png)

Když zakážete marketingovou komunikaci, budete stále dostávat oznámení služby, podle vašeho účtu.
