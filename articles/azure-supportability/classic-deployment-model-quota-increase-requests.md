# <a name="classic-deployment-model"></a>Model nasazení Classic

Model nasazení Classic je starší generace pozemního režimu nasazení Azure vynucuje omezení kvóty virtuálních procesorů globální pro virtuální počítače a škálovací sady virtuálních počítačů. Model nasazení Classic se už nedoporučuje a je nyní nahrazen podle modelu Resource Manager. Další informace o těchto dvou nasazení modelů a využít výhod Resource Manageru najdete na stránce modelu nasazení Resource Manager. Když se vytvoří nový odběr, je k němu přiřazen výchozí kvóta virtuálních procesorů. Kdykoli nového virtuálního počítače je možné nasadit s použitím modelu nasazení Classic, nesmí překročit součet nové i stávající virtuální procesory využití napříč všemi oblastmi kvóty virtuálních procesorů schválené pro model nasazení Classic. Další informace o kvótách na stránce limity služby a předplatného Azure.

Můžete požádat o zvýšení limitu virtuálních procesorů pro model nasazení Classic prostřednictvím příkazu Nápověda a podpora nebo použití + kvóty okno na portálu.
