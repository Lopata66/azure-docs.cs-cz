# <a name="get-started-with-azure-security-center-for-iot"></a>Začínáme se službou Azure Security Center pro IoT

Tento článek obsahuje vysvětlení různých součástí služby Azure Security Center for IoT a vysvětluje, jak začít se službou pomocí dvou možných možností nasazení.

## <a name="deployment-options"></a>Možnosti nasazení

Vyberte si scénář služby, který nejlépe vyhovuje vašim požadavkům na zařízení ioT a prostředí.

### <a name="built-in-deployment"></a>Integrované nasazení

Pomocí bezproblémové integrované možnosti nasazení lze Centrum zabezpečení Azure pro IoT rychle integrovat do vašeho IoT Hubu a poskytnout analýzu zabezpečení konfigurace centra IoT, identity a správy zařízení a vzorů komunikace mezi centrálním zařízením.

Spusťte [integrované nasazení](iot-hub-integration.md) s monitorováním a doporučeními služby IoT Hub.
    <br>

### <a name="enhanced-deployment"></a>Vylepšené nasazení

Pro rozšířené možnosti zabezpečení, nasazení Azure Security Center pro agenty IoT kromě povolení zabezpečení IoT Hub poskytuje shromažďování událostí na základě agenta, analýzy a detekce hrozeb klíčových dat zabezpečení ze zařízení IoT, stejně jako komplexní možnosti správy stavů zabezpečení.

Spusťte [vylepšené nasazení](security-agents.md) s komplexním řešením ochrany před hrozbami a správy stavů zabezpečení založeném na agentech.
