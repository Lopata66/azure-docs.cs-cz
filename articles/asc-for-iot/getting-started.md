# <a name="get-started-with-azure-security-center-for-iot"></a>Začínáme se službou Azure Security Center pro IoT

Tento článek obsahuje vysvětlení různých komponent Azure Security Center pro službu IoT a vysvětluje, jak začít se službou pomocí dvou možných možností nasazení.

## <a name="deployment-options"></a>Možnosti nasazení

Vyberte scénář služby, který nejlépe vyhovuje vašim požadavkům na zařízení a prostředí IoT.

### <a name="built-in-deployment"></a>Integrované nasazení

Použití možnosti bezproblémové, integrovaného nasazení Azure Security Center pro IoT se dá rychle integrovat do vaší IoT Hub a poskytuje analýzu zabezpečení konfigurace služby IoT Hub, identity a správy zařízení a způsobů komunikace centra a zařízení.

Zahajte [integrované nasazení](iot-hub-integration.md) , které nabízí IoT Hub monitorování a doporučení.
    <br>

### <a name="enhanced-deployment"></a>Rozšířené nasazení

Pro rozšířené možnosti zabezpečení nasazování Azure Security Center pro agenty IoT kromě povolení zabezpečení IoT Hub poskytuje shromažďování událostí založené na agentech, analýzu a detekci hrozeb klíčových dat zabezpečení z vašich zařízení IoT a také komplexní možnosti správy stav zabezpečení.

Spusťte [Rozšířené nasazení](security-agents.md) s využitím komplexní ochrany před hrozbami založenými na agentech a řešení pro správu stav zabezpečení.
