# <a name="azure-security-center-for-iot-security-alerts"></a>Výstrahy zabezpečení ve službě Azure Security Center pro IoT

Azure Security Center for IoT průběžně analyzuje vaše řešení IoT pomocí pokročilých analýz a analýzy hrozeb, které vás upozorní na škodlivou aktivitu.
Kromě toho můžete vytvořit vlastní výstrahy na základě znalosti očekávaného chování zařízení.
Výstraha slouží jako ukazatel možného kompromisu a měla by být prošetřena a opravena.

V tomto článku najdete seznam předdefinovaných výstrah, které se můžou aktivovat na vašich zařízeních IoT Hub a nebo IoT.
Kromě integrovaných výstrah umožňuje Azure Security Center for IoT definovat vlastní výstrahy na základě očekávaného ioT hubu nebo chování zařízení.
Další podrobnosti naleznete [v tématu přizpůsobitelné výstrahy](concept-customizable-security-alerts.md).
