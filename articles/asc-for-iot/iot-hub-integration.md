# <a name="get-started-with-built-in-iot-hub-integration"></a>Začínáme s integrovanou integrací IoT Hubu 

Tato možnost umožňuje používat službu bez použití Azure Security Center pro agenty zabezpečení IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Povolení integrované integrace ioT hubu
Chcete-li povolit monitorování správy identit zařízení, zařízení do cloudu a vzory komunikace mezi cloudem a zařízeními, spusťte službu následujícím způsobem: 

1. Otevřete **ioT hub**.
1. Vyberte nabídku **Přehled zabezpečení.** 
1. Klikněte na **Zabezpečit své řešení IoT** a vyplňte přihlašovací formulář. 

Blahopřejeme! Už jste dokončili povolení služby Azure Security Center for IoT ve vašem Centru IoT Hub.
