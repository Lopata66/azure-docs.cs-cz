# <a name="get-started-with-built-in-iot-hub-integration"></a>Začínáme s integrovanou integrací IoT Hub 

Tato možnost umožňuje používat službu bez použití Azure Security Center pro agenty zabezpečení IoT.

## <a name="enable-built-in-iot-hub-integration"></a>Povolit integrovanou integraci IoT Hub
Pokud chcete povolit monitorování komunikačních způsobů správy identit zařízení, zařízení do cloudu a cloudu na zařízení, spusťte službu následujícím způsobem: 

1. Otevřete **IoT Hub**.
1. Vyberte nabídku **Přehled zabezpečení** . 
1. Klikněte na **zabezpečení řešení IoT** a vyplňte formulář pro registraci. 

Blahopřejeme! Dokončili jste povolování Azure Security Center služby IoT v IoT Hub.
