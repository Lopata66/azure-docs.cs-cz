# <a name="release-notes-azure-event-grid-on-iot-edge"></a>Poznámky k verzi: Azure Event Grid na IoT Edge

## <a name="100-preview1"></a>1.0.0-náhled1

Počáteční verze Azure Event Grid na IoT Edge. Zahrnuty funkce:

* Vytvoření tématu
* Vytvoření předplatného událostí
* Rozšířené filtry
* Dávkování výstupu
* Zásady opakování
* Modul k publikování modulu
* Publikovat na WebHook jako cíl
* Publikování do centra IoT Edge Hub jako cíl
* Publikování do cloudové služby Azure Event Grid jako cíl
* Trvalý stav metadat
* Integrace modulu úložiště objektů blob

Tagy:`1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0-náhled2

Preview 2 služby Azure Event Grid na IoT Edge přidáno:

* Konfigurovatelné uchování událostí na disk
* Metriky tématu
* Metriky předplatného událostí
* Publikování do centra událostí jako cíl
* Publikovat do front sběrnice jako cíl
* Publikovat na témata sběrnice jako cíl
* Publikovat do front úložiště jako cíl

`1.0.0-preview2`Tagy: `1.0`, ,`latest`
