# <a name="release-notes-azure-event-grid-on-iot-edge"></a>Poznámky k verzi: Azure Event Grid v IoT Edge

## <a name="100-preview1"></a>1.0.0 – preview1

Počáteční verze Azure Event Grid v IoT Edge Zahrnuté funkce:

* Vytvoření tématu
* Vytvoření odběru událostí
* Rozšířené filtry
* Dávkování výstupu
* Zásady opakování
* Publikování modulu pro modul
* Publikovat do Webhooku jako cíl
* Publikovat do centra IoT Edge jako cíl
* Publikování do cloudové služby Azure Event Grid jako cíl
* Trvalý stav pro metadata
* Integrace modulu BLOB Storage

Značky`1.0.0-preview1`

## <a name="100-preview2"></a>1.0.0 – preview2

Verze Preview 2 Azure Event Grid IoT Edge přidána:

* Konfigurovatelné události trvalého uložení na disk
* Metriky témat
* Metriky odběru událostí
* Publikovat do Event Hubs jako cíl
* Publikovat do Service Bus front jako cíl
* Publikování pro Service Bus témata jako cíl
* Publikovat do front úložiště jako cíl

Značky: `1.0.0-preview2` , `1.0` ,`latest`
