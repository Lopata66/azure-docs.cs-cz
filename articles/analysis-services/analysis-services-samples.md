# <a name="azure-analysis-services-samples"></a>Ukázky služby Azure Analysis Services

Následující ukázkové zdroje vám pomohou získat informace o službě Analysis Services pro vaše prostředí a otestovat je.

## <a name="code-samples"></a>Ukázky kódů

Úložiště [Analysis Services](https://github.com/Microsoft/Analysis-Services) na GitHubu obsahuje ukázky kódu s otevřeným zdrojovým kódem a komunitní projekty. 

## <a name="tabular-model-project-and-database-samples"></a>Ukázky projektu a databáze tabulkového modelu

[Adventure Works for Analysis Services](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks-analysis-services) na GitHubu je nejčastěji používaný ukázkový tabulkový model projektu. Můžete si stáhnout projekt VS nebo dokončenou ukázkovou databázi tabulkového modelu.

## <a name="sample-database-on-azure"></a>Ukázková databáze v Azure

Azure Synapse Analytics (SQL Data Warehouse) poskytuje ukázkovou databázi AdventureWorksDW, která může být zahrnuta do zřízeného prostředku. Další informace najdete [v tématu Úvodní příručka: Vytvoření fondu SQL Azure Synapse Analytics a dotazování na něj](/azure/sql-data-warehouse/create-data-warehouse-portal).

## <a name="sample-databases-on-github"></a>Ukázkové databáze na GitHubu

Tyto ukázkové databáze na GitHubu se můžou použít k vytváření a testování vlastních modelů. 

[Ukázkové databáze služby Adventure Works](https://github.com/Microsoft/sql-server-samples/releases/tag/adventureworks2014)

[Ukázkové databáze širokých světových dovozců](https://github.com/Microsoft/sql-server-samples/releases/tag/wide-world-importers-v1.0)

## <a name="adventure-works-internet-sales-sample-model-in-azure-portal"></a>Ukázkový model adventure works internetového prodeje na webu Azure Portal

Pokud máte server Analysis Services na webu Azure Portal, můžete rychle a snadno vytvořit ukázkový model. Na stránce s přehledem serveru na webu Azure Portal klikněte na **Nový model**a potom v části Zvolit **zdroj dat**vyberte **Ukázková data**.
