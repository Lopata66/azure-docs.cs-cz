# <a name="account-management-overview"></a>Přehled správy účtu

Při vytváření služby CloudSimple vytvoří účet na CloudSimple.  Účet je přidružený k vašemu předplatnému Azure, kde se služba nachází.  Všichni uživatelé s **vlastníka** a **Přispěvatel** mají přístup k portálu CloudSimple role v rámci předplatného.  ID předplatného Azure a ID tenanta přidružené služby CloudSimple se nachází na [stránku účty](https://docs.azure.cloudsimple.com/account/).

## <a name="additional-alert-emails"></a>Další e-maily s výstrahami

ID e-mailu můžete nakonfigurovat v CloudSimple přijímat výstrahy:

* Související služby
* Pro automatické zpracování

## <a name="cloudsimple-operator-access"></a>Přístup CloudSimple – operátor

Můžete řídit přístup k portálu CloudSimple pro pracovníky operace služby.  Pracovníci služby operations přihlášení k portálu při vyplňte lístek podpory.  Operace služby bude opravte všechny problémy hlášeny a akcí provedených jsou k dispozici v protokolech auditu.

## <a name="users"></a>Uživatelé

Všichni uživatelé, kteří mají **vlastníka** a **Přispěvatel** roli v předplatném máte přístup k portálu CloudSimple.  Při přístupu k portálu pro uživatele vytvořit CloudSimple účtu.  Můžete zakázat přístup k portálu CloudSimple pro konkrétní uživatele na stránce účty.
