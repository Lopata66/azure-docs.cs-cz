# <a name="create-resource-pools-for-your-private-cloud-with-azure-subscription-mapping"></a>Vytvoření fondů prostředků pro privátní cloud s mapováním předplatného Azure
Mapování předplatného Azure umožňuje vytvářet fondy prostředků pro váš privátní cloud z dostupných fondů prostředků vSphere. Na portálu CloudSimple můžete zobrazit a spravovat předplatné Azure pro vaše privátní cloudy.

> [!NOTE]
> Mapování fondu zdrojů také mapuje všechny podřízené fondy prostředků. Nadřazený fond prostředků nelze namapovat, pokud jsou již mapovány všechny podřízené fondy prostředků.

1. [Přístup k portálu CloudSimple](access-cloudsimple-portal.md).
2. Otevřete stránku **Prostředky** a vyberte **mapování předplatných Azure**.  
3. Klikněte na **Upravit mapování předplatného Azure**.  
4. Chcete-li mapovat dostupné fondy zdrojů, vyberte je vlevo a klikněte na šipku směřující doprava. 
5. Chcete-li mapování odebrat, vyberte je vpravo a klepněte na šipku směřující doleva. 

    ![Předplatná Azure](media/resources-azure-mapping.png)

6. Klikněte na tlačítko **OK**.
