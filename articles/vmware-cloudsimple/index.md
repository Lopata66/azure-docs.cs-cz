# <a name="azure-vmware-solution-by-cloudsimple"></a>Řešení Azure VMware společnosti CloudSimple

Vítá vás řešení Azure VMware společnosti CloudSimple. Najdete tu podrobné informace ohledně použití CloudSimple a postupu pro zajištění podpory řešení s využitím této služby. Podívejte se na články nebo si projděte obsah.

- Pokud se chcete dozvědět víc o řešení Azure VMware společnosti CloudSimple, projděte si [přehled](cloudsimple-vmware-solutions-overview.md).
- Pokud se chcete dozvědět víc o řešení privátního cloudu CloudSimple, vyberte [přehled privátního cloudu CloudSimple](cloudsimple-private-cloud.md).
- Pokud se chcete dozvědět víc o uzlech CloudSimple, vyberte téma [Uzly](cloudsimple-node.md).
- Pokud chcete začít využívat řešení Azure VMware společnosti CloudSimple, vyberte téma [Inicializace a zřizování kapacity](quickstart-create-cloudsimple-service.md).

|--------------------------|-----------------------------|-------------------------------------------------------------------------------|--------------------|
| **Vytvoření služby**     | Služba CloudSimple         | Konfigurace hraniční sítě služby CloudSimple                                    | portál Azure       |
|                          | Uzel CloudSimple            | Zřizování kapacity pro použití ve službě CloudSimple                          | portál Azure       |
| **Spotřeba služeb**  | Privátní cloud               | Vytváření prostředí VMware                                                    | Portál CloudSimple |
|                          | Virtuální síť/podsíť                 | Vytváření podkladových sítí pro využití VMware                               | Portál CloudSimple |
|                          | Tabulka bran firewall              | Konfigurace pravidel zabezpečení ve směru sever/jih a východ/západ napříč podkladovými sítěmi | Portál CloudSimple |
|                          | Veřejná IP adresa           | Získání veřejných adres pro úlohy VMware                           | Portál CloudSimple |
|                          | Připojení Azure            | Navázání připojení k místním nebo virtuálním sítím v Azure         | Portál CloudSimple |
| **Unifikovaná správa**   | Virtuální počítač CloudSimple | Správa virtuálních počítačů VMware na webu Azure Portal                                | portál Azure       |
