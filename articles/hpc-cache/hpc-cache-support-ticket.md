# <a name="open-a-support-ticket-for-azure-hpc-cache"></a>Otevření lístku podpory pro Azure HPC Cache

K otevření lístku podpory použijte portál Azure. Přejděte na instanci mezipaměti a klikněte na odkaz **Nový požadavek na podporu,** který se zobrazí v dolní části postranního panelu.

Pokud chcete otevřít lístek, když nemáte aktivní mezipaměť, použijte hlavní stránku podpory + podpory z portálu Azure. Otevřete nabídku portálu z ovládacího prvku v levém horním rohu obrazovky, přejděte dolů a klepněte na tlačítko **Nápověda + podpora**.

Zvolte **Nová žádost o podporu** a vyberte **technická** nápověda specifická pro Azure HPC Cache.

Vyberte předplatné ze seznamu.

Chcete-li najít službu Azure HPC Cache, klikněte na tlačítko **Všechny služby** a vyhledejte HPC.

![Snímek obrazovky s žádostí o podporu – karta Základy, částečně vyplněná, jak je popsáno](media/hpc-cache-support-request.png)

Vyplňte zbytek polí s vašimi informacemi a preferencemi, pak odešlete lístek, když jste připraveni.

Po odeslání žádosti obdržíte potvrzovací e-mail s číslem letenky. Pracovník podpory vás bude kontaktovat ohledně žádosti.
