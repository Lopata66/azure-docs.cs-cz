# <a name="whats-new-in-azure-web-application-firewall"></a>Co je nového v bráně firewall webových aplikací Azure?

Brána firewall webových aplikací Azure se průběžně aktualizuje. Chcete-li mít aktuální informace o nejnovějším vývoji, tento článek obsahuje informace o:

- Nejnovější verze
- Známé problémy
- Opravy chyb
- Zastaralé funkce

## <a name="new-features"></a>Nové funkce

|Funkce  |Popis  |Datum přidání  |
|---------|---------|---------|
|Sada pravidel pro zmírnění botů (náhled)|Můžete povolit sadu pravidel zmírňující bot, spolu s crs sadu pravidel, kterou zvolíte. | Listopad 2019 |
|Integrace GeoDB (náhled)|Nyní můžete vytvořit vlastní pravidla omezující provoz podle země původu. | Listopad 2019 |
|Zásady WAF pro pracoviště/identifikátor URI (náhled)|WAF-v2 nyní podporuje použití zásady na schytá, stejně jako pravidla založená na cestě. Viz [Vytvoření zásad WAF](create-waf-policy-ag.md). | Listopad 2019 |
|Vlastní pravidla WAF |WAF_v2 brány aplikací nyní podporuje vytváření vlastních pravidel. Viz [Vlastní pravidla brány aplikace](custom-waf-rules-overview.md). |Červen 2019 |
|Seznam konfigurace a vyloučení WAF     |Přidali jsme další možnosti, které vám pomohou nakonfigurovat waf a snížit falešně pozitivní výsledky. Další informace naleznete v [tématu Omezení velikosti a seznamy vyloučení brány firewall webových aplikací.](application-gateway-waf-configuration.md)|Prosinec 2018|
