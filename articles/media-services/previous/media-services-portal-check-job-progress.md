# <a name="monitor-encoding-job-progress-with-the-azure-portal"></a>Monitorování průběhu úlohy kódování pomocí webu Azure portal

> [!NOTE]
> Do Media Services v2 se nepřidávají žádné nové funkce. <br/>Projděte si nejnovější verzi, [Media Services v3](https://docs.microsoft.com/azure/media-services/latest/). Viz také [pokyny k migraci z v2 na v3](../latest/migrate-from-v2-to-v3.md)

## <a name="overview"></a>Přehled

Při spuštění úlohy, často vyžadují způsob, jak sledovat průběh úlohy. 

Pokud chcete monitorovat průběh úlohy kódování, klikněte na **Nastavení** (v horní části stránky) a pak vyberte **Úlohy**.

![Úlohy](./media/media-services-portal-vod-get-started/media-services-jobs.png)

Můžete kliknout na úlohu zobrazíte další podrobnosti.

![Úlohy](./media/media-services-portal-vod-get-started/media-services-job-progress2.png)

## <a name="next-steps"></a>Další postup
Po kódování úloh je jedna věc, můžete publikovat a přehrát vaše prostředky, jak je popsáno [tady](media-services-portal-publish.md).
