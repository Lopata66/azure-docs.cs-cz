# <a name="manage-smart-groups"></a>Správa inteligentních skupin

[Inteligentní skupiny](https://aka.ms/smart-groups) používají algoritmy strojového učení k seskupení výstrah na základě společného výskytu nebo podobnosti, takže uživatel může nyní spravovat inteligentní skupiny namísto nutnosti spravovat každou výstrahu jednotlivě. Tento článek vás provede, jak získat přístup a používat inteligentní skupiny v Azure Monitoru.

1. Chcete-li zobrazit inteligentní skupiny vytvořené pro instance výstrah, můžete

     1. Klikněte na **inteligentní skupiny** ze stránky **Souhrn výstrah**    
    ![Monitorování](./media/alerts-managing-smart-groups/sg-alerts-summary.jpg)
    
     1. Klikněte na Upozornění inteligentních skupin na stránce Všechna upozornění   
     ![Monitorování](./media/alerts-managing-smart-groups/sg-all-alerts.jpg)

2. Tím přejdete do zobrazení seznamu pro všechny inteligentní skupiny vytvořené přes instance výstrah. Místo prosévání prostřednictvím více výstrah, nyní můžete řešit inteligentní skupiny místo.   
![Monitorování](./media/alerts-managing-smart-groups/sg-list.jpg)

3. Kliknutím na libovolnou inteligentní skupinu se otevře stránka podrobností, kde můžete vidět důvod seskupení spolu s upozorněními členů. Tato agregace umožňuje řešit jednotné inteligentní skupiny namísto prosévání prostřednictvím více výstrah.   
![Monitorování](./media/alerts-managing-smart-groups/sg-details.jpg)
