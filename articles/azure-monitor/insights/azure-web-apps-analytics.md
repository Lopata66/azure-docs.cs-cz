# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Zobrazení analytických dat pro metriky ve všech svých prostředcích Azure Web Appu

![Symbol webových aplikací](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Řešení Azure Web App Analytics přešlo na komunitní podporu.  
>- Řešení už není dostupné na Azure Marketplace, ale dá se nainstalovat ze [šablon Azure QuickStart,](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) kde ho podporuje komunita.
>- Zákazníci, kteří již nainstalovali řešení, jej mohou nadále používat se změnami.  
>- Společnost Microsoft doporučuje používat [Application Insights](../../azure-monitor/app/app-insights-overview.md) ke sledování webových aplikací.
