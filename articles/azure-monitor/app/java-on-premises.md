# <a name="java-codeless-application-monitoring-on-premises---azure-monitor-application-insights---public-preview"></a>Monitorování bezkódových aplikací java v místním prostředí – Azure Monitor Application Insights – public preview

Java codeless application monitoring je o jednoduchosti - neexistují žádné změny kódu, agent Java může být povolen pouze prostřednictvím několika konfiguračních změn.

## <a name="overview"></a>Přehled

Jakmile je povolena, bude agent Java automaticky shromažďovat velké množství požadavků, závislostí, protokolů a metrik z nejpoužívanějších knihoven a architektur.

Postupujte podle [podrobných pokynů](https://docs.microsoft.com/azure/azure-monitor/app/java-in-process-agent) pro všechna prostředí, včetně místních prostředí.
