# <a name="what-is-azure-database-migration-service"></a>Co je Azure Database Migration Service?

Azure Database Migration Service je plně spravovaná služba navržená tak, aby umožňovala bezproblémové migrace z více zdrojů databáze na datové platformy Azure s minimálními prostoji (migrace online).

## <a name="migrate-databases-to-azure-with-familiar-tools"></a>Migrace databází do Azure pomocí známých nástrojů

Služba Azure Database Migration Service integruje některé funkce našich stávajících nástrojů a služeb. Poskytuje zákazníkům komplexní, vysoce dostupné řešení. Služba používá [Pomocníka pro migraci dat](https://aka.ms/dma) ke generování sestav hodnocení, které poskytují doporučení, která vás provedou změnami požadovanými před provedením migrace. Je na vás, abyste provedli jakoukoli požadovanou nápravu. Až budete připraveni zahájit proces migrace, služba Migrace databáze Azure provede všechny požadované kroky. Své projekty migrace můžete v klidu spustit a dál je neřešit, protože proces využívá osvědčené postupy podle zkušeností Microsoftu.

> [!NOTE]
> Použití služby Migrace databáze Azure k provedení migrace online vyžaduje vytvoření instance založené na cenové úrovni Premium.
