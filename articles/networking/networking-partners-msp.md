# <a name="about-azure-networking-msp-partners"></a>O partnerech Azure Networking MSP

[Partnerský program Azure Networking Managed Service Provider (MSP)](https://azure.microsoft.com/blog/enhancing-the-customer-experience-with-the-azure-networking-msp-partner-program/) umožňuje poskytovatelům služeb zaměřeným na síťové služby, telcos a systémovým integrátorům (SIs) nabízet cloudové a hybridní síťové služby zaměřené na portfolio síťových produktů a služeb Azure.

Poskytovatelé síťových služeb Azure jsou specializovanou sadou poskytovatelů spravovaných služeb, kteří řeší potřeby a výzvy v oblasti sítí podnikového cloudu ve všech aspektech cloudových a hybridních sítí. Spravované síťové služby zahrnují jednu nebo více z následujících služeb: architektura sítě, plánování, nasazení, provoz, údržba a optimalizace.

![MSP][0]

Zákazníci si můžou zakoupit nabídky služeb spravovaných pomocí msp na Azure Marketplace nebo přímo z msp. Msps můžete nasadit a spravovat síťové prostředky Azure v předplatných zákazníků a skupiny prostředků pomocí [Azure Lighthouse](https://azure.microsoft.com/services/azure-lighthouse/).
