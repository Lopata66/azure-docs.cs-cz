# <a name="manage-your-azure-maps-account"></a>Správa účtu Azure Maps

Můžete spravovat váš účet Azure Maps pomocí webu Azure portal. Jakmile budete mít účet, můžete implementovat rozhraní API na vašem webu nebo mobilní aplikace.

Pokud ještě nemáte předplatné Azure, vytvořte si [bezplatný účet](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) před tím, než začnete.

## <a name="create-a-new-account"></a>Vytvoření nového účtu

1. Přihlaste se k webu [Azure Portal](https://portal.azure.com).

2. Vyberte **vytvořit prostředek** v levém horním rohu webu Azure portal.

3. Vyhledejte a vyberte **mapy**. Potom vyberte **Vytvořit**.

4. Zadejte informace pro svůj nový účet.

![Zadejte informace o účtu na portálu](./media/how-to-manage-account-keys/new-account-portal.png)

## <a name="delete-an-account"></a>Odstranit účet

Odstranit účet na webu Azure Portal. Přejděte na stránku přehled účtu a vyberte **odstranit**.

![Odstranění účtu na portálu](./media/how-to-manage-account-keys/account-delete-portal.png)

Zobrazí se stránka s potvrzením. Potvrzení odstranění účtu zadáním jejího názvu.
