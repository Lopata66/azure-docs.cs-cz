# <a name="azure-maps-community---open-source-projects"></a>Komunita Map Azure – projekty s otevřeným zdrojovým kódem

Tyto open source iniciativy založené na komunitě jsou vytvářené a spravované týmem Azure Maps. Nejsou součástí standardní nabídky produktů nebo služeb.

V následujícím seznamu jsou uvedeny některé z nejpopulárnějších open source projektů azure maps a ukázky.

| Název projektu | Popis |
|-|-|
| [Ukázky kódu Map Azure](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Kolekce ukázek kódu pro používání Azure Maps ve webových aplikacích. |
| [Ukázky kódu cloudu Azure Maps Gov](https://github.com/Azure-Samples/AzureMapsCodeSamples) | Kolekce ukázek kódu pro používání Azure Maps prostřednictvím Azure Government Cloud. |
| [Azure Maps & ukázky služby Azure Active Directory](https://github.com/Azure-Samples/Azure-Maps-AzureAD-Samples) | Kolekce ukázek, které ukazují, jak používat Azure Active Directory s Azure Maps. | 
| [Implementace prostorové analýzy IoT s využitím Azure Maps](https://github.com/Azure-Samples/iothub-to-azure-maps-geofencing) | Sledování a zachytávání relevantních událostí, ke kterým dochází v prostoru a čase, je běžný scénář IoT. |
| [Ukázky poznámkového bloku Azure Maps Jupyter](https://github.com/Azure-Samples/Azure-Maps-Jupyter-Notebook) | Kolekce ukázek pythonu pomocí služeb Azure Maps REST. |
| [Bot Framework - Bod zájmu dovednost](https://github.com/microsoft/botframework-solutions/tree/488093ac2fddf16096171f6a926315aa45e199e7/skills/csharp/pointofinterestskill) | Dovednosti point of interest poskytují virtuálnímu asistentovi pomocí Azure Maps pomocí služby Azure Bot Service a Bot Framework funkce týkající se bodů zájmu. |

Delší seznam, který zahrnuje komunitou vytvořené open-source projekty je k dispozici [zde](https://github.com/microsoft/Maps/blob/master/AzureMaps.md)

## <a name="supportability-of-open-source-projects"></a>Podpora open-source projektů

Následující příkazy platí ve všech Azure Maps vytvořené a spravované open source projekty a ukázky:

- Open source projekty a ukázky Azure Maps vytvářejí Microsoft a komunita.
- Open source projekty a ukázky Azure Maps spravuje Microsoft a komunita.
- Open source projekty a ukázky Azure Maps používají podporované a doporučené techniky.
- Open source projekty a ukázky Azure Maps jsou komunitní iniciativou – lidé, kteří na ní pracují ve prospěch ostatních a mají také svou běžnou denní práci.
- Open source projekty a ukázky Azure Maps nejsou produktem, a proto není podporován a podporuje ho podpora Premier Support ani jiné oficiální kanály podpory.
- Open source projekty a ukázky Azure Maps jsou podporovány podobným způsobem jako ostatní open source projekty provedené microsoftem s podporou komunity ze strany komunity.
