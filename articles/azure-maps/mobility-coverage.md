# <a name="azure-maps-mobility-coverage"></a>Pokrytí mobility map Azure

Azure Maps nabízí [službu mobility](https://aka.ms/AzureMapsMobilityService). Služba Mobility zlepšuje dobu vývoje aplikací s funkcemi veřejné dopravy, sdílení jízdních kol a sdílení skútrů. Mezi funkce veřejné dopravy patří vyhledávání blízkých stanic, vyhledávání zastávek, vyhledávání tranzitních linek, načítání informací o plánu, načítání příjezdů veřejné dopravy v reálném čase, zastavení seznamu a seznam průvodců trasami. Služba vrací informace o dostupnosti v reálném čase a statické dostupnosti sdíleným vozidlům, jako jsou automobily, kola a skútry. Služba Mobility navíc poskytuje možnosti směrování s multimodálními možnostmi plánování cesty. Multimodální plánování cesty zahrnuje cyklistiku, jízdu a tranzitní možnosti, to vše do jedné cesty. Uživatelé cam přístup podrobné a multimodální krok-za-krokem itineráře a služby upozornění.

Azure Maps nemá stejnou úroveň informací a přesnosti pro všechna města a země nebo oblasti. Možnost volat data o tranzitu závisí na městě. A data nemusí zahrnovat všechny možnosti veřejné dopravy a agentury, které slouží městu.

Následující tabulka obsahuje informace o pokrytí služby Azure Maps Mobility.
