# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Správa cenovou úroveň účtu Azure Maps

Cenová úroveň tohoto účtu Azure Maps na webu Azure portal můžete spravovat. Po vytvoření [účet](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), můžete zobrazit nebo změnit cenovou úroveň pro svůj účet.

Získejte další informace o [Volba správného cenové úrovně ve službě Azure Maps](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-your-pricing-tier"></a>Zobrazit svou cenovou úroveň

Chcete-li zobrazit vaši vybranou cenovou úroveň, přejděte **cenová úroveň** v nabídce nastavení.

![Zobrazení zvolené cenové úrovni](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-a-pricing-tier"></a>Změnit cenovou úroveň.

Po vytvoření účtu Azure Maps můžete upgradovat nebo downgradovat cenovou úroveň pro svůj účet Azure Maps. Chcete upgradovat nebo downgradovat, přejděte **cenová úroveň** v nabídce nastavení. Vyberte cenovou úroveň, která není označena. Vyberte **Uložit** tlačítko Uložit vaši vybranou cenový tarif vrstvy.

![Změnit cenovou úroveň.](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!NOTE]
> Není nutné vygenerovat nové klíče předplatného nebo ID klienta (pro ověřování Azure AD), je-li upgradovat nebo downgradovat cenovou úroveň pro svůj účet Azure Maps.
