# <a name="manage-the-pricing-tier-of-your-azure-maps-account"></a>Správa cenové úrovně účtu Azure Maps

Cenovou úroveň účtu Azure Maps můžete spravovat na webu Azure Portal. Po vytvoření [účtu](https://azure.microsoft.com/free/?WT.mc_id=A261C142F)můžete také zobrazit nebo změnit cenovou úroveň účtu .

Získejte další informace o [výběru správné cenové úrovně v Azure Maps](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier).

## <a name="view-your-pricing-tier"></a>Zobrazení cenové úrovně

Chcete-li zobrazit vybranou cenovou úroveň, přejděte v nabídce nastavení k možnosti **Cenová úroveň.**

[![Zobrazit vybranou](./media/how-to-manage-pricing-tier/view-pricing-tier.png) cenovou úroveň](./media/how-to-manage-pricing-tier/view-pricing-tier.png#lightbox)

## <a name="change-a-pricing-tier"></a>Změna cenové úrovně

Po vytvoření účtu Azure Maps můžete upgradovat nebo snížit úroveň ceny pro váš účet Azure Maps. Chcete-li upgradovat nebo downgrade, přejděte na možnost **Cenová úroveň** v nabídce nastavení. Vyberte cenovou úroveň, která není zvýrazněná. Kliknutím na tlačítko **Uložit** uložte vybranou možnost cenové úrovně.

[![Změna cenové](./media/how-to-manage-pricing-tier/change-pricing-tier.png) úrovně](./media/how-to-manage-pricing-tier/change-pricing-tier.png#lightbox)

> [!NOTE]
> Pokud upgradujete nebo snižujete úroveň ocenění pro váš účet Azure Maps, nemusíte generovat nové klíče předplatného nebo ID klienta (pro ověřování Azure AD).
