# <a name="how-to-manage-pricing-tier-of-your-azure-maps-account"></a>Jak spravovat cenovou úroveň svého účtu Azure Maps

Cenová úroveň tohoto účtu Azure Maps na webu Azure portal můžete spravovat. Po vytvoření [účet](https://azure.microsoft.com/free/?WT.mc_id=A261C142F), můžete zobrazit nebo změnit cenovou úroveň pro svůj účet.

Další informace o Volba správného cenovou úroveň najdete v článku, [Volba správného cenová úroveň](https://docs.microsoft.com/azure/azure-maps/choose-pricing-tier)

## <a name="view-pricing-tier"></a>Cenová úroveň zobrazení

Chcete-li zobrazit vaši vybranou cenovou úroveň, přejděte **cenová úroveň** v nabídce nastavení.

![Zobrazení zvolené cenové úrovni](./media/how-to-manage-pricing-tier/view-pricing-tier.png)

## <a name="change-pricing-tier"></a>Změnit cenovou úroveň

Po vytvoření účtu Azure Maps můžete upgradovat nebo downgradovat cenovou úroveň pro svůj účet Azure Maps. Chcete upgradovat nebo downgradovat, přejděte **cenová úroveň** možnost v nabídce nastavení a vyberte cenovou úroveň, která není označena. Kliknutím na tlačítko Uložit uložte vaši vybranou cenový tarif vrstvy.

![Změnit cenovou úroveň](./media/how-to-manage-pricing-tier/change-pricing-tier.png)

> [!Note]
> Není potřeba generovat nové klíče předplatného, je-li upgradovat nebo downgradovat cenovou úroveň pro svůj účet Azure Maps.
