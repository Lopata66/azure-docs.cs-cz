# <a name="connect-trend-micro-deep-security"></a>Connect Trend Micro Deep Security

Konektor Trend Micro Deep Security umožňuje snadno propojit protokoly hloubkového zabezpečení s Azure Sentinelem, zobrazit řídicí panely, vytvořit vlastní výstrahy a zlepšit vyšetřování. To vám poskytne lepší přehled o sítích a systémech vaší organizace a zlepší temenáš možnosti operací zabezpečení.


## <a name="prerequisites"></a>Požadavky

- Oprávnění ke čtení a zápisu do pracovního prostoru Azure Sentinelu.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Připojení Azure Sentinelu k přímému zabezpečení Trend Micro

1. Na webu Azure Portal přejděte na**datové konektory** **Azure Sentinel** > a pak vyberte konektor **Trend Micro Deep Security.**

2. Vyberte **Otevřít stránku konektoru**.

3. Postupujte podle pokynů a další kroky.
