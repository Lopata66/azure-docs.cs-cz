# <a name="connect-citrix-analytics-security"></a>Připojení Citrix Analytics (zabezpečení) 

Integrace Citrix Analytics (zabezpečení) s Azure Sentinelem vám pomůže exportovat data analyzovaná pro rizikové události z Citrix Analytics (Security) do Azure Sentinelu. Můžete vytvářet vlastní řídicí panely, analyzovat data z jiných zdrojů spolu s těmi z Citrix Analytics (Zabezpečení) a vytvářet vlastní pracovní postupy pomocí logic aplikací pro monitorování a zmírnění událostí zabezpečení. 



## <a name="prerequisites"></a>Požadavky

- Oprávnění ke čtení a zápisu do pracovního prostoru Azure Sentinelu.

- Nárok na řešení zabezpečení společnosti Citrix v citrix cloudu.


## <a name="connect-azure-sentinel-to-citrix-analytics-security"></a>Připojení Azure Sentinelu k řešení Citrix Analytics (zabezpečení)

1. Na webu Azure Portal přejděte na**datové konektory** **Azure Sentinel** > a pak vyberte konektor **Citrix Analytics (Zabezpečení).**

2. Vyberte **Otevřít stránku konektoru**.

3. Postupujte podle pokynů na stránce **Citrix Analytics (Zabezpečení).**
