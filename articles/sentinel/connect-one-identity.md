# <a name="connect-one-identity-safeguard"></a>Connect One Identity Safeguard
Datový konektor One Identity Safeguard vylepšuje standardní konektor Společného formátu událostí (CEF) pomocí řídicích panelů specifických pro privilegované relace. Pomocí tohoto konektoru můžete snadno začít používat události, které vaše zařízení generuje pro vizualizaci, výstrahy, vyšetřování a další.


## <a name="prerequisites"></a>Požadavky

- Oprávnění ke čtení a zápisu do pracovního prostoru Azure Sentinelu.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Připojení Azure Sentinelu k zabezpečení jedné identity

1. Na webu Azure Portal přejděte na**datové konektory** **Azure Sentinel** > a pak vyberte konektor **One Identity Safeguard.**

2. Vyberte **Otevřít stránku konektoru**.

3. Postupujte podle pokynů na stránce **One Identity Safeguard** .
