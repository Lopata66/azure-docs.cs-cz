# <a name="moving-azure-network-watcher-resources-across-regions"></a>Přesunutí prostředků Azure Network Watcher napříč oblastmi

## <a name="moving-the-network-watcher-resource"></a>Přesunutí prostředku Network Watcher
Prostředek Network Watcher představuje back-end službu pro Network Watcher a plně spravuje Azure. Zákazníci je nepotřebují spravovat. Operace přesunutí není u tohoto prostředku podporována.

## <a name="moving-child-resources-of-network-watcher"></a>Přesun podřízených prostředků Network Watcher
Přesunutí prostředků mezi oblastmi není v současné době podporováno u žádného podřízeného prostředku `*networkWatcher*` typu prostředku.
