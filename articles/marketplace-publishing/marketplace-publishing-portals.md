# <a name="portals-you-will-need"></a>Portály, které budete potřebovat

Před zahájením procesu publikování nabídky, Pojďme vás seznámili s různými portály, které budete potřebovat. Níže je stručně shrnuje informace o portálech – Centrum pro vývojáře, portál publikování Azure a webu Azure portal – v pořadí, že bude komunikovat s nimi.                                                                            

## <a name="developer-center"></a>Centrum pro vývojáře
[http://dev.windows.com/registration?accountprogram=azure](http://dev.windows.com/registration?accountprogram=azure)

### <a name="description"></a>Popis
Vytvoření účtu Microsoft Developer Center je jednorázový úkol. Ujistěte se, že společnost ještě nemá účet Centrum pro vývojáře před dalším pokusem ho vytvořit. Během procesu můžeme shromažďovat informace o bankovním účtu, daních a adrese společnosti.

> [!NOTE]
> Pokud publikujete pouze bezplatných nabídek (nebo přeneste your-nabídky používání vlastní licence), jsme nevyžadují daňové a bankovní informace.
> 
> 

### <a name="identityaccount-used"></a>Identity/účtu použít
V ideálním případě by tato hodnota je distribuční seznam nebo skupinu zabezpečení (například azurepublishing @*partnercompany*.com). Distribuční seznam nebo zabezpečení skupiny **musí** zaregistrovat jako účet Microsoft.

> [!TIP]
> Doporučujeme používat distribuční seznam nebo skupinu zabezpečení protože odebere závislost na každý jednotlivec, ačkoli individuálního účtu lze také použít.
> 
> 

## <a name="publishing-portal"></a>Portálu pro publikování
[https://publish.windowsazure.com](https://publish.windowsazure.com)

### <a name="description"></a>Popis

Tato hodnota je na portálu, který používáte pro práci na nabídky a publikujte ho (uvádění na trh, ceny, publikování, certifikace, pokud je k dispozici, atd.).

### <a name="identityaccount-used"></a>Identity/účtu použít
Výše uvedené distribuční seznam nebo skupinu zabezpečení musí použít první přihlášení pro přihlášení k portálu pro publikování. Později můžete jako další spolupracující správce přidat další uživatele. Tento seznam je, jak získá mapované na Centrum pro vývojáře registrační data.

## <a name="azure-portal"></a>portál Azure
[Azure Portal](https://portal.azure.com)

### <a name="description"></a>Popis
Tato hodnota je na portálu, kde najdete postupné a publikované nabídky na webu Azure Marketplace (k dispozici pro virtuální počítače, šablony řešení a služby pro vývojáře využívající Azure Resource Manager).

### <a name="identityaccount-used"></a>Identity/účtu použít
Když jste pracovní nabídky z portálu pro publikování, ID předplatného musí být seznam povolených adres. Stejné předplatné (je uživatelské jméno a heslo, které jsou s ním spojená) je potřeba použít pro přihlášení k tomuto portálu k testování dvoufázové instalace nabídky.
