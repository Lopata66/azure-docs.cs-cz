# <a name="quickstart-prepare-your-network-for-azure-defender-for-iot-deployment"></a>Rychlý Start: Příprava sítě pro nasazení Azure Defenderu pro IoT

## <a name="choose-your-hardware"></a>Volba hardwaru
Azure Defender pro IoT podporuje různá certifikovaná zařízení, včetně těchto:
- Certifikovaná předem nakonfigurovaná zařízení, na kterých je už nainstalovaný software
- Nenakonfigurovaná certifikovaná zařízení, na kterých můžete stáhnout a nainstalovat software snímače
- Specifikace zařízení místních konzol pro správu. Místní Konzola pro správu není k dispozici jako předem nakonfigurované zařízení. 

Další informace najdete v [Průvodci specifikacemi hardwaru](https://aka.ms/AzureDefenderforIoTBareMetalAppliance) .


## <a name="install-components"></a>Nainstalovat komponenty
Po získání hardwarových komponent a stažení softwarového balíčku z Azure Defenderu pro IoT Portal můžete nainstalovat software součásti. Další informace najdete v [instalační příručce](https://aka.ms/AzureDefenderforIoTInstallSensorISO).

## <a name="onboard-sensors-to-the-azure-defender-for-iot-portal"></a>Připojení senzorů k portálu Azure Defender pro IoT 
 Tento článek poskytuje přehled procesů nasazení, zprovoznění a aktualizace pro Azure Defender pro senzory IoT a místní konzolu pro správu. Tento proces se provádí podle pokynů v tématu Vítá vás Azure Defender pro IoT a další stránky.

![Snímek obrazovky se zobrazením úvodní stránky Azure](media/updates/image4.png)

Otevření úvodní stránky:

1. Přihlaste se ke svému účtu Azure a vyhledejte Azure Defender pro IoT.

1. **Nasazení snímače** vyžaduje, abyste buď nakoupili předkonfigurovaný hardware od šipky nebo si koupili vlastní zařízení a nainstalovali na něj software snímače.
1. **Nasazení** místní konzoly pro správu vyžaduje, abyste si nakoupili vlastní zařízení a nainstalovali na něj místní software pro správu. Další informace.




## <a name="network-setup"></a>Nastavení sítě
Po instalaci programu Defender pro součásti systému IoT můžete nakonfigurovat síťové vybavení pro optimální monitorování sítě. Další informace najdete v [Průvodci nastavením sítě](https://aka.ms/AzureDefenderForIoTNetworkSetup).
