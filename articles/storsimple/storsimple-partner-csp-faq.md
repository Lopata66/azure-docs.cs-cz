# <a name="storsimple-for-cloud-solutions-provider-program-frequently-asked-questions"></a>StorSimple pro Cloud Solutions Provider Program: Nejčastější dotazy


## <a name="overview"></a>Přehled
Níže jsou dotazy a odpovědi, které můžete mít jako StorSimple partner při nasazení a správa virtuálních polí StorSimple na webu Azure portal.

Otázky a odpovědi jsou uspořádány do následujících kategorií:

 - [Model StorSimple a řešení poskytovatele programu CSP (Cloud)](#storsimple-and-csp-model)
 - [Nasadit a spravovat virtuální pole jako partner](#deploy-and-manage-storsimple-as-a-partner)
 - [Podpora pro partnery StorSimple pro CSP](#support-for-storsimple-partners)

## <a name="storsimple-and-csp-model"></a>Model StorSimple a CSP

### <a name="what-does-it-mean-to-have-storsimple-as-a-part-of-the-cloud-solutions-provider-csp-program"></a>Co znamená mít StorSimple jako součást programu Cloud Solutions Provider (CSP)?
 StorSimple pro CSP umožňuje partnerům CSP dál prodávat zákazníkům StorSimple Virtual Array. Tento model umožňuje partnerům CSP, aby měli životního cyklu zákazníků začátku do konce se přímé zřízení, fakturaci a podporu cloudových služeb od Microsoftu. 

Další informace najdete v části [StorSimple v programu CSP](https://www.microsoft.com/en-us/server-cloud/products/storsimple/). 

### <a name="is-storsimple-a-usage-based-or-based-on-seat-licenses-like-office-365-and-enterprise-mobility-suite-ems"></a>Se StorSimple založené na využití nebo podle licence licence, jako je Office 365 a Enterprise Mobility Suite (EMS)?
StorSimple je služba založená na využití. Zákazníci, kteří získávají StorSimple Virtual Array prostřednictvím CSP trasy platby za využití virtuální pole a Azure, fakturuje jako samostatných řádcích v CSP faktury. 

### <a name="is-the-storsimple-business-model-similar-to-that-of-other-services-such-as-the-office-365-and-the-enterprise-mobility-suite"></a>Je obchodní model StorSimple podobně jako u jiných služeb, jako je Office 365 a Enterprise Mobility Suite? 
Ano. Model je složité slevy modelu, stejně jako ostatní služby na základě využití v Azure. Velkoobchodní slevy pro StorSimple je podobně jako u ostatních služeb Azure. Tento model poskytuje partnerům CSP příležitost prodávat a řiďte spotřebu Azure prostřednictvím StorSimple Virtual Array.  

### <a name="which-storsimple-skus-are-available-at-launch"></a>Které StorSimple skladové položky jsou k dispozici při spuštění?
Stejné identifikátory SKU, které jsou k dispozici pomocí samostatné služby StorSimple. 

### <a name="is-there-any-difference-between-the-storsimple-directly-purchased-from-microsoft-under-ea-and-storsimple-as-a-part-of-csp"></a>Existuje případný rozdíl mezi StorSimple ve StorSimple zakoupili přímo od Microsoftu v rámci smlouvy EA jako součást CSP?
Ne. V části CSP zákazníci zakoupit další službami nabízenými touto partner CSP spolu se StorSimple Virtual Array (SVA) v rámci jedné faktury. Všimněte si, že pro nákup fyzického zařízení StorSimple 8000 series, zákazníci potřebují projít trasy EA. 

### <a name="in-which-markets-is-storsimple-available-for-csp-at-launch"></a>V které trhy nabízíme StorSimple pro CSP při spuštění?
StorSimple Virtual Array pro zprostředkovatele kryptografických služeb je k dispozici na všech trzích, kde StorSimple je v současnosti dostupné. Další informace najdete v části [seznam oblastí, kde je k dispozici StorSimple](https://azure.microsoft.com/regions/services/).

### <a name="what-kind-of-solutions-can-a-partner-deliver-with-storsimple-and-csp"></a>Jaký druh řešení můžete partnera dodávat StorSimple a CSP? 
Existuje několik řešení:

- Partneři mohou prodávat StorSimple Virtual Array. 
- Partneři mohou poskytovat řešení vycházejí ze StorSimple a jiným službám Azure nebo softwaru jiných společností. Další informace najdete v tématu některé možné [řešení, které existují kolem StorSimple](https://aka.ms/storsimple-build-solutions).
- Můžete také neobjevili nových obchodních modelů poskytováním spravované služby.

### <a name="what-are-the-incentives-available-for-storsimple-csp-partners-do-i-qualify-for-any-csp-program-incentives"></a>Co jsou pobídky pro partnery StorSimple CSP k dispozici? Kvalifikuji pro jakékoli pobídky programu CSP?
Informace o pobídky, přejděte na [pobídky programu CSP ](https://partner.microsoft.com/en-US/cloud-solution-provider/program-details).


## <a name="deploy-and-manage-storsimple-as-a-partner"></a>Nasazení a správa StorSimple jako partner

### <a name="how-can-i-administer-storsimple-virtual-array-in-csp"></a>Jak můžete spravovat StorSimple Virtual Array in CSP? 
Odběry pro StorSimple Virtual Array můžete přidat k účtu vašich zákazníků prostřednictvím partnerského centra. Kromě toho můžete na webu Azure portal k přidání uživatele do předplatná.

### <a name="is-the-azure-portal-approach-the-same-as-other-services-for-csp"></a>Je na webu Azure portal přístup, stejně jako jiné služby pro CSP? 
Ano. Se StorSimple Virtual Array pro CSP, by měla být nejvhodnější pro partnery pro přístup k webu Azure portal prostřednictvím [partnerského centra](https://partnercenter.microsoft.com/) kde můžete spravovat zákazníků a předplatných. Ke správě předplatného StorSimple partnera nebo zákazníka (v závislosti na oprávněních udělených) by měl Přihlaste se k webu Azure portal. 

### <a name="is-microsoft-shipping-a-new-portal-for-storsimple-for-csp"></a>Microsoft dodává nový portál pro StorSimple pro CSP? 
Ne. Budete moct spravovat StorSimple Virtual Array do CSP prostřednictvím webu Azure portal.

### <a name="i-have-provisioned-a-standalone-storsimple-subscription-for-my-customer-do-i-have-to-use-the-azure-portal-to-administer-my-device"></a>Vybaveny samostatného předplatného StorSimple pro své zákazníky. Je nutné spravovat svoje zařízení pomocí webu Azure portal? 
Ano. 

### <a name="is-the-microsoft-account-team-compensated-as-a-result-of-sales-of-storsimple-through-csp"></a>Tým Microsoft Account kompenzována v důsledku sales StorSimple prostřednictvím zprostředkovatele kryptografických služeb?
Ano. Vedoucí prodeje partnera a zákaznického týmu účtu pro koncového zákazníka bude kompenzována pro prodej CSP. Partneři jsou předpokládaná doba realizace v prodeji jejich rozlišené hodnotu a nabídky.
