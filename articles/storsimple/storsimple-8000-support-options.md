# <a name="storsimple-solution-support"></a>Podpora řešení StorSimple

## <a name="storsimple-support"></a>Podpora StorSimple

Microsoft nabízí flexibilní možnosti podpory pro StorSimple podnikové úložiště zákazníkům. Jsme připravení hluboce podpory vysoce kvalitní prostředí, který umožňuje maximalizovat dopadu vaší investice do řešení StorSimple a Microsoft Azure. Jako zákazník StorSimple získáte:

* možnost 24 x 7 k odeslání lístky žádostí o podporu prostřednictvím webu Azure portal.
* Přístup k helpdesku obecných dotazů a podrobné technické podpory pomoct.
* Podpora místních jazyků tam, kde je k dispozici.
* Pošle upozornění, zřizování a správu pro přehledy stavu a výkonu.
* Přístup k zahrnující hlavní, vedlejší verzi, softwarové aktualizace a údržba opravy.
* Podpora pro pole úložiště řady StorSimple 8000 a virtuální pole StorSimple v jednom balíčku.

## <a name="cancellation-policy"></a>Zásady zrušení
Pro získání podpory StorSimple, musí zákazník zakoupit plány podpory Standard nebo Premium po dobu trvání období předplatného. Zrušení nemá za následek poměrnou vrácenou částku. Plány podpory StorSimple jsou snížení oprávnění při výročí uzavření smlouvy EA. Microsoft je však schopna poskytovat podporu pro zákazníky StorSimple bez smlouvy o podpoře platný.

## <a name="renewal-policy"></a>Zásady obnovení


Při nákupu pole úložišť řady StorSimple 8000 je podporované prostřednictvím příštího výročí uzavření smlouvy EA. Zákazník musí obnovení podpory StorSimple při výročí uzavření smlouvy EA. Objednávky plán podpory StorSimple jsou shodných. Zákazníci upozorněni prostřednictvím e-mailu o brzké vypršení platnosti podporu pro pole úložiště řady StorSimple 8000 a se očekává, že se na Microsoft teams účtu/sales nebo Microsoft řešení partnera LSP (Licensing) k obnovení podpory StorSimple.


Azure Standard support nezahrnuje podporu hardwaru StorSimple. Pokud jsou popsané v rámci podpory Premier nebo Unified Microsoft, musíte koupit i obnovení podpory Standard StorSimple. Obnovení podpory StorSimple může být v souladu s výročí uzavření smlouvy EA získáním vyžaduje podporu SKU s počty licencí zahrnují rovna počtu zařízení a množství jednotek objednané se zbývající počet měsíců podporu potřebné až do smlouvy EA Datum výročí služeb, pokud všechny jednotky mají stejné datum vypršení platnosti smlouvy podpory. Pokud jednotky data vypršení platnosti smlouvy jinou podporu, je potřeba zahrnout každé zařízení s podporou jeden SKU s jednotkou objednané množství se zbývající počet měsíců podporu potřebné až do data výročí uzavření smlouvy EA na každé zařízení.
