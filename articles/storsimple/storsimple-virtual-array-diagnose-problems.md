# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Řešení potíží s StorSimple Virtual Array pomocí služby Správce zařízení StorSimple
## <a name="overview"></a>Přehled

Správce zařízení StorSimple poskytuje **diagnostikovat a řešit problémy** nastavení v okně přehledu služby, které zvýrazní některé z nejčastěji se vyskytující problémy, které mohou nastat u vaše virtuální pole a jak je vyřešit. Tento kurz představuje samoobslužné řešení potíží funkce poskytované v rámci služby Správce zařízení StorSimple.

![Diagnostikovat a řešit problémy](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostikovat a řešit problémy

Můžete zobrazit některé běžné problémy s StorSimple Virtual Array a zkontrolujte řešení těchto potíží správné z okno Souhrn služby Správce zařízení StorSimple.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Chcete-li diagnostikovat problémy se vaše virtuální pole

1. Klikněte na tlačítko **diagnostikovat a řešit problémy** nastavení v levém podokně okno Souhrn služby Správce zařízení StorSimple, chcete-li zobrazit seznam běžných problémů.

2. **Rozbalte** na příznaky problému něco se nepovedlo zkontrolovat **doporučené kroky** pro pomoc při řešení potíží. Pokud chcete, můžete také zkontrolovat odkazy podrobnou dokumentaci pro odkaz na další.
   
    ![Řešení problémů](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Pokud nemůžete najít odkaz na váš problém nebo problém vyřešit, kontaktujte Microsoft Support o další pomoc.
