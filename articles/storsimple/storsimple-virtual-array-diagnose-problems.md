# <a name="use-the-storsimple-device-manager-service-to-troubleshoot-the-storsimple-virtual-array"></a>Řešení potíží se službou StorSimple Virtual Array s využitím služby Správce zařízení StorSimple
## <a name="overview"></a>Přehled

Správce zařízení StorSimple poskytuje **diagnostiku a řešení problémů** v rámci okna souhrnu služby, které upozorňuje na některé běžně se vyskytující problémy, které mohou nastat s virtuálnípole a jak je vyřešit. Tento kurz představuje samoobslužné řešení potíží funkce poskytované v rámci služby StorSimple Device Manager.

![Diagnostikovat a řešit problémy](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-main.png)

## <a name="diagnose-and-solve-issues"></a>Diagnostikovat a řešit problémy

Můžete zobrazit některé běžné problémy s StorSimple Virtual Array a zkontrolujte řešení těchto problémů přímo z okna souhrnu služby StorSimple Device Manager.

#### <a name="to-diagnose-an-issue-with-your-virtual-array"></a>Diagnostika problému s virtuálním polem

1. Chcete-li zobrazit seznam běžných problémů, klepněte na tlačítko **Diagnostikovat a vyřešit problémy** v levém podokně okna souhrnu služby Správce zařízení StorSimple.

2. **Rozbalte** příznak problému, se kterým se setkáváte, a projděte si **doporučené kroky,** které vám pomohou při řešení problému. Pokud si přejete, můžete si také prohlédnout podrobné odkazy na dokumentaci, které jsou k dispozici pro další použití.
   
    ![Poradce při potížích s problémem](./media/storsimple-virtual-array-diagnose-problems/diagnose-problems-offline.png)

3. Pokud se vám nedaří najít odkaz na váš problém nebo jej vyřešit, obraťte se na podporu společnosti Microsoft s žádostí o další pomoc.
