# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Threat Modeling Tool vydání aktualizace 7.1.60702.1-7/2/2019

7\.1.60702.1 verze Microsoft Threat Modeling Tool (TMT) byla vydána v červenci 2 2019 a obsahuje následující změny:

- Vylepšení přístupnosti
- Opravy chyb
- Zahrnutí sady vzorníků Open-Source poskytnuté komunitou

## <a name="feature-changes"></a>Změny funkcí

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>K dispozici je nová sada vzorníků zdravotnických zařízení poskytovaná Open Source komunitou.

Komunita Open Source vytvořila sadu vzorníků pro vytváření modelovacích zdravotnických zařízení. Po aktualizaci se nová sada vzorníků zobrazí v rozevírací nabídce Výběr šablony. Informace o tom, jak přispívat vzorníky nebo obsah do šablon, najdete v informacích na [stránce GitHub](https://github.com/Microsoft/threat-modeling-templates)projektu.
