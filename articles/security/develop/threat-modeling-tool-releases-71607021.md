# <a name="threat-modeling-tool-update-release-71607021---722019"></a>Aktualizace nástroje pro modelování hrozeb 7.1.60702.1 - 7/2/2019

Verze 7.1.60702.1 nástroje Microsoft Threat Modeling Tool (TMT) byla vydána v červenci 2 2019 a obsahuje následující změny:

- Vylepšení přístupnosti
- Opravy chyb
- Zahrnutí open-source komunity poskytované šablony set

## <a name="feature-changes"></a>Změny funkcí

### <a name="a-new-medical-devices-stencil-set-provided-by-the-open-source-community-is-available"></a>K dispozici je nová sada šablon zdravotnických prostředků poskytovaná komunitou s otevřeným zdrojovým kódem

Šablona sada pro modelování zdravotnických prostředků byla přispěla open-source komunity. Po aktualizaci se nová sada vzorníků zobrazí v rozevírací nabídce výběru šablony. Informace o přispívání vzorníků nebo obsahu do šablon najděte informace na [stránce GitHub](https://github.com/Microsoft/threat-modeling-templates)projektu .

![Možnost ověření modelu](./media/threat-modeling-tool-releases-71607021/tmt-template-selection.png)

## <a name="system-requirements"></a>Požadavky na systém

  - [Aktualizace Microsoft Windows 10 Anniversary Update](https://blogs.windows.com/windowsexperience/2016/08/02/how-to-get-the-windows-10-anniversary-update/#HTkoK5Zdv0g2F2Zq.97) nebo novější
- Je vyžadována verze rozhraní .NET.
  - [.Net 4.7.1](https://go.microsoft.com/fwlink/?LinkId=863262) nebo novější
