# <a name="threat-modeling-tool-update-release-71510231---1112018"></a>Threat Modeling Tool, vydání aktualizace 7.1.51023.1 – 1. 11. 2018

Jak bylo původně uvedeno v [poznámkách k verzi GA](threat-modeling-tool-releases-71509112.md), vydali jsme aktualizaci (7.1.51023.1) na nástroj Microsoft Threat Modeling Tool, která bude vyžadovat, aby uživatelé verze Preview (náhled klienti s verzí < 7.1.50911.2) upgradovali na podporovanou verzi GA. Tato verze neobsahuje žádné nové funkce nebo opravy.

- Uživatelé verze Náhled automaticky stáhnou upgrade při otevření klienta. Pokud se rozhodnete novou aktualizaci neinstalovat, verze nástroje Náhled se zavře.
- Uživatelé ga verze nástroje budou vyzváni k výběru, zda chtějí upgradovat.
- Uživatelé verze nástroje 2016 nebudou ovlivněni.
