# <a name="threat-modeling-tool-releases"></a>Vydání nástroje modelování hrozeb

Microsoft Threat Modeling Tool je v současné době vydáván jako bezplatná aplikace pro [stahování kliknutím pro](https://aka.ms/threatmodelingtool) systém Windows. Tento mechanismus doručování nám umožňuje posunout nejnovější vylepšení a opravy chyb zákazníkům při každém otevření nástroje.

## <a name="system-requirements"></a>Systémové požadavky

- Podporované operační systémy
  - Aktualizace Microsoft Windows 10 Anniversary Update nebo novější
- Je vyžadována verze rozhraní .NET.
  - .NET 4.7.1 nebo novější
- Další požadavky
  - K přijímání aktualizací nástroje a šablon je nutné připojení k Internetu.
