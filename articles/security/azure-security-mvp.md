# <a name="azure-security-mvp-program"></a>Program MVP pro zabezpečení Azure

Microsoft profesionály (MVP) jsou od předních členů komunity, kteří osoby dokáží EXEMPLÁRNÍ závazku k pomáhají ostatním. MVP povolit ostatním uživatelům, aby maximální využití svých zkušeností s technologiemi Microsoftu. S komunitou a s Microsoftem sdílejí jejich nadšení, praktické znalosti a technické znalosti.

Microsoft Azure nyní rozpoznává komunity odborníků s experty v Azure security. Microsoft MVPs může být udělena MVP v Microsoft Azure v oblasti zabezpečení Azure příspěvek.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Neexistuje žádné srovnávacích testů pro stát MVP. V části je vzhledem k tomu, že se liší podle technologie a jeho životní cyklus. Některá kritéria zahrnuje:

- Dopad kandidát příspěvky k online fórům, jako je například Microsoft Answers, TechNet a MSDN
- Wiki weby a obsah online
- Konference a skupiny uživatelů
- Podcasty, weby, blogy a sociální média
- Články a knih.

Jste odborníky na zabezpečení Azure? Znáte někoho, kdo? Potom [nominace sobě nebo někomu jinému](https://mvp.microsoft.com/Nomination/nominate-an-mvp) stát MVP pro zabezpečení Azure ještě dnes!
