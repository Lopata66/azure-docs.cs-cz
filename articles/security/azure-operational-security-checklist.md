# <a name="azure-operational-security-checklist"></a>Kontrolní seznam provozního zabezpečení Azure
Nasazení aplikace v Azure je rychlá, snadná a nákladově efektivní. Před nasazením cloudové aplikace v produkčním prostředí, které jsou užitečné mít kontrolní seznam pro pomoc při hodnocení aplikace podle seznamu akcí důležité a doporučené provozní zabezpečení je třeba zvážit.

## <a name="introduction"></a>Úvod

Azure nabízí sadu služeb infrastruktury, které můžete použít k nasazení aplikací. Azure Operational Security odkazuje na služby, ovládací prvky a funkce, které jsou dostupné pro uživatele k ochraně svých dat, aplikací a dalších prostředků ve službě Microsoft Azure.

-   Pro maximální těžit z cloudové platformy, doporučujeme využívat služby Azure a na kontrolního seznamu.
-   Organizace, které investovat čas a prostředky provozní připravenosti svoje aplikace před spuštěním posouzení mít mnohem vyšší míra jste spokojen/spokojena než ty, kteří nechtějí. Při provádění této práci, kontrolní seznamy mohou být neocenitelnou mechanismem, k zajištění, že aplikace se vyhodnocují konzistentně a komplexně.
-   Úroveň provozní posouzení se liší v závislosti na úrovni vyspělosti cloudu organizace a fázích vývoje, požadavky dostupnosti a požadavky na citlivosti dat aplikace.
