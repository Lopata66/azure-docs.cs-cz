# <a name="azure-security-mvp-program"></a>Program MVP pro zabezpečení Azure

Nejefektivnějšími specialisty Microsoftu (MVP) jsou vedoucí komunity, kteří prokázali Exemplary závazek pomáhat ostatním. MVP umožňuje ostatním uživatelům využít své zkušenosti s technologiemi Microsoftu. Sdílejí své zaujetí, praktické znalosti a technické zkušenosti s komunitou a Microsoftem.

Microsoft Azure teď rozpoznává odborníky komunity se speciálními znalostmi v zabezpečení Azure. MVP pro odborníky Microsoftu je možné udělit v Microsoft Azure v oblasti příspěvků na zabezpečení Azure.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

Neexistují žádné srovnávací testy, které by se staly MVP. To je součástí, protože se liší podle technologie a jejího životního cyklu. Mezi kritéria patří:

- Dopad příspěvků na kandidát do online fóra, jako jsou Microsoft Answers, TechNet a MSDN
- Wiki a online obsah
- Konference a skupiny uživatelů
- Podcasty, weby, blogy a sociální média
- Články a knihy.
