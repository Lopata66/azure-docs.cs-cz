# <a name="microsoft-security-response-center"></a>Středisko Microsoft Security Response Center

Microsoft Security Response Center (MSRC) se vychází z nejdůležitějších zkušených odborníků na zabezpečení světa. Jednotlivé dny v roce identifikují, monitorují, reagují a odstraňují bezpečnostní incidenty a místní a cloudová ohrožení v průběhu času.

Kromě průběžné práce má tým MSRC k dispozici několik prostředků, takže můžete pochopit, jak efektivně zabezpečit prostředky a nasazení Azure.

## <a name="the-msrc-blog"></a>Blog MSRC

[Blog MSRC](https://blogs.technet.microsoft.com/msrc/) je místem, kde se dozvíte, jak získat nejnovější novinky o tom, co vám MSRC při ochraně před hrozbami v cloudu.

## <a name="white-papers"></a>Dokumenty white paper

MSRC publikoval spoustu [dokumentů White Paper](https://technet.microsoft.com/library/bb969102.aspx) , které vám pomůžou pochopit, co dělají a jak to dělají. Některé poskytují přehled o tom, jak zabezpečujeme Cloud Microsoftu a obsahují užitečné informace o tom, jak můžete používat stejné konfigurace zabezpečení.

## <a name="security-researcher-engagement-and-bounty-programs"></a>Zapojení a Bounty programy Security Researcher

MSRC podporuje spolupráci a vztahy s výzkumnými výzkumníky globálně a zajišťuje tak předběžné zabezpečení produktů společnosti Microsoft.

Programy Microsoft bountye platíte výzkumným pracovníkům pro:

- Nové techniky využívání
- Obrannou linií nápady, které zmírňují nové zneužití
- Identifikace kritických ohrožení zabezpečení v místním a cloudovém softwaru Microsoftu.

Další informace o těchto programech najdete na stránce o [chybách Bounty](https://technet.microsoft.com/security/dn425036) a na [blogu MSRC](https://blogs.technet.microsoft.com/msrc/).

Další informace o MSRC najdete na [domovské stránce centra MSRC](https://technet.microsoft.com/library/dn440717.aspx).
