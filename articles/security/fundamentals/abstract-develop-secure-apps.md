# <a name="develop-secure-cloud-applications-on-azure"></a>Vývoj zabezpečených cloudových aplikací v Azure
## <a name="abstract"></a>Abstraktní
Tento dokument je obecným průvodcem pro bezpečnostní otázky a kontroly, které byste měli zvážit v každé fázi životního cyklu vývoje softwaru při vývoji aplikací pro Cloud. Implementace těchto konceptů před vydáním produktu vám může pomáhat při sestavování bezpečnějšího softwaru. Doporučení uvedená v tomto dokumentu pocházejí z našeho prostředí s využitím zabezpečení Azure a zkušeností našich zákazníků.

Tento dokument je určený jako prostředek pro návrháře softwaru, vývojáře a testery na všech úrovních, kteří sestavují a nasazují zabezpečená řešení Azure.
