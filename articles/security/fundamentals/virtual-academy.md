# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Kurzy zabezpečení Azure od Microsoftu Virtual Academy
Microsoft Virtual Academy poskytuje bezplatné, online školení, které umožňuje vývojářům, IT specialistům a odborníkům v oblasti IT a studentům učit se nejnovější technologie, sestavovat své dovednosti a využít jejich péče.

Na této stránce najdete uspořádanou kolekci kurzů týkajících se zabezpečení Azure. Navštivte [Microsoft Virtual Academy](https://mva.microsoft.com/) , kde najdete všechny kurzy, které jsou k dispozici.

[Vývoj a testování v cloudu](https://mva.microsoft.com/en-us/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311)

Jste vývojáři, kteří potřebují doručovat rychlejší a lepší aplikace? Přesunutí vývojových a testovacích prostředí do cloudu vám umožní dosáhnout přesně toho. Naučte se, jak to udělat, a zjistěte, jaké výhody se přesunují. Další informace najdete v tématu ukázky a prezentace, které ukazují, jak Microsoft Azure můžou podporovat vaše potřeby vývoje a testování. K dispozici jsou lekce týkající se postupů vývoje zabezpečení a nasazení.

[Běžné úlohy pro Linux v Azure](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

Pokud máte dotazy týkající se používání systému Linux na platformě Microsoft Azure, najdete v tomto podrobném kurzu odpovědi. Prozkoumejte některé běžné úkoly s odborníky ze systému [Linux Academy](https://linuxacademy.com/). Přečtěte si o vytvoření virtuálního počítače se systémem Linux v Azure, přístupu k VIRTUÁLNÍmu počítači se systémem Linux pomocí softwaru vzdálené plochy a spuštění virtuálních hostitelů. V tomto kurzu se řeší řada technologií a konfigurací zabezpečení.

[Zabezpečení cloudu](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

V této relaci se dozvíte, jak vám Microsoft může pomáhat plnit globální požadavky na dodržování předpisů, jako jsou ISO 27001/27018, FedRAMP, PCI a HIPAA, s novými ovládacími prvky zabezpečení. Tyto ovládací prvky jsou v rozsahu od neaktivního šifrování dat, správy klíčů, ochrany virtuálních počítačů, protokolování a monitorování, proti malwarovým službám, správě identit, řízení přístupu a dalším.

[Návrh a implementace řešení pro cloudovou datovou platformu](https://mva.microsoft.com/en-us/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527)

Seznamte se s funkcemi a možnostmi řešení Microsoft Cloud data Platform. Získejte přehled o platformě a seznamte se s funkcemi zabezpečení, možnostmi pro vysokou dostupnost, techniky pro monitorování a správu cloudových dat a další. Navíc získáte pokyny k identifikaci kompromisů a rozhodování o návrhu veřejných a hybridních cloudových řešení s využitím funkcí Microsoft Cloud data Platform.

[Správa a zabezpečení identit v cloudu a na mobilním světě](https://mva.microsoft.com/en-us/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797)

V této relaci se dozvíte, jak Azure Active Directory a Microsoft Advanced Threat Analytics pomáhají zabezpečit a spravovat identitu uživatelů, identifikovat porušení zabezpečení před tím, než způsobují poškození, a poskytnou uživatelům jedinou identitu pro přístup k celé podnikové síti. prostředky. Prozkoumejte technologie používané pro zjišťování stínového IT, správu přístupu k aplikacím a monitorování podezřelých aktivit prostřednictvím pokročilých funkcí vytváření sestav zabezpečení, analýzy chování uživatelů, auditování a upozorňování.

[Zabezpečení v cloudu s podporou cloudu](https://mva.microsoft.com/en-us/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474)

Odborníci vás seznámí s plánem zodpovědnosti zákazníka v části [Microsoft Cloud Security for Enterprise Architects](https://www.microsoft.com/download/48121) plakát. Odborníci taky poskytnou doporučení pro modernizacií jednotlivých částí stav zabezpečení, včetně zásad správného řízení, strategií pro zahrnutí, operací zabezpečení, ochrany prostředků s vysokou hodnotou, ochrany informací a zabezpečení uživatelů a zařízení. konkrétní důraz na ochranu administrativního řízení. Naučte se ze stejné architektury, kterou tým Microsoft kyberbezpečnosti používá k vyhodnocení zabezpečení cloudu zákazníků a k sestavování bezpečnostního plánu.

[Microsoft Azure IaaS hloubkové podrobně](https://mva.microsoft.com/en-us/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888)

Naučte se používat možnosti infrastruktury Microsoft Azure. Pokud jste IT specialista, nemusíte mít k dispozici předchozí zkušenosti s Azure. Tento kurz vás provede vytvořením a konfigurací virtuálních počítačů Azure, virtuálních sítí Azure a připojení mezi různými místy, abyste mohli začít pracovat v cloudu. V průběhu tohoto kurzu jsou zahrnuty bezpečnostní funkce a požadavky.

[Začínáme se zabezpečením Azure pro IT specialisty](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474)

V tomto kurzu vyplněném ukázkou tým odborníků na zabezpečení a technici Azure vám povede nad rámec základních certifikátů a zkoumá, co je možné v Azure. Podívejte se, jak navrhovat a používat různé technologie, abyste měli jistotu, že máte zabezpečení a architekturu, kterou potřebujete k úspěšnému spuštění vašich projektů v cloudu. Podrobně se na operace Datacenter, konfiguraci virtuálních počítačů, architekturu sítě a infrastrukturu úložiště.

[Podrobně scénáře a vzory s hloubkou pro Azure Resource Manager](https://mva.microsoft.com/en-us/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557)

Prozkoumejte Azure Resource Manager s využitím týmu expertů, kteří vám předvedou skripty a nástroje, které usnadňují sestavování nebo rozpínání prvků vaší aplikační infrastruktury. Prozkoumejte použití řízení přístupu na základě role (RBAC) k implementaci zabezpečení pomocí Azure Resource Manager.

[Základní dovednosti pro Azure Rights Management Services](https://mva.microsoft.com/en-us/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681)

Zjistěte, proč je ve vaší organizaci nutný požadavek služby Information Protection a jak Rights Management chrání duševní vlastnictví vaší organizace bez ohledu na to, kde se přenáší přes zařízení a Cloud. Využijte praktických zkušeností a odborných znalostí od odborníků Microsoftu.
