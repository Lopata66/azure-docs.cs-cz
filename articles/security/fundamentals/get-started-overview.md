# <a name="azure-security-overview"></a>Přehled zabezpečení Azure

Web informací o zabezpečení Microsoft Azure v Azure.com je vaše místo pro technické odpovědi na vaše otázky týkající se zabezpečení Azure. Pokud vás zajímá dodržování předpisů Azure a způsob, jakým Azure zabezpečuje svou vlastní infrastrukturu, přejděte na web [Microsoft Trust Center](https://microsoft.com/en-us/trustcenter/default.aspx).

Víme, že zabezpečení je úloha v cloudu a jak důležité je, že najdete přesné a včasné informace o zabezpečení Azure. Jedním z nejlepších důvodů, jak používat Azure pro vaše aplikace a služby, je využít výhod nejrůznějších nástrojů zabezpečení a dostupných možností. Tyto nástroje a možnosti usnadňují vytváření zabezpečených řešení na zabezpečené platformě Azure.

Tato stránka a tato lokalita se bude vyvíjet, protože nové služby Azure jsou online a stávající služby se aktualizují. Naše nabídky zabezpečení se neustále zlepšují a my budeme sdílet technické podrobnosti o těchto vylepšeních. Doporučujeme, abyste se do této lokality pravidelně vraceli a viděli novinky.

## <a name="using-the-azure-security-information-site"></a>Používání webu Azure Security Information

Když se podíváte na levou stranu stránky, zobrazí se kolekce kategorií témat zabezpečení. Kliknutím nebo klepnutím na téma zobrazíte články, které pro tuto kategorii máme.

Každá kategorie má článek s přehledem. Cílem přehledu tohoto článku je poskytnout přehled o tom, co Azure má nabídnout v souvislosti s zabezpečením pro danou kategorii. Přehledové články vám pomůžou pochopit produkty, služby a funkce dostupné v Azure, které můžete použít ke zvýšení zabezpečení vašich nasazení. Tyto přehledové články obsahují mnoho odkazů na technický obsah, abyste se mohli dozvědět víc o tématech a jejich implementaci.

Web Azure Security Information v současné době řeší následující kategorie:

* Zdroje a prostředky
* Zabezpečení sítě
* Zabezpečení úložiště
* Zabezpečení virtuálních počítačů
* Ochrana koncového bodu
* Monitorování a správa zabezpečení
* Správa identit
* Služby PaaS
* Specifické pro obor
* Architektura

> [!NOTE]
> Všechny články, které máme na webu Azure Security Information, se zobrazí v navigaci na levé straně stránky. Pokud chcete zobrazit všechny články, klikněte nebo klepněte na **Zobrazit další** v dolní části seznamu kategorií.
>
