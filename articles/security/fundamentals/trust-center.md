# <a name="microsoft-trust-center"></a>Centrum zabezpečení Microsoftu
Web Azure Security Information on Azure.com poskytuje informace, které potřebujete k zajištění bezpečného plánování, návrhu, nasazení, konfigurace a správy cloudových řešení. S centrem zabezpečení Microsoftu máte také informace, které potřebujete mít jistotu, že platforma Azure, na které spouštíte vaše služby, je zabezpečená.

Víme, že když vaše aplikace a data do Azure pověříte, budete mít otázky. Kde je to? Kdo k němu má přístup? Co Microsoft dělá k ochraně? Jak se dá ověřit, jestli Microsoft dělá, co říká?

A máme odpovědi. Vzhledem k tomu, že se jedná o vaše data, rozhodujete se, kdo má přístup, a vy s námi pracovat a rozhodnout se, kde se nachází. K ochraně vašich dat používáme špičkovou technologii zabezpečení a špičkovou kryptografii. Naše dodržování předpisů je nezávisle auditované a transparentní na mnoha úrovních – od toho, jak zpracováváme právní požadavky na vaše zákaznická data na zabezpečení našeho kódu.

Tady najdete informace o tom, co najdete na webu Microsoft Trust Center:

* [Zabezpečení](https://aka.ms/tcsecurity) – Zjistěte, jak jsou všechny služby Microsoft Cloud zabezpečené.
* [Ochrana osobních údajů](https://aka.ms/tcprivacy) – Pochopte, jak Microsoft zajišťuje soukromí vašich dat v cloudu Microsoftu.
* [Dodržování předpisů](https://aka.ms/tccompliance) – Zjistěte, jak Microsoft pomáhá organizacím v souladu s vnitrostátními, regionálními a specifickými požadavky na řízení shromažďování a používání dat jednotlivců.
* [Transparentnost](https://aka.ms/tctransparency) – Prohlédněte si, jak se Microsoft domnívá, že budete kontrolovat data v cloudu a jak vám Microsoft pomůže s tím, jak se tato data zpracovávají, co nejvíce.
* [Produkty a služby](https://aka.ms/tcproductsservices) – Další informace o Microsoft Cloudch produktech a službách na jednom místě
* [Portál pro vztahy důvěryhodnosti služeb](https://aka.ms/tcservicetrportal) – Získejte kopie nezávislých sestav auditu pro cloudové služby Microsoftu, posouzení rizik, osvědčené postupy zabezpečení a související materiály.
* [Co je nového](https://aka.ms/tcwhatsnew) – Zjistěte, co je nového v Microsoft Cloud důvěryhodnosti
* [Prostředky](https://aka.ms/tcresources) – zkoumání dokumentů White Paper, videí a případných studií v důvěryhodném cloudu Microsoftu

[Centrum zabezpečení Microsoftu](https://www.microsoft.com/trustcenter) má za to, co potřebujete pochopit, co je potřeba k zabezpečení Microsoft Cloud.
