# <a name="application-architecture-on-azure"></a>Aplikační architektura v Azure
Abychom vám pomohli zabezpečit cloudová řešení v Microsoft Azure, je rozhodující základ architektury. Architekti, návrháři a implementátori přináší výhody silných znalostí architektury aplikací a služeb. Tato základní znalostní báze vám pomůže pochopit všechny komponenty cloudových řešení a zjednodušit integraci zabezpečení do všech aspektů návrhu a implementace.

Abychom vám pomohli s vyšetřováním a návrhy architektury, máme tyto informace:

* Architektonické plány
* Sada symbolů cloudu a organizace
* Šablona Visia pro prostorový plán

## <a name="architectural-blueprints"></a>Architektonické plány
Microsoft zveřejňuje sadu architektonických plánů na vysoké [](https://aka.ms/azblueprints) úrovni, které ukazují, jak vytvářet konkrétní typy systémů s využitím produktů Microsoftu.
Každý podrobný plán obsahuje:

* Plochý soubor založený na 2D formátu Visio 2003, který můžete stáhnout a upravit
* Barevná 3D perspektiva souborů PDF k zavedení podrobného plánu do méně technických cílových skupin
* Video, které projde 3D verzí

## <a name="cloud-and-enterprise-symbol-set"></a>Sada symbolů cloudu a organizace
[Podívejte se na video s výukou pro Visio a symboly](https://aka.ms/CnESymbolsVideo) a [Stáhněte si sadu symbolů cloudu a podnikových](https://aka.ms/CnESymbols) aplikací, které vám pomůžou vytvořit technické materiály, které popisují Azure, Windows Server, SQL Server a další. Můžete použít symboly v diagramech architektury, školicích materiálech, prezentacích, datových listech, infografikaách, dokumentech White Paper a dokonce i v knihách třetích stran, pokud kniha používá k používání produktů Microsoftu. Nejsou však určeny pro použití v uživatelských rozhraních.

## <a name="3d-blueprint-visio-template"></a>Šablona Visia pro prostorový plán
3D verze plánů [Microsoftu](https://aka.ms/azblueprints) se zpočátku vytvořily v nástroji od jiného výrobce než Microsoftu. Nová šablona Visia 2013 (a novější), která se dodává 5. srpna 2015 jako součást [certifikačního kurzu architektury Microsoft, který je distribuován na EDX.org](https://docs.microsoft.com/azure/architecture/).

Tato šablona je také k dispozici mimo kurz.

* Nejprve si [Prohlédněte výukové video](https://aka.ms/3dBlueprintTemplateVideo) , abyste věděli, co může dělat.
* Stažení [šablony aplikace Microsoft 3D](https://aka.ms/3DBlueprintTemplate) podrobného plánu
* Stáhněte si [cloudové a podnikové symboly](https://docs.microsoft.com/azure/architecture/) pro použití s 3D šablonou.
