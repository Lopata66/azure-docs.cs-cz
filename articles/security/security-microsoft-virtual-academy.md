# <a name="azure-security-courses-from-microsoft-virtual-academy"></a>Kurzy zabezpečení služby Azure z Microsoft Virtual Academy
Microsoft Virtual Academy nabízí bezplatné, online školení vývojáři, IT a odborníkům v oblasti dat a studentům seznamte s nejnovějšími technologiemi, rozvíjet své dovednosti a záloh v kariéře.

Na této stránce najdete kurátorovanou sadu kurzy Azure souvisejících se zabezpečením. Přejděte [Microsoft Virtual Academy](https://mva.microsoft.com/) Chcete-li zobrazit všechny kurzy mít k dispozici.

[Vývoj/testování v cloudu](https://mva.microsoft.com/en-us/training-courses/devtest-in-the-cloud-16274?l=9aAijd7LC_2005190311)

Jste si pro vývojáře, kteří potřebují k poskytování aplikací rychlejší a lepší? Přesunutí vývoje a testovacích prostředí do cloudu vám může pomoci dosáhnout přesně, který! Další informace o získání efektivněji a přečtěte si o výhodách zajišťuje přechodu. Dál uvidíte předvádění a prezentace, které ukazují, jak můžete podpory Microsoft Azure pro vývoj a testování. Zahrnuty jsou postupy vývoje a nasazování lekce týkající se zabezpečení.

[Běžné úlohy pro Linux v Azure](https://mva.microsoft.com/en-us/training-courses/common-tasks-for-linux-on-azure-16191?l=J0Hvb7qJC_1204668937)

Pokud máte dotazy týkající se použití Linux na platformě Microsoft Azure, má tento podrobný kurz odpovědi za vás. Prozkoumejte některé běžné úlohy s odborníky z [Linux Academy](https://linuxacademy.com/). Další informace o vytváření virtuálního počítače (VM) s Linuxem v Azure, přístup k virtuálního počítače s Linuxem pomocí softwaru vzdálené plochy a spuštění virtuálního hostitele. Mnoho technologií zabezpečení a konfigurace se tak vyřeší, v tomto kurzu.

[Zabezpečení cloudu](https://mva.microsoft.com/en-us/training-courses/secure-the-cloud-14037?l=lQIkkst0B_5300115881)

V této relaci se dozvíte, jak Microsoft vám může pomoct splnit požadavky na dodržování předpisů globální, jako jsou ISO 27001 / 27018, FedRAMP, PCI a HIPAA, pomocí nové prvky zabezpečení. Tyto ovládací prvky v rozsahu od šifrování dat v klidovém stavu, správu klíčů, ochranu virtuálního počítače, protokolování a monitorování, antimalwarové služby správy identit, přístup k ovládací prvky a další.

[Návrh a implementace cloudových dat platformy řešení](https://mva.microsoft.com/en-us/training-courses/design-and-implement-cloud-data-platform-solutions-15711?l=jbCdW0j1B_3005244527)

Další funkce a možnosti řešení Microsoft cloud datovou platformu. Získejte přehled platformy a poslechněte si o funkce zabezpečení, možnostech pro vysokou dostupnost, technikách pro monitorování a správu cloudových dat a další. Navíc získáte pokyny, jak odhalovat kompromisy a dělat rozhodnutí pro navrhování veřejných a hybridních cloudových řešení pomocí funkcí platformy data Microsoft cloud.

[Správa a zabezpečení identit v cloudu a mobilním světě](https://mva.microsoft.com/en-us/training-courses/manage-and-secure-identities-in-a-cloud-and-mobile-world-14013?l=GIJ2GcvrB_405192797)

V této relaci se dozvíte, jak Azure Active Directory a Microsoft Advanced Threat Analytics vám pomůže zabezpečit a spravovat identitu uživatele, odhalovat porušení zabezpečení, předtím, než způsobí poškození a poskytnutí jedné identity uživatelé pro přístup k všechna firemní prostředky. Prozkoumejte technologie používané ke zjišťování stínového IT, spravovat přístup k aplikaci a monitorování podezřelých aktivit přes pokročilé zabezpečení vytváření sestav, vypracování analýzy chování uživatelů, auditování a výstrahy.

[Zabezpečení v Cloudovém světě](https://mva.microsoft.com/en-us/training-courses/security-in-a-cloudenabled-world-12725?l=CfLHobAcB_3904300474)

Odborníci vás provedou plánem zodpovědnosti zákazníků v [Microsoft Cloud Security for Enterprise Architects](https://www.microsoft.com/download/48121) plakát. Odborníci také poskytnout doporučení pro modernizaci jednotlivých součástí lepšímu zabezpečení, včetně zásad správného řízení, strategie členství ve skupině, operace zabezpečení, Ochrana prostředků vysoké hodnoty, information protection a uživatel a zabezpečení zařízení, zvláštní důraz na ochranu administrativní řízení. Učte se od stejné architektury, že tým kybernetického zabezpečení Microsoftu používá k vyhodnocení zabezpečení cloudu zákazníků a k jejich vytváření plánu zabezpečení.

[Microsoft Azure IaaS podrobně](https://mva.microsoft.com/en-us/training-courses/microsoft-azure-iaas-deep-dive-14339?l=PtppYVQgB_8300115888)

Další informace o použití funkcí infrastruktury s Microsoft Azure. Pokud jste specialista IT, nemusíte mít předchozí zkušenosti s Azure. Tento kurz vás provede vytvořením a konfigurace virtuálních počítačů Azure, Azure Virtual Networks a připojení mezi místními sítěmi zprovoznit věci a běžící v cloudu. Funkce zabezpečení a důležité informace jsou zahrnuty v rámci kurzu.

[Začínáme s Azure Security pro odborníky v IT](https://mva.microsoft.com/training-courses/getting-started-with-azure-security-for-the-it-professional-11165?l=HfHzCXSAB_7404300474)

V tomto kurzu plného tým odborníků na zabezpečení a technici Azure přejdete nad rámec základní certifikace a popisuje, co je možné v Azure. Zjistit, jak navrhovat a ujistěte se, že máte architektura musíte úspěšně spustit vaše projekty v cloudu a zabezpečení pomocí různých technologií. Ponořte se do provozu datových center, konfigurace virtuálních počítačů, síťové architektury a infrastruktury úložiště.

[Podrobné informace o Azure Resource Manageru scénářům a vzorcům](https://mva.microsoft.com/en-us/training-courses/deep-dive-into-azure-resource-manager-scenarios-and-patterns-13793?l=i1m06ZJYB_7001937557)

Prozkoumejte Azure Resource Manageru s týmu odborníků, kteří se dozvíte, skripty a nástroje, které usnadňují zprovoznění nebo pracovat s prvky vaší aplikační infrastruktury. Prozkoumejte použití řízení přístupu na základě rolí (RBAC) k implementaci zabezpečení pomocí Azure Resource Manageru.

[Azure Rights Management Services Core Skills](https://mva.microsoft.com/en-us/training-courses/azure-rights-management-services-core-skills-10500?l=QLoxMwuCB_1805094681)

Zjistěte, proč je information protection "musí mít" požadavek ve vaší organizaci a jak rights management chrání duševní vlastnictví vaší organizace, bez ohledu na jejich přenosu napříč zařízeními a cloudy. Získejte praktické zkušenosti a technické know-how pomoci odborníků z Microsoftu.
