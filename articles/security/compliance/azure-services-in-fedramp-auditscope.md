# <a name="azure-services-by-fedramp-and-dod-cc-srg-audit-scope"></a>Auditování služby Azure FedRAMP a ministerstva obrany USA kopie SRG oboru

Cloudové služby Microsoftu pro státní správu splnit náročné požadavky služby nám Federal Risk and Authorization Management Program (FedRAMP) a ministerstva obrany USA, od úrovně dopadu informace 2 až 5. Nasazení chráněných služby včetně Azure Government, Office 365 US Státní správy a Dynamics 365 Government, Federální a obrana instituce můžou využívat řadu využijí služby.

Tento článek obsahuje podrobný seznam služeb v oboru cloudu v Azure a Azure Government pro FedRAMP a ministerstva obrany USA kopie SRG nabídek dodržování předpisů.

#### <a name="terminologysymbols-used"></a>Terminologie/symboly použité

* DoD CC SRG = Department of Defense Cloud computingu Security Requirements Guide
* IIL = úroveň dopadu informace
* FedRAMP = Federal Risk and Authorization Management Program  
* : heavy_check_mark: = označuje služby bylo dosaženo tohoto oboru auditu.
