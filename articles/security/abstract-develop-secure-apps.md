# <a name="develop-secure-applications-on-azure"></a>Vývoj zabezpečených aplikací v Azure
## <a name="abstract"></a>Abstraktní
Tento dokument je obecné informace k ovládacím prvkům, které byste měli zvážit v jednotlivých fázích životního cyklu vývoje softwaru při vývoji aplikací pro cloud a bezpečnostní otázky. Implementace těchto konceptů, ještě před vydáním produktu můžete vytvářet bezpečnější software. Doporučení předkládané v tomto dokumentu jsou dostupné z našich zkušeností zabezpečení Azure a prostředí našich zákazníků.

Tento dokument je určen jako prostředek pro profesionální návrháře využívající softwaru, vývojáře a testery na všech úrovních, vytvářet a nasazovat řešení zabezpečení Azure.
