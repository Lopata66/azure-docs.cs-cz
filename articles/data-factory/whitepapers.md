# <a name="azure-data-factory-whitepapers"></a>Azure Data Factory dokumenty White Paper 

Dokumenty White Paper umožňují zkoumání Azure Data Factory na podrobnější úrovni. Tento článek obsahuje seznam dostupných dokumentů White Paper pro službu Azure Data Factory.

| **Dokument White Paper** | **Popis** |
| --- | --- |
|[Azure Data Factory – integrace dat v cloudu](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-data-integration-in-the-cloud/Azure_Data_Factory_Data_Integration_in_the_Cloud.pdf) | Tento dokument popisuje, jak Azure Data Factory vám umožní vytvořit moderní datový sklad, povolit pokročilou analýzu inteligentních aplikací SaaS a přenést své balíčky integrační služby SQL Server do Azure.|
|[Migrace dat z relačního skladu dat místní do Azure pomocí Azure Data Factory](https://azure.microsoft.com/mediahandler/files/resourcefiles/data-migration-from-on-premise-relational-data-warehouse-to-azure-data-lake-using-azure-data-factory/Data_migration_from_on-prem_RDW_to_ADLS_using_ADF.pdf) | Tento papír řeší složitost migrace desítky TB dat z existujícího místního relačního skladu dat (například Netezza, Oracle a Teradata, SQL server) do Azure (například Blob Storage nebo Azure Data Lake Storage) pomocí služby Azure Data Factory. Problémy a osvědčené postupy jsou znázorněny odolnost, výkon, škálovatelnost, řízení a zabezpečení pro cestu příjem velkých objemů dat do Azure pomocí Azure Data Factory. |
|[Azure Data Factory: Služby SSIS v cloudu](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-ssis-in-the-cloud/Azure_Data_Factory_SSIS_in_the_Cloud.pdf)| Tento dokument prochází přes proč byste to migrovat stávající úlohy služby SSIS do Azure Data Factory a adresa úvaze a připomínky. Potom jsme vás provedou technické podrobnosti o vytvoření prostředí Azure-SSIS IR a ukazují, jak nahrát, spouštění a sledování vašich balíčků prostřednictvím služby Azure Data Factory pomocí nástroje, které jsou pravděpodobně znáte jako je SQL Server Management Studio (SSMS). |
|[Azure data Factory – předání parametrů](https://azure.microsoft.com/mediahandler/files/resourcefiles/azure-data-factory-passing-parameters/Azure%20data%20Factory-Whitepaper-PassingParameters.pdf)| Tento článek vás provede jak předávat parametry mezi aktivit a jde o vztah mezi aktivit a kanálu. |
|[Azure data Factory – DevOps](https://azure.microsoft.com/mediahandler/files/resourcefiles/whitepaper-adf-on-azuredevops/Azure%20data%20Factory-Whitepaper-DevOps.pdf)| Tento dokument White Paper popisuje některé z osvědčených postupů po průběžnou integraci a nasazení služby Azure Data Factory. |
