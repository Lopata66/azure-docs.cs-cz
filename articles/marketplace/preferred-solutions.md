# <a name="preferred-solutions-in-azure-marketplace-and-appsource"></a>Upřednostňovaná řešení na Azure Marketplace a AppSource

Upřednostňované řešení společnosti Microsoft je cloudová aplikace vybraná pro svou kvalitu, výkon a schopnost řešit potřeby zákazníků v určité oblasti vertikálního odvětví nebo řešení. Tým odborníků společnosti Microsoft ověřuje řešení od partnerů se specifickými osvědčenými kompetencemi a schopnostmi. Tato řešení jsou dostupná v našich obchodech na cloudmarketplace, Azure Marketplace a AppSource, stejně jako na webu Azure Portal. Upřednostňovaná řešení na AppSource mohou být objevena oborovými vertikálami. Upřednostňovaná řešení na Azure Marketplace a na portálu Azure se můžou zjidužovat napříč kategoriemi horizontálních řešení.
