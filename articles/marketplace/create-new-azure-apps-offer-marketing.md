# <a name="how-to-market-your-azure-application-offer"></a>Jak uvádět nabídku aplikací Azure na trh

Tento článek popisuje další marketingové možnosti, které můžete vybrat, Pokud prodáváte nabídku aplikací Azure prostřednictvím Microsoftu. Vaši nabídku můžete společně prodávat pomocí Microsoftu a znovu ji prodávat prostřednictvím [programu Cloud Solution Providers (CSP)](cloud-solution-providers.md)nebo obojího.

## <a name="co-sell-with-microsoft"></a>Společný prodej s Microsoftem

Poskytování informací o **společném prodeji** na kartě Microsoft je zcela volitelné. Je ale potřeba, _abyste dosáhli svého stavu připraveného_ k spoluprodeji a _IP_ . Prodejní týmy Microsoftu využívají tyto informace k získání dalších informací o řešení při vyhodnocování svých potřeb pro zákazníky. Informace, které na této kartě zadáte, nejsou k dispozici přímo pro zákazníky.

Podrobnosti a pokyny, jak nakonfigurovat **společný prodej pomocí karty Microsoftu** , najdete v tématu [možnost společného prodeje v partnerském centru](partner-center-portal/commercial-marketplace-co-sell.md).

## <a name="resell-through-csps"></a>Prodávejte dál prostřednictvím CSP

Pokud se rozhodnete, že vaše nabídka bude v programu Cloud Solution Provider (CSP) k dispozici, mohou poskytovatelé služeb prodávat vaše produkty jako součást řešení svým zákazníkům v rámci balíčku. Další informace o opětovném prodeji vaší nabídky pomocí programu CSP a podrobné pokyny ke konfiguraci opětovného **prodeje prostřednictvím karty CSP** najdete v tématu [program Cloud Solution Provider](cloud-solution-providers.md).
