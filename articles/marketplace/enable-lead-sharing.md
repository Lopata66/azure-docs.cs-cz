# <a name="enable-lead-sharing"></a>Povolení sdílení potenciálních zákazníků
Zajištění, že je povolená správa potenciálních zákazníků pro váš zápis na [Azure Marketplace](https://azuremarketplace.microsoft.com) nebo [Microsoft AppSource](https://appsource.microsoft.com) vám může pomoci získat kontaktní informace pro následující cílové skupiny:

| Průčelí | Cílová skupina |
|:--- |:--- |
| AppSource | Firemní zákazníci |
| Azure Marketplace | Všichni zákazníci |

Vaši zájemci poskytují základ pro oslovení kampaní uvedených v následující tabulce, souvisejících prodejních pohybů vašich pracovníků prodeje v terénu a informací o tom, jak si vaše aplikace nebo nabídka vede:

| Průčelí | Kampaň |
|:--- |:--- |
| AppSource | Nurture |
| Azure Marketplace | Výroba v poptávce |

Pro zájemce postupujte takto:
*   Kvalifikujte zájemce a skóre zájemců jako prodejních příležitostí.
*   Pečujte o zájemce o vstup do prodejního kanálu.
*   Srovnejte své úsilí s celkovou strategií marketingové kampaně.

Vaši zájemci prokazují zájem zákazníků o následující nabídky:

| Průčelí | Výpis |
|:--- |:--- |
| AppSource | <table> <tr><td>Aplikace</td></tr> <tr><td>Konzultační služby</td></tr> </table> |
| Azure Marketplace | <table> <tr><td>Nabídka</td></tr> <tr><td>Technologie</td></tr> </table> |

Měli byste mít zaveden proces, který bude sledovat zájemce, jak je popsáno v následující tabulce:

| Průčelí | Leads |
|:--- |:--- |
| AppSource | <table> <tr><td>Váš CRM</td></tr> <tr><td>Microsoft pole prodejce CRM</td></tr> </table> |
| Azure Marketplace | <table> <tr><td>Azure Marketplace</td></tr> <tr><td>Microsoft pole prodejce CRM</td></tr> </table> |

Vaši zájemci v obchodech na marketplace se liší od potenciálních zákazníků kvalifikovaných pro prodejce společnosti Microsoft, kteří jsou součástí programu Co-Sell.

Další informace o přístupu k programu Co-Sell naleznete v [tématu Propagace firmy u společnosti Microsoft](./promote-your-business-with-microsoft.md).
