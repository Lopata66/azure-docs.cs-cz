# <a name="co-sell-partner-engagement"></a>Zapojení partnerů do společného prodeje

Pokud jste partner, můžete použít program pro spoluprodejci společnosti Microsoft k přímé spolupráci se společností Microsoft na společných prodejních příležitostech. Tento program pomáhá našim partnerům oslovit rozsáhlou komunitu zákazníků a vytvářet nové prodejní příležitosti, které vám pomůžou zrychlit obchodní růst.

## <a name="partner-selling-motions"></a>Prodejní pohyb partnerů

Společnost Microsoft nabízí tři prodejní pohyby, které jsou pro partnery:

* *Prodej prostřednictvím partnera*: technologii Microsoftu můžete dál prodávat prostřednictvím svých prodejců.
* *Služby, které společně prodávají*: prodejci a partneři spolupracují na prodeji vašeho projektu nebo spravovaných služeb pomocí technologie Microsoft Embedded.
* *Společný prodej*: prodejci a partneři spolupracují na tom, aby mohli prodávat vaše opakované a zabalené IP řešení. *

\*Některá z těchto řešení pro spoluprodejní IP můžou dostávat další pobídky, když je úspěšně proprodáváte a nasadíte s prodejními týmy pole společnosti Microsoft.
