# <a name="microsoft-commercial-marketplace-faqs"></a>Nejčastější dotazy ke komerčnímu marketplace Microsoftu

Odpovědi na běžné otázky týkající se komerčního trhu.

## <a name="faq-for-customers"></a>Nejčastější dotazy pro zákazníky

### <a name="what-you-need-to-know-about-the-commercial-marketplace"></a>Co potřebujete vědět o komerčním trhu

**Co je Azure Marketplace?**

[Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace) poskytuje přístup a informace o řešeních a službách dostupných od Microsoftu a našich partnerů. Zákazníci můžou objevovat, vyzkoušet nebo kupovat cloudová softwarová řešení postavená na Azure nebo pro Azure. Náš katalog 8 000+ výpisů poskytuje stavební bloky Azure, jako jsou virtuální počítače (Virtuální počítače), rozhraní API, aplikace Azure, šablony řešení a spravované aplikace, aplikace SaaS, kontejnery a poradenské služby.

**Kdo jsou zákazníci Azure Marketplace?**

Azure Marketplace je navržené pro odborníky v oblasti IT a vývojáře cloudu, kteří se zajímají o komerční IT software a služby.

**Jaký typ produktů je momentálně dostupný na Azure Marketplace?**

Azure Marketplace nabízí technická řešení a služby od Microsoftu a partnerů vytvořených pro rozšíření produktů a služeb Azure. Katalog řešení zahrnuje několik kategorií, včetně:

* základní operační systémy
* databáze
* security
* identity
* Sítí
* blockchain
* vývojářské nástroje

### <a name="azure-marketplace-for-customers"></a>Azure Marketplace pro zákazníky

**Jak na Azure Marketplace začnu?**

Širokou škálu podnikových aplikací a řešení, které jsou certifikované a optimalizované pro spouštění v Azure, najdete na [Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/apps). Azure Marketplace je taky přístupné prostřednictvím [portálu Azure] při [**vytváření prostředku**](https://portal.azure.com/#create/hub).

**Jaké jsou hlavní výhody Azure Marketplace?**

Na Azure Marketplace můžou zákazníci objevovat technické aplikace vytvořené pro Azure nebo postavené na Azure. Kombinuje trh řešení a služeb Microsoft Azure do jediné, sjednocené platformy a pomocí několika kliknutí objevuje, zkouší, kupuje nebo nasazuje řešení.

**Jak si můžu koupit produkty z Azure Marketplace?**

Nabídky Azure Marketplace lze zakoupit prostřednictvím:

* [Webová prodejna](https://azuremarketplace.microsoft.com/marketplace/apps)
* [Portál Azure] [Portál Azure]
* [Rozhraní příkazového řádku Azure (CLI)](/cli/azure/?view=azure-cli-latest)

>[!Note]
>Předplacené kredity a jiné formy peněžního závazku nelze použít k úhradě licenčních poplatků za software, ale lze je použít k úhradě souvisejících poplatků za využití Azure. Výjimky jsou uvedeny v [azure peněžní závazek](https://azure.microsoft.com/updates/azure-marketplace-third-party-reseller-services-now-use-azure-monetary-commitment/).

**Můžu si vybrat, která oblast Azure(y) nasadím svůj nákup na Azure Marketplace?**

Zákazník má možnost nasazení do libovolné oblasti datového centra Azure, kterou povolí. Doporučujeme vybrat umístění datových center, která jsou nejblíže vašim službám, pro optimalizaci výkonu a kontroly rozpočtu.

**Pokud omylem odstraním nákup na Azure Marketplace, můžu akci *vrátit zpět?***

Ne, smazání je konečné. Pokud je předplatné omylem odstraněno, lze ho odkoupit. Veškeré nevyužité funkce nebo předplacené služby jsou ztraceny.

**Jsem upozorněn, když se pokusím odstranit nákup azure marketplace, který používá některá z mých aplikací?**

Ne, Azure neposkytuje žádné upozornění při odstranění nákupu, i když se aktuálně používá nebo je aplikace závislá.

**Pokud má můj nákup na Azure Marketplace nějaké závislosti na jiných aktivech, jako je web Azure, musím je spravovat?**

Závislosti se automaticky nespravují pro nabídky Azure Marketplace. Pečlivě zkontrolujte popis nákupu Azure Marketplace před použitím k určení, zda existují nějaké závislosti potřebné před nasazením řešení.

**Můžu si koupit řešení Azure Marketplace od poskytovatele cloudových řešení Azure?**

Pokud vydavatel nakonfiguroval svou nabídku tak, aby byla dostupná prostřednictvím kanálu Zprostředkovatele cloudových řešení (CSP), mají partneři poskytovatele cloudových řešení možnost řešení dále prodávat.

**Které země nebo oblasti jsou podporované pro nákup aplikací a služeb prodávaných/zřazených prostřednictvím Azure Marketplace?**

Azure Marketplace je k dispozici zákazníkům Azure v zemích nebo oblastech uvedených v [zásadách účasti](/legal/marketplace/participation-policy).

**Jaké měny azure marketplace podporuje?**

Transakce mohou být prováděny v následujících 17 měnách: AUD, BRL, CAD, CHF, DKK, EUR, GBP, INR, JPY, KRW, NOK, NZD, RUB, SEK, TWD, USD, RMB.

### <a name="deploying-a-solution-from-azure-marketplace"></a>Nasazení řešení z Azure Marketplace

**Nasadil jsem virtuální počítač Azure Marketplace (VM) do předplatného a teď chci migrovat předplatné z jednoho účtu Azure do jiného. Je to v současné době podporováno?**

Chcete-li migrovat předplatné Azure, včetně virtuálních virtuálních měn a služeb Azure Marketplace, odstraňte nebo zrušte jakékoli předchozí předplatné Azure před přisunutím k novému účtu Azure. Po dokončení migrace se výsledné poplatky za použití účtují pomocí způsobu platby nového registrovaného účtu.

**Chci migrovat předplatné virtuálního počítače Azure Marketplace (VM) do smlouvy Enterprise. Je to v současné době podporováno?**

Chcete-li migrovat předplatné virtuálního počítače Azure Marketplace (VM) do smlouvy Enterprise, zastavte nebo zrušte jakékoli předchozí předplatné před migrací. Po dokončení migrace vašeho účtu Azure a přidružených předplatných můžete znovu zakoupit virtuální počítač nebo službu Azure Marketplace.  Výsledné poplatky za použití se účtují čtvrtletně podle smlouvy Enterprise.

### <a name="pricing-and-payment"></a>Ceny a platby

**Jaké jsou ceny předplatných Azure Marketplace?**

Ceny se liší v závislosti na typu produktů a specifikacích vydavatele. Licenční poplatky za software a náklady na využití Azure se účtují zvlášť prostřednictvím předplatného Azure.

*Oddělená:*

+ *Model BYOL (Bring-your-own-license):* Při získání softwarové licence přímo od vydavatele nebo prodejce nejsou účtovány žádné další poplatky související se softwarem.

*Svázaný:*

Předplatné Azure je součástí cen řešení nezávislých dodavatelů softwaru (ISV) vydavatele.

*Účtovány:*

+ *Zdarma:* SKU zdarma. Za licenční poplatky za software ani za použití nabídky se neúčtují žádné poplatky.

+ *Zkušební verze bezplatného softwaru:* Nabídka, která je zdarma po omezenou dobu. Za použití během zkušebního období se neúčtují žádné poplatky za softwarové licence vydavatele. Po vypršení platnosti se automaticky převede na placenou nabídku na základě standardních sazeb vydaných vydavatelem.

+ *Na základě využití:* Ceny jsou účtovány nebo účtovány na základě rozsahu využití nabídky. U image virtuálních počítačů se účtuje hodinový poplatek. U služeb pro vývojáře a api se účtuje za měrnou jednotku definovanou nabídkou.

+ *Paušální poplatek:* Předplatné SaaS lze oceňovat jako paušální poplatek, který se účtuje měsíčně nebo ročně. To může také zahrnovat další fakturační dimenze, které se účtují podle spotřeby (například šířka pásma, e-maily nebo vstupenky). 

+ *Na uživatele:* Předplatné SaaS lze oceňovat na základě pro jednotlivé uživatele, které se účtují měsíčně nebo ročně. 

Podrobnosti o cenách specifických pro nabídku najdete na stránce podrobností o řešení na [https://azure.microsoft.com/pricing/](https://azure.microsoft.com/pricing/) [portálu Azure] nebo v jeho rámci.

> [!Note]
> S výjimkou měsíčních poplatků se poplatky za využití Azure vztahují na všechny cenové modely, pokud není uvedeno jinak.

**Jak mám poskytnout klíč softwarové licence pro řešení NA WEBU A jakou roli hraje Azure Marketplace?**

Za získání a vynucení licenčních pověření pro řešení BYOL odpovídá vydavatel. U nabídek virtuálního počítače se po spuštění aplikace obvykle v aplikaci vydavatele vyskytuje získání licenčního klíče. Při použití nabídky virtuálního počítače nasazené prostřednictvím šablony aplikačního řešení Azure lze šablonu Správce prostředků nakonfigurovat tak, aby uživatele vyzýval k řadě vstupů, včetně pověření licencí.

Toto jsou nejběžnější možnosti podle typu nabídky:

*Nabídka virtuálního počítače:*

+ *Možnost 1:* K získání licenčního klíče obvykle dochází v aplikaci vydavatele po spuštění aplikace.

+ *Možnost č. 2:* Licenční klíč zadá koncový uživatel (přes příkazový řádek / webové rozhraní poskytované nabídkou) po nasazení nabídky virtuálního počítače ve vybraném předplatném. Licence může být klíč nebo soubor, jak určí vydavatel.

*Aplikace Azure (šablona řešení a spravované aplikace):*

+ *Možnost 1:* Šablonu Resource Manager lze nakonfigurovat tak, aby vás vyzvala k zadání řady vstupů včetně pověření licencí. To lze provést jako licenční soubor (file upload) nebo klíč (vstup textového pole) před nasazením nabídky v předplatném koncového uživatele.

+ *Možnost č. 2:* Licenční klíč můžete zadat pomocí příkazového řádku / webového rozhraní poskytovaného nabídkou.  Provádí se po nasazení nabídky Azure Apps ve vybraném předplatném. Licence může být klíč nebo soubor, jak určí vydavatel.

**Jaké druhy zkoušek jsou podporovány?**

Vydavatelé můžou přidat jeden měsíc zdarma pro placené nabídky SaaS a jeden nebo tři měsíce spotřeby virtuálních her zdarma. Bezplatné zkušební nabídky jsou výpisy s výzvou k akci k zahájení zkušební verze. Ty vedou zákazníka na web definovaný vydavatelem k nastavení zkušebního prostředí. Zkušební verze lze také přidat k placeným nabídkám, kde je první měsíc zdarma. 

**Musím mít v e-souboru platební nástroj (například kreditní kartu) pro nasazení bezplatné úrovně nebo pro vlastní nabídku licence (BYOL)?**

Ne. K nasazení nabídky Free Tier nebo BYOL není vyžadován platební nástroj. Nabídka bezplatné zkušební verze však vyžaduje platební prostředek. Výpisy, které obsahují tlačítka get **it now** nebo bezplatné **ho softwaru,** se nasadí do vybraného předplatného Azure.  Tyto výpisy se účtují pomocí registrovaného způsobu platby vybraného účtu. Poplatky za využití Azure se účtují odděleně od licenčních poplatků za software.

**Pokud mají dotazy týkající se cen za nabídky prodávané na Azure Marketplace, kdo nepřímý kontakt se zákazníkem se smlouvou Enterprise (EA) kontaktuje?**

Nepřímí zákazníci smlouvy Enterprise (EA) musí kontaktovat svého poskytovatele licenčních řešení (LSP) a vyžádat si všechny cenové otázky na Azure Marketplace.

**Můžu řídit přístup svých zaměstnanců na Azure Marketplace a oprávnění k nákupu?**

Ano, u zákazníků smlouvy Enterprise (EA) může správce registrace vypnout oprávnění k nákupu pro všechny účty při registraci a zapnout je zpět na dostatečně dlouhou dobu na nákup.

**Jaké způsoby platby jsou podporovány pro nákupy na komerčním trhu?**

Zákazníci si mohou zakoupit nabídky z komerčního trhu pomocí kreditních karet. Pokud máte existující předplatné Azure, nákupy z Azure Marketplace budou používat způsob platby nakonfigurovaný na účtu a zobrazí se na stejné faktuře jako samostatná řádková položka. Některé nabídky spotřebovávají peněžní závazek Azure, ale většina nákupů na komerčním trhu nečerpá závazky smlouvy Enterprise, i když spotřeba infrastruktury Azure bude.

**Můžu na nabídky Azure Marketplace použít kredity předplatného Azure nebo peněžní prostředky na závazky na svém účtu?**

Konkrétní nabídky Azure Marketplace můžou využívat kredity předplatného Azure nebo fondy peněžních závazků. Úplný seznam produktů zapojených do tohoto programu najdete v tématu [Peněžní závazek Azure](https://azure.microsoft.com/updates/azure-marketplace-third-party-reseller-services-now-use-azure-monetary-commitment/). Tyto nabídky nezahrnují možnosti BYOL nebo BYOS. Všechny ostatní nabídky Azure Marketplace nemohou používat kredity předplatného Azure ani peněžní závazek: jako je bezplatný měsíční zkušební kredit, měsíční kredity MSDN, kredity z propagačních akcí Azure, zůstatky peněžních závazků a další bezplatné kredity poskytované z Azure.

**Vztahují se na nákupy na Azure Marketplace slevy na multilicence?**

Ne. Vydavatel, který vlastní řešení na Azure Marketplace, může nastavit ceny.  Standardní slevy na multilicence Microsoftu se na nákupy na Azure Marketplace nevztahují.

**Kde si můžu zobrazit podrobnosti o předplatném Azure Marketplace a fakturační údaje?**

MOSP – [Microsoft Online Subscription Program](https://azure.microsoft.com/support/legal/subscription-agreement/?country=us&language=en) (web direct) zákazníci mohou zobrazit podrobnosti o nákupu Marketplace na kartě Faktury v části Správa nákladů a fakturace na [portálu Azure].

Zákazníci s podnikovou smlouvou (EA) si můžou zobrazit podrobnosti o nákupu na Marketplace na kartě Faktury v části Správa nákladů a fakturace na portálu [Azure Portal].

Partneři poskytovatele cloudových řešení (CSP) mohou zobrazit podrobnosti o nákupu marketplace pro každého zákazníka v Centru partnerů na kartě Historie objednávek po výběru společnosti v zobrazení Zákazníci.

**Jak zruším doplněk Azure Marketplace k virtuálnímu počítači Azure?**

Vzhledem k tomu, že doplněk je přidružený k virtuálnímu počítači (VM) Azure Virtual Machine (VM), chcete-li zrušit nákup Azure Marketplace, nejprve virtuální počítač zastavte jeho odstraněním. Tím se zastaví veškeré využití předplatného a poplatky na nákup Azure Marketplace.

**Jak často se mi účtují za nákupy na Azure Marketplace?**

Všechny nabídky Azure Marketplace, které se neodečítají od peněžního závazku, se účtují měsíčně v prodlení. Roční předplatné SaaS se účtuje jednou za celoroční služby.

MOSP – [Microsoft Online Subscription Program](https://azure.microsoft.com/support/legal/subscription-agreement/) (web direct) zákazníkům se účtuje měsíčně ze stejné platební karty, která je v souboru pro jejich profil předplatného Azure. Roční předplatné SaaS se účtuje jednou za celoroční služby.

**Jak můžu přesunout své nákupy na Azure Marketplace z předplatného MOSP na své přímé předplatné Smlouvy Enterprise (EA)?**

I když většinu předplatných Microsoftu lze snadno převést na smlouvu Enterprise, nákupy na Azure Marketplace v rámci těchto předplatných nelze.

Chcete-li migrovat další služby zakoupené z Azure Marketplace do předplatného EA, nejprve zrušte aplikace z existujícího předplatného MOSP a znovu odkupte tyto aplikace v rámci předplatného EA. Tímto způsobem můžete odeslat žádost o úvěr o vrácení peněz během potenciálního měsíce překrývajícího se pokrytí mezi předplatnými služeb Marketplace – vytvořte [lístek podpory](https://portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview).

**Jaký je rozdíl mezi *cenou*, *cenou softwaru*a *celkovou cenou* ve struktuře nákladů na nabídky virtuálních strojů na Azure Marketplace?**

*Cena* se vztahuje na náklady na virtuální počítač Azure spustit software. *Cena softwaru* označuje náklady na software vydavatele Marketplace spuštěný na virtuálním počítači Azure. *Celková cena* se vztahuje na kombinované celkové náklady na virtuální počítač Azure a software vydavatele Marketplace spuštěné na virtuálním počítači Azure.

**Jak zjistím, kolik z nákupu na Azure Marketplace jsem použil?**

Odhadované informace o využití najdete na [portálu Azure]. Tyto odhadované informace o využití nemusí zahrnovat nedávné činnosti a mohou být založeny na odhadech odvozených z minulé spotřeby. Během verze Public Preview nemusí být tato funkce k dispozici pro všechny nákupy a může se lišit v závislosti na typu produktu.

### <a name="customer-support"></a>Zákaznická podpora

**Na koho se mám obrátit kvůli obecným problémům s podporou Azure Marketplace?**

Chcete-li požádat o obecnou podporu aplikací týkající se použití nebo řešení potíží, obraťte se přímo na vydavatele aplikace.

Pokud chcete s nákupem na Azure Marketplace řešit účty a předplatné, kontaktujte [podporu Azure](https://support.microsoft.com/getsupport?oaspworkflow=start_1.0.0.0&wf=0&wfname=productselection&prid=16230&forceorigin=esmc&ccsid=636694515623707953).

**Na koho se mám obrátit, když potřebuji technickou podporu k řešení zakoupenému na Azure Marketplace?**

Obraťte se na poskytovatele vydavatele a požádejte o veškerou technickou podporu produktu. Kontaktní informace vydavatele nebo odkaz na web podpory najdete na stránce s podrobnostmi o řešení na Azure Marketplace.

**Na koho se mám obrátit s žádostí o fakturační podporu/dotazy týkající se řešení třetí strany zakoupeného na Marketplace?**

Obraťte se na podporu Microsoftu na [webu Azure Support](https://support.microsoft.com/getsupport?oaspworkflow=start_1.0.0.0&wf=0&wfname=productselection&prid=16230&forceorigin=esmc&ccsid=636694515623707953).

**Na koho se mám obrátit, pokud mám dotazy týkající se cen nebo podmínek pro partnerská řešení prodávaná na Azure Marketplace?**

Obraťte se na poskytovatele vydavatele a požádejte o veškerou technickou podporu produktu. Kontaktní informace vydavatele nebo odkaz na web podpory najdete na každé stránce podrobností o řešení na Azure Marketplace.

**Pokud nejsem spokojen(a), mohu nákup vrátit?**

Nákupy provedené z [Azure Marketplace](https://azuremarketplace.microsoft.com/) nelze vrátit, ale lze je zrušit nebo odstranit. Nabídky založené na spotřebě se účtují podle využití, takže když se zastaví, poplatky se zastaví také. Předplatná jsou zrušena a nebudou se účtovat po aktuálním fakturačním období. Pokud je předplatné zrušeno krátce po zakoupení (24 hodin měsíčně a 14 dní pro roční), je poskytnuta plná náhrada.

Zákazníci musí přímo kontaktovat vydavatele v případě jakýchkoli technických problémů souvisejících se službou Marketplace nebo nákupem. Kontaktní informace vydavatele nebo odkaz na web podpory najdete na stránce s podrobnostmi o řešení na Azure Marketplace.

**Jak se zachází s dodatky licencí při zvýšeném střednědobém horizontu?**

Licence přidané k existujícímu předplatnému jsou po zbytek doby trvání předplatného poměrné.

**Jak se zpracovává odebrání licencí v polovině období?**

**Je možné změnit plány v polovině období?**
