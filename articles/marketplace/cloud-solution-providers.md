# <a name="cloud-solution-providers"></a>Poskytovatelé cloudových řešení

Nabídky softwaru může oslovit miliony kvalifikovaný zákazníkům Microsoftu, kteří obsluhují partneři v programu Cloud Solution Provider (CSP), kromě zveřejnění nabídky prostřednictvím [prodejních míst web společnosti Microsoft](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Vydavatelé konfigurace nabídky pro dostupnost v programu CSP na základě vyjádření souhlasu se (výchozí je vyloučena), pro nové nabídky nebo některý z existujících umožňuje partnerům prodávat vaše produkty a vytvořit dodávat řešení pro zákazníky.

Vydavatelé nesou odpovědnost za poskytování podpory typu break-fix koncovým zákazníkům a poskytuje mechanismus pro partneři CSP a/nebo můžete požádat o podporu zákazníků. Je osvědčené postupy k poskytování PARTNEŘI CSP dokumentaci pro uživatele, školení a služeb oznámení o stavu/výpadcích (podle vhodnosti) tak, aby partneři CSP jsou vybaveny pro zpracování žádostí o vrstvy 1 podporu od zákazníků.

V následující tabulce jsou uvedeny typy nabídek a cenové modely, které můžete přejít k programu CSP:

| **Typ nabídky**    | **BYOL**  |  **Free** | **PRŮBĚŽNÉ PLATBY**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuální počítače  | Ano | Ano | Ano |
| Containers | neuvedeno | Ano | neuvedeno |
| Šablony řešení | neuvedeno | Ano | neuvedeno |
| Managed Applications | neuvedeno | Ano | Ano |
| Software-as-a-Service (SaaS) | neuvedeno | Ano | Ano |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>Jak nakonfigurovat nabídku

Zprostředkovatel kryptografických služeb program vyjádřit výslovný souhlas nastavení nakonfigurované v prostředí vytvoření nabídky partnerského centra nebo portál partnerů cloudu. ([Další informace o prostředí měnících vydavatele. ](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

V portál Cloud Partner je nastavena na kartě Marketplace nebo z prodejních míst. V partnerském centru je nastavena na kartě nabídky nastavení v části Podrobnosti o nastavení.

![Zprostředkovatel kryptografických služeb opt-in prostředí v CPP](media/marketplace-publishers-guide/csp-opt-in.png)

Jak vidíte níže, v partnerském centru vydavatelé mohou přihlásit a jejich nabídky zpřístupnit v programu CSP zaškrtnutím zaškrtávacího políčka, **zpřístupnit své nabídky do programu CSP**.

![Zprostředkovatel kryptografických služeb vyjádřit výslovný souhlas prostředí do partnerského centra](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>Sdílení s partnery CSP prodej a podporu materiály

Chcete-li povolit partneři v programu Cloud Solution Provider k co nejefektivnějšímu představují vaší nabídky a zapojení ve vaší organizaci, odešlete prodeje a podporují materiály, které budou k dispozici je prodejcům v systému. Tyto prostředky nebude vystavena pro zákazníky v marketplace prodejní místa.

Jak je znázorněno níže, portál Cloud Partner, mohou vydavatelé zadejte adresu URL, který je hostitelem příslušné marketingové materiály a kanál kontaktní informace na kanál CSP:
