# <a name="cloud-solution-providers"></a>Poskytovatelé cloudových řešení

Nabídky softwaru teď můžete oslovit miliony kvalifikovaný zákazníkům Microsoftu, kteří obsluhují partnery Cloud Solution Provider (CSP) s minimálními marketingové a prodejní investice.  Vydavatelé můžou nyní využívat kanály partner CSP, kromě zveřejnění nabídky prostřednictvím [prodejních míst web společnosti Microsoft](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Vydavatelé mohou rozhodnout, jestli k vyjádření výslovného souhlasu nabídku dostupnosti v kanálu partner CSP, pro nové nabídky nebo některý z existujících. Tato možnost umožňuje partnerům CSP prodávat vaše produkty a vytvořit jako součást balíčku řešení pro zákazníky.

Vydavatelé nesou odpovědnost za poskytování podpory typu break-fix koncovým zákazníkům a poskytuje mechanismus pro partneři CSP a/nebo můžete požádat o podporu zákazníků. Je osvědčené postupy k poskytování PARTNEŘI CSP dokumentaci pro uživatele, školení a služeb oznámení o stavu/výpadcích (podle vhodnosti) tak, aby partneři CSP jsou vybaveny pro zpracování žádostí o vrstvy 1 podporu od zákazníků.

V následující tabulce jsou uvedeny typy nabídek a cenové modely, které se mohou přihlásit a partnerský kanál CSP:

| **Typ nabídky**    | **BYOL**  |  **Free** | **PRŮBĚŽNÉ PLATBY**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuální počítače  | Ano | Ano | Ano |
| Containers | neuvedeno | Ano | neuvedeno |
| Šablony řešení | neuvedeno | Ano | neuvedeno |
| Managed Applications | neuvedeno | Ano | Ano |
| IT řešení SaaS (podle ceny za web) | neuvedeno | Ano | Ano |
|   |   |   |

## <a name="how-to-opt-in"></a>Tom, jak vyjádřit výslovný souhlas

CSP partnera kanál vyjádřit výslovný souhlas se našlo nastavení na kartě Marketplace nebo z prodejních míst v rámci prostředí pro vytváření nabídka portál partnerů cloudu.
