# <a name="cloud-solution-providers"></a>Poskytovatelé cloudových řešení

Nabídky softwaru můžou získat přístup k miliony kvalifikovaných zákazníků Microsoftu poskytovaných partnery v programu Cloud Solution Provider (CSP) Kromě veřejné dostupnosti nabídek přes [Web prodejní místa Microsoftu](https://docs.microsoft.com/azure/marketplace/comparing-appsource-azure-marketplace).

Vydavatelé konfigurují nabídky pro dostupnost v programu CSP na základě výslovného souhlasu (výchozí je vyloučený), pro novou nabídku nebo stávající, umožňující partnerům prodávat vaše produkty a vytvářet řešení pro zákazníky.

Vydavatelé zodpovídají za poskytování podpory opravných řešení koncovým zákazníkům a poskytování mechanismů pro partnery CSP a/nebo zákazníky, kteří vám pomůžou kontaktovat podporu. Osvědčenými postupy je poskytnout partnerům CSP používání dokumentace k uživateli, školení a upozornění na stav a výpadek služeb (podle potřeby), aby partneři CSP mohli zpracovávat žádosti o podporu vrstvy 1 od zákazníků.

Následující tabulka obsahuje typy nabídek a cenové modely, které se můžou zúčastnit programu CSP:

| **Typ nabídky**    | **BYOL**  |  **Free** | **PAYG**   |
| :---------------- | :---------|:----------|:-----------|
| Virtuální počítače  | Ano | Ano | Ano |
| Kontejnery | Není dostupné. | Ano | Není dostupné. |
| Šablony řešení | Není dostupné. | Ano | Není dostupné. |
| Managed Applications | Není dostupné. | Ano | Ano |
| Software jako služba (SaaS) | Není dostupné. | Ano | Ano |
|   |   |   |

## <a name="how-to-configure-an-offering"></a>Postup konfigurace nabídky

Nastavení výslovných přihlášení k programu CSP se konfiguruje v partnerském centru nebo v prostředí pro vytváření nabídek portál partnerů cloudu. (Další[informace o změně prostředí pro publikování.](https://www.microsoftpartnercommunity.com/t5/Azure-Marketplace-and-AppSource/Cloud-Marketplace-In-Partner-Center/m-p/9738#M293))

V portál partnerů cloudu se nastavuje na kartě Marketplace nebo prezentace. V partnerském centru je nastavená na kartě nastavení nabídky v části Podrobnosti o nastavení.

![Prostředí pro výslovný souhlas zprostředkovatele CSP v CPP](media/marketplace-publishers-guide/csp-opt-in.png)

Jak je vidět níže, v partnerském centru se můžou vydavatelé přihlásit k zpřístupnění nabídek v programu CSP tím, že zaškrtnutím políčka **zpřístupní nabídku moje nabídka v programu CSP**.

![Prostředí s výslovným souhlasem CSP v partnerském centru](media/marketplace-publishers-guide/pc-csp-opt-in.png)

## <a name="sharing-sales-and-support-materials-with-csp-partners"></a>Sdílení materiálů pro prodej a podporu pomocí partnerů CSP

Pokud chcete, aby partneři v programu Cloud Solution Provider efektivně představovali vaši nabídku a mohli se zapojit do vaší organizace, posílejte prodejní a podpůrné materiály, které budou k dispozici pro prodejce. Tyto prostředky nebudou zveřejněny zákazníkům v prodejní místa na webu Marketplace.

Jak je vidět níže, v portál partnerů cloudu můžou vydavatelé zadat adresu URL, která hostuje relevantní marketingové materiály a kontaktní informace kanálu k kanálu CSP:

![Informace o prorukou CSP](media/marketplace-publishers-guide/cpp-csp-information.png)
