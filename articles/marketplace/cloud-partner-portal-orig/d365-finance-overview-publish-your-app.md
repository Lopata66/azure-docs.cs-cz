# <a name="bring-your-microsoft-dynamics-365-business-central-app-into-microsoft-appsource"></a>Přeneste aplikaci Centrální obchodní aplikace Microsoft Dynamics 365 do Microsoft AppSource

K dispozici je spousta výhody používání Dynamics 365 Business Central jako platforma pro vývojáře aplikací:

-   Obohaťte Dynamics 365 Business Central, ověřené Microsoft online řešení s svou odbornost.
-   Využijte značky Dynamics 365 – značky, kterou milionům uživatelů znáte a kterým důvěřujete.
-   Oslovte více zákazníků pro své obchodní aplikace pomocí Microsoft AppSource.
-   Dosažení více z platformy, která nabízí moderní prostředí a nabízí škálování.
-   Vytvoření balíčku se firemní aplikace, jako je například Microsoft PowerApps, Microsoft Flow, Power BI, Cortana Intelligence a mnoho dalších.

Zapojit vaše centrální obchodní aplikace do Microsoft AppSource:

1.  Vytváření účtů.
2.  Spojte se s námi o svůj nápad aplikace.
    - Vývoj technické aspekty vaší aplikace
    - Vývoj marketingové aspekty vaší aplikace
4.  *Publikování aplikace.*
