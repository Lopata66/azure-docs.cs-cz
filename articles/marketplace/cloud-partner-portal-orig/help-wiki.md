# <a name="dynamics-365-for-operations-help-wiki"></a>Dynamics 365 pro operace pomáhají Wiki

Microsoft Dynamics 365 pro operace se Microsoft\'s obchodní aplikací pro podniky. Umožňuje uživatelům rychle dělejte chytřejší rozhodnutí pomocí inteligentních uživatelského rozhraní, která poskytuje přehledy v reálném čase a data. Umožňuje firmám k transformaci a inovace poskytováním prověřené obchodní logiky, která umožňuje změnit návrh jejich obchodní procesy rychleji. Firmám poskytuje možnost rozšířit svým vlastním tempem a uzavírat obchody téměř odkudkoliv, kdykoliv, což jim umožní škálovat své operace globálně pro obchodní potřeby.

Tato uživatelská příručka vám můžou pomoct při tom, jak provést Dynamics 365 pro operace pracovat pro vaši firmu.

Zobrazit [Dynamics 365 pro operace pomáhají Wiki](https://docs.microsoft.com/dynamics365/operations/) pro další informace.
