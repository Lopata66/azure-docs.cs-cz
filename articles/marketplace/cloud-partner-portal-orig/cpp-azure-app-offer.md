## <a name="publishing-process"></a>Proces publikování

Tyto články vás provedou procesem vytvoření a publikování spravované aplikace Azure:

1. [Zaregistrovat účet na webu Microsoft Developer Center](./cloud-partner-portal-dev-center-accounts-registration.md) stát schválení pro publikování na webu Azure Marketplace.
2. [Publikovat novou nabídku aplikací Azure](./cloud-partner-portal-managed-app-publish.md) na webu Azure Marketplace.
3. [Ujistěte se, vaši nabídku live](./cloud-partner-portal-managed-app-offer-go-live.md)!
