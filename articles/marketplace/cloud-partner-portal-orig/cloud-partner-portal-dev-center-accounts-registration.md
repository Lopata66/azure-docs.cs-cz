<a name="create-a-microsoft-developer-account"></a>Vytvoření účtu Microsoft Developer
====================================

Tento článek popisuje, jak se stát schválené Microsoft Developer pro publikování na webu Azure Marketplace.

## <a name="create-a-microsoft-account"></a>Vytvořit účet Microsoft

Chcete-li zahájit proces publikování, budete potřebovat k dokončení **Microsoft Developer Center** registrace. Budete používat stejný účet registrované na **[portál partnerů cloudu](https://cloudpartner.azure.com/)** zahájíte proces publikování.

### <a name="general-account-guidelines"></a>Účet pro obecné pokyny

Doporučujeme vám, že máte jenom jeden účet Microsoft pro svoje nabídky na webu Azure Marketplace. Tento účet nesmí být specifické pro služeb nebo nabídky.

Adresa, která tvoří uživatelské jméno by měly být ve vaší doméně a řídit váš IT tým. Pomocí tohoto účtu se má počítat publikování souvisejících aktivit.

>[!WARNING]
>Slova, jako je "Azure" a "Microsoft" nejsou podporovány pro registraci účtu Microsoft. Vyhněte se použití těchto slov a dokončete vytváření účtů a proces registrace.

### <a name="company-account-guidelines"></a>Pokyny pro účet společnosti

Pokud pro přístup k účtu po přihlášení pomocí účtu Microsoft, které se otevřelo účet bude potřebovat víc než jedna osoba, postupujte podle následujících pokynů.

>[!IMPORTANT]
>Umožňuje několika uživatelům přístup ke svému účtu Dev Center doporučujeme pomocí Azure Active Directory pro přiřazení rolí pro jednotlivé uživatele. Když se přihlásíte pomocí svých osobních bude moct účet přihlašovacích údajů Azure AD. Další informace najdete v tématu [spravovat uživatele účtu](https://docs.microsoft.com/windows/uwp/publish/manage-account-users).

-   Vytvoření účtu Microsoft pomocí e-mailovou adresu, která patří do vaší společnosti\'s domény, ale ne k někdo. Například windowsapps\@fabrikam.com.
-   Omezit přístup k tomuto účtu Microsoft na nejmenší možný počet vývojářů.
-   Nastavení firemního e-mailu distribučního seznamu, který zahrnuje všem uživatelům, kteří potřebují přístup k účtu pro vývojáře a přidat tuto e-mailovou adresu ke svým bezpečnostním údajům. To umožňuje všichni zaměstnanci v seznamu a získat zabezpečovací kódy, pokud nepotřebujete spravovat bezpečnostní údaje účtu Microsoft. Pokud nastavení distribučního seznamu není vhodná, vlastník jednotlivé e-mailový účet se musí být dostupné pro přístup k a sdílet kód zabezpečení po zobrazení výzvy (například při přidání nové bezpečnostní údaje k účtu nebo když musí být přistupovat z nového zařízení.)
-   Přidáte telefonní číslo společnosti, který nevyžaduje, aby rozšíření a je přístupné členům týmu klíče.
-   Obecně platí mají vývojáři používat důvěryhodná zařízení pro přihlášení k účtu vaší společnosti pro vývojáře. Všichni členové týmu klíče by měl mít přístup k těmto důvěryhodná zařízení. Tím se sníží potřebu zabezpečovací kódy, které má být odeslána při přístupu k účtu.
-   Pokud je potřeba povolit přístup k účtu z nedůvěryhodné počítače, omezte, která přístup k maximálně pět vývojářů. V ideálním případě tyto vývojáře by měla přístup k účtu z počítače, které sdílejí stejné zeměpisné a síťové umístění.
-   Často zkontrolujte vaše [informace o zabezpečení vaší společnosti](https://account.live.com/proofs/Manage) k Ujistěte se, že je aktuální.

>[!IMPORTANT]
>Váš účet vývojáře by měla přístupná především z důvěryhodných počítačů. To je důležité, protože není omezený počet kódů generovaných každý účet za týden. Taky umožňuje bezproblémové přihlašování.
>
>Další informace najdete v tématu [zabezpečení a další vývojářské účet pokyny](https://msdn.microsoft.com/windows/uwp/publish/opening-a-developer-account#additional-guidelines-for-company-accounts).

### <a name="to-create-a-microsoft-account"></a>Chcete-li vytvořit účet Microsoft

1.  Otevřete nový Chrome Incognito nebo Internet Explorer se službou InPrivate relaci procházení k zajištění, že nejste přihlášení k existujícímu účtu.
2.  Zaregistrujte se e-mailu (pomocí předchozích pokynů) jako účet Microsoft prostřednictvím tohoto [odkaz](https://signup.live.com/signup.aspx). Dokončení registrace následující pokyny:

    - Při registraci vašeho účtu jako účet Microsoft, je třeba zadat platné telefonní číslo pro systém na poslat ověřovací kód účtu jako textovou zprávu nebo automatizovaného hovoru.
    - Při registraci vašeho účtu jako účet Microsoft, budete muset zadat id platné e-mailu pro příjem automatizovaných e-mailu pro ověření účtu.
    - Ověřit e-mailovou adresu odeslání do distribučního seznamu.

    Teď jste připraveni používat nový účet Microsoft na webu Microsoft Developer Center.

## <a name="register-your-account-in-microsoft-developer-center"></a>Zaregistrujte svůj účet ve službě Microsoft Developer Center

Microsoft Developer Center, se použije k registraci informací o společnosti jednou. Žadateli o registraci musí být platný zástupce společnosti a musíte zadat svoje osobní údaje jako způsob, jak ověřit svoji identitu. Osoba registrace musíte použít účet Microsoft, jež jsou sdílena pro společnosti, **a portál Cloud Partner musí použít stejný účet.** Měli byste zkontrolovat zajistit, aby že vaše společnost ještě nemá účet Microsoft Developer Center před dalším pokusem ho vytvořit. Během procesu budeme shromažďovat informace o adrese společnosti, informace o bankovním účtu a daňové údaje. Ty se obvykle dají získat od finančních nebo podnikových kontaktů.

>[!IMPORTANT]
>Pokud chcete po absolvování různých fází nabídky vytváření a nasazování, je třeba provést následující součásti profil pro vývojáře.

| Registrace společnosti  | Musí mít         | Musí mít     | Musí mít                             | Musí mít             |

>[!NOTE]
>Používání vlastní licence (BYOL) se podporuje jenom pro virtuální počítače a je považován za bezplatné nabídky.

### <a name="register-your-company-account"></a>Zaregistrujte svůj účet společnosti

1. Otevřete nový Internet Explorer InPrivate nebo Incognito Chrome relaci procházení k zajištění, že nejste přihlášení k osobního účtu.

2. Přejděte [Windows Dev Center](https://dev.windows.com/registration?accountprogram=azure) sami zaregistrovat jako prodejce. Přečtěte si prosím následující důležitá Poznámka: než budete pokračovat.

   ![Ověření účtu Microsoft](./media/cloud-partner-portal-create-dev-center-registration/seller-dashboard-verify.jpg)

    >[!IMPORTANT]
    >Zajistěte, aby byl na e-mailu id nebo distribuční seznam (distribuční seznam doporučujeme odebrat závislost jednotlivců) který budete používat pro registraci v Centru pro vývojáře v nejprve zaregistrovat jako účet Microsoft. Pokud ne, pak zaregistrujte prosím pomocí tohoto odkazu. Navíc nelze použít žádné id e-mailu v rámci domény společnosti Microsoft pro vývojáře pro registraci. "

   ![Přihlašovací dev center](./media/cloud-partner-portal-create-dev-center-registration/seller-dashboard-login.jpg)

3. Spusťte Průvodce "Pomozte nám chránit váš účet", chcete-li ověřit svoji identitu pomocí telefonní číslo nebo e-mailovou adresu.

4. Informace o registraci účtu, vyberte vaše **účtu země/oblast** z rozevíracího seznamu a pak vyberte **Další**.

   ![Vyberte zemi nebo oblast](./media/cloud-partner-portal-create-dev-center-registration/imgRegisterCo_04.png)

    >[!WARNING]
    >"Zákazník z" zemích nebo oblastech: Pokud chcete prodávat vaše služby na Azure Marketplace, musí být z jednoho z schválené "prodej z" zemích nebo oblastech je znázorněno v rozevíracím seznamu registrovaných entity. Toto omezení je platební a daňové z důvodů. Další informace najdete v zásadách pro účast na webu Marketplace.

5. Vyberte **společnosti** jako "Účtu typu" a pak vyberte **Další**.

    >[!IMPORTANT]
    >Pro lepší pochopení typů účtů a rozhodnout, jaký typ je pro vás nejvhodnější, zobrazit typy účtů stránky, umístění a poplatky uvedené v následující snímek obrazovky.

    ![Typy účtů pro prodejci](./media/cloud-partner-portal-create-dev-center-registration/imgRegisterCo_05.png)

6. Zadejte **zobrazovaný název vydavatele**. Toto je obvykle název vaší společnosti.

    >[!NOTE]
    >Zobrazovaný název vydavatele zadaný v Centru pro vývojáře se nezobrazí v Tržišti Azure Marketplace po nabídky je uveden. Ale tyto informace potřebné k dokončení procesu registrace.

7. Zadejte **kontaktní informace** pro ověření účtu.

    >[!IMPORTANT]
    >Je nutné zadat přesné kontaktní informace, protože ho se použije v našich proces ověření pro vaši společnost, chcete-li možné ji schválit v Centru pro vývojáře. "

8. Zadejte kontaktní informace **společnosti schvalovatel**. Schvalovatel společnosti je osoba, která můžete ověřit, že máte oprávnění k vytvoření účtu služby v Centru pro vývojáře jménem svojí organizace. Po zadání těchto informací, vyberte **Další** přesunete **platby části**.

    ![Identifikujte společnosti schvalovatele](./media/cloud-partner-portal-create-dev-center-registration/imgRegisterCo_08.png)

9. Zadejte platební údaje pro váš účet. Pokud máte propagační kód, které zahrnují náklady na registrace, zadejte, který tady. V opačném případě poskytnete informace o platební kartu (nebo PayPal v podporované trhy). Vyberte **Další** do konečné **revize**.

   ![Registrace plateb](./media/cloud-partner-portal-create-dev-center-registration/imgRegisterCo_09.png)

10. Projděte si informace o vašem účtu a potvrďte, že je všechno správně. Přečtěte si a přijměte podmínky a ujednání o [smlouvy vydavatele Microsoft Azure Marketplace](https://go.microsoft.com/fwlink/?LinkID=699560). Zaškrtněte políčko označující vám přečetli a přijali tyto podmínky.

11. Vyberte **Dokončit** potvrďte svou registraci. Na e-mailovou adresu přijde potvrzovací zpráva.

12. Pokud jste v úmyslu publikovat jenom bezplatné nabídky, vyberte [přejít na portál Cloud Partner](https://cloudpartner.azure.com/) a přeskočit k "Registraci účtu na portálu cloud partner" v tomto článku.

### <a name="commercial-offers"></a>Obchodní nabídky

Pokud se chystáte publikovat obchodní nabídky, jako jsou nabídky virtuálních počítačů pomocí hodinu fakturační model, budete muset poskytnout daňové a bankovní informace. Na to, přihlaste se do vašeho účtu středisko pro vývojáře a vyberte **aktualizovat informace o vašem účtu**. Postupujte podle pokynů v následující části "Přidání bankovnictví a daň informací o".

>[!IMPORTANT]
>Nebude moct obchodní nabídky přejít do produkčního prostředí bez zadání bankovním účtu a daňové údaje.

Pokud chcete aktualizovat banka a daňové údaje později, pak můžete přeskočit na "Registraci účtu na portálu cloud partner" v tomto článku.

>[!NOTE]
>Doporučujeme vzhledem k tomu může zabrat určitý čas ověřit informace o daních poskytuje bankovním účtu a daňové údaje co nejdříve.

### <a name="add-banking-and-tax-information"></a>Přidat bankovnictví a daňové údaje

Pokud chcete publikovat obchodní nabídky k nákupu, budete muset přidat platební a daňové údaje a odeslat je k ověření v Centru pro vývojáře.

**K poskytnutí informací bank**

1.  Přihlaste se k [Microsoft Developer Center](https://dev.windows.com/registration?accountprogram=azure) s vaším účtem Microsoft.
2.  Vyberte **výběr účtu** v nabídce vlevo v části **zvolte způsob platby**vyberte **bankovním účtu** nebo **PayPal**.

    >[!NOTE]
    >Pokud budete mít obchodní nabídky, které zákazníci zakoupit na webu Marketplace, jedná se o účet, ve kterém obdržíte výběr pro tyto nákupy.
3.  Zadejte údaje o a poté vyberte **Uložit**.

    >[!IMPORTANT]
    >Pokud je potřeba aktualizovat nebo změnit váš výběr účtu, postupujte podle předchozích kroků nahradit aktuální informace o nové informace.
    >
    >Změna účtu výběr zpozdit vaší platby podle až jeden cyklus platby. Toto zpoždění dochází, protože budeme potřebovat ověřit změnu účtu stejně jako při prvním nastavování účtu výběr. Budete stále platby za celou částku po váš účet byl ověřen; všechny platby z důvodu pro aktuální platební cyklu přibudou další.

4.  Vyberte **Další**.

**K poskytnutí informací daně**

1.  Přihlaste se k [Microsoft Developer Center](https://dev.windows.com/registration?accountprogram=azure) pomocí svého účtu Microsoft (v případě potřeby).
2.  V nabídce vlevo vyberte **daňové profilu**.
3.  Na **nastavit váš formulář** stránky:
    - Vyberte zemi nebo oblast, kde máte trvalý rezidence.
    - Vyberte zemi nebo oblast, kde podržte primární citizenship.
    - Vyberte **Další**.
4.  Zadejte své daňové údaje a pak vyberte **Další**.

>[!WARNING]
>Nebudete moci nasdílet do vaší obchodní nabídky do produkčního prostředí bez zadání bankovním účtu a daňové údaje ve vašem účtu Microsoft Developer Center.

## <a name="register-your-account-in-the-cloud-partner-portal"></a>Zaregistrujte svůj účet na portálu cloud partner

Můžete použít [portál partnerů cloudu](https://cloudpartner.azure.com/) k publikování a správě vašich nabídek.

1.  Otevřete nový Chrome Incognito nebo Internet Explorer se službou InPrivate relaci procházení k zajištění, že nejste přihlášení k osobního účtu.
2.  Přejděte na [portál partnerů cloudu](https://cloudpartner.azure.com/).
3.  Pokud jste nový uživatel a přihlásí se do [portál partnerů cloudu](https://cloudpartner.azure.com/) poprvé, pak musíte se přihlásit pomocí stejné id e-mailu, který je registrovaný pomocí účtu Dev Center. Tím se zajistí, že jsou propojeny účtu Dev Center a účet portálu partnerů cloudu k sobě navzájem.

Později přidáte ostatní členy společnost, kteří pracují na aplikaci. Můžete je přidat jako přispěvatele nebo vlastníci portál partnerů cloudu podle kroků v další části.

Pokud jsou přidány jako přispěvatele nebo vlastníka do portálu portál cloud partner, pak se můžete přihlásit pomocí vašeho účtu.

>[!TIP]
>Zásady účasti, jsou popsané na webu Azure.
