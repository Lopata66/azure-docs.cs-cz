<a name="seller-insights-definitions"></a>Definice Insights prodejce
=======================

Následující tabulka obsahuje definice pro celou řadu termíny používané ve službě Insights prodejce.

|  **Termín**                 |  **Definice**                                                                                                                              |
|  --------------------------------------------    |  ---------------------------------------------------------------------------------------------------------------------------------           |
| **Typ licence pro Azure**                               | Typ licenční smlouvy k nákupu Azure použít zákazníky.  Označované také jako kanál.                                                  |
| **Typ licence Azure: Cloud Solution Provider**      | Azure a nabídky Marketplace přes svého poskytovatele Cloud Solution Provider, který funguje jako svého prodejce opatřuje finanční koncového zákazníka.                 |
| **Typ licence Azure: Enterprise**                   | Azure a nabídky Marketplace prostřednictvím smlouvy Enterprise, podepsaný přímo s Microsoftem opatřuje finanční koncového zákazníka.                  |
| **Typ licence Azure: Organizace prostřednictvím prodejce**  | Azure a nabídky Marketplace prostřednictvím prodejce, který usnadňuje jejich smlouvy Enterprise s Microsoftem opatřuje finanční koncového zákazníka.     |
| **Typ licence Azure: Průběžné platby**                | Azure a nabídky Marketplace prostřednictvím smlouvy "Platit průběžně", podepsaný přímo s Microsoftem opatřuje finanční koncového zákazníka.                |
| **Poplatek za dobu (kopie)**                              | Určená pro zákazníky, v *zákazníka měny* (kopie) pro účely fakturace.                                                                 |
| **Poplatek za dobu (PC)**                               | Určená pro zákazníky, v *výběr měny* (PC).                                                                      |
| **Datum poplatku**                                      | Datum, kdy byl zákazníkovi poplatky počítají (obvykle následuje období využití).                                             |
| **Název Instance cloudu**                              | Cloud od Microsoftu v němž došlo k nasazení virtuálního počítače.                                   |
| **Název Instance cloudu: Azure Global**                | Veřejné globálního cloudu společnosti Microsoft.                           |
| **Název Instance cloudu: Azure Government**        | Cloud Microsoftu určené pro státní správu pro jednu z následujících zemí: Čína, Německo nebo Spojených států amerických.                           |
| **Stav shromažďování**  | Pro konkrétní poplatek, nejnovější stav životního cyklu fakturace a kolekce.  Stav může být například kolekce v průběhu shromáždí, případně refundaci.                  |
| **Základní velikost**  | Počet virtuálních jader, které jsou prezentované podle virtuálního počítače, které nabídky je spuštěn na.               |
| **Město zákazníka**  | Název města zákazníka k dispozici. Může to být jiný než města v rámci předplatného Azure zákazníka.  |
| **Jazyk odběratele komunikace**        | Jazyk upřednostňuje zákazníka pro komunikaci.                                  |
| **Název společnosti zákazníka**                  | Název společnosti poskytované zákazníka. Může to být jiný než města v rámci předplatného Azure zákazníka.                                  |
| **Zemi zákazníka**                       | Název země/oblasti k dispozici zákazník.  Může to být jiný než zemi/oblast v rámci předplatného Azure zákazníka.                               |
| **Zákazník měny (kopie)**                 | Měna preferované zákazníkem pro ceny a fakturace.                            |
| **Zákaznických e-mailů**                         | E-mailovou adresu poskytované koncového zákazníka.  To může být jiná než e-mailovou adresu v rámci předplatného Azure zákazníka.                   |
| **Křestní jméno odběratele**                              | Křestní jméno zákazníka k dispozici.  Může to být jiný než název zadaný v rámci předplatného Azure zákazníka.                    |
| **Příjmení odběratele**                               | Příjmení poskytované zákazníka.  Může to být jiný než název zadaný v rámci předplatného Azure zákazníka.                    |
| **ID zákazníka**                               | Jedinečný identifikátor přiřazený pro zákazníka.  Zákazník může mít nula nebo více předplatných Azure Marketplace.                    |
| **Typ platby zákazníků**                            | Typ platební nástroj využívaných zákazníkem.  Například to může být karty, faktury nebo elektronické zaškrtnutí.                             |
| **Poštovní směrovací číslo zákazníka**                             | PSČ zákazníka k dispozici.  Může to být jiný než poštovní směrovací číslo místa k dispozici v rámci předplatného Azure zákazníka.                                                                 |
| **Stav zákazníka**                                   | Stav (adresa), poskytuje zákazníka.  Může to být jiný než poslední název zadaný v rámci předplatného Azure zákazníka.                                                                 |
| **Datum získání**                                    | První datum zákazník jakékoli využili nabídku nákupu někým jiným než vámi.                 |
| **Datum ztráty**                                        | Datum posledního zákazník poslední všechny nabídky, které jste dříve zakoupili bylo zrušeno.                                                         |
| **Odhadovaný poplatek rozšířené (kopie)**                   | Odhadované rozšířené poplatek za množství jednotek využití pro danou skladovou jednotku (v místní měně zákazníka). Tato hodnota nemusí být přesné z důvodu chyby zaokrouhlení nebo zkrácení.                            |
| **Odhadovaný poplatek rozšířené (PC)**                   | Odhadované rozšířené poplatek za množství jednotek využití pro danou skladovou jednotku, podle cizích měn převod na datum využití se počítá (v místní měně vydavatele). Tato hodnota nemusí být přesné z důvodu chyby zaokrouhlení nebo zkrácení.                  |
| **Odhadované výběr (PC)**                            | Odhadované poplatky za množství jednotek využití pro danou skladovou jednotku, podle cizích měn převod na datum, kdy využití se počítá (v místní měně vydavatele). Tato hodnota nemusí být přesné z důvodu chyby zaokrouhlení nebo zkrácení.                        |
| **Odhadovaná cena (PC)**                             | Odhadovaná cena za jednotku využití pro danou skladovou jednotku, podle cizích měn převod na datum využití se počítá (v místní měně vydavatele). Tato hodnota nemusí být přesné z důvodu chyby zaokrouhlení nebo zkrácení.                                             |
| **Je nového zákazníka**                                  | Hodnota vám dá vědět, pokud je nový zákazník používání nebo zakoupení jeden nebo více z Marketplace nabízí první (nebo nemusíte).  Hodnota bude "Ano", pokud je stejný kalendářního měsíce pro "Získat datum".  Hodnota bude "Ne", pokud zákazník zakoupila některou z nabídek před hlášené kalendářního měsíce.                                                       |
| **Je ve verzi Preview SKU**                                  | Hodnota vám dá vědět, pokud jste označili jako "preview" SKU. Pokud odpovídajícím způsobem označená SKU a předplatná pouze Azure autorizovaný vy můžete nasadit a použít tuto bitovou kopii, budou hodnotu "Yes".  Hodnota bude "Ne", pokud skladová položka nebyla identifikována jako "verze preview".                                          |
| **Je propagační kontakt vyjádřit výslovný souhlas**                   | Hodnota vám dá vědět, pokud zákazník proaktivně výslovného souhlasu pro propagační kontaktu vydavatele. V tuto chvíli jsme nejsou nabízí ten samý možnost pro zákazníky, aby nám sdělily "Ne" na panelu. Po nasazení této funkce začneme odpovídajícím způsobem aktualizuje.                                          |
| **Typ licence Marketplace**                         | Metodu fakturace nabídky Marketplace.                             |
| **Typ licence Marketplace: Fakturuje v rámci Azure**   | Společnost Microsoft je agenta pro tuto nabídku Marketplace a účtuje zákazníkům vaším jménem. (Průběžné platby platební karty nebo faktury Enterprise)       |
| **Typ licence Marketplace: Používání vlastní licence** | Virtuální počítač vyžaduje licenční klíč zákazníka k nasazení k dispozici. Microsoft není fakturovat zákazníků pro jejich využití této nabídky na webu Marketplace. |
| **Typ licence Marketplace: Zdarma**                   | Nabídka Marketplace byl nakonfigurován jako zdarma pro všechny uživatele. Microsoft není fakturovat zákazníků pro jejich použití v rámci této nabídky Marketplace.    |
| **Typ licence Marketplace: Microsoft jako prodejce**  | Společnost Microsoft se pro tuto nabídku Marketplace svého prodejce.                               |
| **Název společnosti předplatného Marketplace**  | Název společnosti zákazníka v rámci svého předplatného Azure k dispozici.                           |
| **ID předplatného Marketplace**            | Jedinečný identifikátor spojený s předplatným Azure zákazníka použít k nákupu nabídky Marketplace.  Dříve bylo GUID předplatného Azure.  V některých případech jako SAAS pořadí vyžádáním žádné předplatné.           |
| **Microsoft Fee (CC)**                               | Microsoft poplatky za transakce v měně zákazníka.                              |
| **Název nabídky**                                | Název nabídky Marketplace.                                                             |
| **Typ nabídky**                                | Typ nabídky webu Microsoft Marketplace.  Například může být spravované aplikace, virtuální počítač licence nebo kontejneru.                           |
| **Datum zrušení objednávky**                         | Datum, kdy byla zrušena pořadí Marketplace.                                                       |
| **ID objednávky**                                  | Jedinečný identifikátor objednávky zákazníků služby webu Marketplace.  Nabídky podle využití virtuálního počítače nejsou přidružené objednávky.                 |
| **Datum nákupní objednávky**                       | Datum vytvoření objednávky Marketplace.                                                        |
| **Množství objednávky**                       | Množství objednávek se stejným SKU spojené s předplatným Azure.  Pro SAAS objednávky představuje počet licencí, které jsou přidružené k této instanci objednávky.                                            |
| **Stav objednávky**                              | Stav pořadí Marketplace v době, kdy data poslední aktualizaci.  Například pořadí může být aktivní nebo zrušila.             |
| **Stav objednávky: Aktivní**                           | Zákazník koupil objednávky a nebyl zrušen jejich pořadí.   |
| **Stav objednávky: Bylo zrušeno**                        | Zákazník jste dříve zakoupili pořadí a následně bylo zrušeno jejich pořadí.        |
| **Výběr částka (PC)**                             | Velikost placené vám v preferované výběr měny (počítače).                                       |
| **Výběr měny (PC)**                           | Měna použitá pro vaše výplaty.                                                                |
| **Výběr data**                                    | Datum platby byla vyslána žádost od Microsoftu pro váš zvolený finanční instituce.                                                                         |
| **Výběr stavu**                                  | Stav výběr v době, kdy data poslední aktualizaci.                                   |
| **Výběr stavu: V průběhu**               | Transakce není připraven pro výběr. (Další informace najdete v tématu stav kolekce)                             |
| **Výběr stavu: Hotovostní platba**                          | Transakce byla zahrnuta v posledních výpočtu výběr. Placené kladné hodnoty a Celková dlužná částka se odečíst od záporné hodnoty. |
| **Výběr stavu: Nadcházející výběr**                   | Transakce je připravený pro výběr a zahrne do dalšího výpočtu dostupný výběr.                      |
| **Cena (kopie)**                                   | Cena za jednotku využití pro danou skladovou jednotku (v místní měně zákazníka).                                                                  |
| **Poskytovatele e-mailu**                                 | E-mailovou adresu poskytovatele účastnící se vztahu mezi společnostmi Microsoft a koncového zákazníka. Pokud má zákazník organizace prostřednictvím prodejce, bude prodejce.  Pokud se jedná o Cloud Solution Provider (CSP), bude poskytovatel CSP.                                                                 |
| **Název poskytovatele**                                 | Název zprostředkovatele účastnící se vztahu mezi zákazníky Microsoftu a end. Pokud má zákazník organizace prostřednictvím prodejce, bude prodejce.  Pokud se jedná o Cloud Solution Provider (CSP), bude poskytovatel CSP.                                                                 |
| **Identifikátor URI prostředku**                                  | Cestu, kam se nasadí artefaktů nasazení.                                            |
| **SKU**                                           | Název skladové položky definované během publikování. Nabídka může mít mnoho skladové položky, ale SKU může být přidružen pouze jediné nabídky.              |
| **Typ SKU fakturace**                                 | Metodu fakturace SKU.                                                                                                             |
| **Typ SKU fakturace: Používání vlastní licence**         | Virtuální počítač nebo službu vyžaduje licenční klíč zákazníka k nasazení k dispozici. Microsoft není fakturovat zákazníků pro jejich používání těchto nabídek na webu Marketplace.   |
| **Typ SKU fakturace: Microsoft As Reseller**          | Společnost Microsoft se pro tuto SKU svého prodejce.   |
| **Typ SKU fakturace: Zdarma**                           | SKU je nakonfigurovaný to bude zdarma pro všechny uživatele. Microsoft není fakturovat zákazníků pro jejich použití pro tuto skladovou jednotku.                           |
| **Typ SKU fakturace: Placené**                           | Společnost Microsoft je agenta pro tuto SKU a účtuje zákazníkům vaším jménem. (Průběžné platby platební karty nebo faktury Enterprise)                   |
| **Typ SKU fakturace: Zkušební verze**                          | Zákazník je v jejich zkušebního období a se převede na placené, je-li zrušit nebo odstranit.                                         |
| **Částka daně (kopie)**                                  | Částka daně použijí ve fakturaci za zákazníka v měně zákazníka (kopie).                                                               |
| **Datum transakce**                                 | Datum transakce se zaznamenávají do vytváření sestav výběr.                                                                           |
| **Typ transakce**                               | Typ transakce, která hlásí.  Například to může být poplatek, náhrada nebo úpravu výběr.                                                                                        |
| **Zkušební verze koncové datum**                                 | Data zkušební období pro tuto objednávku se ukončí nebo skončila.                                                                           |
| **Datum využití**                                       | Došlo k používání zákazníky datum.                                                                                                             |
| **Množství využití**                                       | Využití ohlášených zákazníky SKU. Pro Image virtuálních počítačů představují záznamů využití ohlášené období pro tento virtuální počítač velikost a SKU.                                               |
| **Referenční informace k využití**                                       | Identifikátor pro jeden nebo více dnů využití ze strany zákazníků pro danou skladovou Položku přidruženou ke položka v sestavě výběr.                                               |
| **Typ využití**                                       | Popis měří využití. (Normalizovaná využití nebo nezpracované využití)                                                              |
| **Typ použití: Normalizovaná využití**                | Hodin využití normalizovány na účet pro počet jader virtuálního počítače, které jsou zahrnuté ve využití: [počet jader virtuálního počítače] x [hodin nezpracovaná používání]. Virtuální počítače označený jako "SHAREDCORE" jako násobitel [počet jader virtuálního počítače] použít 1/6 (nebo 0.1666).                                        |
| **Typ použití: Nezpracovaná využití**                        | Hodin využití pro vaši nabídku Marketplace.  Zobrazí se z hlediska hodin "kalendáře".  Je počet virtuálních počítačů čas už běží.                           |
| **Využití jednotek**                                   | Jednotka měření pro uvedené využití. Virtuální počítače jsou vždy prováděno pomocí hodinové jednotky měření.                               |
| **Velikost virtuálního počítače**                                       | Představuje velikost hardwaru virtuálního počítače, v souladu s nabídku Azure. Mezi příklady patří `Basic_A0`, `Standard_A11`, `Standard_D12`, a `Standard_G4`.   |
