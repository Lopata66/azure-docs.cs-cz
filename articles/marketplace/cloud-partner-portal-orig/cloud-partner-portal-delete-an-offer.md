<a name="delete-an-offer-or-sku-from-azure-marketplace"></a>Odstranění nabídky nebo skladovou Položku z Azure Marketplace
==========================================

Z různých důvodů se můžete rozhodnout odebrat svou nabídku z webu Marketplace. Odebrání nabídky zajišťuje, že noví zákazníci už vaši nabídku nebudou moci zakoupit ani nasadit, ale nemá žádný vliv na stávající zákazníky.
Ukončení nabídky je proces ukončení služby a/nebo licenční smlouvy mezi vámi a vašimi stávajícími zákazníky. Pokyny a zásady týkající se odebrání a ukončení nabídky se řídí [smlouvy vydavatele Microsoft Marketplace](https://go.microsoft.com/fwlink/?LinkID=699560) (viz oddíl
7) a [zásadách pro účast](https://azure.microsoft.com/support/legal/marketplace/participation-policies/) (viz část 6.2). Tento článek hovoří o tom, jiné podporované odstranit scénáře a kroky, které si můžete pro ně.

<a name="delete-a-live-sku-from-azure-marketplace"></a>Odstranit živé skladovou Položku z Azure Marketplace
----------------------------------------

Živé SKU můžete odstranit z Azure Marketplace pomocí následujících kroků:

1.  Přihlaste se k [portál partnerů cloudu](https://cloudpartner.azure.com/).

2.  Vyberte z nabídky **všechny nabídky** kartu.

3.  V podokně na levé straně obrazovky vyberte **SKU** kartu.

4.  Vyberte jednotku SKU, kterou chcete odstranit a klikněte na tlačítko Odstranit pro dané skladové jednotce.

5.  [Znovu publikujte](./cloud-partner-portal-make-offer-live-on-Azure-Marketplace.md) nabídky na webu Azure Marketplace.

Po tato nabídka je za provozu na webu Azure Marketplace, které skladová jednotka Potřebuje budou odstraněny z Azure Marketplace a webu Azure portal.

<a name="roll-back-to-a-previous-sku-version"></a>Vrátit zpět na předchozí verzi SKU
----------------------------------

Aktuální verze za provozu skladovou Položku z Azure Marketplace můžete odstranit pomocí následujících kroků. Po dokončení procesu SKU bude vrácena zpět jeho předchozí verzi.

1.  Přihlaste se k [portál partnerů cloudu](https://cloudpartner.azure.com/).

2.  Vyberte z nabídky **všechny nabídky** kartu.

3.  V podokně na levé straně obrazovky vyberte **SKU** kartu.

4.  Odstranit nejnovější **disková verze** ze seznamu publikovaných diskových verzí.

5.  [Znovu publikujte](./cloud-partner-portal-make-offer-live-on-Azure-Marketplace.md) nabídky na webu Azure Marketplace.

Až tato nabídka je za provozu na webu Azure Marketplace, aktuální verze uvedené SKU se odstraní z Azure Marketplace a na webu Azure portal.
Skladovou Položku, bude vrácena zpět jeho předchozí verzi.

<a name="delete-a-live-offer"></a>Odstranit živé nabídku
-------------------

Existují různé aspekty, které je třeba se postarat v případě žádost o odebrání za provozu nabídky. Postupujte podle kroků níže zobrazíte pokyny od týmu podporu k odebrání za provozu nabídky z Azure Marketplace:

1.  Lístek podpory pomocí tohoto [odkaz](https://go.microsoft.com/fwlink/?linkid=844975), nebo v pravém horním rohu portál partnerů cloudu klikněte na podporu.

2.  Vyberte svůj typ konkrétní nabídky v **typ problému** seznamu a **odebrání publikované nabídky** v **kategorie** seznamu.

3.  Odešlete žádost.

Tým podpory vás provede odstranění nabídky.

>[!NOTE]
>Odstraňuje se skladová položka/nabídka nebude mít vliv aktuální nákupy SKU a nabídce. Tyto SKU nabídky budou nadále fungovat jako předtím. Však nebudou k dispozici pro všechny nové nákupy do budoucna.
