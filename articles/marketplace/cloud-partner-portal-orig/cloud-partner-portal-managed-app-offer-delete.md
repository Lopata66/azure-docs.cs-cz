<a name="delete-an-offersku-from-azure-marketplace"></a>Odstranit nabídka/SKU z Azure Marketplace 
==========================================

Z různých důvodů se můžete rozhodnout odebrat svou nabídku z webu Marketplace. Noví zákazníci už mohou koupit a nasadit vaši nabídku, ale stávající zákazníky to neovlivní.
Ukončení nabídky je proces ukončení služby a/nebo licenční smlouvy mezi vámi a vašimi stávajícími zákazníky. Pokyny a zásady týkající se odebrání a ukončení nabídky se řídí [smlouvy vydavatele Microsoft Marketplace](http://go.microsoft.com/fwlink/?LinkID=699560) (viz oddíl
7) a [zásadách pro účast](https://azure.microsoft.com/support/legal/marketplace/participation-policies/) (viz část 6.2). Odstraňte jiné podporované scénáře a jsou popsané kroky, které můžete provést.

<a name="delete-a-live-sku-from-azure-marketplace"></a>Odstranit živé skladovou Položku z Azure Marketplace 
----------------------------------------

Živé skladovou Položku z Azure Marketplace můžete odstranit pomocí následujících kroků:

1.  Přihlaste se k [portál partnerů cloudu](http://cloudpartner.azure.com).
2.  Na kartě všechny nabídky vyberte vaši nabídku.
3.  V podokně na levé straně obrazovky vyberte kartu skladové položky.
4.  Vyberte SKU, kterou chcete odstranit a klikněte na tlačítko Odstranit pro dané skladové jednotce.
5.  Opakované publikování nabídky na webu Azure Marketplace.

Po tato nabídka je za provozu na webu Azure Marketplace, které skladová jednotka Potřebuje budou odstraněny z Azure Marketplace a webu Azure portal.

<a name="roll-back-to-a-previous-sku-version"></a>Vrátit zpět na předchozí verzi SKU 
----------------------------------

Aktuální verze za provozu skladovou Položku z Azure Marketplace můžete odstranit pomocí následujících kroků. Po dokončení procesu SKU bude vrácena zpět jeho předchozí verzi.

1.  Přihlaste se k [portál partnerů cloudu](http://cloudpartner.azure.com).
2.  Na kartě všechny nabídky vyberte vaši nabídku.
3.  V podokně na levé straně obrazovky vyberte kartu skladové položky.
4.  Odstraníte ze seznamu publikovaných balíčků na nejnovější verzi balíčku.
5.  Opakované publikování nabídky na webu Azure Marketplace.

Až tato nabídka je za provozu na webu Azure Marketplace, aktuální verze uvedené SKU se odstraní z Azure Marketplace a na webu Azure portal.
Skladovou Položku, bude vrácena zpět jeho předchozí verzi.

<a name="delete-a-live-offer"></a>Odstranit živé nabídku 
-------------------

Postupujte podle kroků níže zobrazíte pokyny od týmu podporu k odebrání za provozu nabídky z Azure Marketplace:

1.  Lístek podpory pomocí tohoto [odkaz](https://go.microsoft.com/fwlink/?linkid=844975)
2.  Vyberte možnost nabídky Správa potíží zadejte seznam úpravy nabídky a skladové položky již v produkčním prostředí v seznamu kategorií.
3.  Odešlete žádost.

Tým podpory vás provede odstranění nabídky.

Odstranění skladové položky/nabídka nebude mít vliv aktuální nákupy SKU/nabídka již provedli. Budou nadále fungovat jako předtím, ale nebudou k dispozici pro všechny nové nákupy do budoucna.
