# <a name="azure-saas-application-offer"></a>Nabídka Azure aplikace SaaS

<table> <tr> <td>Tato část vysvětluje, jak publikovat software-as-a-service (SaaS) aplikací Microsoftu <a href="https://azuremarketplace.microsoft.com/marketplace/">Azure Marketplace</a> nebo <a href="https://appsource.microsoft.com">Microsoft AppSource</a>. Aplikace SaaS použijte, pokud vaše řešení se nasadí ve vlastním předplatném Azure a Zákazníci se připojují prostřednictvím rozhraní návrhu a správě testování aplikace. <a href="https://azure.microsoft.com/services/active-directory">Azure Active Directory</a> (AAD) umožňuje využít vaše stávající zkušební prostředí. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Publikování výhody

Výpis vašich aplikací na webu Microsoft marketplace výhody patří:

- Oslovení 100 milionů uživatelů Azure Active Directory v Office 365 a Dynamics 365.
- Prodejní tým prodloužit: oslovit obchodní uživatele po celém světě a získejte prodejní kanál, který zaujme uživatele, pomáhá generovat zájemce a zahajuje konverzace s novým zákazníkům napříč odvětvími.

- Získejte užitečné přehledy: budeme sdílet přehledy o jaký je výkon vaší aplikace na AppSource, co funguje dobře a jak můžete ještě zlepšit prodejní postupy.


## <a name="publishing-process"></a>Proces publikování

Vytvoření a publikování vaší aplikace na webu Microsoft Azure Marketplace nebo Microsoft AppSource, tato část vás provede ale následující kroky:

1. Potvrďte, že splňujete požadavky, včetně registrace partnera: [Dev Center registrace](https://dev.windows.com/en-us/registration?accountProgram=Azure) *nebo* [registrace programu Microsoft Partner Network](https://partner.microsoft.com/).
2.  Definujte svoji nabídku marketplace a plány.
3.  Poskytnout marketingové obsahu a cenách.
4.  Testování a certifikaci vaší nabídky [portál partnerů cloudu](https://cloudpartner.azure.com).
5.  Přechod na živý přenos v rámci vaší nabídky.
