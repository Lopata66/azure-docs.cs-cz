# <a name="azure-virtual-machine-offer"></a>Nabídka virtuálních počítačů Azure

Tato funkce byla přesunuta do [partnerského centra](https://partner.microsoft.com/pcv/)Microsoftu. Další informace najdete v tématu [Vytvoření nabídky virtuálních počítačů Azure](../../partner-center-portal/azure-vm-create-offer.md).
