# <a name="azure-saas-application-offer"></a>Nabídka aplikace Azure SaaS

[Portál partnerů cloudu](https://cloudpartner.azure.com/) už nepodporuje vytváření a správu nabídek aplikací SaaS.  Místo toho se tato funkce přesunula na [portál partnerského centra](https://partner.microsoft.com/pcv/)Microsoftu.  Další informace najdete v tématu [Vytvoření nové nabídky SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Další kroky

Než vytvoříte novou nabídku aplikace SaaS, měli byste si přečíst [příručku pro publikování aplikací SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) a zkontrolovat kroky portálu pro partnery, které jsou uvedené v [kontrolním seznamu pro vytváření nabídek](../../partner-center-portal/offer-creation-checklist.md).
