# <a name="azure-saas-application-offer"></a>Nabídka aplikací Azure SaaS

[Portál cloudových partnerů](https://cloudpartner.azure.com/) již nepodporuje vytváření a správu nabídek aplikací SaaS.  Místo toho byla tato funkce přesunuta na [portál Microsoft Partner Center](https://partner.microsoft.com/pcv/).  Další informace naleznete [v tématu Vytvoření nové nabídky SaaS](../../partner-center-portal/create-new-saas-offer.md). 


## <a name="next-steps"></a>Další kroky

Před vytvořením nové nabídky aplikací SaaS byste si měli přečíst [Průvodce nabídkou publikování aplikací SaaS](../../marketplace-saas-applications-technical-publishing-guide.md) a přečíst si kroky portálu Partnerského centra uvedené v [kontrolním seznamu vytváření nabídek](../../partner-center-portal/offer-creation-checklist.md).
