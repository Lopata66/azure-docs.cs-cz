# <a name="saas-application-contacts-tab"></a>Kartu kontakty aplikace SaaS

Tento článek popisuje, jak používat kartu kontakty k identifikaci kontakty pro zákazníky, kteří používají vaši nabídku.

## <a name="identify-contacts"></a>Identifikujte kontakty

Následující snímek obrazovky, použijte jako vodítko k poskytování požadované informace. Všechna pole označená hvězdičkou (*) musí být vyplněna.

![Konstrukce a podporu formulář Kontakty](./media/saas-offer-contacts.png)

1. Pod technickým kontaktem zadejte informace pro následující pole:

   - Name – zadejte název technický kontakt pro vaši aplikaci. Tento kontakt obdrží technické komunikaci od Microsoftu.
   - E-mailu – zadejte e-mailová adresa technickým kontaktem.
   - Telefon – zadejte telefonní číslo technickým kontaktem. ISO telefonní číslo zápisy jsou podporovány. Další informace najdete v tématu [E.123](https://en.wikipedia.org/wiki/E.123)
  
2. V části kontakt podpory zadejte informace pro následující pole:

   - Name – zadejte jméno kontaktu podpory pro vaši aplikaci. Tento kontakt obdrží zpráv týkající se podpory od Microsoftu.
   - E-mailu – zadejte e-mailovou adresu kontaktu podpory pro vaši aplikaci.
   - Telefon – zadejte telefonní číslo kontaktu podpory. ISO telefonní číslo zápisy jsou podporovány. Další informace najdete v tématu [E.123](https://en.wikipedia.org/wiki/E.123)
   - Adresa URL podpory – zadejte adresu URL na vaše stránka podpory.

3. Vyberte **Uložit** po dokončení formuláře.
