# <a name="azure-application-contacts-tab"></a>Kartu kontakty aplikace Azure

Použití **kontakty** karty poskytují technické a zákazníka informace o podpoře pro vaše Dynamics 365 Customer Engagement nabídky.

![Kartu kontakty nabídky.](./media/dynce-contacts-tab.png)


## <a name="specify-customer-and-engineering-contacts"></a>Zadejte zákazníka a engineering kontaktů

Zadejte následující hodnoty k definování primární **technickým kontaktem** a pro vaši nabídku. Povinná pole jsou uvést hvězdičkou (*).

|    Pole        |  Popis                    |
|    ---------    |  ---------------                |
|  **Název\***       |  Název technický kontakt pro vaši aplikaci. Tento kontakt obdrží technické komunikaci od Microsoftu.   |
|  **E-mailu\***      |  E-mailovou adresu technický kontakt  |
|  **Telefon\***      |  Telefonní číslo technický kontakt  |
|  |  |

Zadejte následující hodnoty k definování primární **kontakt podpory** a pro vaši nabídku.

|    Pole        |   Popis                   |
|    ---------    |  ---------------                |
|  **Název\***       |  Jméno kontaktu podpory zákazníků pro vaši aplikaci. Tento kontakt obdrží zpráv týkající se podpory od Microsoftu.   |
|  **E-mailu\***      |  E-mailovou adresu kontaktu podpory zákazníků  |
|  **Telefon\***      |  Telefonní číslo kontaktu podpory zákazníků  |
|  |  |
