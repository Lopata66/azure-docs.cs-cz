# <a name="dynamics-365-for-customer-engagement-offer-settings-tab"></a>Dynamics 365 pro kartu Nastavení nabízejí zapojení zákazníků

Tento článek popisuje, jak nakonfigurovat nastavení nabídky pro Dynamics 365 Customer Engagement aplikace.


**Dynamics 365 for Customer Engagement** > **nová nabídka** otevře se stránka s fokusem na **nabízejí nastavení** kartu.  Hvězdičku (*) připojeným k názvu pole označuje, že je to požadováno.

![Karta nastavení nabídky pro Dynamics 365 for Customer Engagement](./media/dyn365ce-offer-settings-tab.png)


## <a name="offer-settings-fields"></a>Nabídka nastavení pole

Následující tabulka popisuje pole na této kartě. 

|    Pole         |       Popis                                                            |
|  ---------       |     ---------------                                                          |
| **ID nabídky**       | Jedinečný identifikátor (v rámci profilu vydavatele) pro tuto nabídku. Tento identifikátor se nebude zobrazovat v sestavách adresy URL a přehledů produktů. Má maximální délku než 50 znaků a můžou používat malé alfanumerické znaky a pomlčky (-). (Identifikátor nemůže končit spojovníkem.) **Poznámka:** Toto pole nelze změnit po nabídku přejde za provozu, protože se váže na základní adresu URL nabídky marketplace.  |
| **ID vydavatele**     | Jedinečný identifikátor vaší organizace na webu Azure Marketplace. Všechny nabídky by měly být přidruženy s vaším ID vydavatele. Tuto hodnotu nelze změnit, až se uloží nabídky. |
| **Název**          | Zobrazovaný název nabídky. Tento název se zobrazí na webu Marketplace AppSource. Může mít maximálně 50 znaků. Doporučujeme používat pro svůj produkt rozpoznat název značky. Pokud je jak produktu na trh, nezahrnujte název vaší organizace. Pokud jsou marketing této nabídky do jiných webových stránek a publikace, ujistěte se, že název je ve všech publikacích přesně stejné. |
|  |  |

Vyberte **Uložit** nabídky nastavení uložte.
