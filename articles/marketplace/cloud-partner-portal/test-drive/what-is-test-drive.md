<a name="what-is-test-drive"></a>Co je testovací verze?
===================

Testovací jednotka je skvělým způsobem, jak předvést vaši nabídku potenciálním zákazníkům tím, že jim poskytne možnost \'vyzkoušet před zakoupením\'a výsledkem je vyšší konverze a generace vysoce kvalifikovaných potenciálních zákazníků.

Po poskytnutí kontaktních údajů můžou zákazníci získat přístup k předem vytvořeným prostředím testovacích jednotek: praktická zkušební verze\'ch klíčových funkcí produktu a výhody, které jsou ve scénáři implementace reálného světa.

Testovací verze vám umožní dostat produkt do životního prostředí a vytvořit vysoce kvalifikované potenciální zákazníky v procesu.

<a name="how-does-a-test-drive-work"></a>Jak funguje testovací jednotka?
---------------------------

Potenciální zákazník zjistí vaši aplikaci na webu Marketplace, přihlásí se a souhlasí s podmínkami použití. V tomto okamžiku zákazník obdrží předem nakonfigurovaná prostředí, aby se mohl pokusit o určitý počet hodin, zatímco obdržíte vysoce kvalifikovaný zájemce, se kterým budete pokračovat.

Bez ohledu na to, jak komplexní je vaše aplikace, vám Microsoft Test Drive pomůže zajistit život produktu pro zákazníka. Dnes nabízíme tři různé typy testovacích jednotek na základě typu produktu, scénáře a Marketplace, na kterém jste.

- **[Azure Resource Manager](./azure-resource-manager-test-drive.md)** : Azure Resource Manager Test Drive je šablona nasazení, která obsahuje všechny prostředky Azure, které tvoří řešení vytvořené vydavatelem. Produkty, které vyhovují tomuto scénáři, jsou ty, které používají jenom prostředky Azure.
- **[Aplikace logiky](./logic-app-test-drive.md)** : testovací jednotka aplikace logiky je šablonou nasazení, která zahrnuje všechny komplexní architektury řešení. Všechny aplikace Dynamics nebo vlastní produkty by měly používat tento typ testovacích jednotek.
- **Power BI**: Power BI testovací jednotka je jednoduše vložené propojení na vlastní sestavený řídicí panel. Libovolný produkt, který chce pouze předvést interaktivní Power BI vizuál, by měl používat tento typ testovacích jednotek.
    Vše, co je třeba nahrát, je vaše vložená Power BI adresa URL.

<a name="what-goes-on-in-the-background"></a>Co se na pozadí nachází?
-------------------------------

Služba testovacích jednotek je navržená tak, aby nepřetržitě podporovala a poskytovala zákazníkům bez nutnosti jakéhokoli ručního úsilí. Jako vydavatelem úlohy je Správa a konfigurace nastavení testovacích jednotek z portálu pro partnery theCloud a toto nastavení bude pro vaše zákazníky k dispozici přímo.

To znamená, že po nastavení konfigurací pro testovací verzi se každá testovací jednotka změní na spravovanou instanci, která bude nasazena na vyžádání pro zákazníka požadujícího IT. Po přiřazení instance testovací jednotky je testovací jednotka dostupná pro použití v nastaveném časovém intervalu a pak se odstraní, aby se vytvořila místnost pro jiného zákazníka.
