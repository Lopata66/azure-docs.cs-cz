# <a name="all-publishers-page"></a>Stránka Všichni vydavatelé

[ **Všichni vydavatelé** ](https://cloudpartner.azure.com/#publishers) stránka zobrazuje seznam všech registrovaných Microsoft cloud vydavatelů, včetně jejich vydavatele jedinečný název a zobrazovaný název. Můžete výsledky seřadit podle sloupce nebo zadejte řetězec k filtrování výsledků.  Na této stránce se zobrazí také **přidat vydavatele** tlačítko, ale toto tlačítko je povolená jenom pro správce portálu.  Vyberte vydavatele v tomto seznamu se zobrazí profil pro daného vydavatele.  Další informace najdete v tématu [Správa profilu vydavatele](./../../cloud-partner-portal-orig/cloud-partner-portal-manage-publisher-profile.md).
