# <a name="insights-page"></a>Stránka Přehledy

Stránka [**Přehledy**](https://cloudpartner.azure.com/#insights) obsahuje následující sady řídicích panelů, které vám pomohou pochopit a maximalizovat prodej nabídek na marketplace.  Ke každému řídicímu panelu se přistupuje prostřednictvím stejnojmenné karty v horní části stránky.


|  **Řídicího panelu**    |  **Zobrazený obsah**               |
|  -------------    |  ---------------------               |
| **Souhrn**       | grafy, trendy a hodnoty souhrnných dat, které shrnují aktivitu na trhu pro nabídky vydavatele |
| **Výplata**        | výplaty a související transakce v grafických a stahovatelných formátech |
| **Objednávky & použití** | objednávky a informace o použití v grafických a stahovatelných formátech |
| **Zákazník**      | informace o zákaznících, včetně jejich nákupního profilu |
| **Nasazení**    | informace o úspěchu a neúspěchu nasazení v grafických formátech i formátech na úrovni událostí |
| **Stažené soubory**     | seznam žádostí o stažení za posledních 30 dní |
| **Analýza**     | přehled webové analýzy a analýzy výkonnosti kampaně |
|  |  |


## <a name="summary-dashboard"></a>Souhrnný řídicí panel
Řídicí panel Souhrn je první karta a je výchozí karta zobrazená při přechodu na stránku Přehledy.  Tento záměrný návrh odráží skutečnost, že tento řídicí panel poskytuje majitelům stránek nejširší pohled na prodejní aktivitu jejich nabídek.  Poskytuje grafické znázornění trendu normalizovaného použití; měsíční používání; trendové využití podle zeměpisného trhu, nabídky a zákazníka; a výplatní trend.  Výstup lze přizpůsobit zadáním počátečního a koncového data analýzy.  Následující obrázek znázorňuje ukázkový souhrnný řídicí panel pro Contoso pro červen až srpen 2018.
