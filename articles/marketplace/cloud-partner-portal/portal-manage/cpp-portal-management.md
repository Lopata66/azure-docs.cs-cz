# <a name="cloud-partner-portal-management"></a>Partnerský portál pro správu cloudu

Tato část popisuje typy operací, které jsou k dispozici pro vytváření a správě účtů, aby jeho používání [portál partnerů cloudu](https://cloudpartner.azure.com).  Jsou vyžadovány pro tyto účty [Azure Marketplace](https://azuremarketplace.microsoft.com) a [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Základní informace

Než začnete používat portál Cloud Partner, měli byste seznámit najdete v následujících článcích:

- [Azure Marketplace zásady a podmínky](../../policies-terms.md) obsahuje seznam různých právní a technické požadavky a pravidla, která platí pro uživatele a vydavatelé [Azure Marketplace](https://azuremarketplace.microsoft.com) a [AppSource](https://appsource.microsoft.com).
- [Stát vydavatelem cloudové Tržiště](../../become-publisher.md) najdete zde přehled všechny kroky potřebné k stát oficiální aplikaci Microsoft publisher cloudu.

- [Správa profilu vydavatele Azure Marketplace a AppSource](./cpp-manage-publisher-profile.md) vysvětluje, jak používat portál Cloud Partner zobrazit a spravovat svůj profil vydavatele Microsoft marketplace.
