# <a name="cloud-partner-portal-management"></a>Správa portál partnerů cloudu

Tato část popisuje typy operací, které jsou k dispozici pro vytváření a správu účtů potřebných k použití [portál partnerů cloudu](https://cloudpartner.azure.com).  Tyto účty jsou vyžadovány pro [Azure Marketplace](https://azuremarketplace.microsoft.com) a [AppSource](https://appsource.microsoft.com).  


## <a name="background-information"></a>Základní informace

Než začnete používat portál partnerů cloudu, měli byste se seznámit s následujícími články:

- [Azure Marketplace zásady a podmínky](../../policies-terms.md) obsahují seznam různých právních a technických požadavků a pravidel, které se vztahují na uživatele a vydavatele na [Azure Marketplace](https://azuremarketplace.microsoft.com) a [AppSource](https://appsource.microsoft.com).
- [Stane se vydavatelem cloudového tržiště](https://docs.microsoft.com/azure/marketplace/become-publisher) , který poskytuje přehled všech kroků požadovaných k tom, aby se stal oficiálním vydavatelem Microsoftu pro Cloud.


## <a name="account-creation-and-management"></a>Vytváření a Správa účtů

Tato část obsahuje následující články, které popisují, jak spravovat účty vydavatelů Marketplace.

- [Předpoklady pro správu účtů](./cpp-manage-prerequisites.md) popisují, jak se stát registrovaným vydavatelem cloudu. požadavek na používání portálu cloud partner.
- [Správa Azure Marketplace a profil vydavatele AppSource](./cpp-manage-publisher-profile.md) vysvětluje, jak pomocí portál partnerů cloudu zobrazit a spravovat váš profil vydavatele na webu Microsoft Marketplace. 
- [Správa uživatelů na portál partnerů cloudu](./cpp-manage-users.md) přidání uživatelů k účtu pro publikování do cloudu a způsobu správy přístupu, aby mohli více lidí pracovat na stejných nabídkách.
