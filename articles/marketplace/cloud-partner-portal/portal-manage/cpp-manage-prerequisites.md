# <a name="prerequisites-for-managing-accounts-on-the-cloud-partner-portal"></a>Požadavky na správu účtů na portál Cloud Partner 

Než budete moct použít [portál partnerů cloudu](https://cloudpartner.azure.com/) pro všechny funkce, včetně správy účtů, je třeba dodržovat sérii kroků, jako Microsoft cloud vydavatelé zaregistrovat sami sebe i jejich organizace.


## <a name="account-terms-and-guidelines"></a>Podmínky pro účet a pokyny

Pečlivě si prostudujte práva a povinnosti pod [smlouvou o poskytování služeb společnosti Microsoft](https://www.microsoft.com/servicesagreement) a [Terms](https://www.microsoft.com/legal/intellectualproperty/copyright) před vytvořením účtu.  

Pečlivě zvolte název svého účtu.  Představuje vaší organizaci, neměl by obsahovat všechny slova autorským nebo sporné.  Například Microsoft neumožňuje použití jeho autorským podmínky, například "Azure" a "Microsoft", v názvech účtů.  Další informace najdete v tématu [pokyny pro Azure Marketplace](../../guidelines.md)


## <a name="registration-steps"></a>Kroky registrace

V následující tabulce jsou uvedeny kroky potřebnými k plně zaregistrovat jako vydavatel Microsoft cloud.  Úplné pokyny najdete v tématu [stát vydavatelem cloudové Tržiště](../../become-publisher.md). 


|                   Krok                   |  Výsledný registraci nebo prostředků                     |
|                  ------                  |  -----------------------------------                    |
| 1. Registrace v programu Microsoft Partner Network |  ID Microsoft Partner Network pro vaši organizaci |
| 2. Vytvoření účtu Microsoft *           |  A ověřit účet Microsoft s koncovkou @outlook.com, @live.comatd. |
| 3. Odeslat marketplace registrační formulář | Vydavatelský účet na portál Cloud Partner      |
| 4. Přihlaste se k portálu Cloud Partner        | Oprávnění k vytvoření a publikování nabídek, Správa uživatelů, marketplace shromažďovat metriky, atd. |
| 5. Zaregistrovat v rozhraní Microsoft Developer Center * | Účet ověřený Dev Center; oprávnění k provádění transakcí marketplace  |
|   |   |

\* Tyto kroky jsou požadované pro transact nabídky Azure Marketplace, ale důrazně doporučujeme pro všechny nabídky.  Po vytvoření účtu Microsoft a zaregistrujte se jako Microsoft Developer, dostanete další Microsoft zákazníka a materiály pro vývojáře, jakož i volitelné partnerské nabídky.  


## <a name="next-steps"></a>Další postup

Po dokončení kroků registrace byste měli být schopni se přihlásit [portál partnerů cloudu](https://cloudpartner.azure.com/).  Můžete použít tento portál a Správa profilů vydavatele a spravovat uživatelské účty.
