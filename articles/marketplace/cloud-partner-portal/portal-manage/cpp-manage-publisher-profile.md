# <a name="manage-your-azure-marketplace-and-appsource-publisher-profile"></a>Správa profilu vydavatele Azure Marketplace a AppSource

Tento článek vysvětluje, jak používat [portál partnerů cloudu](https://cloudpartner.azure.com/) zobrazení a Správa profilu vydavatele.  Musíte být registrovaný partnera společnosti Microsoft cloudu pro přístup k tomuto portálu.  Další informace najdete v tématu [začít pracovat s portálem cloud partner](../../cloud-partner-portal-orig/cloud-partner-portal-getting-started-with-the-cloud-partner-portal.md).

V tuto chvíli jste dokončili kroky, abyste se na vydavatele Azure Marketplace a AppSource. Profil vydavatele je zaregistrovaná na portál partnerů cloudu po schválení vaší žádosti o partnera. Svůj profil vydavatele se budou vztahovat na všechny nabídky a skladové položky publikované z účet použitý během registrace partnera.


## <a name="publisher-profile"></a>Profil vydavatele

Svůj profil vydavatele odlišuje vaší společnosti na Azure Marketplace a AppSource. Zahrnuje ID vydavatele, zobrazovaný název a vlastníka email(s). Dobře spravovaného profilu se zvýší viditelnost a pomozte uživatelům marketplace snadno identifikovat a vyberte příslušné nabídky.

> [!NOTE]
> ID vydavatele a registrovaného vlastníka e-mailu se upravovat po publikování vaší nabídky první. Ale můžete zobrazit svůj profil a upravit zobrazovaného názvu publisher z portál partnerů cloudu.  <br/>
> Přidávání uživatelů (přispěvatelům a vlastníkům) do profilu publikování z části uživatelé na levém navigačním podokně v části Uživatelé portál partnerů cloudu.


### <a name="view-and-manage-your-publisher-profile"></a>Umožňuje zobrazit a spravovat svůj profil vydavatele

Chcete-li zobrazit a spravovat svůj profil vydavatele v pravé horní části nabídek, klikněte na tlačítko **profilu vydavatele** kartu.

![Vydavatel profilu kartu vydavatele profilu](./media/publisherprofilenew.png)

Vaše **profilu** zobrazí se dialogové okno.

![Podrobnosti o profilu vydavatele – podrobnosti o profilu vydavatele](./media/publisherprofiledetails.png)


### <a name="link-your-dev-center-account"></a>Propojení účtu Dev Center

Můžete také propojit existujícího účtu Dev Center s svůj profil vydavatele na portál partnerů cloudu.  Nejdřív přihlásit k portálu pomocí stejné e-mailovou adresu použije k registraci účtu Dev Center. Pak zobrazí stránku svého profilu vydavatele stav účtu Dev Center, e-mail vlastníka účtu Dev Center a název účtu Dev Center.

> [!NOTE]
> Registrace účtu středisko pro vývojáře je povinná pro publikování placené trh skladové položky.

Pokud nemáte účet Centrum pro vývojáře, můžete vytvořit účet, jak je popsáno v [registraci v centru vývojářů pro](https://docs.microsoft.com/azure/marketplace/register-dev-center).  Pak přidejte registrované e-mailovou adresu jako vlastníka na portál partnerů cloudu pro vaši nabídku. Jako vlastníka můžete přidat pouze registrované e-mailovou adresu do profilu publikování z **uživatele** oddílu. Poté Přihlaste se k portálu pomocí stejné e-mailovou adresu a propojí váš účet vývojáře.

![Podrobnosti o profilu vydavatele – popis profilu vydavatele](./media/publisherprofiledescription.png)
