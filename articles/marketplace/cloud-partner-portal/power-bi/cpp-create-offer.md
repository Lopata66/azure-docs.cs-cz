# <a name="create-a-power-bi-application-offer"></a>Vytvořte nabídku aplikace pro Power BI

Tato část obsahuje kroky potřebné k vytvoření nové nabídky aplikace Power BI pro [AppSource](https://appsource.microsoft.com). Každé nabídky se zobrazí jako vlastní entity v AppSource.  Když vytvoříte novou nabídku v [portál partnerů cloudu](https://cloudpartner.azure.com/), budete muset zadat čtyři skupiny prostředků pro vaši nabídku.

|   Skupiny prostředků      | Popis                                                                         |
| ----------------   | ----------------                                                                    |
| Nabídka nastavení     | Primární identifikace a název nabídky.                                      |
| Technické informace     | Adresa URL instalační program použitý k instalaci aplikace v pracovním prostoru klienta Power BI. Další informace o tom, jak vygenerovat tuto adresu URL, najdete v tématu [dokumentace k aplikaci Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).   |
| Prezentace podrobnosti | Obsahuje marketing, právní a může vést správy prostředků. Marketingové prostředky zahrnují popis nabídky a loga.  Právní prostředky zahrnují další právní dokumentace, zásady ochrany osobních údajů a podmínky použití.  Zásada umožňuje průběžné správy můžete určit způsob zpracování zájemci z AppSource portálu pro koncové uživatele. |
| Kontakty           | Obsahuje informace o podpoře kontaktní údaje a zásady                                     |
|    |     |


## <a name="new-offer-form"></a>Formulář nové nabídky

Jakmile vaše přihlaste na portál Cloud Partner, klikněte na tlačítko **+ nová nabídka** položka na levém řádku nabídek.  V nabídce výsledný klikněte na **aplikace Power BI** zobrazíte **nová nabídka** formuláře a zahájit proces definice prostředků pro nové nabídky aplikace.

![Položka nabídky Nabídka Power BI](./media/new-offer-menu.png)

> [!WARNING] 
> Pokud **aplikace Power BI** možnost není zobrazena, nebo není povoleno a pak váš účet nemá oprávnění vytvořit tento typ nabídky. Zkontrolujte prosím, že jste splnili veškeré [požadavky](./cpp-prerequisites.md) pro tento typ nabídky, včetně registrace pro účet pro vývojáře.
