# <a name="create-power-bi-app-technical-assets"></a>Vytvoření aplikace Power BI technické prostředky

Primární technické asset budete potřebovat pro tento typ nabídky se aplikace Power BI, což je kolekce primární datové sady, sestavy a (nebo) řídicí panel a volitelné připojených služeb a vložené datové sady. (Aplikace power BI jsou výsledkem vývoje typ prostředku, dříve označovaný jako [balíčky obsahu](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction).)  Úplné informace o vývoji těchto aplikací, najdete v části [aplikace Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Získání adresy URL instalace

Vytvoření aplikace Power BI v pouze k dispozici v rámci [služby Power BI](https://powerbi.microsoft.com/) a vyžaduje přihlášení pomocí [licenci Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Jakmile aplikaci služby je vytvořený a otestovaný v Power BI, uložte adresu URL instalace aplikace, která generuje Power BI.  Je nutné zadat tuto adresu URL v [technické informace o kartě](./cpp-technical-info-tab.md) portál partnerů cloudu odeslání formuláře.
 

## <a name="next-steps"></a>Další postup

Po vytvoření a testování vaší aplikace Power BI a vytvořit související nabídky můžete [publikování nabídky Power BI](./cpp-publish-offer.md).
