# <a name="power-bi-apps-contacts-tab"></a>Power BI Apps kontakty kartu

Použití **kontakty** karty **nová nabídka** stránku s cílem poskytnout uživatele a technické podpory prostředků pro vaši nabídku. Je rozdělený do dvou částí: **Technické** a **zákaznickou podporu**.

![Kartu kontakty](media/contacts-tab.png)


## <a name="contacts-fields"></a>Pole kontakty 

Následující tabulka popisuje název a účel z těchto polí. 

|   Pole               |   Popis                                                            |
|-----------------------|--------------------------------------------------------------------------|
| *Technický kontakt* | Slouží jako technický kontakt mezi AppSource a vaši organizaci    |
| **Název**              | Jméno osoby nebo skupiny, která slouží jako podpora technické/inženýrství |
| **E-mail**             | E-mailovou adresu Tento technický kontakt                                  |
| **Telefon**             | Telefonní číslo technické podpory                                       |
| *Kontakt na podporu*     | Přijímá lístky podpory otevírány zákazníky v AppSource            |
| **Název**              | Jméno osoby nebo skupiny, která slouží jako Zákaznická podpora              |
| **E-mail**             | E-mailovou adresu podpory zákazníků                                        |
|  |  |


## <a name="next-steps"></a>Další postup

Po dokončení nabídky aplikace Power BI, by měl potvrďte, že jste vytvořili všechny přidružené [technických prostředků](./cpp-create-technical-assets.md) k této nabídce zaregistrují.
