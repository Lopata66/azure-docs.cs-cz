# <a name="manage-azure-and-appsource-marketplace-offers"></a>Správa nabídek Azure a AppSource Marketplace

Tato část představuje obecné operace používané ke správě nabídek pro Azure a AppSource Marketplaces.  Vysvětluje například, jak pomocí [portálu partnerů cloudu](https://cloudpartner.azure.com/) vytvořit a publikovat nabídku marketplace.  Tato část také vysvětluje technické problémy, které jsou společné pro většinu typů nabídek, jako jsou textové a obrazové prostředky.


## <a name="standard-offer-operations"></a>Standardní nabídka

Portál partnerů cloudu umožňuje vydavateli provádět následující základní operace v nabídce.

|     Operace      |  Popis                                           |
|     ---------      |  -----------                                           |
| [Vytvoření nabídky](./cpp-create-offer.md)   | Vytvoří položku čekající nabídky v úložišti registrace, ale automaticky ji nepublikuje na tržišti Microsoftu. | 
| [Zobrazení stavu nabídky](./cpp-view-status-offer.md)   | Poskytuje stav publikování nabídky.  Na portálu je více míst, kde se zobrazují informace o stavu. |
| [Publikování nabídky](./cpp-publish-offer.md) | Ověří a pak zveřejní dokončenou nabídku na příslušném webu Microsoft Marketplace.  Nabídka je pak řekl, aby byl *živý*. |
| [Aktualizace nabídky](./cpp-update-offer.md)   | Umožňuje změnu určitých vlastností stávající nabídky.  Platí pro nevyřízené i publikované nabídky. |
| [Odstranění nabídky](./cpp-delete-offer.md)   | Odstraní existující nevyřízenou nebo živou nabídku.  | 
|  |  |
