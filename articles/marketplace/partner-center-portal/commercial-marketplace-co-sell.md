# <a name="co-sell-option-in-partner-center"></a>Možnost spoluprodeje v Partnerském centru

Poskytnutí informací na kartě Co-sell je zcela volitelné. Je nutné dosáhnout Co-sell Ready a IP Co-sell Ready status. Informace, které poskytnete, budou prodejními týmy společnosti Microsoft použity k tomu, aby se dozvěděly další informace o vašem řešení při vyhodnocování jeho vybavení potřebám zákazníků. Není k dispozici přímo zákazníkům. Další informace naleznete v tématu [sell through Microsoft](https://partner.microsoft.com/membership/sell-with-microsoft).

Možnost spoluprodeje v Partnerském centru je v současné době k dispozici pro některé typy nabídek. Podpora pro další typy nabídek bude k dispozici v průběhu času.

| **Typ nabídky** | **Stránka pro spoluprodej je k dispozici v Partnerském centru** |
|    -----------------------    |    -----------------------------  |
| Software jako služba (SaaS) |     Všeobecná dostupnost duben 2020      |
| Dynamics 365 pro zapojení zákazníků & PowerApps      |     Nově dostupné      |
| Dynamics 365 for Operations     |     Nově dostupné       |
| Dynamics 365 Business Central      |  Všeobecná dostupnost duben 2020     |
| Azure Application     |     Všeobecná dostupnost duben 2020     |
| Virtuální počítač Azure      |     Bude oznámeno       |
| Kontejner Azure     |     Bude oznámeno       |
| Konzultační služba      |     Bude oznámeno       |
| Modul IoT Edge      |     Bude oznámeno       |
| Aplikace Služby Power BI     |     Bude oznámeno       |
|  |

## <a name="listing"></a>Výpis

Spoluprodej výpisy pomohou prodejním týmům Microsoftu prodávat vaše řešení širšímu publiku.

- Vyberte platformy Microsoftu, se kterými je vaše nabídka postavená, rozšiřuje se s ní nebo s ní integruje.
- Vyberte segmenty trhu, na které vaše nabídka cílí.
- Vyberte [typ řešení](#solution-type) pro vaši nabídku.
- Vyberte si až tři [oblasti řešení](#solution-areas) a podoblasti.

### <a name="solution-type"></a>Typ řešení

Typy řešení pomáhají definovat scénáře, které vaše nabídka je určena k řešení.

| **Typ řešení**    | **Popis**  |
| :------------------- | :-------------------|
| Zařízení (hardware) | Nabídka, která zahrnuje vytváření nebo prodej hardwaru od výrobce zařízení. |
| IP (aplikace) | Aplikace nebo jiný materiál licencovaný autorskými právy pro použití zákazníkem. Například: program CRM, který může být licencován a instalován místně. |
| Služba | Praktické odborné znalosti pro konkrétní jednorázový projekt, často poskytované prostřednictvím konzultantů. Například: nastavení databáze zákazníků pro klienta (s klientem převzetí odpovědnosti za provoz databáze po dodání). |
| Spravovaná služba | Praktické odborné znalosti pro cloudový projekt, obvykle průběžně. Například: Poskytování platformy a nástrojů pro spuštění online databáze s průběžnou správou poskytovanou poskytovatelem spravovaných služeb. |

### <a name="solution-areas"></a>Oblasti řešení

Oblasti řešení pomáhají dále definovat vaše řešení.

Pro vaši nabídku můžete přidat až tři oblasti řešení. Pro každou oblast řešení můžete zvolit až dvě podoblasti řešení. Chcete-li vybrat více `Ctrl` podkategorií, použijte klíč (v systému Windows) nebo příkazový klíč (v systému Mac OS).

## <a name="contacts"></a>Kontakty

Kontaktní informace umožňují prodejním týmům společnosti Microsoft a prodejcům kanálů požadovat další informace od příslušného zdroje ve vaší organizaci. Kontaktní informace jsou k dispozici všem prodejním týmům společnosti Microsoft. Pokud se rozhodnete zpřístupnit svou nabídku v programu CSP, budou tyto kontaktní informace k dispozici také prodejcům kanálů.

Tento dokument obsahuje pokyny k různým rolím pro partnery a o tom, jak si mezi nimi vybrat. Kontakty můžete poskytnout stažením šablony CSV z Centra partnerů, vyplněním každého řádku informacemi o kontaktu a importem tohoto souboru CSV. Importem souboru CSV se přepíší existující kontakty.

Můžete také exportovat existující seznam kontaktů nabídky a potom v tomto souboru provést změny.

* *Jméno (povinné)*: Jméno kontaktu.
* *E-mailová adresa (povinné)*: E-mailová adresa kontaktu.
* *Funkce (povinné)*: Pracovní pozice.
* *Role (povinné)*: V šabloně použijte jednu z následujících rolí v tomto poli.

    | **Role**    | **Popis**  |
    | :------------------- | :-------------------|
    | *:::no-loc text="Partner marketing":::* | Tato role se zaměřuje na marketing vašeho řešení a spolupráci na marketingových aktivitách s prodejními týmy microsoftu a prodejci kanálů. Hlavním kontaktním místem pro marketingové aktivity a nabízení obsahu, jako jsou popisy produktů, obrázky a videa. |
    | *:::no-loc text="Partner sales":::* | Tato role se zaměřuje na prodej vašeho řešení a spolupráci na prodeji s prodejními týmy microsoftu a prodejci kanálů. U řešení pro spoluprodej uveďte alespoň jeden partnerský prodejní kontakt na region, ve kterém chcete být co-sell Ready. Stejný partnerský obchodní kontakt se může týkat více oblastí. |
    | *:::no-loc text="Partner technical sales":::* | Podporuje technickou architekturu a aspekty nasazení během prodejního cyklu, poprodejní integrace a období nasazení. |
    | *:::no-loc text="Partner customer success manager":::* | Obvykle podporuje zákazníky po nasazení, aby jim pomohla získat co nejvíce z vašeho řešení a zvýšit jeho nároky v rámci organizace zákazníka. |

* *Země/oblasti (povinné):*  Při vyplňování šablony použijte [dva kódy země nebo oblasti, jak jsou uvedeny](./commercial-marketplace-co-sell-countries.md). Pokud se kontakt vztahuje na všechny země/oblasti, použijte třípísmenný kód "OOO". Pokud kontakt pokrývá více než jednu zemi nebo oblast, zadejte každý ze dvou kódů písmen oddělených čárkou (například zadejte "US, CA, FR" bez uvozovek do šablony).

    Země nebo oblasti by měly odrážet území každého kontaktu. Prodejní týmy a prodejci kanálů společnosti Microsoft budou tyto informace využívat k vyžádání informací nebo spolupráci na prodejích v rámci konkrétní země nebo oblasti.

* *Státy/provincie (nepovinné):*  Při vyplňování šablony použijte formát XX-XX uvedený v [tabulce Stavy/provincie](./commercial-marketplace-co-sell-states.md).
