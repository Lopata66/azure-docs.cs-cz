# <a name="azure-marketplace-and-appsource-roadmap"></a>Plán Azure Marketplace a AppSource

Tento dokument poskytuje přehled o co se chystá pro Azure Marketplace AppSource a související partnera programy. Pojednává o některých důležitých funkcích, které jsme přidali a hrubý časový rámec, kdy můžete očekávat, že k jejich zobrazení. Není úplný seznam všech nových funkcí, ale cílem je poskytnout přehled o našich investic do klíče. Tyto sady funkcí a časové rámce doručování jsou aktuální a můžou se změnit. Na této stránce se aktualizují jednou měsíčně.
