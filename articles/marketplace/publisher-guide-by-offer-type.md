# <a name="publishing-guide-by-offer-type"></a>Průvodce publikováním podle typu nabídky

Jakmile [se rozhodnete pro možnost publikování](https://docs.microsoft.com/azure/marketplace/determine-your-listing-type), jste připraveni vybrat typ nabídky, který bude použit k prezentaci nabídky.

 *Typ nabídky* definuje strukturu nabídky, která zahrnuje metadata, artefakty a další obsah použitý k prezentaci nabídky na trhu.

Před vytvořením nabídky je nutné zvolit typ nabídky. Typ nabídky bude odpovídat typu řešení, aplikace nebo nabídky služeb, které chcete publikovat, a také jeho zarovnání produktů a služeb společnosti Microsoft.

Jeden typ nabídky lze nakonfigurovat odlišně, aby bylo možné povolit různé možnosti publikování, výzvy k akci, zřizování nebo ceny. Možnost publikování a konfigurace typu nabídky také zarovnávají způsobilost nabídky a technické požadavky.

Nezapomeňte si před vytvořením nabídky projít požadavky na nárok na příkladu a typ nabídky a požadavky na technické publikování.

## <a name="list-of-offer-types"></a>Seznam typů nabídek

Typy nabídek Azure Marketplace jsou uvedeny v následující tabulce.

| **Typ nabídky**    | **Popis**  |
| :------------------- | :-------------------|
| [**Virtuální počítače**](./marketplace-virtual-machines.md) | Typ nabídky virtuálního počítače použijte při nasazení virtuálního zařízení do předplatného přidruženého k zákazníkovi. |
| [**Šablony řešení**](./marketplace-solution-templates.md) | Použijte šablonu řešení (označovanou také jako aplikace Azure) typ nabídky, když vaše řešení vyžaduje další nasazení a automatizaci konfigurace nad rámec jednoho virtuálního počítače. Šablony řešení můžou využívat mnoho různých druhů prostředků Azure, včetně virtuálních počítačích, ale nejen na virtuální počítače.  |
| [**Spravované aplikace**](./marketplace-managed-apps.md) | Použijte aplikaci Azure: typ nabídky spravované aplikace, když jsou vyžadovány následující podmínky: <br> <ul> <li> Nasadíte buď řešení založené na předplatném pro zákazníka pomocí virtuálního počítače nebo celé řešení založené na IaaS. </li> <li>Vy nebo váš zákazník požadujete, aby řešení bylo řízeno partnerem. </li> <ul> |
| [**Aplikace SaaS**](./marketplace-saas-applications-technical-publishing-guide.md) | Pomocí typu nabídky aplikace SaaS umožníte zákazníkovi zakoupit si technické řešení založené na SaaS jako předplatné. |
| [**Kontejnerové nabídky**](./marketplace-containers.md) | Typ nabídky Container použijte, když vaše řešení je image kontejneru Dockeru zřízená jako služba kontejneru Azure založená na Kubernetes. |
| [**Moduly Azure IoT Edge**](./iot-edge-module.md) | Moduly Azure IoT Edge jsou nejmenší výpočetní jednotky spravované ioT Edge a mohou obsahovat služby Microsoftu (například Azure Stream Analytics), služby třetích stran nebo vlastní kód specifický pro řešení. |
| [**Konzultační služby**](./consulting-services.md) | Poradenské služby pomáhají propojit zákazníky se službami a podporovat a rozšiřovat jejich používání služeb Azure, Dynamics 365 nebo Power Suite.|
| [**Office 365, Dynamics 365 a Power BI**](./appsource-offer-publishing-guide.md) | Můžete publikovat nabídky AppSource, které budou vycházet z Dynamics 365, Office 365, Power BI a Power Apps nebo je rozšíří.|
