# <a name="third-party-software-compatible-with-azure-ad-domain-services"></a>Software jiných výrobců kompatibilní se službou Azure AD Domain Services

## <a name="one-identity-authentication-services"></a>Jedna služba ověřování identity

![Jedna identita](./media/compatible-software-icons/OneIdentity.png)

Jedna služba ověřování identit umožňuje systémům Unix, Linux a Mac OS X používat přístup, ověřování a autorizaci stávající infrastruktury služby Active Directory (AD). Ověřovací služby teď podporují službu Azure Active Directory Domain Services, která umožňuje prostředkům, které nejsou windows, využívat stejnou platformu nové generace, jakou už používají vaše stávající řešení SaaS.

[Jedna služba ověřování identity](https://www.quest.com/products/authentication-services/)

[Článek kb kb – použití jedné služby ověřování identity se službou Azure AD Domain Services](https://support.quest.com/authentication-services/kb/208427)


## <a name="red-hat-enterprise-linux"></a>Red Hat Enterprise Linux

![Red Hat](./media/compatible-software-icons/RedHat.png)

Red Hat Enterprise Linux je přední světová podniková linuxová platforma postavená tak, aby vyhovovala potřebám dnešního moderního podniku. Red Hat Enterprise Linux je preferovanou volbou pro podnikové úlohy virtuálních strojů (VM) v Microsoft Azure. Red Hat Enterprise Linux je otevřená, spolehlivá a bezpečná platforma určená pro zákazníky, kteří chtějí flexibilitu nasazení pro své důležité podnikové úlohy – od datového centra až po cloud Azure – podpořenou úzce integrovanou podporou na podnikové úrovni od Red Hat a Microsoft.

[Připojte se k virtuálnímu počítači Red Hat Enterprise Linux 7.2 ke službě Azure AD Domain Services](active-directory-ds-admin-guide-join-rhel-linux-vm.md)
