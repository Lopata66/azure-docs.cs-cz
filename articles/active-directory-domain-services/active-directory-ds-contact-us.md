# <a name="azure-ad-domain-services---contact-us"></a>Azure Active Directory Domain Services – kontaktujte nás
## <a name="contact-the-product-team"></a>Kontaktováním produktového týmu
Pokud máte problémy s vaší spravované domény, zkontrolujte, pokud podle kroků popsaných v [Průvodce odstraňováním potíží](active-directory-ds-troubleshooting.md) problém vyřešit. Pokud stále máte potíže, neváhejte nás kontaktovat.

Vám může e-mailové adrese: [Azure AD Domain Services zpětnou vazbu](mailto:aaddsfb@microsoft.com).

Ujistěte se, že složku zahrnujete následující příkaz, takže jsme prošetření problému.

* **ID/directory ID tenanta** pro váš adresář Azure AD. ID tenanta je identifikátor GUID se zobrazí v poli ID adresáře na [stránku vlastnosti pro adresář](https://ms.portal.azure.com/#blade/Microsoft_AAD_IAM/ActiveDirectoryMenuBlade/Properties)
* **Název domény DNS** nakonfigurujete vaší spravované domény služby AAD Domain Services.

## <a name="provide-feedback"></a>Poslat názor
Vítáme váš názor na Azure AD Domain Services prostřednictvím  **[kanálu Azure Active Directory User Voice](https://feedback.azure.com/forums/169401-azure-active-directory/)**.

Zkontrolujte tuto vám pre-jako nedokončené svůj dotaz nebo zpětné vazby pomocí slova **"AADDS"**, aby se nás kontaktovat.
