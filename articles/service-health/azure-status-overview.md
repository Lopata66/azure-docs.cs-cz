# <a name="azure-status"></a>Stav Azure
[Stav Azure](https://azure.microsoft.com/status/) vám poskytuje globální přehled o stavu služeb Azure. Stav služby Azure umožňuje získat nejčerstvější informace na dostupnost služby. Všichni mají přístup ke stavu Azure a můžete zobrazit všechny služby, které hlásit jejich stav služby. 

![Stránce stavu Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Aktualizace stavu Azure
Na stránce stavu Azure aktualizována v reálném čase jako stav služeb Azure změnit. Pokud ponechat otevřené stránce stavu Azure může řídit rychlost, jakou se aktualizuje na stránce s novými daty. V horní části se zobrazí čas poslední stránka se aktualizuje.

![Aktualizace stavu Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Historie stavu Azure
Na stránce stavu Azure vždy zobrazovat nejnovější informace o stavu, ale když zobrazíte starší události pomocí [stránky s historií stavu Azure](https://azure.microsoft.com/status/history/).

Zde můžete vidět události až do 90 dnů v minulosti, včetně předběžný hlavní příčinu, zmírnění škod a další kroky pro Azure.

## <a name="rss-feed"></a>Informační kanál RSS
Stav služby Azure také poskytuje [informačního kanálu RSS](https://azure.microsoft.com/en-us/status/feed/) změn stavu služby Azure, které se můžete přihlásit k odběru.
