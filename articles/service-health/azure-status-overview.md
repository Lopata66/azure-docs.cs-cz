# <a name="azure-status-overview"></a>Přehled stavu Azure

[Stav Azure](https://status.azure.com/status/) poskytuje globální zobrazení stavu služeb Azure a oblastí. Se stavem Azure můžete získat informace o dostupnosti služeb. Stav Azure je k dispozici všem uživatelům k zobrazení všech služeb, které hlásí stav jejich služby, stejně jako incidenty s rozsáhlým dopadem. Pokud jste však aktuální uživatel Azure, důrazně doporučujeme, abyste používali přizpůsobené prostředí ve [službě Azure Service Health](https://aka.ms/azureservicehealth). Azure Service Health zahrnuje všechny výpadky, nadcházející plánované aktivity údržby a informační zpravodaje služeb.

![Stavová stránka Azure](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Aktualizace stavu Azure

Stránka stavu Azure se aktualizuje v reálném čase, jak se mění stav služeb Azure. Pokud necháte stránku stavu Azure otevřenou, můžete řídit rychlost, jakou se stránka aktualizuje s novými daty. V horní části můžete vidět, kdy byla stránka naposledy aktualizována.

![Aktualizace stavu Azure](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Historie stavu Azure

Zatímco stránka stavu Azure vždy zobrazuje nejnovější informace o stavu, můžete zobrazit starší události pomocí [stránky Historie stavu Azure](https://status.azure.com/status/history/).

Počínaje událostmi, které byly publikovány v listopadu 2019, stránka historie stavu zobrazuje události až 365 dní v minulosti, včetně předběžné hlavní příčiny, zmírnění a dalších kroků pro Azure.

## <a name="rss-feed"></a>Informační kanál RSS

Stav Azure také poskytuje [rss kanál](https://status.azure.com/status/feed/) změn stavu služeb Azure, ke kterým se můžete přihlásit.
