# <a name="azure-cdn-coverage-by-metro"></a>Pokrytí Azure CDN podle metra 
> [!div class="op_single_selector"]
> * [Lokality POP podle oblastí](cdn-pop-locations.md)
> * [Lokality POP podle zkratek](cdn-pop-abbreviations.md)
> 


Tento článek uvádí aktuální metro s umístěními bodu přítomnosti (POP) seřazených podle oblasti pro produkty Sítě pro doručování obsahu Azure (CDN). Každý metro může obsahovat více než jeden POP. Například Azure CDN od Microsoftu má 130 POP v 80 metros. 

> [!IMPORTANT]
> Umístění ve městě POP pro **Azure CDN z Akamai** nejsou jednotlivě zveřejněny.  
> 
> Vzhledem k tomu, že každý produkt Azure CDN má odlišný způsob vytváření infrastruktur CDN, společnost Microsoft doporučuje nepoužívat umístění POP k rozhodnutí, který produkt Azure CDN použít. Místo toho zvažte jeho funkce a výkon koncového uživatele. Otestujte výkon s každým produktem Azure CDN a vyberte správný produkt pro vaše uživatele.
