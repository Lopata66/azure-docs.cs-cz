# <a name="azure-cdn-coverage-by-metro"></a>Azure CDN pokrytí Metro 
> [!div class="op_single_selector"]
> * [Lokality POP podle oblastí](cdn-pop-locations.md)
> * [Lokality POP podle zkratek](cdn-pop-abbreviations.md)
> 


Tento článek uvádí aktuální metrem obsahující point-of-presence (POP) umístění, seřazené podle oblasti, pro produkty Azure Content Delivery Network (CDN). Každý Metro může obsahovat více než jeden POP. Azure CDN od společnosti Microsoft má například 129 POP napříč 65 metrem. 

> [!IMPORTANT]
> Umístění POP Město **Azure CDN od Akamai** nejsou uvedeny jednotlivě.  
> 
> Protože všech produktů Azure CDN má různé způsob vytváření infrastruktury jeho CDN, společnost Microsoft doporučuje nepoužívat místa POP se rozhodnout, který produkt využít Azure CDN. Místo toho zvažte jeho funkcí a výkonu koncového uživatele. Testování výkonu u každého produktu Azure CDN k výběru správné produktu pro vaše uživatele.
