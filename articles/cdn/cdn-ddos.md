# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS Protection

Content delivery network poskytuje ochranu před útoky DDoS záměrné. Kromě globálních titul a chránit před útoky volumentric Azure CDN má další ochranu před útoky DDoS, jak je uvedeno níže, ne dalších poplatků.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN od společnosti Microsoft

Azure CDN od společnosti Microsoft je chráněn [Azure Basic DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Je integrovaná do platformy Microsoft Azure CDN ve výchozím nastavení a bez dalších poplatků. Úplné škálování a kapacitu Azure CDN od společnosti Microsoft globálně nasazené sítě poskytuje ochranu před útoky na běžné sítě prostřednictvím monitorování a v reálném čase zmírnění neustále v provozu. Základní ochranu před útoky DDoS také chrání před nejběžnější, často ke kterým dochází vrstvy 7 DNS dotazu záplavy a vrstvy 3 a 4 odměrné útoky této cílové koncové body CDN. Tato služba má také prověřené záznamu sledování v ochranu Microsoft enterprise a službami pro veřejnost ve velkém měřítku útoky.

## <a name="azure-cdn-from-verizon"></a>Azure CDN od Verizonu

Azure CDN od Verizonu je chráněný platformou pro Verzion proprietární před útoky DDoS omezení rizik. Je integrovaná do Azure CDN od Verizonu ve výchozím nastavení a bez dalších poplatků. Poskytuje základní ochranu proti nejběžnější, často ke kterým dochází vrstvy 7 DNS dotazu záplavy a vrstvy 3 a 4 odměrné útoky této cílové koncové body CDN.
