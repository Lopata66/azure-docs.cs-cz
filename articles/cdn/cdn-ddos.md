# <a name="azure-cdn-ddos-protection"></a>Azure CDN DDoS Protection

Síť pro doručování obsahu poskytuje ochranu DDoS od návrhu. Kromě globální kapacity absorbovat objemové útoky má Azure CDN další ochranu DDoS, jak je uvedeno níže, bez dalších nákladů.

## <a name="azure-cdn-from-microsoft"></a>Azure CDN od Microsoftu

Azure CDN od Microsoftu je chráněná [Azure Basic DDoS](https://docs.microsoft.com/azure/virtual-network/ddos-protection-overview). Je integrován do Azure CDN z platformy Microsoft ve výchozím nastavení a bez dalších nákladů. Úplné škálování a kapacita azure cdn z globálně nasazené sítě Microsoftu poskytuje ochranu proti běžným útokům síťové vrstvy prostřednictvím nepřetržitého sledování provozu a zmírňování stavu v reálném čase. Základní ochrana DDoS také chrání proti nejběžnějším, často se vyskytujícím záplavám dotazů DNS vrstvy 7 a objemovým útokům vrstvy 3 a 4, které cílí na koncové body CDN. Tato služba má také prokazatelné zkušenosti v ochraně podnikových a spotřebitelských služeb společnosti Microsoft před rozsáhlými útoky.

## <a name="azure-cdn-from-verizon"></a>Azure CDN od společnosti Verizon

Azure CDN od společnosti Verizon je chráněna proprietární platformou pro zmírnění DDoS společnosti Verzion. Je integrován do Azure CDN od Verizon ve výchozím nastavení a bez dalších nákladů. Poskytuje základní ochranu proti nejběžnější, často se vyskytující vrstva 7 DNS dotaz povodně a vrstva 3 a 4 volumetrické útoky, které cílí na koncové body CDN.

## <a name="azure-cdn-from-akamai"></a>Azure CDN od Akamai

Azure CDN od Akamai je chráněn akamai proprietární platformy pro zmírnění DDoS. Je integrován do Azure CDN z Akamai ve výchozím nastavení a bez dalších nákladů. Poskytuje základní ochranu proti nejběžnější, často se vyskytující vrstva 7 DNS dotaz povodně a vrstva 3 a 4 volumetrické útoky, které cílí na koncové body CDN.
