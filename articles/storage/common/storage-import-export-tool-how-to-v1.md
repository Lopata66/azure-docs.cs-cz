# <a name="using-the-azure-importexport-tool-classic-deployment-model"></a>Použití nástroje pro import a export Azure (klasický model nasazení)

Nástroj pro import a export Azure (WAImportExport.exe) se používá k vytváření a správě úloh pro službu Import a export Azure, což umožňuje přenášet velké množství dat do nebo z azure blob storage.

Tato dokumentace je pro klasický model nasazení nástroje Azure Import/Export Tool. Informace o použití nejnovější verze nástroje naleznete [v tématu Použití nástroje Pro import a export Azure](../storage-import-export-tool-how-to.md).

Následující články ukazují, jak:

- Nainstalujte a nastavte nástroj importu a exportu.
- Připravte si pevné disky na úlohu, ve které importujete data z disků do úložiště objektů blob Azure.
- Zkontrolujte stav úlohy pomocí kopírovat soubory protokolu.
- Oprava úlohy importu.
- Oprava úlohy exportu.
- Poradce při potížích nástroj pro import a export Azure, v případě, že jste měli problém během procesu.
