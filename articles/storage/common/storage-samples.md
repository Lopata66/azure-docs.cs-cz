# <a name="azure-storage-samples"></a>Ukázky úložiště Azure

Pomocí níže uvedených odkazů můžete zobrazit a stáhnout ukázkový kód a aplikace služby Azure Storage.

## <a name="azure-code-samples-library"></a>Knihovna ukázek kódu Azure

Knihovna [Ukázky kódu Azure](https://azure.microsoft.com/documentation/samples/?service=storage) obsahuje ukázky pro Azure Storage, které můžete stáhnout a spustit místně. Ukázková knihovna kódu poskytuje ukázkový kód ve formátu ZIP. Případně můžete procházet a klonovat úložiště GitHub pro každou ukázku.

## <a name="net-samples"></a>Ukázky .NET

Chcete-li prozkoumat ukázky rozhraní .NET, stáhněte si [klientskou knihovnu úložiště .NET](https://www.nuget.org/packages/WindowsAzure.Storage/) z NuGet. Klientská knihovna úložiště .NET je také k dispozici v [azure sdk pro .NET](https://azure.microsoft.com/downloads/).

* [Ukázky pro Azure Storage s použitím .NET](storage-samples-dotnet.md)

## <a name="java-samples"></a>Ukázky Java

Chcete-li prozkoumat ukázky jazyka Java, stáhněte si [klientskou knihovnu úložiště Java](https://github.com/azure/azure-storage-java).

* [Ukázky pro Azure Storage s použitím Javy](storage-samples-java.md)

## <a name="python-samples"></a>Ukázky Pythonu

Chcete-li prozkoumat ukázky Pythonu, stáhněte si [klientskou knihovnu úložiště Pythonu](https://github.com/azure/azure-storage-python).

* [Ukázky Azure Storage pomocí Pythonu](storage-samples-python.md)

## <a name="nodejs-samples"></a>Ukázky Node.js

Chcete-li prozkoumat ukázky souboru Node.js, stáhněte si [klientskou knihovnu úložiště Node.js](https://github.com/Azure/azure-storage-node).

* [Ukázky úložiště Azure pomocí javascriptu/node.js](storage-samples-javascript.md)

## <a name="c-samples"></a>Ukázky jazyka C++

Chcete-li prozkoumat ukázky C++, stáhněte si [klientskou knihovnu úložiště C++](https://www.nuget.org/packages/wastorage/) z NuGet.

* [Začínáme s objekty BLOB](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/BlobsGettingStarted)
* [Začínáme s tabulkami](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/TablesGettingStarted)
* [Začínáme s frontami](https://github.com/Azure/azure-storage-cpp/tree/master/Microsoft.WindowsAzure.Storage/samples/QueuesGettingStarted)

## <a name="azure-cli"></a>Azure CLI

Chcete-li prozkoumat ukázky příkazového příkazového příkazu k webu Azure, [nejprve nainstalujte příkazové příkazové příkazové nebo tonové.](/cli/azure/install-azure-cli)

* [Začínáme s azure CLI](/cli/azure/get-started-with-azure-cli)
* [Ukázky azure úložiště pomocí azure cli](/azure/storage/blobs/storage-samples-blobs-cli)
* [Oblíbené články o používání azure cli s Azure Storage](/cli/azure/popular-articles-using-the-azure-cli#storage)
