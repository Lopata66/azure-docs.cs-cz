# <a name="using-the-azure-importexport-tool"></a>Použití nástroje pro import a export Azure

Nástroj pro import a export Azure (WAImportExport.exe) se používá k vytváření a správě úloh pro službu Import a export Azure, což umožňuje přenášet velké množství dat do nebo z azure blob storage.

Tato dokumentace je pro nejnovější verzi nástroje Azure Import/Export Tool. Informace o použití klasického modelu nasazení najdete [v tématu Použití nástroje Pro import a export Azure v1](storage-import-export-tool-how-to-v1.md).

Následující články ukazují, jak:  

- Nainstalujte a nastavte nástroj pro import a export Azure.
- Připravte si pevné disky na úlohu, ve které importujete data z disků do úložiště objektů blob Azure.
- Zkontrolujte stav úlohy pomocí kopírovat soubory protokolu.
- Oprava úlohy importu.
- Oprava úlohy exportu.
- Poradce při potížích s nástrojem pro import a export Azure.
