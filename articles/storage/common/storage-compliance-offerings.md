# <a name="azure-storage-compliance-offerings"></a>Nabídky dodržování předpisů azure storage

Microsoft Azure & Azure Storage nabízí nejkomplexnější sadu certifikací a osvědčení ze všech poskytovatelů cloudových služeb, které organizacím pomáhají dodržovat národní, regionální a oborové požadavky, kterými se řídí shromažďování a používání dat jednotlivců.

Níže uvedené nabídky dodržování předpisů najdete na Azure Storage, abyste zajistili, že vaše služba bude regulována používáním služby Azure Storage. Platí pro následující nabídky Azure Storage: Objekty BLOB, Soubory, Fronty, Tabulky, Disky, Cool Storage a Úložiště Premium.
