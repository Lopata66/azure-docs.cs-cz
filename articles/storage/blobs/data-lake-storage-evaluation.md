# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Pokyny pro zákazníky Azure Data Lake Storage Gen1

Azure Data Lake Storage Gen2 Preview nabízí nativní objektu Blob úložiště funkce, včetně ovládání datových vrstev na data, správu životního cyklu a pokročilé zabezpečení sítě, navíc k dispozici v Data Lake Storage Gen1 systému souborů Hadoop kompatibilní.

Pokud jste stávající zákazník Gen1 úložiště Data Lake, Microsoft doporučuje, abyste k vyhodnocení, všechny nové a vylepšené možnosti systému Data Lake Storage Gen2. Data Lake Storage Gen1 navíc budou i nadále podporovány. Data Lake Storage Gen1 a Gen2 můžou existovat společně, takže existující prostředky a aplikace, které využívají Data Lake Storage Gen1 i nadále fungovat bez aktualizací. Můžete vyvíjet plán k upgradu vašeho řešení na Gen2 úložiště Data Lake vám to nejlépe hodí. Pokyny k upgradu, najdete v části [Upgrade vlastních analytických řešení pro velké objemy dat na verzi Preview služby Azure Data Lake Storage Gen2](data-lake-storage-upgrade.md).

Pokud vyhodnocujete aktuálně Gen1 úložiště Data Lake, Microsoft doporučuje, abyste k účasti ve verzi preview Gen2 a vyhodnocovat u nich nové a vylepšené možnosti nabízené službou Data Lake Storage Gen2. Podobně pokud jste stávající zákazník úložiště objektů Blob, Microsoft doporučuje, abyste k vyhodnocení, zda je systém souborů Hadoop kompatibilní vhodné pro potřeby analýzy.
