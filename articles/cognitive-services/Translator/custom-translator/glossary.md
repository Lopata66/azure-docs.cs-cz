# <a name="custom-translator-glossary"></a>Slovníček glosář na zakázku

Glosář [vlastního překladače](https://portal.customtranslator.azure.ai) vysvětluje termíny, se kterými se můžete setkat při práci s vlastním překladačem.

| **Slovo nebo fráze**       | **Definice**                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
|--------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Zdrojový jazyk          | Zdrojový jazyk je jazyk, se kterým začínáte a který chcete převést do jiného jazyka (dále jen "cíl").                                                                                                                                                                                                                                                                                                                                                         |
| Cílový jazyk          | Cílový jazyk je jazyk, který má strojový překlad poskytnout po přijetí zdrojového jazyka.                                                                                                                                                                                                                                                                                                                                               |
| Jednojazyčný soubor         | Jednojazyčný soubor má jeden jazyk, který není spárován s jiným souborem jiného jazyka.                                                                                                                                                                                                                                                                                                                                                                 |
| Paralelní soubory           | Paralelní soubor je kombinací dvou souborů s odpovídajícím textem. Jeden soubor má zdrojový jazyk. Druhý má cílový jazyk.                                                                                                                                                                                                                                                                                                                                         |
| Zarovnání věty       | Paralelní datová sada musí mít zarovnané věty s větami, které představují stejný text v obou jazycích. Například ve zdrojovém paralelním souboru by první věta měla teoreticky mapovat na první větu v cílovém paralelním souboru.                                                                                                                                                                                                                               |
| Zarovnaný text             | Jedním z nejdůležitějších kroků ověření souboru je zarovnání vět v paralelních dokumentech. Věci jsou vyjádřeny odlišně v různých jazycích. Také různé jazyky mají různé slovo objednávky. Tento krok provádí zarovnání vět se stejným obsahem, aby je bylo možné použít pro školení. Zarovnání nízké věty označuje, že s jedním nebo oběma soubory může být něco v nepořádku. |
| Rozdělení slov/ neprolomení | Rozdělení slov je funkce označení hranic mezi slovy. Mnoho systémů zápisu používá mezeru k označení hranice mezi slovy. Nerozdělení aplikace odkazuje na odebrání jakékoli viditelné značky, která mohla být vložena mezi slova v předchozím kroku.                                                                                                                                                                                                  |
| Oddělovače               | Oddělovače jsou způsoby, jak je věta rozdělena do segmentů nebo vymezena okraj mezi větami. Například v anglických prostorech vymezují slova, dvojtečky a středníky vymezují věty a tečky oddělují věty.                                                                                                                                                                                                                                         |
| Školicí soubory           | Školicí soubor se používá k výuce systému strojového překladu, jak mapovat z jednoho jazyka (zdroj) do cílového jazyka (cíl). Čím více dat můžete poskytnout, tím lépe bude systém fungovat při překladu.                                                                                                                                                                                                               |
| Ladění souborů             | Tyto soubory jsou často náhodně odvozeny z trénovací sady (pokud nevyberete žádnou sadu ladění). Automaticky vybrané věty se používají k vyladění systému a ujistěte se, že funguje správně. Pokud se rozhodnete vytvořit vlastní tuningové soubory, ujistěte se, že se jedná o náhodnou sadu vět napříč doménami, pokud chcete vytvořit univerzální překladový model.                                                                                 |
| Testování souborů            | Tyto soubory jsou často odvozené soubory, náhodně vybrané z trénovací sady (pokud nevyberete žádnou testovací sadu). Účelem těchto vět je vyhodnotit přesnost modelu překladu. Jedná se o věty, které chcete zajistit, aby systém přesně překládá. Takže možná budete chtít vytvořit testovací sadu a nahrát ji do překladače, abyste zajistili, že tyto věty budou použity v hodnocení systému (generování skóre BLEU).   |
| Kombinovaný soubor               | Typ souboru, ve kterém jsou zdrojové a přeložené věty obsaženy ve stejném souboru. Podporované formáty souborů (".tmx", ".xliff", ".xlf", ".lcl", ".xlsx").                                                                                                                                                                                                                                                                                                                       |
| Archivní soubor             | Soubor, který obsahuje další soubory. Podporované formáty souborů (zip, gz, tgz).                                                                                                                                                                                                                                                                                                                                                                                                |
| Skóre BLEU               | [BLEU](what-is-bleu-score.md) je standardní metoda hodnocení "přesnosti" nebo přesnosti překladového modelu. Ačkoli existují jiné metody hodnocení, Microsoft Translator spoléhá na metodu BLEU pro hlášení přesnosti vlastníkům projektu.
