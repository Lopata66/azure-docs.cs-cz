# <a name="glossary"></a>Glosář

## <a name="a"></a>A

### <a name="acoustic-model"></a>Akustický Model

Akustický model je klasifikátor, který označuje krátké zvukové fragmenty do jednoho několika fonémů, nebo zvukových jednotek v daném jazyce. Například slovo „speech“ je tvořeno čtyřmi fonémy „s p iy ch“. Tyto klasifikace se provádějí řádově 100krát za sekundu

## <a name="c"></a>C

### <a name="conversational-model"></a>Konverzační modelu

Vhodné pro rozpoznávání řeči, kterým se mluví ve stylu konverzační modelu. Microsoft Conversational AM je přizpůsobené pro zpracování řeči, které jsou obvykle zaměřená na jinou osobu.

### <a name="subscription-key"></a>Klíč předplatného

Klíč předplatného je řetězec, který je třeba zadat jako parametr řetězce dotazu, aby bylo možné volat jakékoli služby Custom Speech Service model. Klíč předplatného se získá z [webu Azure Portal](https://portal.azure.com/#create/Microsoft.CognitiveServices/apitype/CustomSpeech) a získat jednou je možné najít "Moje předplatné zobrazeno" na portálu služby Custom Speech Service.
