# <a name="bing-visual-search-sdk"></a>Pro vizuální vyhledávání Bingu SDK

Pro vizuální vyhledávání Bingu poskytuje následující klientské sady SDK, které můžete použít k rozhraní API pro vizuální vyhledávání.

* [Ukázky .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Balíček NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [Knihovny .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) pro definice a závislosti
* [Ukázky Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Knihoven Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) pro definice a závislosti
* [Ukázky v Javě](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Knihovny Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) pro definice a závislosti
* [Ukázky Pythonu](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Knihovny jazyka Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) pro definice a závislosti

Ukázky sady SDK pro jednotlivé jazyky zahrnují soubor ReadMe s předpoklady pro použití sady SDK spolu s kroky pro instalaci a spuštění ukázky.

Pro tyto ukázky, budete muset spustit si předplatné v cenové úrovni S9, jak je znázorněno v [Cognitive Services ceny – rozhraní API Bingu pro vyhledávání](https://azure.microsoft.com/en-us/pricing/details/cognitive-services/search-api/). 

K zahájení předplatného na webu Azure Portal:
1. Zadejte do textového pole v horní části webu Azure Portal s textem "BingSearchV7" `Search resources, services, and docs`.  
2. V části Marketplace v rozevíracím seznamu vyberte `Bing Search v7`.
3. Zadejte `Name` pro nový prostředek.
4. Vyberte `Pay-As-You-Go` předplatného.
5. Vyberte `S9` cenovou úroveň.
6. Klikněte na tlačítko `Enable` k zahájení odběru.

### <a name="sdk-tools"></a>Nástroje sady SDK
Nástroje sady SDK a prozkoumejte možnosti pro vizuální vyhledávání Bingu najdete v tomto úložišti Git: [nástrojů pro vizuální vyhledávání Bingu](https://github.com/Azure-Samples/bing-visualsearch-tools)
