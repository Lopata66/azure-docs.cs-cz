# <a name="how-to-sign-up-for-the-translator-speech-api"></a>Jak se zaregistrovat pro rozhraní Translator Speech API

Postupujte podle těchto kroků k registraci pro rozhraní Translator Speech API.

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-translator-speech-deprecation-note.md)]

## <a name="1-sign-into-azure"></a>1: Přihlaste se k Azure

- Nemáte účet? [Zaregistrujte si účet Microsoft Azure](http://azure.com/)
- Už máte účet? [Přihlásit se](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: Přihlaste se k Microsoft Translatoru odběru

- Po přihlášení k Azure, přejděte [Azure Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices) oddílu
- V části "Typ rozhraní API" Vyberte rozhraní Translator Speech API
- V části cenová úroveň vyberte cenovou úroveň, která nejlépe vyhovuje vašim potřebám
    - Každé předplatné má úroveň free. Úroveň free má stejné funkce a funkce jako placených plánů a nemá žádné datum vypršení platnosti.
    - Ve svém účtu můžete mít jenom jedno bezplatné předplatné
- Vyplňte zbývající část formuláři a klikněte na tlačítko Vytvořit
- Předplatné je připravené k okamžitému použití

## <a name="3-authentication-key"></a>3: ověřovací klíč

- Načíst ověřovací klíč tak, že přejdete do nabídky všechny prostředky > klikněte na předplatné
- Hodnota klíče se používá pro ověřování, ponechte hodnotu klíče důvěrné. Hodnota klíče, která potřebujete při vývoji vaší aplikace

- [Fórum podpory služby Microsoft Translator](http://www.aka.ms/TranslatorForum)
