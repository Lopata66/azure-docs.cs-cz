# <a name="sapi-phone-set-usage"></a>Použití telefonního sady SAPI

Microsoft definuje telefonní sadu SAPI pro sedm jazyků (EN-US, fr-FR, de-DE, ES-ES, ja-JP, zh-CN, zh-TW). Telefonní sada SAPI se poměrně snadno používá a zapamatuje se hlavně pro nativní reproduktory. Například ja-JP jako telefon používá kana, zh-CN jako telefonický zápis použijte PinYin. V některých případech ale není dostatečně výkonná, například telefonní sada SAPI v systému fr-FR nemůže představovat francouzskou spolupráci.
