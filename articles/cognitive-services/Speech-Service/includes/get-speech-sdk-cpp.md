#### <a name="c-nuget-package"></a>Balíček NuGet pro C++

Sadu C++ Speech SDK lze nainstalovat ze **Správce balíčků** pomocí následujícího `Install-Package` příkazu.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
