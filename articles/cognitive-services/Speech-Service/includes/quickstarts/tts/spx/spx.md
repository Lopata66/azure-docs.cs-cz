## <a name="run-the-speech-cli"></a>Spuštění funkce CLI pro rozpoznávání řeči

Teď jste připraveni spustit rozhraní příkazového řádku pro rozpoznávání řeči od textu.

V příkazovém řádku přejděte do adresáře, který obsahuje binární soubor CLI rozpoznávání řeči, a zadejte:

```bash
spx synthesize --text "The speech synthesizer greets you!"
```

Rozhraní příkazového řádku pro rozpoznávání řeči vytvoří přirozený jazyk v angličtině prostřednictvím mluvčího počítače.
