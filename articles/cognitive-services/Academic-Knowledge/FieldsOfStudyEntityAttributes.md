# <a name="field-of-study-entity"></a>Pole studijní entity

> [!NOTE]
> Následující atributy jsou specifické pro pole entity studie. (Ty = 6 ')

Název | Popis | Typ | Operace
--- | --- | --- | ---
CC      |Pole počtu citací celkem v studiu    |Datový typ Int32      |Žádný  
DFN     |Pole zobrazovaného názvu studie            |Řetězec     |Žádný
ECC     |Pole celkového počtu odhadovaných citací|Datový typ Int32      |Žádný
FL      |Úroveň v oblastech hierarchie studie     |Datový typ Int32      |Rovná se, mezi
FN      |Pole normalizovaného názvu studie         |Řetězec     |Rovná se
FC. Ukazující  |Podřízené pole ID studie                |Int64      |Rovná se
FC. VISTASCAN   |Podřízené pole s názvem studie              |Řetězec     |Rovná se
FP.FId  |Nadřazené pole s ID studie               |Int64      |Rovná se
FP.FN   |Nadřazené pole s názvem studie             |Řetězec     |Rovná se
ID      |ID entity                              |Int64      |Rovná se
PC    | Pole počtu publikací celkem pro studii | Datový typ Int32 | Žádný
