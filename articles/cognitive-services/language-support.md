# <a name="natural-language-support-for-azure-cognitive-services"></a>Podpora přirozeného jazyka pro Azure Cognitive Services

Azure Cognitive Services vám umožní sestavovat aplikace, které najdete v článku, slyšet, mluvit s a pochopit vaši uživatelé. Mezi tyto služby více než deseti tři podporují následující jazyky, uživatelé si můžou komunikovat s vaší aplikací přirozeným způsobem.

V tomto článku se dělí na dva oddíly. První seznam jazyků core běžně podporovány v rámci služeb Azure Cognitive Services. Druhý oddíl se zaměřuje na podpory dalších jazyků pro každou službu. Jazyk dostupnost se může lišit podle země/oblasti a na trhu.

## <a name="core-languages"></a>Základní jazyků

Tyto základní jazyky jsou podporovány v rámci služeb Azure Cognitive Services:

* Čínština
* Angličtina
* Francouzština
* Němčina
* italština
* Japonština
* Korejština 1
* Portugalština
* Španělština

> [!NOTE]
> 1 LUIS, Video Indexer, Analýza textu a převod řeči na text se nepodporuje.
