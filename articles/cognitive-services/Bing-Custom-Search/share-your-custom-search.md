# <a name="share-your-custom-search-instance"></a>Sdílet vaše instance vlastního vyhledávání

Můžete snadno povolit spolupráci, úpravu a testování vaší instance při sdílení se členy vašeho týmu. Vaše instance můžete sdílet s kýmkoli pomocí jenom e-mailová adresa. Chcete-li sdílet instanci:

- Přihlaste se k [vlastního vyhledávání](https://customsearch.ai)
- Vyberte instanci vlastního vyhledávání
- Klikněte na ikonu nastavení (zobrazí se jako ozubené kolo). 
- V části **sdílet vaše Instance**, zadejte e-mailovou adresu osoby, klikněte na tlačítko a sdílet vaše instance s **sdílet**. 

Po přidání e-mailovou adresu, se přidá **instanci sdílet s** seznamu. Postup opakujte pro každou osobu, která chcete sdílet vaše instance s. 

Chcete-li přidat uživatele e-mailu do seznamu, není nezbytné pro vlastní vyhledávání účtu. Musí se zaregistrovat pro vlastní vyhledávání, než se ale udělat změny konfigurace. Po sdílení instance s někým, zobrazí se v jejich seznam instancí vlastního vyhledávání. Jenom jedna osoba může změna instance současně. Pokud se pokusíte změnit instanci, která upravuje někdo jiný, zobrazí se upozornění. Instance je sdílet s maximálně 10 uživatelů.

## <a name="stop-sharing"></a>Ukončit sdílení

Pokud chcete ukončit sdílení instance s někým, použijte ikonu odebrat ze seznamu odebrat e-mailová adresa. To také odstraníte tuto instance ze svého seznamu instancí.
