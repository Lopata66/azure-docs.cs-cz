K připojení aplikace budete potřebovat klíč a koncový bod od Analýza textu prostředku. Získejte je vytvořením jednoho z níže uvedených prostředků:

* <a href="https://azure.microsoft.com/try/cognitive-services/my-apis/?api=text-analytics" title="vytvořit prostředek zkušebního Analýza textu" target="_blank">vytvořit prostředek <span class="docon docon-navigate-external x-hidden-focus"></span> zkušební verze</a>
    * Není potřeba žádné předplatné Azure. 
    * Platí po dobu sedmi dnů zdarma. Po registraci bude na [webu Azure](https://azure.microsoft.com/try/cognitive-services/my-apis/)dostupný zkušební klíč a koncový bod. 
    * To je skvělý způsob, pokud chcete vyzkoušet rozhraní API pro analýzu textu, ale nemáte předplatné Azure.
    
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesTextAnalytics"  title="vytvořit prostředek Analýza textu"  target="_blank">vytvořit Analýza textu prostředku <span class="docon docon-navigate-external x-hidden-focus"></span></a>
    * K dispozici, dokud prostředek neodstraníte.
    * Pomocí cenové úrovně Free můžete službu vyzkoušet a upgradovat ji později na placenou úroveň pro produkční prostředí.
     
* <a href="https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne"  title="vytvořit prostředek pro více služeb" target="_blank">vytvoření prostředku <span class="docon docon-navigate-external x-hidden-focus"></span> s více službami</a>
    * K dispozici, dokud prostředek neodstraníte.  
    * Pro přístup k více Cognitive Services použijte stejný klíč a koncový bod.
