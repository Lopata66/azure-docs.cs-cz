# <a name="what-is-the-linguistic-analysis-api-preview"></a>Co je rozhraní API pro jazykovou analýzu? (Preview)

> [!IMPORTANT]
> Dne 9. srpna 2018 došlo k vyřazení jazykové analýzy ve verzi Preview z provozu. Ke zpracování a analýze textu doporučujeme používat [moduly analýzy textu služby Azure Machine Learning](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics).

Rozhraní API pro jazykovou analýzu poskytuje přístup k nástrojům pro zpracování přirozeného jazyka, které identifikují strukturu textu. Aktuální vydaná verze poskytuje tři druhy analýzy:

- [Rozdělení a tokenizace vět](Sentences-and-Tokens.md)
- [Označování částí řeči](POS-tagging.md)
- [Analýza konstituentů](Constituency-Parsing.md)

## <a name="getting-started"></a>Začínáme

Až se seznámíte s výše uvedenými hlavními typy analýzy, můžete se podívat do koncového bodu se [seznamem dostupných analyzátorů](AnalyzersMethod.md) a způsobem jejich [pojmenování](Analyzer-Names.md) a potom do koncového bodu pro [analýzu nového vstupu](AnalyzeMethod.md).
