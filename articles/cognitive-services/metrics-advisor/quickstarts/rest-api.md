# <a name="quickstart-use-the-rest-apis-to-customize-your-solution"></a>Rychlý Start: použití rozhraní REST API k přizpůsobení řešení

Začněte s REST API Poradce pro metriky. Pomocí těchto kroků nainstalujete balíček a vyzkoušíte ukázkový kód pro základní úlohy.

Použijte poradce metriky k provedení těchto akcí:

* Přidání datového kanálu ze zdroje dat
* Ověřit stav ingestování
* Konfigurace zjišťování a výstrah 
* Dotaz na výsledky detekce anomálií
* Diagnostika anomálií
