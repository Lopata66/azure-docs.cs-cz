# <a name="visualize-your-graph-data-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Vizualizace dat grafu v rozhraní Gremlin API služby Azure Cosmos DB pomocí řešení pro vizualizaci dat

Následující řešení jsou doporučení [Apache Tinkerpop komunity](http://tinkerpop.apache.org/#poweredby) pro vizualizaci grafu Gremlin.

## <a name="linkurious-enterprise"></a>Linkurious Enterprise

![Linkurious Logo](./media/graph-visualization/linkurious-logo.jpg)

Linkurious Enterprise používá grafu technologie a vizualizaci dat pro komplexní datové sady na interaktivní vizuální sítě. Platforma se připojí ke zdrojům dat a umožňuje výzkumníků bez problémů přejít mezi miliardami entit a vztahů. Výsledkem je nová schopnost detekovat podezřelé relace bez opomenou s dotazy nebo tabulky.

Interaktivní vizualizace rozhraní Linkurious Enterprise nabízí snadný způsob, jak prozkoumat komplexních datových připojení. Vyhledat konkrétní entity, rozbalte připojení odhalit skryté vztahy a použití rozložení rozplétání složité sítě podle vaší volby. Linkurious je teď kompatibilní s Gremlin API služby Azure Cosmos DB. Je vhodné pro scénáře, vizualizace grafu na začátku do konce a podporuje čtení a zápis možnosti z uživatelského rozhraní. 

![Ukázka Linkurious](./media/graph-visualization/linkurious-demo.gif)

**Další kroky:** Můžete požádat [ukázku Linkurious pomocí služby Azure Cosmos DB](https://linkurio.us/contact/).

## <a name="cambridge-intelligence"></a>Cambridge Intelligence

![Logo Cambridge Intelligence](./media/graph-visualization/ci-logo.png)


Sady Cambridge Intelligence grafu vizualizace nástrojů teď podporují službu Azure Cosmos DB:
- [Orámování pro vývojáře v JavaScriptu](https://cambridge-intelligence.com/keylines/) 
- [Znovu Graph pro vývojáře v React](https://cambridge-intelligence.com/regraph/)

![Ukázka Cambridge Intelligence](./media/graph-visualization/ci-demo-2.gif)

Sady nástrojů umožňují navrhovat grafu výkonné vizualizace a analýza aplikací pro případy použití. Využijte výkonný vykreslování WebGL a pečlivě vytvořený kód a poskytuje tak uživatelům rychlý a přehledné vizualizace zkušeností.

Že jsou kompatibilní s prohlížečem, zařízení, serveru nebo databáze a jsou dostupné podrobné kurzy, plně zdokumentovaných rozhraní API a ukázky interaktivní.
