# <a name="visualize-graph-data-stored-in-azure-cosmos-db-gremlin-api-with-data-visualization-solutions"></a>Vizualizace dat grafu uložených v rozhraní AZURE Cosmos DB Gremlin API s řešeními pro vizualizaci dat

Data uložená v rozhraní Azure Cosmos DB Gremlin API můžete vizualizovat pomocí různých řešení pro vizualizaci dat. Následující řešení jsou doporučena [komunitou Apache Tinkerpop](https://tinkerpop.apache.org/#poweredby) pro vizualizaci dat grafu.

## <a name="linkurious-enterprise"></a>Linkurious Podnik

![Linkurious Logo](./media/graph-visualization/linkurious-logo.jpg)

[Linkarious Enterprise](https://linkurio.us/product/) využívá technologii grafů a vizualizaci dat k přeměně složitých datových sad na interaktivní vizuální sítě. Platforma se připojuje k vašim zdrojům dat a umožňuje vyšetřovatelům bezproblémovou navigaci mezi miliardami entit a vztahů. Výsledkem je nová schopnost detekovat podezřelé relace bez žonglování s dotazy nebo tabulkami.

Interaktivní rozhraní Linkurious Enterprise nabízí snadný způsob zkoumání komplexních dat. Můžete vyhledat konkrétní entity, rozšířit připojení, abyste odhalili skryté vztahy, a použít rozvržení podle vašeho výběru k rozmotání složitých sítí. Linkarious Enterprise je teď kompatibilní s rozhraním Azure Cosmos DB Gremlin API. Je vhodný pro scénáře vizualizace grafu začátku do konce a podporuje možnosti čtení a zápisu z uživatelského rozhraní. Můžete požádat [o ukázku Linkurious s Azure Cosmos DB](https://linkurio.us/contact/)

![Linkurious Demo](./media/graph-visualization/linkurious-demo.gif)

## <a name="cambridge-intelligence"></a>Cambridge inteligence

![Logo Cambridge Intelligence](./media/graph-visualization/ci-logo.png)

Sady nástrojů pro vizualizaci [grafů cambridge intelligence](https://cambridge-intelligence.com/products/) teď podporují Azure Cosmos DB. Azure Cosmos DB podporuje následující dvě vizualizační sady nástrojů:

- [KeyLines pro vývojáře JavaScriptu](https://cambridge-intelligence.com/keylines/)

- [Re-Graph pro vývojáře React](https://cambridge-intelligence.com/regraph/)

![Cambridge Intelligence Ukázka](./media/graph-visualization/ci-demo-2.gif)

Tyto sady nástrojů umožňují navrhovat vysoce výkonné aplikace pro vizualizaci a analýzu grafů pro váš případ použití. Využívají výkonné vykreslování webové grafické knihovny (WebGL) a pečlivě vytvořený kód, aby uživatelům poskytli rychlé a zasvěcené prostředí pro vizualizaci. Tyto nástroje jsou kompatibilní s jakýmkoli prohlížečem, zařízením, serverem nebo databází a jsou dodávány s podrobnými kurzy, plně zdokumentovanými rozhraními API a interaktivními ukázkami.

![Cambridge Intelligence Ukázka](./media/graph-visualization/ci-demo-1.gif)
