# <a name="azure-cosmos-db-query-cheat-sheets"></a>Tabulky pro podvody s dotazy Azure Cosmos DB

**Pracovní listy dotazů Azure Cosmos DB** vám pomůžou rychle psát dotazy na vaše data zobrazením běžných databázových dotazů, operací, funkcí a operátorů v snadno tiskových referenčních listech PDF. Cheat listy obsahují referenční informace pro SQL, MongoDB, tabulka a Gremlin API. 

Vyberte si ze stažení velikosti Letter nebo A3. 

## <a name="letter-sized-cheat-sheets"></a>Dopis-velké cheat listy

Pokud chcete tisknout na papír velikosti letter (8,5 x 11), stáhněte si [informační listy dotazů velikosti Azure Cosmos DB.](https://go.microsoft.com/fwlink/?LinkId=623215)

![Pracovní listy dotazů Azure Cosmos DB – velikost i velikost písmen, s dotazy a funkcemi rozhraní API SQL API, JavaScriptu, MongoDB, Gremlinu a rozhraní API pro tabulky](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Nadrozměrné cheat listy
Stáhněte si [listy pro dotazování velikosti Azure Cosmos DB A3,](https://go.microsoft.com/fwlink/?linkid=870413) pokud budete tisknout pomocí plotru nebo velkoplošné tiskárny na papír velikosti A3 (11,7" x 16,5").

![Pracovní listy dotazů Azure Cosmos DB – velikost i 3, s dotazy a funkcemi rozhraní SQL API, JavaScript, MongoDB, Gremlin a Table API](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
