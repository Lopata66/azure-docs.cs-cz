# <a name="billing-questions"></a>Dotazy k fakturaci

## <a name="how-does-billing-work-for-azure-security-center"></a>Jak funguje fakturace pro Azure Security Center?
Security Center se nabízí s Azure Defenderem a bez těchto akcí:

- **Bez použití Azure defenderu**Security Center poskytuje přehled o stavu zabezpečení vašich prostředků Azure, základní zásady zabezpečení, doporučení zabezpečení a integraci s produkty a službami zabezpečení od partnerů.

- **Azure Defender** přidává funkce ochrany před hrozbami, které zahrnují výstrahy zabezpečení, analýzy hrozeb, analýzu chování, detekci anomálií a sestavy týkající se hrozeb. Můžete začít bezplatnou 30denní zkušební verzi Azure Defenderu. [Přečtěte si další informace o upgradu](security-center-pricing.md).

## <a name="how-can-i-track-who-in-my-organization-enabled-azure-defender-changes-in-azure-security-center"></a>Jak mohu sledovat, kdo v organizaci povolil změny v Azure Defenderu v Azure Security Center
K předplatným Azure může mít několik správců oprávnění ke změně nastavení cen. Chcete-li zjistit, který uživatel provedl změnu, použijte protokol aktivit Azure. 

Další informace najdete [tady](https://techcommunity.microsoft.com/t5/Security-Identity/Tracking-Changes-in-the-Pricing-Tier-for-Azure-Security-Center/td-p/390832).
