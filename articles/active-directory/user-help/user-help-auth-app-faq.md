# <a name="frequently-asked-questions-faqs-about-the-microsoft-authenticator-app"></a>Nejčastější dotazy týkající se aplikace Microsoft Authenticator

Tento článek odpovídá na běžné otázky týkající se aplikace Microsoft Authenticator. Pokud odpověď na svou otázku nevidíte, přejděte na [fórum aplikací Microsoft Authenticator](https://social.technet.microsoft.com/Forums/en-US/home?forum=MicrosoftAuthenticatorApp).

Aplikace Microsoft Authenticator nahradila aplikaci Azure Authenticator a je doporučenou aplikací při použití Azure Multi-Factor Authentication. Aplikace Microsoft Authenticator je k dispozici pro [Android](https://app.adjust.com/e3rxkc_7lfdtm?fallback=https%3A%2F%2Fplay.google.com%2Fstore%2Fapps%2Fdetails%3Fid%3Dcom.azure.authenticator) a [iOS](https://app.adjust.com/e3rxkc_7lfdtm?fallback=https%3A%2F%2Fitunes.apple.com%2Fus%2Fapp%2Fmicrosoft-authenticator%2Fid983156458).

| Proč vám aplikace Microsoft Authenticator umožňuje schválit žádost bez odemknutí zařízení? | Nemusíte odemykat zařízení, abyste mohli schvalovat žádosti o ověření, protože vše, co potřebujete, je, abyste dokázali, že máte telefon u sebe.
