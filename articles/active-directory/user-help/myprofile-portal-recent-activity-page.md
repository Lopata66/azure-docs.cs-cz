# <a name="view-your-recent-sign-in-activity-on-the-my-profile-preview-portal"></a>Zobrazit nedávné přihlašovacích aktivit na portálu Můj profil (preview)
Můžete zobrazit všechny nedávné aktivity přihlášení pomocí pracovního nebo školního účtu z **poslední aktivitu na** stránku **Můj profil** portálu.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-recent-sign-in-activity"></a>Zobrazit poslední přihlašovací aktivita
1. Přihlaste se ke svému pracovnímu nebo školnímu účtu a potom přejděte k vaší https://myprofile.microsoft.com/ stránky.

2. Vyberte **poslední aktivitu na** z levého navigačního podokna nebo vyberte **kontrolní aktivita** propojit **poslední aktivitu na** bloku.

    ![Stránka Moje profilu zobrazující zvýrazněný odkazy nedávné aktivity](media/myprofile-portal/myprofile-portal-recent-activity-links.png)

3. Rozbalte položku a prostudujte si všechna položek přihlášení, ujistěte se, že rozpoznat každé z nich. Pokud zjistíte přihlášení položku, která nebude vypadat povědomě, důrazně doporučujeme, že abyste si změnili heslo k ochraně vašeho účtu, pokud došlo k ohrožení.

    ![Stránce s nedávnou aktivitou s Rozšířené podrobnosti o přihlášení](media/myprofile-portal/myprofile-portal-recent-activity-page.png)

    >[!Note]
    >Pokud si myslíte, že došlo k napadení vašeho účtu, přečtěte si téma [jak získat přístup k ohrožení bezpečnosti účtu Microsoft](https://support.microsoft.com/help/10494/microsoft-account-how-to-access-a-compromised-account) další pomoc s resetování hesla nebo odblokování účtu.

## <a name="next-steps"></a>Další postup

- Po zobrazení poslední přihlašovací aktivitu, můžete vybrat možnost zobrazovat a spravovat vaše [bezpečnostní údaje](user-help-security-info-overview.md), propojená [zařízení](myprofile-portal-devices-page.md) nebo [organizace](myprofile-portal-organizations-page.md), nebo pokud chcete zobrazit jak vaše organizace [používá data](myprofile-portal-privacy-page.md).
