# <a name="how-to-configure-endpoints"></a>Konfigurace koncových bodů

Koncové body ověřování pro vaši aplikaci najdete na [webu Azure Portal](https://portal.azure.com).

-   Přejděte na [Azure Portal](https://portal.azure.com).

-   V levém navigačním podokně klepněte na **položku Azure Active Directory**.

-   Klikněte na **Registrace aplikací** a zvolte **Koncové body**.

-   Tím otevřete stránku **Koncové body,** která obsahuje seznam všech koncových bodů ověřování pro vašeho tenanta.

-   Použijte koncový bod specifický pro ověřovací protokol, který používáte, ve spojení s ID aplikace k vytvoření požadavku na ověření specifického pro vaši aplikaci.
