# <a name="how-to-configure-endpoints"></a>Postup konfigurace koncových bodů

Koncové body ověřování pro vaši aplikaci najdete v [Azure Portal](https://portal.azure.com).

-   Přejděte na [Azure Portal](https://portal.azure.com).

-   V levém navigačním podokně klikněte na **Azure Active Directory**.

-   Klikněte na **Registrace aplikací** a vyberte **koncové body**.

-   Tím otevřete stránku **koncové body** , kde najdete seznam všech koncových bodů ověřování pro vašeho tenanta.

-   Použijte koncový bod specifický pro protokol ověřování, který používáte, ve spojení s ID aplikace, abyste mohli vytvořit žádost o ověření specifickou pro vaši aplikaci.
