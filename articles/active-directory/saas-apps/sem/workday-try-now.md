# <a name="workday-and-azure-active-directory"></a>Pracovní den a Azure Active Directory

Připojte pracovní den pomocí služby Azure Active Directory (Azure AD) s využitím naší předem připravené integrace a automaticky zřídí nebo zruší zřízení uživatelských účtů, když se zaměstnanci připojí, přesunou nebo odejdou.

![Zobrazuje logo aplikace v Workday.](./media/workday.png)

> [!div class="nextstepaction"]
> [Vyzkoušejte si Workday pomocí Azure AD >](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Co se dá dělat s Workday a Azure AD?

- **Automatizace zřizování uživatelů** – automaticky přiřazují aplikace zaměstnancům na základě jejich skupiny, role nebo jiných atributů v Workday.
- **Synchronizovat profily zaměstnanců** – Ujistěte se, že se data o zaměstnancích mezi Azure AD a Workday budou udržovat v aktuálním stavu.
- **Zjednodušení připojování a** oddálení – Automatické zřizování uživatelských účtů pro nová přijetí, aby mohly být produktivní na den a snížily rizika zabezpečení tím, že se při tom, co zaměstnanci odejdou, automaticky zruší.
- **Snadné jednotné přihlašování** Eliminujte uživatelské jméno a hesla a Umožněte uživatelům, aby se přihlásili pomocí svých účtů organizace hostovaných ve službě Azure AD.

## <a name="get-started"></a>Začínáme

Nastavte [zřizování uživatelů](https://aka.ms/WorkdayProvisioning) a [jednotné přihlašování](https://aka.ms/WorkdaySSODoc) pro Workday pomocí našich jednoduchých průvodců konfigurací.
