# <a name="box-and-azure-active-directory"></a>Box a Azure Active Directory

Zrychlete přijímání krabic napříč vaší organizací a zvyšte přijetí uživatelů s naší předem připravenou integrací. Připojení box k Azure Active Directory (Azure AD) umožňuje uživatelům rychlou spolupráci.

![Zobrazuje logo aplikace box.](./media/box.png)

> [!div class="nextstepaction"]
> [Zkuste box s Azure AD >](https://aka.ms/BoxTryItNow)

## <a name="what-can-you-do-with-box-and-azure-ad"></a>Co se dá dělat s polem a Azure AD?

- **Snadné jednotné přihlašování** – eliminuje více uživatelských jmen a hesel a zjednoduší přístup tím, že se uživatelům umožní přihlásit se k boxu pomocí svých účtů Azure AD.
- **Automatizace zřizování uživatelů** – umožňuje automatizovat vytváření, údržbu a odebírání uživatelských účtů do boxu, když se uživatelé připojí, přesunou nebo odejdou z vaší organizace.
- **Zabezpečte, řídí a monitorujte přístup** – zabezpečený přístup k boxu pomocí zásad podmíněného přístupu, řízení přístupu prostřednictvím kontrol přístupu a monitorování přístupu pomocí protokolů přihlášení.

## <a name="get-started"></a>Začínáme

Pomocí našich jednoduchých průvodců konfigurací nastavte [zřizování uživatelů](https://aka.ms/BoxSSODoc) a [jednotné přihlašování](https://aka.ms/BoxProvisioningDoc) pro box.
