# <a name="servicenow-and-azure-active-directory"></a>ServiceNow a Azure Active Directory

Zrychlete přijímání ServiceNow napříč vaší organizací a zvyšte přijetí uživatelů díky naší předem připravené integraci. Připojte ServiceNow k Azure Active Directory (Azure AD) a Zjednodušte svou správa IT služeb.

![Zobrazuje logo aplikace ServiceNow.](./media/servicenow.png)

> [!div class="nextstepaction"]
> [Vyzkoušejte si ServiceNow s Azure AD >](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>Co se dá dělat s ServiceNow a Azure AD?

- **Snadné jednotné přihlašování** – eliminuje více uživatelských jmen a hesel a zjednoduší přístup tím, že se uživatelé budou moct přihlašovat k ServiceNow pomocí svých účtů Azure AD.
- **Automatizace zřizování uživatelů** – automatizace vytváření, údržby a odebírání uživatelských účtů do ServiceNow, když se uživatelé připojí, přesunou nebo odejdou z vaší organizace.
- **Zabezpečte, řídí a monitorujte přístup** – zabezpečený přístup k ServiceNow pomocí zásad podmíněného přístupu, řízení přístupu prostřednictvím kontrol přístupu a monitorování přístupu pomocí protokolů přihlášení.

## <a name="get-started"></a>Začínáme

Nastavte [zřizování uživatelů](https://aka.ms/ServiceNowProvisioning) a [jednotné přihlašování](https://aka.ms/ServiceNowSSO) pro ServiceNow pomocí našich jednoduchých konfiguračních průvodců.
