# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet a Azure Active Directory

Zrychlete přijímání Smartsheet napříč vaší organizací a zvyšte přijetí uživatelů díky naší předem připravené integraci. Připojení Smartsheet k Azure Active Directory (Azure AD) umožňuje uživatelům rychlou spolupráci.

![Zobrazuje logo aplikace Smartsheet.](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Vyzkoušejte si Smartsheet s Azure AD >](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Co se dá dělat s Smartsheet a Azure AD?

- **Snadné jednotné přihlašování** – eliminuje více uživatelských jmen a hesel a zjednoduší přístup tím, že se uživatelé budou moct přihlašovat k Smartsheet pomocí svých účtů Azure AD.
- **Automatizace zřizování uživatelů** – automatizace vytváření, údržby a odebírání uživatelských účtů do Smartsheet, když se uživatelé připojí, přesunou nebo odejdou z vaší organizace.
- **Zabezpečte, řídí a monitorujte přístup** – zabezpečený přístup k Smartsheet pomocí zásad podmíněného přístupu, řízení přístupu prostřednictvím kontrol přístupu a monitorování přístupu pomocí protokolů přihlášení.

## <a name="get-started"></a>Začínáme

Nastavte [zřizování uživatelů](https://aka.ms/SmartsheetProvisioningDoc) a [jednotné přihlašování](https://aka.ms/SmartsheetSSODoc) pro Smartsheet pomocí našich jednoduchých konfiguračních průvodců.
