# <a name="how-to-configure-an-application-proxy-application-to-use-pingaccess"></a>Konfigurace aplikace Proxy aplikace pro použití technologie PingAccess

Naše spolupráce s PingAccess nyní umožňuje rozšířit výhody proxy aplikací na aplikace pomocí ověřování na základě záhlaví. Pokud vaše aplikace záhlaví nepoužívají, přečtěte si naši dokumentaci k [jednotnému přihlašování,](application-proxy-configure-single-sign-on-with-kcd.md) kde najdete podrobnosti o dalších možnostech.

## <a name="overview-of-steps-and-recommended-documents"></a>Přehled kroků a doporučených dokumentů

Chcete-li nakonfigurovat aplikaci pomocí technologie PingAccess, existují čtyři kroky:

1.  Konfigurace konektorů proxy aplikací

2.  Vytvoření aplikace proxy aplikace Azure AD

3.  Stáhnout & Konfigurace pingaccessu

4.  Konfigurace aplikací v aplikaci PingAccess

Podrobnosti o každém z těchto kroků naleznete v [dokumentaci k jednotnému přihlašování s záhlavími](application-proxy-configure-single-sign-on-with-ping-access.md).
