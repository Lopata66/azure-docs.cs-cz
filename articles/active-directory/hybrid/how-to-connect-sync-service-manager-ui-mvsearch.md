# <a name="sync-service-manager-metaverse-search"></a>Správce synchronizace služby vyhledávání Metaverse

![Správce synchronizace služby](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Karta vyhledávání metaverse je užitečné při řešení problémů souvisejících s daty. V horní polovině můžete vytvořit dotaz na základě kombinace atributů. Pokud jste spokojeni s dotazem, klikněte na tlačítko **hledání**. Výsledkem je viditelný v mřížce dolů. Můžete vybrat sloupce, které by měl být viditelný spolu s **nastavení sloupců**.

Ve výsledcích hledání vyberte objekt a **vlastnosti** zobrazíte [vlastnosti objektu úložiště metaverse](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
