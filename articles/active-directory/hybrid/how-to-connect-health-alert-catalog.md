# <a name="azure-active-directory-connect-health-alert-catalog"></a>Katalog výstrah služby Azure Active Directory Connect 

Služba Azure AD Connect Health odesílá výstrahy označují, že infrastruktura identit není v pořádku. Tento článek obsahuje názvy výstrah, popisy a nápravné kroky pro každou výstrahu. <br />
Chyba, upozornění a předběžné upozornění jsou tři fáze výstrah, které jsou generovány ze služby Connect Health. Důrazně doporučujeme provést okamžitá opatření u aktivovaných výstrah. <br />
Výstrahy stavu Azure AD Connect se vyřeší za podmínek úspěchu. Agenti stavu Azure AD Connect pravidelně zjišťují a hlásí podmínky úspěchu službě. U několika výstrah je potlačení založeno na čase. Jinými slovy, pokud stejný chybový stav není pozorován do 72 hodin od generování výstrahy, výstraha je automaticky vyřešena.

Poslední data zpracovaná službou Health Service jsou starší než 2 hodiny. | Ujistěte se, že agenti stavu mají odchozí připojení k požadovaným koncovým bodům služby.
