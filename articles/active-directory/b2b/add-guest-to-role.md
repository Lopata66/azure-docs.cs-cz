# <a name="grant-permissions-to-users-from-partner-organizations-in-your-azure-active-directory-tenant"></a>Udělení oprávnění uživatelům z partnerských organizací ve vašem tenantovi Služby Azure Active Directory

Uživatelé spolupráce Služby Azure Active Directory (Azure AD) B2B se do adresáře přidají jako uživatelé typu Host a oprávnění typu hosta v adresáři jsou ve výchozím nastavení omezena. Vaše firma může potřebovat některé uživatele typu Host k obsazení rolí s vyššími oprávněními ve vaší organizaci. Pro podporu definování rolí s vyššími oprávněními mohou být uživatelé typu Host přidáni do všech rolí, které si přejete, na základě potřeb vaší organizace.
