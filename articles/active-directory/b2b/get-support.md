# <a name="getting-support-for-b2b-collaboration"></a>Získání podpory pro spolupráci B2B

Po přečtení článku dokumentace, provedli se správnými věcmi, ale nedaří se vám něco na práci? Vytvořit lístek podpory (vyžaduje plán podpory):

1. V [na webu Azure portal](https://portal.azure.com), přejděte do okna nápovědy a podpory a vyberte Nová žádost o podporu:
  - Typ problému: technické
  - Předplatné: Vyberte vliv na předplatné
    - Služba: Active Directory
    - Plán podpory: Zvolte plán podpory relevantní

  ![Nová žádost o podporu](media/get-support/new-support-request.png)

2. Popište svůj problém:
  - Zvolte odpovídající závažnost, která zohledňuje, které potřebuje.
  - Zvolte typ problému jako Správa uživatelů a skupin
  - Zvolit kategorii jako přidání uživatelů (B2B)
  - Zahrnout všechny chybové zprávy jako ID korelace, ovlivněných uživatelů a tak dále.

  ![popis problému lístku podpory](media/get-support/problem-description.png)

3. Zástupce podpory můžete požádat o další informace o řešení přidáte svoje kontaktní informace.
