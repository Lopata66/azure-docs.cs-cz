# <a name="find-help-and-open-a-support-ticket-for-azure-active-directory"></a>Vyhledání nápovědy a otevření lístku podpory pro Službu Azure Active Directory
Microsoft poskytuje globální technickou podporu, předprodejní, fakturační a předplacenou podporu pro Azure Active Directory (Azure AD). Podpora je k dispozici online i telefonicky pro placené a zkušební předplatné Microsoft Azure. V dalších jazycích je k dispozici podpora po telefonu a online podpora k fakturaci. 

## <a name="find-help-without-opening-a-support-ticket"></a>Získání pomoci bez otevření lístku podpory

Než budete vytvářet lístek podpory, zkuste najít odpovědi a informace v následujících zdrojích informací. 

* Obsah, jako jsou informace o návodu nebo ukázky kódu pro odborníky v oblasti IT a vývojáře, naleznete [v technické dokumentaci na docs.microsoft.com](https://docs.microsoft.com/azure/active-directory/).

* [Technická komunita Společnosti Microsoft](https://techcommunity.microsoft.com/) je místem, kde mohou naši partneři a zákazníci v oddělení IT a zákazníků spolupracovat, sdílet a učit se. [Centrum technické komunity společnosti Microsoft](https://techcommunity.microsoft.com/t5/Community-Info-Center/ct-p/Community-Info-Center) se používá pro oznámení, příspěvky na blogu, interakce s odborníky a další interakce s odborníky . Můžete se také [připojit ke komunitě, aby předložily své nápady](https://techcommunity.microsoft.com/t5/Communities/ct-p/communities).


## <a name="open-a-support-ticket"></a>Otevření lístku podpory

Pokud se vám nedaří najít odpovědi pomocí svépomocných prostředků, můžete otevřít lístek online podpory. Měli byste otevřít každý lístek podpory pouze pro jeden problém, abychom vás mohli připojit k technikům podpory, kteří jsou odborníky na předmět vašeho problému. Technické týmy Azure Active Directory také upřednostňují svou práci na základě vygenerovaných incidentů, takže často přispíváte k vylepšením služeb.

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-azure-portal"></a>Jak otevřít lístek podpory pro Azure AD na webu Azure Portal

> [!NOTE]
> V případě problémů s fakturací nebo předplatným je nutné použít [Centrum pro správu Microsoftu 365](https://admin.microsoft.com).
> 

1. Přihlaste se k [portálu Azure a](https://portal.azure.com) otevřete **Službu Azure Active Directory**.
   
2. Posuňte se dolů na **Poradce při potížích + Podpora** a vyberte Nový **požadavek na podporu**.
   
3. V okně **Základy** vyberte pro **typ problému** **položku Technical**.

4. Vyberte **předplatné**.

5. V **případě služby**vyberte **položku Azure Active Directory**.

6. Vytvořte **souhrn** pro požadavek. Souhrn musí být méně než 140 znaků.
  
7. Vyberte **typ problému**a pak vyberte kategorii pro tento typ. V tomto okamžiku jsou také nabízeny svépomocné informace pro kategorii problémů.
  
7. Přidejte zbývající informace o problému a klepněte na tlačítko **Další**. 

8. V tomto okamžiku jsou vám v čepeli **Řešení** nabídnuta svépomocná řešení a dokumentace. Pokud žádný z řešení tohoto problému nevyřeší, klepněte na tlačítko **Další**. 

9. V okně **Podrobnosti** vyplňte požadované podrobnosti a vyberte [závažnost](https://azure.microsoft.com/support/plans/response/). 
 
![image](https://user-images.githubusercontent.com/13383753/76565580-1c284900-6468-11ea-8c0f-85af98097b6f.png)
 
10. Zadejte své kontaktní údaje a vyberte **další**. 


8. Zadejte kontaktní informace a vyberte **Vytvořit**.
  ![Snímek obrazovky s váma nepomocnou kategorií](./media/active-directory-troubleshooting-support-howto/open-support-ticket.png)

### <a name="how-to-open-a-support-ticket-for-azure-ad-in-the-microsoft-365-admin-center"></a>Jak otevřít lístek podpory pro Azure AD v Centru pro správu Microsoftu 365

> [!NOTE]
> Podpora azure ad v [Centru pro správu Microsoft 365 je](https://admin.microsoft.com) nabízena jenom pro správce. 

1. Přihlaste se do [Centra pro správu Microsoft u 365](https://admin.microsoft.com) pomocí účtu, který má licenci Enterprise Mobility + Security (EMS).

2. Na dlaždici **Podpora** vyberte **Nový požadavek na službu**:

3. Na stránce **Přehled podpory** vyberte Správa identit nebo **Správa** **uživatelů a domén**:

4. V **části Funkce**vyberte funkci Azure AD, pro kterou chcete podporu.

5. V **části Příznak**vyberte vhodný příznak, shrňte problém a uveďte příslušné podrobnosti a pak vyberte **Další**.

6. Vyberte jeden z nabízených samoobslužných zdrojů nebo vyberte **Ano, pokračovat** nebo **Ne, zrušit požadavek**.

7. Pokud budete pokračovat, budete požádáni o další podrobnosti. Můžete připojit všechny soubory, které představují problém, a pak vybrat **další**.

8. Zadejte své kontaktní údaje a vyberte **Odeslat žádost**.
