# <a name="rest-api-specification-for-azure-front-door-service"></a>Specifikace REST API pro službu Azure branou
Služba Azure branou vám umožňuje definovat, spravovat a monitorovat globální směrování pro webový provoz optimalizace pro zajištění nejlepšího výkonu a rychlé globální převzetí služeb při selhání pro zajištění vysoké dostupnosti. S branou můžete transformovat globální příjemce (více oblastí) a podnikové aplikace do robustní a výkonné individuální moderních aplikací, rozhraní API a obsah, který najdou v Azure.


Následující tabulka obsahuje odkaz pro rozhraní REST API pro službu Azure branou.

Verze rozhraní API: 2018-08-01

| Operace skupiny | Popis |
|-------------|------------|
| [Přední dveře](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/frontdoor.json) | Definuje, jak můžete vytvořit a spravovat konfiguraci branou |
| [Zásady brány Firewall webových aplikací](https://github.com/Azure/azure-rest-api-specs/blob/master/specification/frontdoor/resource-manager/Microsoft.Network/preview/2018-08-01-preview/webapplicationfirewall.json) | Definuje, jak spravovat zásady brány firewall webových aplikací pro vaše branou |
