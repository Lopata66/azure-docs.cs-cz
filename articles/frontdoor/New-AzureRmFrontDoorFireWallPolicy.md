# Nové AzureRmFrontDoorFireWallPolicy
Vytvoření zásad WAF

## SYNTAXE

```
New-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> -Name <String> [-EnabledState <PSEnabledState>]
 [-Mode <PSMode>] -Customrule <PSCustomRule[]> [-ManagedRule <PSManagedRule[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## POPIS
**New-AzureRmFrontDoorFireWallPolicy** rutina vytvoří novou zásadu waf služby Azure v zadané skupině prostředků v rámci aktuálního předplatného
