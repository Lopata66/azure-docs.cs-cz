# <a name="overview-of-the-pci-dss-v321-blueprint-sample"></a>Přehled ukázky podrobného plánu PCI-DSS v3.2.1

Ukázka podrobného plánu PCI-DSS v3.2.1 je sada zásad, které pomáhají zajistit dodržování předpisů PCI-DSS v3.2.1. Tento podrobný plán pomáhá zákazníkům řídit cloudová prostředí s využitím úloh PCI-DSS.
Podrobný plán PCI-DSS nasazuje základní sadu zásad pro libovolnou architekturu nasazenou v Azure, která vyžaduje tuto akreditaci.

## <a name="control-mapping"></a>Mapování kontrol

Oddíl mapování kontrol poskytuje podrobné informace o zásadách zahrnutých v této iniciativě a o tom, jak tyto zásady pomáhají splňovat různé kontroly definované standardem PCI-DSS v3.2.1. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje případné nedodržení přiřazených zásad.

Po přiřazení tohoto podrobného plánu si prohlédněte úroveň dodržování předpisů vašich prostředí Azure na řídicím panelu dodržování předpisů služby Azure Policy.
