# <a name="overview-of-the-nist-sp-800-53-r4-blueprint-sample"></a>Přehled ukázky podrobného plánu NIST SP 800-53 R4

Ukázka podrobného plánu NIST SP 800-53 R4 poskytuje ochranné mantinely zásad správného řízení s využitím [Azure Policy](../../../policy/overview.md), které pomáhají vyhodnotit specifické kontroly NIST SP 800-53 R4. Tento podrobný plán pomáhá zákazníkům nasadit základní sadu zásad pro libovolnou architekturu nasazenou v Azure, která musí implementovat kontroly NIST SP 800-53 R4.

## <a name="control-mapping"></a>Mapování kontrol

Oddíl mapování kontrol poskytuje podrobné informace o zásadách zahrnutých v tomto podrobném plánu a o tom, jak tyto zásady řeší různé kontroly v rámci standardu NIST SP 800-53 R4. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje případné nedodržení přiřazených zásad. Další informace najdete v tématu [Azure Policy](../../../policy/overview.md).
