# <a name="overview-of-the-hipaa-hitrust-blueprint-sample"></a>Přehled ukázky podrobného plánu HIPAA HITRUST

Ukázky podrobného plánu HIPAA HITRUST poskytuje sadu ochranných mantinelů zásad správného řízení s využitím [Azure Policy](../../../policy/overview.md), které pomáhají zajistit osvědčení [HIPAA HITRUST](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html). 

## <a name="blueprint-sample"></a>Ukázka podrobného plánu

Ukázka podrobného plánu pomáhá zákazníkům nasadit základní sadu zásad pro libovolnou architekturu nasazenou v Azure, která vyžaduje osvědčení pro architekturu HIPAA HITRUST nebo dodržování příslušných předpisů. Oddíl [mapování kontrol](./control-mapping.md) poskytuje podrobné informace o zásadách zahrnutých v této iniciativě a o tom, jak tyto zásady pomáhají splňovat různé kontroly definované architekturou HIPAA HITRUST. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje dodržování přiřazených zásad.
