# <a name="overview-of-the-azure-security-benchmark-blueprint-sample"></a>Přehled ukázky podrobného plánu srovnávacího testu zabezpečení Azure

Ukázka podrobného plánu srovnávacího testu zabezpečení Azure poskytuje ochranné mantinely zásad správného řízení s využitím [Azure Policy](../../../policy/overview.md), které pomáhají vyhodnotit specifické kontroly [srovnávacího testu zabezpečení Azure](../../../../security/benchmarks/overview.md). Tento podrobný plán pomáhá zákazníkům nasadit základní sadu zásad pro libovolnou architekturu nasazenou v Azure, ve které chtějí implementovat kontroly srovnávacího testu zabezpečení Azure.

## <a name="control-mapping"></a>Mapování kontrol

Oddíl mapování kontrol poskytuje podrobné informace o zásadách zahrnutých v tomto podrobném plánu a o tom, jak tyto zásady řeší různé kontroly v rámci srovnávacího testu zabezpečení Azure. Prostředky při přiřazení k architektuře vyhodnocuje Azure Policy a zjišťuje případné nedodržení přiřazených zásad. Další informace najdete v tématu [Azure Policy](../../../policy/overview.md).
