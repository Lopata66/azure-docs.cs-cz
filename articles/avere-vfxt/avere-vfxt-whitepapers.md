# <a name="azure-vfxt-for-azure-whitepapers-and-other-resources"></a>Azure vFXT pro azure bílé knihy a další prostředky

V tomto článku jsou uvedeny případové studie, technologické dokumenty white paper, infografiky a další prostředky, které vám pomůžou pochopit, jak Avere vFXT pro Azure zapadá do vašeho pracovního postupu.

## <a name="deploy-agisoft-photoscan-on-azure"></a>Nasazení aplikace Agisoft PhotoScan v Azure

**Odkaz:** [Nasazení agisoftového fotoskenu v Azure pomocí Avere vFXT pro Azure nebo BeeGFS](https://azure.microsoft.com/mediahandler/files/resourcefiles/deploy-agisoft-photoscan-on-azure-with-azere-vfxt-for-azure-or-beegfs/AgiSoft%20PhotoScan%20on%20Azure%20using%20Avere%20vFXT%20or%20BeeGFS.pdf)

Tato příručka obsahuje podrobné pokyny pro instalaci fotogrammetrického softwaru Agisoft PhotoScan, který je podporován úložištěm Avere vFXT nebo paralelním souborovým systémem BeeGFS. Autor, Paulo Marques da Costa z AzureCAT, ukazuje, jak nastavit PhotoScan na virtuálních počítačích Azure (VMs). Vysoce výkonné úložiště urychluje dobu zpracování a jsou zahrnuty výsledky jeho srovnávacích testů. Toto prostředí lze škálovat nahoru a dolů podle potřeby a podporuje terabajty úložiště bez obětování výkonu.

**Odkaz:** [Infografika: Vytvoření hybridního cloudu pro úlohy založené na souborech](https://azure.microsoft.com/resources/building-a-hybrid-cloud-for-file-based-hpc-workloads/)
