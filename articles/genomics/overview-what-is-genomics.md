# <a name="what-is-microsoft-genomics"></a>Co je Microsoft Genomics?
Microsoft Genomics nabízí cloudovou implementaci Burrows-Wheeler Aligner (BWA) a Genome Analysis Toolkit (GATK) pro sekundární analýzu. Služba je certifikována iso a je v souladu s předpisy HIPAA a nabízí předvídatelnost cen pro vaše potřeby sekvenování genomu. Přečtěte si naše rychlé starty, kurzy a dokumentaci naučte, jak používat službu Microsoft Genomics a integrovat ji s naším rozhraním API.

## <a name="support-your-most-demanding-sequencing-needs"></a>Podpořte své nejnáročnější potřeby sekvenování
Namísto správy vlastních datových center využijte výhod škálování a zkušeností microsoftu při spouštění úloh v měřítku exabajtů. Naše cloudová implementace BWA-GATK je velmi shodou s kanálem osvědčených postupů Broad Institute.


## <a name="keep-your-business-running"></a>Udržujte svou firmu v chodu
Microsoft Genomics nabízí 99,99% dostupnost smlouvy o úrovni služeb (SLA) pro příjem požadavků pracovního postupu. Další informace o [sla Microsoft](https://azure.microsoft.com/support/legal/sla/genomics/v1_0/)Genomics .


## <a name="secure-your-data"></a>Zabezpečení dat
Služba Microsoft Genomics má certifikaci ISO 27001, ISO 27018 a ISO 9001 a je v souladu s předpisy HIPAA. Informace o zabezpečení platformy Azure najdete v [Centru zabezpečení Azure](https://www.microsoft.com/trustcenter/security).
