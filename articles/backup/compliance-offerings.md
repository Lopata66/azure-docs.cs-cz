# <a name="azure-backup-compliance-offerings"></a>Nabídky dodržování předpisů azure backup

Microsoft Azure & Azure Backup nabízí komplexní sadu certifikací a atestationů, které organizacím pomáhají dodržovat národní, regionální a oborové požadavky, kterými se řídí shromažďování a používání dat jednotlivců.

Níže uvedené nabídky dodržování předpisů pro Azure Backup najdete, abyste zajistili, že vaše služba bude regulována při používání služby Azure Backup.
