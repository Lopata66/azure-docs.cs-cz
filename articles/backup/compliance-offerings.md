# <a name="azure-backup-compliance-offerings"></a>Azure Backup nabídky dodržování předpisů

Aby organizace dodržovaly národní, regionální a specifické požadavky na konkrétní odvětví pro shromažďování a používání dat jednotlivců, Microsoft Azure & Azure Backup nabízí komplexní sadu certifikací a osvědčení.

Můžete najít níže uvedené nabídky dodržování předpisů pro Azure Backup, abyste zajistili, že bude vaše služba při používání služby Azure Backup regulována.
