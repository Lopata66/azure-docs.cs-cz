# <a name="update-the-azure-backup-central-reporting-content-pack"></a>Aktualizace balíčku obsahu Azure Backup Central Reporting 

[Balíček obsahu Azure Backup](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) lze použít k zobrazení sestav o centrálním zálohování. Balíček obsahu se pravidelně aktualizuje, aby bylo možné přidat další funkce a opravit chyby. V tomto článku se dozvíte, jak aktualizovat balíček obsahu. Také se dozvíte, jak odložit aktualizaci a zobrazit aktualizace provedené v průběhu času.

## <a name="get-updates-to-the-content-pack"></a>Získat aktualizace balíčku obsahu

### <a name="get-the-updated-content-pack"></a>Získat aktualizovaný balíček obsahu
Pokud jste ve své kopii balíčku obsahu neudělali žádné změny, aktualizuje se automaticky. Když se balíček obsahu změní, obdržíte oznámení v Power BI a e-mailové oznámení. Aktualizovaný balíček obsahu si můžete stáhnout snadno. 

### <a name="postpone-the-update"></a>Odložit aktualizaci
Osvědčeným postupem je Import balíčku obsahu do [vlastního pracovního prostoru](https://youtu.be/26zyOtyHPJM?t=1m57s). Nyní můžete sestavy upravovat.
Jak už bylo zmíněno, v případě změny balíčku obsahu se zobrazí oznámení v Power BI. Balíček obsahu si můžete stáhnout později. 

## <a name="coming-soon"></a>Již brzy
   
Azure Backup balíček obsahu se aktualizuje tak, aby podporoval více úloh. Úlohy zahrnují Azure SQL Database zálohování virtuálních počítačů IaaS a nástroje System Center Data Protection Manager. Tato podpora přičítá k aktuální podpoře záloh Azure Backup a virtuálních počítačů Azure. Tato podpora znamená, že můžete zobrazit a analyzovat všechna vaše Zálohovaná data na jednom centrálním místě. [Sestavy je také možné přizpůsobit](https://youtu.be/26zyOtyHPJM) tak, aby vyhovovaly potřebám vaší organizace.

Nakonfigurované sestavy, které jsou součástí Azure Backup balíčku obsahu, se mění. Změny usnadňují vytváření sestav napříč úlohami. K dispozici je vás zajímá náhled na nadcházející sadu sestav.
