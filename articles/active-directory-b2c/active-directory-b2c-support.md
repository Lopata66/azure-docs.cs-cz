# <a name="azure-active-directory-b2c-file-support-requests"></a>Azure Active Directory B2C: Soubor žádosti o podporu
Podávejte požadavky na podporu pro Azure Active Directory (Azure AD) B2C na webu Azure portal pomocí následujících kroků:

1. Přepnout do jiného tenanta, který má předplatné Azure spojené s ním z vašeho tenanta B2C. Obvykle je váš tenant zaměstnance nebo výchozímu tenantu vytvoří za vás při registraci předplatného Azure. Další informace najdete v tématu [jak předplatné služby Azure souvisí s Azure AD](../active-directory/fundamentals/active-directory-how-subscriptions-associated-directory.md).

    ![Azure portal se zvýrazněnou výběr tenanta](./media/active-directory-b2c-support/support-switch-dir.png)

3. Po přepnutí tenantů, klikněte na tlačítko **Nápověda a podpora**.

    ![Nápověda a podpora dlaždice zvýrazněný na webu Azure portal](./media/active-directory-b2c-support/support-support.png)

4. Klikněte na tlačítko **nová žádost o podporu**.

    ![Novou dlaždici žádost o podporu zvýrazněný na webu Azure portal](./media/active-directory-b2c-support/support-new.png)
5. V **Základy** okno, použijte následující informace a klikněte na tlačítko **Další**.

   * **Typ problému** je **technické**.
   * Zvolte odpovídající **předplatné**.
   * **Služba** je **služby Active Directory**.
   * Zvolte odpovídající **plán podpory**. Pokud ho nemáte, můžete si zaregistrovat pro jednu [tady](https://azure.microsoft.com/support/plans/).

     ![Základní informace o stránce s tlačítkem Další na webu Azure portal](./media/active-directory-b2c-support/support-basics.png)
6. V **problém** okno, použijte následující informace a klikněte na tlačítko **Další**.

   * Zvolte odpovídající **závažnost** úroveň.
   * **Typ problému** je **B2C**.
   * Zvolte odpovídající **kategorie**.
   * Popište svůj problém v **podrobnosti** pole. Zadejte podrobnosti, jako je název tenanta B2C, popis problému, chybových zpráv, korelace ID (Pokud je k dispozici) a tak dále.
   * V **časový rámec** pole, zadejte datum a čas (včetně časového pásma), ke které došlo k problému.
   * V části **nahrání souboru**, nahrát všechny snímky obrazovky a soubory, které si myslíte, že by pomáhají při řešení problému.

     ![Stránka problém s tlačítkem Další na webu Azure portal](./media/active-directory-b2c-support/support-problem.png)
7. V **kontaktní informace** okně Přidat kontaktní informace. Klikněte na možnost **Vytvořit**.

    ![Obraťte se na stránce s informacemi s tlačítkem vytvořit na webu Azure portal](./media/active-directory-b2c-support/support-contact.png)
8. Po odeslání žádosti o podporu, ji můžete sledovat kliknutím **Nápověda a podpora** na úvodním panelu a potom **spravovat žádosti o podporu**.

## <a name="known-issue-filing-a-support-request-in-the-context-of-a-b2c-tenant"></a>Známý problém: Podání žádosti o podporu v rámci tenanta B2C

> [!IMPORTANT]
