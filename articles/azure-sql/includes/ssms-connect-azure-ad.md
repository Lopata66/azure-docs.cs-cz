> V prosinci 2021 nebudou verze SSMS starší než 18,6 ověřovány prostřednictvím Azure Active Directory s MFA.
>
> Abyste mohli i nadále využívat Azure Active Directory ověřování pomocí MFA, budete potřebovat [SSMS 18,6 nebo novější](https://docs.microsoft.com/sql/ssms/download-sql-server-management-studio-ssms).
