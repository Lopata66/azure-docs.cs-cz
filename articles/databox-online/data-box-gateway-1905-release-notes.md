# <a name="azure-data-box-edge-and-azure-data-box-gateway-1905-release-notes"></a>Zpráva k vydání verze služby Azure Data Box Edge a Azure Data Box brány 1905

## <a name="overview"></a>Přehled

Následující zpráva k vydání verze Identifikujte kritické otevřené problémy a vyřešené problémy 1905 verzi pro Azure Data Box Edge a Azure Data Box brány.

Zpráva k vydání verze se průběžně aktualizuje, a při zjištění zásadních problémů vyžadujících alternativní řešení, se přidají. Před nasazením brány Data pole Edge/dat pole, pečlivě si prostudujte informace obsažené v poznámkách k verzi.

Tato verze odpovídá verzí softwaru:

- **Brána pole dat 1905 (1.6.887.626)**
- **Data Box Edge 1905 (1.6.887.626)**

> [!NOTE]
> Aktualizace 1905 lze použít pouze na zařízení Data Box Edge, na kterých běží verze GA softwaru.

## <a name="whats-new"></a>Co je nového

- **Vylepšení protokolování pole Programmable Gate pole FPGA ()** – v této verzi jsme změnili protokolování a vylepšení výstrahy související s FPGA. To je požadovaná aktualizace pro Edge pole dat, pokud používáte funkci výpočetní Edge s FPGA. Další informace najdete v tématu Jak [transformovat data pomocí hraniční výpočetní prostředí na hranici vaší datové pole](data-box-edge-deploy-configure-compute-advanced.md).

## <a name="known-issues-in-ga-release"></a>Známých problémech ve verzi GA

Žádné nové problémy jsou verze, které jste si poznamenali pro tuto verzi. Všechny verze poznamenat, že potíže mají přeneseny z předchozích verzí. Chcete-li zobrazit seznam známých problémů, přejděte na [známých problémech ve verzi GA](data-box-gateway-release-notes.md#known-issues-in-ga-release).
