# <a name="service-bus-messaging-samples"></a>Ukázky zasílání zpráv service bus
Ukázky zasílání zpráv služby Service Bus ukazují klíčové funkce zasílání [zpráv service bus](https://azure.microsoft.com/services/service-bus/). V současné době můžete najít ukázky na následujících místech:

| Programovací jazyk | SDK nebo ukázky umístění |
| -------------------- | ----------------------- |
| .NET, Java a správa | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Service Bus Explorer

Kromě toho [Průzkumník service bus](https://github.com/paolosalvatori/ServiceBusExplorer) je ukázka hostovaná na GitHubu, která umožňuje připojení k oboru názvů služby Service Bus a snadnou správu entit zasílání zpráv. Nástroj poskytuje pokročilé funkce, jako je například funkce importu a exportu a možnost testovat entity zasílání zpráv a přenosové služby. Úplný zdroj aplikace Service Bus Explorer a dokumentaci najdete na [GitHubu](https://github.com/paolosalvatori/ServiceBusExplorer).
