# <a name="azure-database-for-mysql-videos"></a>Videa Azure Database for MySQL

[Přehled: Databáze Azure pro PostgreSQL a MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

Azure Database for PostgreSQL a Azure Database for MySQL sdružují databázové moduly a možnosti komunitní edice plně spravované služby, takže se můžete soustředit na své aplikace, aniž byste museli databázi spravovat. Nalaďte se, abyste získali rychlý přehled o výhodách používání služby a podívejte se na některé funkce v akci.

[Hluboký ponor o schopnostech spravovaných služeb pro MySQL a PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

Azure Database for PostgreSQL a Azure Database for MySQL sdružují databázové moduly komunitní edice a možnosti plně spravované služby. Nalaďte se a získejte podrobné informace o tom, jak tyto služby fungují – jak zajišťujeme vysokou dostupnost a rychlé škálování (během několika sekund), abyste mohli vyhovět potřebám svých zákazníků. Dozvíte se také o některých základních investicích do zabezpečení a celosvětové dostupnosti.

[Jak začít s novou službou Azure Database for MySQL](https://channel9.msdn.com/events/Build/2017/B8045)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8045/player]

V tomto videu z konference Microsoft //Build z května 2017 se dozvíte o spravované nabídce MySQL od Microsoftu v Azure. Video prochází strategií Microsoftu pro podporu open source databázových systémů v Azure. Video popisuje, co pro vás jako vývojáře znamená vyvíjet nebo nasazovat aplikace, které používají MySQL v Azure. Toto video ukazuje přehled architektury služby a ukazuje, že Azure Database for MySQL je integrovaná s dalšími službami Azure, jako jsou webové aplikace.
