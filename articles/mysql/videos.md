# <a name="azure-database-for-mysql-videos"></a>Azure Database for MySQL videa

[Přehled: Azure Database for PostgreSQL a MySQL](https://channel9.msdn.com/Events/Connect/2017/T147)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T147/player]

Azure Database for PostgreSQL a Azure Database for MySQL – dejte dohromady community edition databázovým strojům a možnosti plně spravovaná služba, takže se můžete soustředit na své aplikace namísto toho, aby pro správu databáze. Nalaďte si získat rychlý přehled toho, jaké výhody používání služby a některé z možností v akci.

[Podrobný pohled na možnosti spravovaných služeb pro MySQL a PostgreSQL](https://channel9.msdn.com/Events/Connect/2017/T148)

>[!VIDEO https://channel9.msdn.com/Events/Connect/2017/T148/player]

Azure Database for PostgreSQL a Azure Database for MySQL pohromadě community edition databázovým strojům a možnosti plně spravovaná služba. Nalaďte si zobrazíte podrobné informace o fungování těchto služeb – jak zajistit vysokou dostupnost a Rychlé škálování (v sekundách), takže lze splnit potřeby vašich zákazníků. Budete také se dozvíte některé základní investic v zabezpečení a dostupnost po celém světě.

[Jak začít pracovat s nová služba Azure Database for MySQL](https://channel9.msdn.com/events/Build/2017/B8045)

>[!VIDEO https://channel9.msdn.com/Events/Build/2017/B8045/player]

V tomto videu od května 2017 konferenci //Build Microsoft, přečtěte si o nabídku Microsoftu spravované MySQL v Azure. Video vás provede strategie společnosti Microsoft pro podporu systémů Open Source databáze v Azure. Video popisuje, co znamená pro vás jako vývojář, abyste mohli vyvíjet a nasazovat aplikace, které používají MySQL v Azure. Toto video ukazuje přehled architektury služby a ukazuje, že Azure Database for MySQL je integrovaná s ostatními službami Azure, jako jsou například webové aplikace.
