# <a name="content-delivery-network-on-azure-with-azure-content-delivery-network"></a>Síť pro doručování obsahu v Azure pomocí sítě pro doručování obsahu Azure
[Azure Content Delivery Network](https://azure.microsoft.com/services/cdn/) nabízí vývojářům globální řešení pro rychlé doručování obsahu s vysokou šířkou pásma, jako jsou videa, obrázky nebo statické soubory, uživatelům ukládáním jejich obsahu do stovek bodů přítomnosti po celém světě. Síť pro doručování obsahu může také urychlit dynamický obsah, který nelze uložit do mezipaměti, aby se zkrátila doba načítání, šetřila šířka pásma a zvýšila se odezva. Tyto výhody získáte bez ohledu na to, zda vyvíjíte nebo spravujete weby nebo mobilní aplikace nebo kódujete a distribuujete streamovaná média, herní software, aktualizace firmwaru nebo koncové body IoT.

## <a name="azure-content-delivery-network-features"></a>Funkce sítě doručování obsahu Azure
  - Bezproblémově pracujte se službami Azure, jako jsou webové aplikace, mediální služby, úložiště a cloudové služby.
  - Snadno zvládejte dopravní špičky a těžké náklady.
  - Integrujte se společnostmi Akamai a Verizon a vytvářejte a přizpůsobujte strategii doručování obsahu.
  - Použijte dynamickou akceleraci webu.
  - Řízení chování ukládání obsahu do mezipaměti sítě pomocí pravidel ukládání do mezipaměti.
  - Získejte podporu vlastní domény HTTPS.
  - Zobrazení protokolů diagnostiky Azure zobrazíte základní analýzy a exportujte je do úložiště objektů Blob Azure, Azure Event Hubs nebo Azure Monitor.
  - Zlepšete výkon kompresí souborů.
  - Pomocí geografické filtrace můžete omezit obsah sítě doručování obsahu podle země nebo oblasti.
