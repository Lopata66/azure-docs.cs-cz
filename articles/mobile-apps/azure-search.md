# <a name="ai-powered-cloud-service-with-azure-search"></a>Cloudová služba s podporou AI s Azure Search
[Azure Search](https://azure.microsoft.com/services/search/) je cloudové řešení hledání jako služby, které vývojářům poskytuje rozhraní API a nástroje pro přidávání bohatých možností vyhledávání přes soukromý, heterogenní obsah ve webových, mobilních a podnikových aplikacích. Váš vlastní kód vyvolá ingestování dat (indexování) pro vytvoření a načtení indexu. Na druhé straně kód aplikace vystavuje požadavky na dotazy a zpracovává odpovědi. Možnosti vyhledávání jsou ve vašem klientovi definované pomocí funkce z Azure Search, při provádění dotazů prostřednictvím trvalého indexu, který vytvoříte, vlastníte a uložíte v rámci služby.

## <a name="azure-search-features"></a>Azure Search funkce
- Fulltextové **vyhledávání v podobě** fulltextového vyhledávání, jednoduché syntaxe dotazů a syntaxe dotazů Lucene
- **Jednoduché profily vyhodnocování** pro modelování relevance jako funkce hodnot v dokumentech samotných.
- **Geografické vyhledávání** umožňuje uživatelům prozkoumat data na základě blízkosti výsledků hledání do fyzického umístění.
- **Funkce uživatelského prostředí s funkcí** automatického dokončování, návrhy hledání, řazení a stránkování.
- **Hledání v rozpoznávání** obrázků a textu lze použít pro kanál indexování k extrakci textových informací z nezpracovaného obsahu.
- **Znalostní bázi Store** k uložení obohacení vytvořených během indexování.
- Připojte se k Azure SQL Database, Azure Cosmos DB nebo úložišti objektů BLOB v Azure jako **zdroje dat** pro extrakci prohledávatelných obsahu v primárních úložištích dat.
- Pomocí **Průvodce importem dat** můžete nakonfigurovat indexery, **Návrháře indexů** , aby pomohly index, a **Průzkumník služby Search** pro testování dotazů a upřesnění profilů vyhodnocování.
