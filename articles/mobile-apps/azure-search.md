# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Cloudová služba s podporou AI s využitím Azure Kognitivní hledání
[Azure kognitivní hledání](https://azure.microsoft.com/services/search/) je cloudové řešení hledání jako služby, které vývojářům poskytuje rozhraní API a nástroje pro přidávání bohatých možností vyhledávání přes soukromý, heterogenní obsah ve webových, mobilních a podnikových aplikacích. Váš vlastní kód vyvolá ingestování dat (indexování) pro vytvoření a načtení indexu. Na druhé straně kód aplikace vystavuje požadavky na dotazy a zpracovává odpovědi. Možnosti vyhledávání jsou ve vašem klientovi definované pomocí funkcí z Azure Kognitivní hledání. provádění dotazů se provádí prostřednictvím trvalého indexu, který vytvoříte, vlastníte a ukládáte do své služby.

## <a name="azure-cognitive-search-features"></a>Funkce Azure Kognitivní hledání
- **Hledání textu od ruky**: poskytuje fulltextové vyhledávání, jednoduchou syntaxi dotazů a syntaxi dotazů Lucene.
- **Jednoduché profily vyhodnocování**: relevance modelu jako funkce hodnot v dokumentech samotných.
- Geografická **hledání**: zkoumá data na základě blízkosti výsledků hledání do fyzického umístění.
- **Funkce uživatelského prostředí**: Poskytněte automatické dokončování, návrhy hledání, řazení a stránkování.
- **Hledání rozpoznávání**: lze použít na kanál indexování k extrakci textových informací z nezpracovaného obsahu pro analýzu obrázků a textu.
- **Znalostní báze Knowledge Store**: šetří rozšíření vytvořená během indexování.
- **Zdroje dat**: extrakce vyhledávaného obsahu v primárních úložištích dat, když jste připojeni k Azure SQL Database, Azure Cosmos DB a úložišti objektů BLOB v Azure.
- **Průvodce importem dat**: konfiguruje indexery. 
- **Návrhář indexu**: vystupuje index. 
- **Průzkumník vyhledávání**: testuje dotazy a dotáže se na profily vyhodnocování.

Přečtěte si podrobný popis [funkcí](/azure/search/search-what-is-azure-search#feature-descriptions)služby Azure kognitivní hledání.
