# <a name="ai-powered-cloud-service-with-azure-cognitive-search"></a>Cloudová služba založená na umělou ai s Azure Cognitive Search
[Azure Cognitive Search](https://azure.microsoft.com/services/search/) je cloudové řešení pro vyhledávání jako službu, které vývojářům poskytuje rozhraní API a nástroje pro přidání bohatého vyhledávacího prostředí přes soukromý, heterogenní obsah ve webových, mobilních a podnikových aplikacích. Vlastní kód vyvolá ingestování dat (indexování) k vytvoření a načtení indexu. Na druhé straně kód aplikace vydává požadavky na dotazy a zpracovává odpovědi. Prostředí vyhledávání je definováno ve vašem klientovi pomocí funkcí z Azure Cognitive Search, s prováděním dotazu přes trvalý index, který vytvoříte, vlastníte a uložíte ve vaší službě.

## <a name="azure-cognitive-search-features"></a>Funkce Azure cognitive search
- **Hledání volného tvaru textu**: Poskytuje fulltextové vyhledávání, jednoduchou syntaxi dotazu a syntaxi lucenedotazu.
- **Jednoduché profily hodnocení**: Relevance modelu jako funkce hodnot v samotných dokumentech.
- **Geosearch**: Zkoumá data na základě blízkosti výsledku hledání k fyzickému umístění.
- **Funkce uživatelských prostředí**: Poskytují automatické dokončování, návrhy hledání, řazení a stránkování.
- **Kognitivní vyhledávání**: Lze použít na indexovací kanál pro extrahování textových informací z nezpracovaného obsahu pro analýzu obrazu a textu.
- **Úložiště znalostí**: Uloží obohacení vytvořené během indexování.
- **Zdroje dat:** Extrahujte prohledávatelný obsah v úložištích primárních dat, když jste připojení k Azure SQL Database, Azure Cosmos DB a úložišti objektů Blob Azure.
- **Průvodce importem dat**: Konfiguruje indexery. 
- **Návrhář indexu**: Postaví index. 
- **Průzkumník hledání**: Testuje dotazy a vylepšuje profily vyhodnocování.
