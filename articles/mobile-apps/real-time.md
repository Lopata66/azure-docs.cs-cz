# <a name="build-real-time-applications-with-azure-signalr"></a>Sestavování aplikací v reálném čase pomocí služby Azure Signal

[Signál Azure](https://azure.microsoft.com/services/signalr-service/) vám umožňuje snadno přidávat do aplikací funkce v reálném čase. Tato funkce v reálném čase umožňuje službě nabízet aktualizace obsahu připojeným mobilním aplikacím. V důsledku toho se aplikace aktualizují bez nutnosti dotazování serveru nebo odeslání nových požadavků HTTP na aktualizace.

## <a name="azure-signalr-features"></a>Funkce služby Azure Signal
- **Podpora širokého klienta** – funguje se širokou škálou klientů, jako jsou webové a mobilní prohlížeče, desktopové aplikace, proces serveru, zařízení IoT a herní konzoly.
- Služba signalizace umožňuje **více instancím** spolupracovat pro škálování na miliony připojení mobilních aplikací. Podporuje také více globálních oblastí pro účely horizontálního dělení, vysoké dostupnosti nebo zotavení po havárii.
- **Nabízí bohatá rozhraní API pro různé vzory zasílání zpráv** , které odesílají zprávy do konkrétního připojení, všechna připojení nebo podmnožinu připojení, která patří konkrétnímu uživateli, nebo která se umístila do libovolné skupiny.
