# <a name="build-real-time-applications-with-azure-signalr-service"></a>Vytváření aplikací v reálném čase pomocí služby Azure SignalR

Se [službou Azure SignalR](https://azure.microsoft.com/services/signalr-service/)můžete snadno přidávat funkce v reálném čase do aplikací. Tato funkce v reálném čase umožňuje službě nabízení aktualizací obsahu do připojených mobilních aplikací. V důsledku toho jsou aplikace aktualizovány bez nutnosti dotazování serveru nebo odesílání nových požadavků HTTP pro aktualizace.

## <a name="azure-signalr-service-features"></a>Funkce služby Azure SignalR
- Má širokou klientskou podporu pro širokou škálu klientů, jako jsou webové a mobilní prohlížeče, desktopové aplikace, serverový proces, zařízení IoT a herní konzole.
- Umožňuje více instancí spolupracovat škálovat na miliony připojení mobilních aplikací. Podporuje také více globálních oblastí pro účely sytí, vysoké dostupnosti nebo zotavení po havárii.
- Nabízí bohatá rozhraní API pro různé vzory zasílání zpráv pro odesílání zpráv na určité připojení, všechna připojení nebo podmnožinu připojení, která patří určitému uživateli nebo byla umístěna do libovolné skupiny.
