# <a name="azure-cognitive-services-to-solve-business-problems-using-artificial-intelligence"></a>Azure Cognitive Services k řešení obchodních problémů pomocí umělých inteligentních funkcí
[Azure Cognitive Services](https://azure.microsoft.com/services/cognitive-services/) poskytuje výkonné služby pro zajištění vašich aplikací pomocí inteligentních algoritmů, abyste mohli vidět, slyšet, mluvit, pochopit a interpretovat uživatele, kteří potřebují transformovat své podnikání pomocí AI. Mobilní aplikace můžete nastavit jako inteligentní, poutavé a kontextové pro uživatele. Rozhraní API, sady SDK a služby umožňují vývojářům přidat AI do své nabídky a pomáhat sestavovat inteligentní aplikace bez přímých dovedností v oblasti AI a datových věd.

## <a name="cognitive-services-capabilities"></a>Možnosti Cognitive Services
- **Vidění**
    - Označování obrázků, miniatur
    - Optické rozpoznávání znaků (rozpoznávání rukopisu)
    - Detekce tváře
    - Detekce emoce
    - Video přehledy
    - Moderování obrázků a videí
    - Přizpůsobené rozpoznávání obrázků

- **Řečí**
    - Převod řeči na text (řeč přepisu)
    - Vlastní přepis řeči (složitá slova, prostředí s vysokou úrovní šumu)
    - Převod textu na řeč
    - ID mluvčího a ověřování
    - Překlady řeči v reálném čase

- **Jazyk**
    - Porozumění kontextovým jazykům – přizpůsobená analýza záměrů
    - Analýza mínění, detekce klíčových frází
    - Překlad textu až na 60 jazyků
    - Moderování textu
    - Kontrola pravopisu

- **Rozhodnutí** 
    - Moderátor obsahu
    - Individuální doručování obsahu
    - Detektor anomálií

- **Search**
    - Automatické návrhy hledání 
    - Rozšíření informací o entitě
    - Komplexní novinky, hledání obrázků, videí, webů a místních podnikových entit
    - Přizpůsobené prostředí pro vyhledávání

- Podpora pro **kontejnery** pro Cognitive Services
