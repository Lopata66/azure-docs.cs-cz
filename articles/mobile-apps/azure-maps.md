# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Geoprostorové rozhraní API pro přidání map, prostorových analýz a řešení mobility pomocí Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) poskytuje platformu umístění nativně v cloudu Azure, nabízí portfolio geoprostorové služby, které používají k zajištění přesného geografického kontextu pro webové a mobilní aplikace pomocí nových dat mapování. Umožňuje vývojářům, podnikům a nezávislým výrobcům softwaru vytvářet aplikace pracující s polohou a řešení IoT, mobility, logistiky a sledování prostředků.

## <a name="azure-maps-features"></a>Azure Maps funkce
- **Hledání** poskytuje podrobné informace o sousedních místech zájmu (POI), vykreslování umístění na mapě a zeměpisném (nebo obráceném) adresách pro získání odpovídajících souřadnic zeměpisné šířky a délky.
- Vylepšete dobu trvání cesty a zabraňte Gridlock používání **provozu**.
- Integrací jasných a snadno čitelných **map** do vašeho řešení.
- **Směrování** a prezentuje nejkratší nebo nejrychlejší dostupné trasy – do více cílů najednou, nebo poskytují specializované trasy a fakta pro cyklisty a komerční vozidla.
- **Časová pásma** usnadňují uživatelům, aby viděli, jak dlouho je kdekoli na světě.
- **Prostorové operace** pro vylepšení inteligentních informací o poloze.
- **Geografická poloha** pro vyhledání země IP adresy.
- **Datová služba** pro nahrávání a ukládání geoprostorovéch dat pro použití s prostorovou operací nebo kompozicí imagí.
- **Mobilita** poskytuje v reálném čase informace o poloze pro nejbližší veřejné tranzitní služby, včetně informací o zastavení, trasách a odhadech doby trvání cesty.
