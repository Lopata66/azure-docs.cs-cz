# <a name="geospatial-apis-to-add-maps-spatial-analytics-and-mobility-solutions-with-azure-maps"></a>Geoprostorová rozhraní API pro přidání map, prostorových analýz a řešení mobility pomocí Azure Maps

[Azure Maps](https://azure.microsoft.com/services/azure-maps/) poskytuje platformu pro určování polohy nativně v cloudu a nabízí portfolio geoprostorových služeb, které využívají čerstvá data mapování k poskytování přesného geografického kontextu webovým a mobilním aplikacím. Vývojáři, podniky a nepřesnou mez itauři můžou pomocí Azure Maps vytvářet aplikace pro zjišťování polohy a řešení Pro mobilitu, logistiku a sledování majetku.

## <a name="azure-maps-features"></a>Funkce Azure Maps
- **Vyhledávání**: Poskytuje podrobnosti o blízkých bodech zájmu, vykresluje místa na mapě a používá geokódové (nebo reverzní geokódové) adresy k získání odpovídajících souřadnic zeměpisné šířky a délky.
- **Provoz**: Zlepšuje dobu jízdy a pomáhá vyhnout se zablokování.
- **Mapy**: Poskytuje přehledné a přehledné mapy pro integraci do vašeho řešení.
- **Směrování**: Představuje nejkratší nebo nejrychlejší trasy, které jsou k dispozici pro více cílů v určitém čase. Směrování také poskytuje specializované trasy a fakta pro chodce, cyklisty a užitková vozidla.
- **Časová pásma**: Usnadňuje zjistit, kolik je hodin kdekoli na světě.
- **Prostorové operace:** Vylepšete svou lokalizační inteligenci.
- **Geolokace**: Vyhledá zemi IP adresy.
- **Datová služba**: Nahrává a ukládá geoprostorová data pro použití s prostorovými operacemi nebo kompozicí obrazu.
- **Mobilita**: Poskytuje informace o poloze v reálném čase o blízkých službách veřejné dopravy. Zastávky, informace o trase a odhady doby jízdy jsou zahrnuty.
