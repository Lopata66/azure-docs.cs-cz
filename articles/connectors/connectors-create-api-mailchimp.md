# <a name="get-started-with-the-mailchimp-connector"></a>Začínáme s konektor MailChimp
MailChimp je služba SaaS, která umožňuje podnikům spravovat a automatizovat e-mailové marketingové aktivity, včetně posílání marketingových e-mailů, automatických zpráv a cílené kampaně.

Můžete začít tak, že teď vytvoříte aplikaci logiky, najdete v článku [vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-mailchimp"></a>Vytvoření připojení k Mailchimpu
Chcete-li vytvořit Logic apps s využitím platformy MailChimp, musíte nejdřív vytvořit **připojení** pak zadejte podrobnosti pro následující vlastnosti:

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| Podpisový |Ano |Zadat přihlašovací údaje k MailChimpu |
