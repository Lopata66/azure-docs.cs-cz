# <a name="get-started-with-the-mailchimp-connector"></a>Začínáme s konektorem MailChimp

MailChimp je služba SaaS, která podnikům umožňuje spravovat a automatizovat e-mailové aktivity marketingu, včetně odesílání marketingových e-mailů, automatizovaných zpráv a cílené kampaně.

Můžete začít vytvořením aplikace logiky hned teď v tématu [Vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-mailchimp"></a>Vytvoření připojení k MailChimp

Pokud chcete vytvářet aplikace logiky pomocí MailChimp, musíte nejdřív vytvořit **připojení** a pak zadat podrobnosti pro následující vlastnosti:

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| Podpisový |Ano |Zadat přihlašovací údaje pro MailChimp |
