# <a name="get-started-with-the-mailchimp-connector"></a>Začínáme s konektorem MailChimp

MailChimp je služba SaaS, která umožňuje firmám spravovat a automatizovat e-mailové marketingové aktivity, včetně odesílání marketingových e-mailů, automatizovaných zpráv a cílených kampaní.

Můžete začít vytvořením aplikace logiky teď, najdete v [tématu Vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-mailchimp"></a>Vytvoření připojení k mailchimpu

Chcete-li vytvořit aplikace logiky s MailChimp, musíte nejprve vytvořit **připojení** a pak zadat podrobnosti o následujících vlastnostech:

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| Podpisový |Ano |Zadejte pověření MailChimp |
