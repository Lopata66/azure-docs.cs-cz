# <a name="get-started-with-the-office-365-users-connector"></a>Začínáme s konektoru uživatelé Office 365.
Připojte se k Office 365 uživatelům získat profily, Hledat uživatele a další. Uživatelé Office 365 můžete:

* Vytvoření obchodní toku na základě dat, který jste získali od uživatelů Office 365. 
* Použití akce, které získat přímé podřízené získat profil uživatele správce a další. Tyto akce získat odpověď a poté zpřístupní výstup pro další akce. Třeba získat přímé podřízené osoby a pak provést tyto informace a aktualizace databáze SQL Azure. 

Můžete začít tak, že teď vytvoříte aplikaci logiky, najdete v článku [vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-office-365-users"></a>Vytvoření připojení k uživatelům Office 365
Při přidání tohoto konektoru logic Apps musíte přihlásit ke svému účtu uživatelé Office 365 a povolit aplikacím logiky připojíte ke svému účtu.

> [!INCLUDE [Steps to create a connection to Office 365 Users](../../includes/connectors-create-api-office365users.md)]
> 
> 

Po vytvoření připojení můžete zadat vlastnosti uživatelé Office 365, jako je ID uživatele. **Reference k rozhraní REST API** v tomto článku popisuje tyto vlastnosti.
