# <a name="get-started-with-the-microsoft-translator-connector"></a>Začínáme s konektorem Microsoft Translator

Připojte se k microsoft translatoru a překládejte text, zjišťujte jazyk a další. S microsoft translatorem můžete: 

* Sestavte si svůj obchodní tok na základě dat, která získáte od Microsoft Translator. 
* Pomocí akcí můžete překládat text, zjišťovat jazyk a další. Tyto akce získat odpověď a potom zpřístupnit výstup pro jiné akce. Například při vytvoření nového souboru v Dropbox, můžete přeložit text v souboru do jiného jazyka pomocí Microsoft Translator.

Můžete začít vytvořením aplikace logiky teď, najdete v [tématu Vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).
