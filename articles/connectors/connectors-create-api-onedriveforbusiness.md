# <a name="get-started-with-the-onedrive-for-business-connector"></a>Začínáme s Onedrivem pro firmy konektoru
Připojte se k Onedrivu pro firmy a spravovat soubory. Můžete provádět různé akce, třeba nahrávat, aktualizovat, zobrazovat a odstraňovat soubory.

Můžete začít tak, že teď vytvoříte aplikaci logiky, najdete v článku [vytvoření aplikace logiky](../logic-apps/quickstart-create-first-logic-app-workflow.md).

## <a name="create-a-connection-to-onedrive-for-business"></a>Vytvoření připojení k Onedrivu pro firmy
K vytváření aplikací logiky s Onedrivem pro firmy, musíte nejprve vytvořit **připojení** pak zadejte podrobnosti pro následující vlastnosti:

| Vlastnost | Požaduje se | Popis |
| --- | --- | --- |
| Podpisový |Ano |Zadat přihlašovací údaje k OneDrivu pro firmy |

Po vytvoření připojení, můžete k provedení akce a naslouchat aktivační procedury popsané v tomto článku.
