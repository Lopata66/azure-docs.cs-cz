# <a name="additional-documentation"></a>Další dokumentace 

Některé prostředky mimo tento docs.microsoft.com webu vám můžou pomoct pochopit hybridní mezipaměť Microsoft Azure FXT Edge Filer a pracovat s ní.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Dokumentace avere vFXT pro azure cache

[Avere vFXT pro Azure](https://docs.microsoft.com/azure/avere-vfxt/) je podobný produkt mezipaměti clusteru, který používá virtuální uzly v Microsoft Azure. Je vytvořen pomocí portálu Azure, ale používá stejný konfigurační software ovládacího panelu jako hardwarová cache Azure FXT Edge Filer.

* [Průvodce řídicím panelem](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) – vysvětluje funkce karty Řídicí panel ovládacího **panelu**
