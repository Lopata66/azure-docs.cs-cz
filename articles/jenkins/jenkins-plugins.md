# <a name="jenkins-plugins-for-azure"></a>Moduly plug-in Jenkinse pro Azure

Následující moduly plug-in Jenkins podporovat různé funkce pro použití s Azure.
  
| Modul plug-in Jenkinse | Popis                                   |
|------------------------------------------------------------------------------|
| [Modul plug-in Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Modul plug-in Jenkinse pro nasazení služby Azure App Service (v současné době podporuje pouze webová aplikace). | 
| [Modul plug-in Azure AD](https://plugins.jenkins.io/azure-ad)                       | Modul plug-in Jenkinse, který podporuje ověřování a autorizace pomocí služby Azure Active Directory. | 
| [Modul plug-in Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Modul plug-in Azure Container Agents vám mohou pomoci při spuštění kontejneru jako agenta v Jenkinsu | 
| [Modul plug-in Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Modul plug-in Jenkinse pro konfigurace nasadit do Azure Container Service (AKS). | 
| [Modul plug-in Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Modul plug-in Jenkinse pro správu přihlašovacích údajů Azure. | 
| [Modul plug-in Azure Function](https://plugins.jenkins.io/azure-function)           | Tento modul plug-in používat k nasazení do funkce Azure Functions, musíte mít instanční objekt Azure do instance Jenkinse. | 
| [Modul plug-in Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Modul plug-in Jenkinse pro Service Fabric s Linuxem Azure projekty. |
| [Plug-in Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | Artefakty k sestavení modulu plug-in pro nahrávání nebo stahování sestavení závislosti z Microsoft Azure Blob storage. | 
| [Modul plug-in Azure VM agents](https://plugins.jenkins.io/azure-vm-agents)         | Modul plug-in Jenkinse k vytváření agentů Jenkinse ve virtuálních počítačích Azure (pomocí šablony Azure Resource Manageru). | 
| [Modul plug-in sady škálování virtuálních počítačů Azure](https://plugins.jenkins.io/azure-vmss)           | Nastaví modul plug-in Jenkinse pro nasazení imagí virtuálních počítačů škálování virtuálních počítačů Azure. |
