# <a name="jenkins-plugins-for-azure"></a>Moduly plug-in Jenkinse pro Azure

Následující moduly plug-in Jenkinse podporují různé funkce pro použití s Azure.
  
| Modul plug-in Jenkinse | Popis                                   |
|------------------------------------------------------------------------------|
| [Modul plug-in Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Modul plug-in Jenkinse pro nasazení Azure App Service (aktuálně podporuje jenom webovou aplikaci). | 
| [Modul plug-in Azure AD](https://plugins.jenkins.io/azure-ad)                       | Modul plug-in Jenkinse, který podporuje ověřování & autorizaci prostřednictvím Azure Active Directory. | 
| [Modul plug-in Azure artefakt Manageru](https://plugins.jenkins.io/azure-artifact-manager) | Modul plug-in Azure artefakt Manager je správcem artefaktů, který umožňuje ukládat artefakty do Azure Blob Storage. Modul plug-in Azure artefakt Manager funguje transparentně pro Jenkinse a vaše úlohy, jako výchozí správce artefaktů. | 
| [Modul plug-in Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Modul plug-in Azure Container Agents vám může pomáhat při spuštění kontejneru jako agenta v Jenkinse. | 
| [Modul plug-in úlohy Azure Container Registry](https://plugins.jenkins.io/azure-container-registry-tasks)       | Modul plug-in Jenkinse pro odeslání žádosti Docker-Build do [Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Modul plug-in Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Modul plug-in Jenkinse pro nasazení konfigurací do Azure Container Service (AKS) | 
| [Modul plug-in Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Modul plug-in Jenkinse pro správu přihlašovacích údajů Azure | 
| [Modul plug-in Azure Function](https://plugins.jenkins.io/azure-function)           | Modul plug-in Jenkinse pro nasazení funkce Azure Functions | 
| [Modul plug-in Azure IoT Edge](https://plugins.jenkins.io/azure-iot-edge)           | Modul plug-in Azure IoT Edge umožňuje snadno nastavit kanál CI/CD pro vývoj řešení IoT Edge na Jenkinse. | 
| [Modul plug-in Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Modul plug-in Jenkinse pro projekty Azure Service Fabric pro Linux |
| [Plug-in Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | modul plug-in pro nahrání artefaktů sestavení do nebo stahování závislostí sestavení z Microsoft Azure úložiště objektů BLOB. | 
| [Modul plug-in agentů virtuálních počítačů Azure](https://plugins.jenkins.io/azure-vm-agents)         | Modul plug-in Jenkinse k vytváření agentů Jenkinse na virtuálních počítačích Azure (prostřednictvím šablony Azure Resource Manager). | 
| [Modul plug-in Azure Virtual Machine Scale set](https://plugins.jenkins.io/azure-vmss)           | Modul plug-in Jenkinse pro nasazení imagí virtuálních počítačů do služby Azure Virtual Machine Scale Sets. |
