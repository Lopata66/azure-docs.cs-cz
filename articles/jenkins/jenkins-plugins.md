# <a name="jenkins-plugins-for-azure"></a>Moduly plug-in Jenkins pro Azure

Následující moduly plug-in Jenkins podporují různé funkce pro použití s Azure.
  
| Jenkins plugin | Popis                                   |
|------------------------------------------------------------------------------|
| [Modul plug-in Azure App Service](https://plugins.jenkins.io/azure-app-service)     | Jenkins plugin pro nasazení služby Azure App Service (aktuálně podporuje jenom Web App). | 
| [Modul plug-in Azure AD](https://plugins.jenkins.io/azure-ad)                       | Modul plug-in Jenkins, který podporuje autorizaci ověřování & prostřednictvím služby Azure Active Directory. | 
| [Plugin Azure Artifact Manager](https://plugins.jenkins.io/azure-artifact-manager) | Modul plug-in Azure Artifact Manager je Správce artefaktů, který umožňuje ukládat artefakty do úložiště objektů blob Azure. Azure Artifact Manager plugin funguje transparentně Jenkins a vaše úlohy, je to jako výchozí Správce artefaktů. | 
| [Modul plug-in Azure Container Agents](https://plugins.jenkins.io/azure-container-agents) | Modul plug-in Azure Container Agents vám může pomoct spustit kontejner jako agent v Jenkinsi | 
| [Plugin úlohy registru kontejnerů Azure](https://plugins.jenkins.io/azure-container-registry-tasks)       | Jenkins plugin pro odeslání docker-build požadavek [na Azure Container Registry](/azure/container-registry/container-registry-tasks-overview). |
| [Modul plug-in Azure Container Service](https://plugins.jenkins.io/azure-acs)       | Jenkins plugin pro nasazení konfigurací do služby Azure Container Service (AKS). | 
| [Modul plug-in Azure Credential](https://plugins.jenkins.io/azure-credentials)      | Jenkins plugin pro správu přihlašovacích údajů Azure. | 
| [Modul plug-in Azure Function](https://plugins.jenkins.io/azure-function)           | Jenkins plugin pro nasazení funkce Azure. | 
| [Azure IoT Edge , plugin](https://plugins.jenkins.io/azure-iot-edge)           | Azure IoT Edge plugin usnadňuje nastavení kanálu CI/CD pro vývoj řešení IoT Edge na Jenkins. | 
| [Modul plug-in Azure Service Fabric](https://plugins.jenkins.io/service-fabric)     | Jenkins plugin pro projekty Linux Azure Service Fabric. |
| [Modul plug-in Azure Storage](https://plugins.jenkins.io/windows-azure-storage)     | plugin pro nahrávání artefaktů sestavení do úložiště objektů Blob Microsoft Azure nebo stahování z nich. | 
| [Modul plug-in Azure VM Agents](https://plugins.jenkins.io/azure-vm-agents)         | Jenkins plugin pro vytvoření jenkinsových agentů ve virtuálních počítačích Azure (pomocí šablony Azure Resource Manager). | 
| [Plugin pro škálování virtuálních strojů Azure](https://plugins.jenkins.io/azure-vmss)           | Jenkins plugin pro nasazení ibi virtuálních počítačů do škálovacích sad virtuálních strojů Azure. |
