# <a name="sql-data-warehouse-data-management-partners"></a>SQL Data Warehouse partneři pro správu dat
Pokud chcete vytvořit řešení datového skladu, vyberte si z široké škály špičkových nástrojů. Tento článek popisuje partnerské společnosti Microsoftu s nástroji pro správu dat a řešeními podporujícími Azure SQL Data Warehouse.

Nexus je k dispozici jako dotazovací Nástroj pro SQL Data Warehouse. Nexus slouží k dotazování interních a cloudových počítačů a spojování dat napříč různými platformami. Pomocí svého uživatelského rozhraní, které bylo vyvinuto na základě rozsáhlého uživatelského prostředí, může Modelátor pracovat na modelech databáze snadno a pohodlně. Kromě toho Infolibrarian poskytuje správu metadat, řízení dat a softwarová řešení pro správu prostředků pro správu a publikování metadat z nejrůznějších nástrojů a technologií. |[Stránka produktu][infolibrarian_website]<br> [Azure Marketplace][infolibrarian_marketplace]<br> |

Díky využití Azure SQL Data Warehouse a RedPoint můžete maximalizovat hodnotu vašich strukturovaných a nestrukturovaných dat a zajistit tak přizpůsobené kontextovou interakci Hyper-v, která je potřeba k zapojení dnešního zákazníka z Omni kanálu.
