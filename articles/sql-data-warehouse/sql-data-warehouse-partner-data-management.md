# <a name="sql-data-warehouse-data-management-partners"></a>SQL Data Warehouse partnerů pro správu dat
K vytvoření vašeho řešení datového skladu, vyberte z široké škály špičkové nástroje. Tento článek se soustředí na partnera společnosti Microsoft pomocí nástroje pro správu dat a řešení Azure SQL Data Warehouse podporuje.

## <a name="data-management-partners"></a>Partnerů pro správu dat
| Partner | Popis | Odkaz na web nebo produkt |
| ------- | ----------- | -------------------- |
| ![Coffing datového skladu][1] |**Coffing datového skladu**<br>Datové sklady coffing poskytuje Nexus Chameleon, nástroj se po dobu 10 let návrhu vyhrazený pro dotazování systémy. Nexus je k dispozici jako nástroj pro dotaz pro SQL Data Warehouse. Pomocí Nexus dotazování interně a počítačů a dat v cloudu napříč různými platformami. Bod klikněte na tlačítko sestava! |[Stránka produktu][coffing_website]<br> |
| ![Inbrein][2] |**Inbrein MicroERD**<br>Inbrein MicroERD poskytuje nástroje, které je potřeba vytvořit přesný datový model, snížit redundanci dat, zvyšují produktivitu a dodržovat standardy. S použitím jeho uživatelské rozhraní, který byl vyvinut podle rozsáhlé uživatelské prostředí, můžete pracovat Modelátor na modely DB snadné a pohodlné. Můžete průběžně využívat nové a vylepšené funkce MicroERD přes příkazový řádek funkční vylepšeními a aktualizacemi. |[Stránka produktu][inbrein_website]<br> |
| ![Infolibrarian][3] |**Infolibrarian (Server správy metadat)**<br>InfoLibrarian katalogy, ukládá a spravuje metadat, které vám pomůžou vyřešit klíče problémové body správy dat. Kromě toho poskytuje Infolibrarian správy metadat, zásady správného řízení dat a řešení pro správu softwarových prostředků ke správě a publikování metadat z nejrůznějších nástrojů a technologií. |[Stránka produktu][infolibrarian_website]<br> [Azure Marketplace][infolibrarian_marketplace]<br> |
| ![Globální Redpoint][4] |**Správa dat RedPoint**<br>Správa dat RedPoint umožňuje obchodníci jejich data k podpoře zapojení zákazníků napříč kanál při provádění strukturovaná a Nestrukturovaná data správy použít. S využitím Azure SQL Data Warehouse a RedPoint můžete maximalizovat hodnotu strukturovaných a nestrukturovaných dat k zajištění hyper přizpůsobené, kontextové interakce potřebné pro zapojení dnešní omni kanál zákazníka. Přetáhněte myší rozhraní díky navrhování a provádění procesů správy dat snadno. |[Stránka produktu][redpoint_dm_website]<br> [Azure Marketplace][redpoint_dm_marketplace]<br> |
| ![SENTRY jeden][5] |**SentryOne (datový Sklad Sentry)**<br>Pomocí inteligentní data přesun řídicí panel a události kalendáře vždy víte přesně co je dopadu na vaše úlohy. Účelem je poskytnout přehled o dotazy a úlohy, ve kterých k načtení zálohování nebo obnovení dat, které nikdy starat o vašich prostředků Azure na maximum. |[Stránka produktu][sql_sentry_website]<br>[Azure Marketplace][sql_sentry_marketplace]<br> |
