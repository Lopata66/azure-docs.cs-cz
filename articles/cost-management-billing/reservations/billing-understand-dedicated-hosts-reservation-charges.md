# <a name="how-the-azure-reservation-discount-is-applied-to-azure-dedicated-hosts"></a>Jak se na službu Azure Dedicated Host uplatňuje sleva za rezervaci Azure

Když si zakoupíte rezervovanou instanci služby Azure Dedicated Host, sleva za rezervaci se automaticky uplatní na vyhrazené hostitele, kteří vyhovují rezervovaným atributům a množství. Rezervace pokrývá náklady na výpočetní prostředky vyhrazených hostitelů.

## <a name="how-reservation-discount-is-applied"></a>Jak se uplatňuje sleva za rezervaci

*Nevyužité slevy se nenahrazují*. Pokud tedy nemáte na každou hodinu odpovídající prostředky, o rezervované množství za danou hodinu přijdete. Nevyužité rezervované hodiny vám nemůžeme převést.

Když vyhrazeného hostitele odstraníte, sleva za rezervaci se automaticky uplatní na další vyhovující prostředek v zadaném rozsahu. Pokud se v zadaném rozsahu nenajdou žádné vyhovující prostředky, o rezervované hodiny  *přijdete*.

## <a name="reservation-discount-for-dedicated-hosts"></a>Sleva za rezervaci pro službu Dedicated Host

Rezervovaná instance služby Azure Dedicated Host nabízí slevu na výpočetní infrastrukturu, kterou využívají vyhrazení hostitelé. Tato sleva se vztahuje na vyhrazené hostitele bez ohledu na to, jestli je virtuální počítače využívají. Rezervace nepokrývá další náklady, jako jsou náklady na licence, využití sítě nebo spotřebu úložiště virtuálním počítačem nasazeným ve vyhrazeném hostiteli.
