# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Připojování nebo odpojování svazku pro virtuální počítače s Windows nebo Linuxem 

Můžete připojit nebo odpojit podle potřeby svazek pro virtuální počítače s Windows nebo Linux.  Pokyny připojení pro virtuální počítače s Linuxem jsou k dispozici souborů NetApp Azure.  

1. Klikněte na tlačítko **svazky** okně a potom vyberte svazek, pro kterou chcete připojit. 
2. Klikněte na tlačítko **pokyny pro připojení** z vybraných svazků a pak postupujte podle pokynů k připojení svazku. 

    ![Pokyny k připojení systému souborů NFS](../media/azure-netapp-files/azure-netapp-files-mount-instructions-nfs.png)

    ![Pokyny protokolu SMB pro připojení](../media/azure-netapp-files/azure-netapp-files-mount-instructions-smb.png)
