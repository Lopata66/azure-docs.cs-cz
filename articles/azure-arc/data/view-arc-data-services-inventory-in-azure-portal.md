# <a name="view-inventory-of-your-instances-in-the-azure-portal"></a>Zobrazení inventáře vašich instancí v Azure Portal

Jakmile nahrajete [metriky, protokoly](upload-metrics-and-logs-to-azure-monitor.md)nebo [využití](view-billing-data-in-azure.md), můžete svou instanci zobrazit jako prostředek Azure z Azure Portal. Provedete to tak, že přejdete na speciální adresu URL a přejdete na [https://portal.azure.com](https://portal.azure.com) snímky obrazovky níže.

1. Přejít na všechny služby
1. Hledání typů instancí databáze
1. Přidat k oblíbeným položkám
1. V levém navigačním panelu klikněte na typ instance.
1. Umožňuje zobrazit vaše instance ve stejném zobrazení jako vaše jiné prostředky Azure SQL nebo Azure PostgreSQL s vlastním škálováním (pomocí filtrů získat podrobné zobrazení).
