# <a name="windows-virtual-desktop-partner-integrations"></a>Integrace partnerů virtuálního klienta Windows

Tento článek obsahuje seznam partnerů virtuálního klienta Windows v abecedním pořadí.

## <a name="active-cypher"></a>Aktivní šifrováním

Aktivní šifrovací je poskytovatel služeb s přidanou hodnotou virtuální plochy Windows, který pro vícevrstvou IT s šifrovací technologii, která k ochraně před porušením a ztráty dat organizace. Díky aktivní Cypher inovativní šifrovací technologii, která podporuje celkové strategie ochrany souboru ke spuštění virtuálního klienta Windows v Azure.

Při použití s virtuální plochy Windows, aktivní Cypher poskytuje další funkce, které umožňují usnadnění a mobilitu dat uživatele zároveň je možné zabránit nechtěnému úniku. S aktivní Cypher stávající zásady zabezpečení skupiny a oprávnění ve službě Azure Information Protection můžete využít k šifrování souborů uživatele při používání virtuální plochy Windows.

[Přejdete na web partnera.](https://activecypher.com/)

## <a name="cloudjumper"></a>CloudJumper

CloudJumper je poskytovatel služeb s přidanou hodnotou virtuální plochy Windows, který pro vícevrstvou poskytovatelů řešení a organizace IT se softwarem holistické zřizovat a spravovat prostředí virtuálního klienta Windows. Se softwarem CloudJumper, může oddělení IT spravovat každé vrstvě nasazení virtuální plochy Windows. Doručování úlohy a aplikace je automatické, zajištění, že uživatelé můžete rychle získat své ploše kdekoli na jakémkoli zařízení.

CloudJumper na software, Cloud pracovní prostor Management Suite zvyšuje hodnotu virtuální plochy Windows díky zjednodušení nasazení a úlohy průběžné správy v Azure. V podokně ze skla, může oddělení IT zřizovat, spravovat a optimalizovat infrastrukturu pro uživatele pracovních prostorů. Společnosti CloudJumper jednoduchý skript aktivací stroj se integruje s IT služby platformy a automatizovat úkoly při zřizování virtuálního klienta Windows. Kromě toho rozhraní API CloudJumper povolit další rozšíření a integrace s jinými systémy enterprise jako ServiceNow a BMC připravené.

[Přejdete na web partnera.](https://cloudjumper.com/wvd/)

## <a name="lakeside-software"></a>Lakeside softwaru

Lakeside softwaru je poskytovatel služby s přidanou hodnotou virtuální plochy Windows, který pro vícevrstvou IT týmy se softwarem pro sledování výkonu a o vyhodnocení připravenosti na migraci do Azure zatížení uživatele. Pomocí tohoto softwaru získává IT jasnější přehled o využití a prostředků využití aplikace ke zjednodušení procesu migrace. Lakeside Software shromažďuje data za každý pracovní prostor k vytvoření komplexní sestavu v uživatelském prostředí umožňující rychlé řešení potíží a optimalizaci prostředků.

Digitální prostředí Lakeside softwarové řešení, SysTrack, monitorování pomáhá poskytovat skvělé uživatelské prostředí pomocí sledování výkonu a identifikaci ideální úlohy pro migraci. SysTrack funguje a jak můžete rozšířit hodnotu virtuální plochy Windows prostřednictvím posouzení určení správné velikosti a nepřetržité monitorování prostředí uživatele.

[Přejdete na web partnera.](https://www.lakesidesoftware.com/assessments/wvd)

## <a name="liquidware"></a>Liquidware

Liquidware je poskytovatel služeb s přidanou hodnotou virtuální plochy Windows, která poskytuje software, který spravuje a optimalizuje nasazení virtuální plochy Windows. Poskytuje sada Liquidware Essentials doručování aplikací prostřednictvím vrstev, správu uživatelského prostředí a viditelnost klíče uživatelské prostředí a provádět diagnostiku. S řešeními pro vyhodnocení připravenosti na migraci a analýzu metriky využití Liquidware poskytuje bezproblémové virtuální desktopové prostředí pro koncové uživatele.

Liquidware Essentials rozšiřuje výhody virtuální plochy Windows efektivně sběru profily uživatelů a shromažďování dat klíče uživatele zjednodušit migraci uživatelské prostředí do Azure. Kromě toho Liquidware Essentials zjednodušuje Správa imagí ve sjednocení profily uživatelů a vrstvení aplikace na základě konfigurovatelných práv nastavení pro správu.

[Přejdete na web partnera.](https://www.liquidware.com/solutions/solutions-platform/microsoft)

## <a name="nerdio"></a>Nerdio

Nerdio je Platforma Azure IT služby automation, který umožňuje snadno nasadit a spravovat virtuální plochy Windows. Poskytuje Nerdio znalostní báze a technologie pro nasazení, ceny, balíček, spravovat a optimalizovat nasazení u zákazníků Azure – s virtuální plochy Windows front a-System center.

Nerdio zvyšuje hodnotu virtuální plochy Windows tak, že usnadňuje zřizování prostředků Azure a zjednodušení nasazení. S Nerdio pro Azure, může oddělení IT automatické nasazení a správě kompletního prostředí pro Azure, včetně virtuální plochy Windows, v části dvě hodiny.

[Přejdete na web partnera.](https://getnerdio.com/academy/windows-virtual-desktop-explained-for-msps/)

## <a name="thinprint"></a>ThinPrint

ThinPrint je zprostředkovatel služby s přidanou hodnotou virtuální plochy Windows, který nabízí jednoduchý a zabezpečený cloudový tisk z virtuálního počítače Windows. Pomocí služeb a softwaru můžete používat stávající infrastrukturu tisku tisk z cloudu. ThinPrint umožňuje připojení k oběma místní a síťových tiskáren, usnadňuje uživatelům v office a práce vzdáleně tisknout.

Řešení společnosti ThinPrint ezeep zvyšuje hodnotu virtuální plochy Windows tím, že připojení k existující podnikovou infrastrukturou tisku. ezeep poskytuje uživatelům kontrolu nad tisk v podnikové síti bez ohledu na to, kde jsou. Pomocí ezeep, můžou uživatelé překleňte propast mezi virtuální plochy Windows a tisk hardwaru.

[Přejdete na web partnera.](http://www.ezeep.com/wvd-printing)

## <a name="workspot"></a>Workspot

Workspot je poskytovatel služeb s přidanou hodnotou virtuální plochy Windows, který pro vícevrstvou podniky s vysoce výkonné stolních počítačů a pracovních stanic v Azure. S Workspot, zřizování infrastruktury je automatické, což znamená, že uživatelé měli přístup k jejich prostředí virtuálního klienta Windows kdekoli na světě s vysokou dostupností.

Workspot zvyšuje hodnotu virtuální plochy Windows díky zjednodušení procesu zřizování cloudové infrastruktury klasické pracovní plochy. S Workspot prostředky je možné snadno škálovat nahoru a dolů a vyhovět potřebám různých případech uživatelů a používá. Workspot můžete optimalizovat nasazení pro vysoce výkonné GPU pracovní stanice pro nezbytné pro CAD a technické uživatele, stejně jako aplikace Windows a stolních počítačů s Windows 10 pro všechny firemní uživatele.

[Přejdete na web partnera.](https://www.workspot.com/wvd)
