# <a name="connect-with-windows-7-and-windows-10"></a>Spojte se s Windows 7 a Windows 10

> Týká se Windows 7 a Windows 10.

Je k dispozici ke stažení klienta, která poskytuje přístup k prostředkům Windows Virtual Desktop Preview ze zařízení s Windows 7 a Windows 10.

## <a name="install-the-client"></a>Instalace klienta

[Stáhněte si](https://go.microsoft.com/fwlink/?linkid=2068602) a instalaci klienta do vašeho místního počítače. To vyžaduje oprávnění správce.

## <a name="subscribe-to-a-feed"></a>Přihlášení odběru informačního kanálu

Získání seznamu prostředky dostupné pro vás z vašeho místního počítače prostřednictvím přihlášení odběru informačního kanálu zadané váš správce.

K odběru informačního kanálu:

1. Spusťte klienta z seznam všech aplikací, podívejte se do pro vzdálenou plochu.
2. Vyberte pracovní nebo školní účet na hlavní stránce. To bude k přihlášení k odběru výchozí adresa URL informačního kanálu.
   - V případě potřeby se můžete také zadat jinou adresu URL **místo toho použijte adresu URL** možnost.
3. Přihlaste se pomocí uživatelského účtu.

> [!NOTE]
> Pokud váš správce nastavil e-mailu zjišťování, klient bude automaticky vyhledávat váš kanál adresy URL, pokud zadáte e-mailovou adresu.

Po úspěšném ověření, měli byste vidět seznam pro vás dostupné materiály.

Prostředky můžete spustit pomocí jedné ze dvou způsobů.

- Přejděte na hlavní stránce klienta a vyberte prostředek.
- Spusťte prostředek běžným způsobem další aplikace v nabídce Start.
  -  Můžete také vyhledat aplikaci v panelu vyhledávání.

## <a name="update-the-client"></a>Aktualizace klienta

Pokud je dostupná nová verze klienta, budete upozorněni na klienta a Centrum akcí Windows. Vyberte oznámení k zahájení procesu aktualizace.
