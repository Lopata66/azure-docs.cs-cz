# <a name="connect-with-the-macos-client"></a>Připojení s klientem macOS

> Platí pro: macOS 10.12 nebo novější

K prostředkům Windows Virtual Desktop můžete přistupovat ze svých zařízení macOS pomocí našeho klienta ke stažení. Tato příručka vám řekne, jak nastavit klienta.

## <a name="install-the-client"></a>Instalace klienta

Chcete-li začít, [stáhněte](https://apps.apple.com/app/microsoft-remote-desktop/id1295203466?mt=12) a nainstalujte klienta na zařízení s macOS.

## <a name="subscribe-to-a-feed"></a>Přihlásit se k odběru informačního kanálu

Přihlaste se k odběru zdroje, který vám správce dal, abyste získali seznam spravovaných zdrojů, které máte k dispozici na vašem zařízení s macOS.

Přihlášení k odběru informačního kanálu:

1. Vyberte **Přidat pracovní prostor** na hlavní stránce, abyste se připojili ke službě a načetli prostředky.
2. Zadejte adresu URL informačního kanálu. Může se jedná o adresu URL nebo e-mailovou adresu:
   - Pokud používáte adresu URL, použijte adresu, kterou vám dal správce. Za normálních <https://rdweb.wvd.microsoft.com>okolností je adresa URL .
   - Chcete-li používat e-mail, zadejte svou e-mailovou adresu. To říká klientovi, aby vyhledal adresu URL přidruženou k vaší e-mailové adrese, pokud váš správce nakonfiguroval server tímto způsobem.
3. Vyberte **Přidat**.
4. Po zobrazení výzvy se přihlaste pomocí svého uživatelského účtu.

Po přihlášení byste měli vidět seznam dostupných zdrojů.

Jakmile se přihlásíte k odběru informačního kanálu, bude se obsah zdroje pravidelně automaticky aktualizovat. Prostředky mohou být přidány, změněny nebo odebrány na základě změn provedených správcem.
