# <a name="connect-with-the-web-client"></a>Připojení k webovému klientovi

Webový klient umožňuje přístup k prostředkům virtuální plochy systému Windows z webového prohlížeče bez zdlouhavého procesu instalace.

>[!NOTE]
>Webový klient nemá aktuálně podporu mobilního operačního serveru.

## <a name="supported-operating-systems-and-browsers"></a>Podporované operační systémy a prohlížeče

Zatímco jakýkoli prohlížeč podporující HTML5 by měl fungovat, oficiálně podporujeme následující operační systémy a prohlížeče.

| Prohlížeč           | Podporovaný operační systém                     | Poznámky               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows, macOS, Linux            | Verze 55 nebo novější |
| Google Chrome     | Windows, macOS, Linux, Chrome OS |                     |

## <a name="access-remote-resources-feed"></a>Přístup ke vzdálenému kanálu zdrojů

V prohlížeči přejděte do [webového klienta Virtuální plochy Windows](https://rdweb.wvd.microsoft.com/webclient) a přihlaste se pomocí svého uživatelského účtu.

>[!NOTE]
>Pokud jste se už přihlásili pomocí jiného účtu Azure Active Directory, než který chcete použít pro virtuální plochu Windows, měli byste se buď odhlásit, nebo použít okno soukromého prohlížeče.

Po přihlášení byste nyní měli vidět seznam zdrojů. Prostředky můžete spustit tak, že je vyberete stejně jako normální aplikaci na kartě **Všechny prostředky.**
