# <a name="using-sap-on-linux-virtual-machines-in-azure"></a>Použití SAP na virtuálních počítačích s Linuxem v Azure
Cloud Computing je často používaný termín, který získává v oblasti IT čím dál větší význam – od malých firem až po velké a nadnárodní společnosti. Microsoft Azure je platforma cloudových služeb od společnosti Microsoft, která nabízí široké spektrum nových možností. Nyní mohou zákazníci rychle zřizovat a zrušit aplikace jako cloudové služby, takže už nejsou omezeni technickými nebo rozpočtovými limity. Namísto investování času a prostředků do hardwarové infrastruktury se společností můžou zaměřit na aplikace, obchodní procesy a jejich výhody pro zákazníky a uživatele.

S virtuálními počítači Microsoft Azure Microsoft nabízí komplexní infrastruktura jako služba (IaaS). Služba Azure Virtual Machines teď v rámci IaaS podporuje aplikace využívající SAP NetWeaver. Následující dokumenty White Paper popisují postupy plánování a implementace aplikací využívajících SAP NetWeaver na virtuálních počítačích Windows v Azure. Můžete také implementovat aplikací využívajících SAP NetWeaver na [Windows virtual machines](../../virtual-machines-windows-classic-sap-get-started.md?toc=%2fazure%2fvirtual-machines%2fwindows%2fclassic%2ftoc.json).

## <a name="sap-netweaver-on-azure-suse-linux-virtual-machines"></a>SAP NetWeaver na virtuálních počítačích Azure SUSE Linuxem
Název: Testování SAP NetWeaveru na virtuálních počítačích Microsoft Azure se SUSE Linuxem

Souhrn: Neexistuje žádné oficiální podpora SAP pro používání systému SAP NetWeaver na virtuálních počítačích Azure s Linuxem v daném okamžiku. Zákazníci však může být vhodné provést testování nebo zvažte spuštění SAP firma ukázkovou nebo systémy školení na virtuálních počítačích Azure s Linuxem za předpokladu, není nutné pro kontaktování podpory SAP. Tento článek by měly pomoci nastavení virtuálních počítačů Azure operačním systémem SUSE Linux pro spuštění SAP a poskytuje několik základních informací, pokud se chcete vyhnout běžným nástrahám potenciální.
