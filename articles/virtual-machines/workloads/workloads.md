# <a name="workloads-on-azure-virtual-machines"></a>Úlohy na virtuálních počítačích Azure

Virtuální počítače v Azure můžete podporovat mnoho úloh. Každá úloha může mít specializovaná pokyny k jeho optimální návrhu pro nasazení na virtuální počítače v Azure. Tady je některé pokyny, které nabízíme pro specializované úlohy.

- [Vysokovýkonné výpočetní prostředí](https://docs.microsoft.com/azure/architecture/topics/high-performance-computing/) virtuální počítače k řešení problémů složitých, jsou náročné na výpočetní prostředky v Azure používá CPU nebo GPU optimalizované. 
- [SAP](./sap/get-started.md) v Azure můžete spouštět nejdůležitější úlohy SAP a scénářů na škálovatelné a kompatibilní prověřená v podnikovém platformě.
- [Oracle](./oracle/oracle-considerations.md) podporuje spouštění verze Oracle DB 12.1 Standard a Enterprise v Azure pro Image virtuálního počítače založené na Oracle Linuxu. 
- [Změna hostování sálového počítače](./mainframe-rehosting/overview.md)
