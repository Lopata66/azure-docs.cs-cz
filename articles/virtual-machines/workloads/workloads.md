# <a name="workloads-on-azure-virtual-machines"></a>Úlohy v Azure Virtual Machines

Virtuální počítače v Azure můžou podporovat mnoho úloh. Každé zatížení může mít specializované pokyny týkající se optimálního návrhu nasazení na virtuálních počítačích v Azure. Tady jsou některé doprovodné materiály, které můžeme nabídnout pro specializované úlohy.

- [Vysoce výkonné výpočetní](./hpc/overview.md) prostředí v Azure používá virtuální počítače optimalizované pro procesory nebo GPU k řešení komplexních a výpočetních problémů náročných na výpočetní výkon.
- [SAP](./sap/get-started.md) v Azure může provozovat nejdůležitější úlohy a scénáře SAP na škálovatelné, kompatibilní a podnikové platformě.
- [Oracle](./oracle/oracle-considerations.md) podporuje v Azure image virtuálních počítačů s operačním systémem Oracle DB 12,1 Standard a Enterprise a na základě Oracle Linux. 
- [Opětovné hostování sálového počítače](./mainframe-rehosting/overview.md) 



**Další postup**

Získejte informace o virtuálních počítačích Azure pomocí průvodců rychlým startem virtuálních počítačů Azure pro [Linux](../linux/quick-create-cli.md) a [Windows](../windows/quick-create-powershell.md).
