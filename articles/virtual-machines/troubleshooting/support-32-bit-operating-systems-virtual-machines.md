# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Podpora 32bitových operačních systémů ve virtuálních počítačích Azure

Microsoft Azure teď umožňuje uživatelům přenést své 32bitové operační systémy Windows do Azure. Vzhledem k tomu, že některé z těchto operačních systémů již dosáhly smlouvy o podpoře ukončení životnosti, společnost Microsoft pro ně nemusí nabídnout další podporu. Podpora se také nenabízí pro operační systémy založené na Linuxu nebo Berkeley Software Distribution (BSD), které běží na virtuálním počítači Microsoft Azure .)
