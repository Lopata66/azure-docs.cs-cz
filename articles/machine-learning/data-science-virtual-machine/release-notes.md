# <a name="azure-data-science-virtual-machine-release-notes"></a>Poznámky k verzi virtuálního počítače Azure Data Science

V tomto článku se dozvíte o vydání virtuálních strojů Azure Data Science. Úplný seznam zahrnutých nástrojů spolu s čísly verzí naleznete na [této stránce](./tools-included.md).

Podívejte se na [seznam známých problémů,](reference-known-issues.md) kde se dozvíte o známých chybách a řešeních.

## <a name="2020-02-24"></a>2020-02-24

Datové vědy Virtuální počítač obrázky pro [Ubuntu 18.04](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.ubuntu-1804?tab=Overview) a [Windows 2019](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.dsvm-win-2019?tab=Overview) obrázky jsou nyní k dispozici.

- Power BI Desktop 2.73.55xx
- Firefox 69.0.2
- PyCharm 19,2,3
