# <a name="downloadable-infographic-machine-learning-basics-with-algorithm-examples"></a>Infografika ke stažení: Základy služby Machine Learning s příklady algoritmů

Stáhněte si tento snadno infografikaý přehled základních informací o strojovém učení, abyste se seznámili s oblíbenými algoritmy, které se používají k zodpovězení běžných otázek strojového učení. Příklady algoritmů umožňují strojové studiu začátečník pochopit, které algoritmy se mají použít a k čemu se používají.

## <a name="popular-algorithms-in-machine-learning-studio-classic"></a>Oblíbené algoritmy v Machine Learning Studio (klasické)

[Azure Machine Learning Studio (Classic)](https://studio.azureml.net/) přináší rozsáhlou knihovnu algoritmů pro prediktivní analýzy. Tato infografika identifikuje čtyři oblíbené řady pro algoritmy – regresi, detekci anomálií, clusteringu a klasifikaci – a poskytuje odkazy na pracovní příklady v [Azure AI Gallery](https://gallery.azure.ai/). Galerie obsahuje příklady experimentů a kurzů, které ukazují, jak lze tyto algoritmy použít v mnoha reálných řešeních.
