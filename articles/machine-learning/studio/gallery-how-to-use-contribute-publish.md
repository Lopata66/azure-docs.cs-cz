# <a name="share-and-discover-resources-in-the-azure-ai-gallery"></a>Sdílení a vyhledání prostředků v galerii Azure AI

**[Galerie Azure AI](http://gallery.cortanaintelligence.com)**  je komunitní web pro zjišťování a sdílení řešení vytvořené pomocí Azure AI.
Galerie má širokou škálu prostředků, které můžete použít k vývoji vlastních řešení pro analýzu.

[!INCLUDE [machine-learning-free-trial](../../../includes/machine-learning-free-trial.md)]

## <a name="what-can-i-find-in-the-gallery"></a>Co lze najít v galerii?
Galerie Azure AI obsahuje širokou škálu prostředků, které můžete použít k vývoji vlastních řešení pro analýzu.

* **[Experimenty](gallery-experiments.md)**  – The galerie obsahuje širokou škálu experimenty, které byly vyvinuty v Azure Machine Learning Studio. Jsou nejrůznější, od rychlé testování konceptu experimenty, které ukazují konkrétní služby machine learning technika na plně vyvinuli řešení problémů složitých strojového učení.
* **[Poznámkové bloky Jupyter](gallery-jupyter-notebooks.md)**  – zahrnout kód, vizualizace dat, poznámkové bloky Jupyter a dokumentace v jediném interaktivní plátno.
  Poznámkové bloky ve galerii poskytují kurzy a podrobné vysvětlení pokročilých machine learningu, techniky a řešení.

* **[Řešení](gallery-solutions.md)**  – rychle vytvářet řešení Cortana Intelligence z předkonfigurovaných řešení, referenčních architektur a vzorů návrhu. Aby byly vaše vlastní prostřednictvím zahrnutých pokynů nebo doporučeného partnera.
* **[Kurzy](gallery-tutorials.md)**  – kurzů je k dispozici několik vás provede technologie machine learning a koncepty, nebo k popisu rozšířené metody k řešení potíží, různé machine learning.

Tyto základní prostředky Galerie je možné seskupit logicky v několika různými způsoby:

* **[Kolekce](gallery-collections.md)**  – kolekce umožňuje seskupit dohromady experimenty, rozhraní API a další galerie položek, které řeší konkrétní řešení nebo koncept.
* **[Obory](gallery-industries.md)**  – The odvětví oddíl Galerie sdružuje různé prostředky, které jsou specifické pro takové odvětví jako maloobchodní prodej, výroby, bankovnictví a zdravotní péče.

Nakonec **[soutěže](gallery-competitions.md)** poskytovat skvělé příležitosti soutěžit s komunitou data scientsts řešit složité problémy pomocí Cortana Intelligence Suite.

## <a name="discover-and-learn"></a>Vyhledání a osvojení
Kdokoli může procházení a vyhledávání různé druhy prostředků v galerii přispěl Microsoft a komunita pokročilých analýz.
Další informace a začít na řešení vlastní problémy analýzy dat pomocí nich.
K vašemu pracovnímu prostoru Machine Learning Studio můžete také stáhnout experimentů a poznámkové bloky Jupyter.

Můžete snadno vyhledat naposledy publikované a oblíbených prostředků v galerii, nebo můžete vyhledávat podle názvu, značky, algoritmy a dalších atributů.
Klikněte na tlačítko **Procházet vše** v hlavičce galerii a vyhledat upřesnění na levé straně stránky vyberte a zadejte hledaný text v horní části.

Zobrazit příspěvky od konkrétního autora kliknutím na jméno autora z v rámci všech dlaždic:

![Zobrazení a autora příspěvků](./media/gallery-how-to-use-contribute-publish/view-by-author.png)

Můžete přidat komentář, poskytovat zpětnou vazbu nebo klást otázky pomocí oddílu pro komentáře na každé stránce prostředku.
Prostředek zájmu můžete sdílet i s přáteli nebo kolegy pomocí možností sdílení LinkedIn a Twitter.
Může také e-mailu odkazy na tyto prostředky chcete pozvat další uživatele zobrazíte na stránkách.

![Sdílet tuto položku s přáteli](./media/gallery-how-to-use-contribute-publish/comment-and-share.png)

## <a name="contribute-to-the-gallery"></a>Přispívání do galerie
Když jste přihlášení stane členem Galerie komunity. Díky tomu můžete vytvářet příspěvky vlastní položky galerie, takže ostatní můžou těžit z řešení, které jste se seznámili.

Stejně jako ostatní vytvářet přes svůj příspěvek v galerii, můžete postupovat podle počtu zobrazení a soubory ke stažení podstatu vašeho příspěvku:

![Sdílet tuto položku s přáteli](./media/gallery-how-to-use-contribute-publish/view-and-download-counts.png)

Uživatelé mohou také přidat komentáře a sdílet s ostatními členy komunity odborníků přes data vašeho příspěvku.
Můžete přihlásit pomocí nástroje diskuze, jako je Disqus a přijímat oznámení pro komentáře na vaše příspěvky.

![Postupujte podle komentáře uživatelů](./media/gallery-how-to-use-contribute-publish/follow-comments.png)

Následující položky se dá přispívat do galerie -, použijte tyto odkazy pro další informace:

* **[Kolekce](gallery-collections.md#contribute)**
* **[Experimenty](gallery-experiments.md#contribute)**
* **[Kurzy](gallery-tutorials.md#contribute)**
