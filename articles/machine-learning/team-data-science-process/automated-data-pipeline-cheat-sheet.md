# <a name="cheat-sheet-for-an-automated-data-pipeline-for-azure-machine-learning-predictions"></a>Tahák k automatizovanému datovému kanálu pro predikce služby Azure Machine Learning
**Cheatsheet automatizovaného datového kanálu Microsoft Azure Machine Learning** vám pomůže procházet technologie, které můžete použít k získání dat do webové služby Machine Learning, kde je můžete skórovat podle modelu prediktivní analýzy.

V závislosti na tom, zda jsou vaše data místní, v cloudu nebo streamování v reálném čase, jsou k dispozici různé mechanismy pro přesunutí dat do koncového bodu webové služby pro vyhodnocování.
Tento tahák vás provede rozhodnutími, která musíte učinit, a nabízí odkazy na články, které vám pomohou vyvinout vaše řešení.
