# <a name="team-data-science-process-project-planning"></a>Plánování projektu Team Data Science Process

Proces vědecké ho procesu týmových dat (TDSP) poskytuje životní cyklus pro strukturu vývoje vašich projektů datové vědy. Tento článek obsahuje odkazy na šablony aplikace Microsoft Project a Excel, které vám pomohou naplánovat a spravovat tyto fáze projektu.

Životní cyklus popisuje hlavní fáze, které projekty obvykle provádějí, často iterativně:

- Obchodní porozumění
- Získávání a porozumění dat
- Modelování
- Nasazení
- Přijetí zákazníkem

Popis každé z těchto fází naleznete [v tématu Životní cyklus procesu vědecké ho společnosti týmových dat](https://docs.microsoft.com/azure/machine-learning/team-data-science-process/lifecycle).

 
## <a name="microsoft-project-template"></a>Šablona aplikace Microsoft Project

Šablona aplikace Microsoft Project pro proces vědecké ho řízení týmových dat je k dispozici zde: [Šablona aplikace Microsoft Project](https://github.com/Azure/Azure-MachineLearning-DataScience/blob/master/Team-Data-Science-Process/Project-Planning-and-Governance/Advanced%20Analytics%20Microsoft%20Project%20Plan.mpp) 

Po otevření plánu klikněte na odkaz zcela vlevo pro TDSP. Změňte název a popis a přidejte všechny ostatní týmové prostředky, které potřebujete. Odhadněte data požadovaná ze svých zkušeností.
