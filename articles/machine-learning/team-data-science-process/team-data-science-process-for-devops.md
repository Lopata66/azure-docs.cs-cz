# <a name="team-data-science-process-for-developer-operations"></a>Vědecké zpracování týmových dat pro vývojářské operace

Tento článek popisuje funkce vývojářské operace (DevOps), které jsou specifické pro pokročilé analýzy a implementaci Cognitive Services řešení. Tyto školicí materiály se týkají TDSP (Team data Sciences) a softwaru a nástrojů společnosti Microsoft a open source, které jsou užitečné pro plánování, spouštění a doručování řešení pro datové vědy. Odkazuje na témata, která se týkají DevOps sada nástrojů, která je specifická pro datové vědy a projekty a řešení AI.
