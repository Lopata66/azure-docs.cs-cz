# <a name="team-data-science-process-for-developer-operations"></a>Proces vědecké ho spoje pro vývojová data

Tento článek zkoumá funkce pro vývojářské operace (DevOps), které jsou specifické pro implementaci řešení Advanced Analytics a Cognitive Services. Tyto školicí materiály implementují proces team data science (TDSP) a microsoft a open source software a sady nástrojů, které jsou užitečné pro představu, provádění a poskytování řešení pro datové vědy. Odkazuje na témata, která pokrývají řetězec nástrojů DevOps, který je specifický pro projekty a řešení pro datové vědy a ai.
