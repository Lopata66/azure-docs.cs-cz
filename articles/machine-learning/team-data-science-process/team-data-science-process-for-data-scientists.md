# <a name="team-data-science-process-for-data-scientists"></a>Proces týmové datové vědy pro datové vědce

Tento článek obsahuje pokyny k sadě cílů, které se obvykle používají k implementaci komplexních řešení pro datové vědy s technologiemi Azure. Jste vedeni:

- pochopení analytického pracovního vytížení
- pomocí procesu vědecké ho svědecké ho spoje
- pomocí Azure Machine Learning
- základy přenosu a ukládání údajů
- poskytování dokumentace ke zdroji dat
- používání nástrojů pro zpracování analýzy

Tyto školicí materiály se vztahují k procesu teamdata science (TDSP) a microsoftu a open-source softwaru a nástrojů, které jsou užitečné pro představu, provádění a poskytování řešení pro datové vědy.
