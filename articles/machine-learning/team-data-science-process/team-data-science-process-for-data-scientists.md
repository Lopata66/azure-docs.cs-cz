# <a name="team-data-science-process-for-data-scientists"></a>Vědecké zpracování týmových dat pro odborníky přes data

- Principy úlohu analýza 
- použití vědeckého zpracování týmových dat
- používání Azure Machine Learning
- Základy úložiště a přenosu dat
- poskytuje dokumentaci zdroje dat.
- pomocí nástrojů pro zpracování analýzy

Vědecké zpracování týmových dat (TDSP) a Microsoft a open source softwaru a sady nástrojů, které jsou užitečné při plánování, provádění a poskytováním řešení datové vědy se vztahují tyto školicích materiálů.
