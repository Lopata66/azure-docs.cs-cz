# <a name="explore-data-in-the-team-data-science-process"></a>Zkoumání dat v rámci vědeckého zkoumání týmových dat

Zkoumání dat je krok v [procesu vědecké vědy o týmových datech](overview.md).

Následující články popisují, jak prozkoumat data ve třech různých prostředích úložiště, které se obvykle používají v procesu datové vědy:

* Prozkoumejte data [kontejneru objektů blob Azure](explore-data-blob.md) pomocí balíčku [Pandas](https://pandas.pydata.org/) Python.
* Prozkoumejte data [serveru SQL Server](explore-data-sql-server.md) pomocí SQL a pomocí programovacího jazyka, jako je Python.
* Prozkoumejte data [tabulky Hive](explore-data-hive-tables.md) pomocí dotazů Hive.

Kromě toho [video, předběžné zpracování dat v Azure Machine Learning Studio](https://azure.microsoft.com/documentation/videos/preprocessing-data-in-azure-ml-studio/), popisuje běžně používané moduly pro čištění a transformaci dat.
