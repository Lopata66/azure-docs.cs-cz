# <a name="microsoft-azure-new-support-request-file-upload-utility-guidelines"></a>Pokyny pro odesílání nástrojů pro novou žádost o podporu Microsoft Azure

Když [vytvoříte žádost o podporu](https://portal.azure.com/#create/Microsoft.Support), možnost nahrání souboru vám pomůže snadno nahrát diagnostické soubory nebo jiné soubory, které považujete za relevantní pro incident.  

> [!NOTE]
> Chcete-li chránit své soukromí, neuvádějte do nahrávání žádné osobní údaje.
>
>

Při použití možnosti nahrávání souborů postupujte podle těchto pokynů:

* Všechny soubory musí mít příponu názvu souboru, například *DOCX* nebo *.xlsx*. V tabulce naleznete povolené přípony názvů souborů.
* Název souboru nesmí být delší než 110 znaků.
* Nelze nahrát více než jeden soubor.
* Soubory nesmí být větší než 4 MB.
