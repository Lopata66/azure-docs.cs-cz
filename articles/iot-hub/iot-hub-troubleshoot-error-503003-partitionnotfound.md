# <a name="503003-partitionnotfound"></a>503003 PartitionNotFound

Tento článek popisuje příčiny a řešení pro **503003 PartitionNotFound** chyby.

## <a name="symptoms"></a>Příznaky

Požadavky na službu IoT Hub se nezdaří s chybou **503003 PartitionNotFound**.

## <a name="cause"></a>Příčina

Tato chyba je interní ioT Hub a je pravděpodobně přechodné. Viz [Příčinou chyb interního serveru služby IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#cause).

## <a name="solution"></a>Řešení

Viz [řešení chyb interního serveru služby IoT Hub](iot-hub-troubleshoot-error-500xxx-internal-errors.md#solution).
